Duplizieren eines Faxempfängers

Mit der Faxfunktion können mehrere Datensätze für einen Empfänger und eine Faxnummer angelegt werden. Wenn unterschiedliche Mailbox-Informationen, E-Mail-Adressen oder Telefonnummern gespeichert werden sollen, kann ein duplizierter Datensatz hinzugefügt werden.

Wenn ein duplizierter Datensatz hinzugefügt wird, wird ein Bestätigungsfenster angezeigt.

  1. Zum Hinzufügen eines duplizierten Empfängers auf Ja klicken.
  2. Zum Zurückkehren zum vorherigen Fenster auf Nein klicken.
