Пользовательская цветовая коррекция

При выборе опции «Пользовательская автонастройка цвета» открывается окно «Пользовательская цветовая коррекция» для точной настройки цвета опытными пользователями.

Примечание: Не все перечисленные опции поддерживаются всеми принтерами. Некоторые опции предназначены только для определенных моделей и конфигураций принтера, операционных систем и драйверов.

Настройка цветов:

  1. Выполните одно или несколько из следующих действий:
    • Для настроек «Офисная печать», «Соответствие печати» и «Цветовая заливка» выполните следующее:
      • Выберите опции в меню «Профиль изображения» и «Профиль текста и графики».
      • Для настройки яркости, контраста, насыщенности и уровней цвета выберите Дополнительные параметры.
    • Для настройки «Цвет CIE» выполните одно из следующих действий:
      • Выберите Всегда применять встроенные профили.

        Всегда применять встроенные профили – принтер использует встроенные профили документов, при этом существует риск применения нестандартных цветовых схем. Применение встроенных профилей может привести к ухудшению качества печати. В случае особых данных, таких как данные измерения цвета, использование собственных цветовых схем во встроенном профиле обеспечивает наилучшее качество печати.

        Встроенный профиль представляет собой словарь цветопередачи (Color Rendering Dictionary, CRD), передаваемый из приложения вместе с данными и описывающий характеристики данных по цвету.

      • Выберите Никогда не применять встроенные профили.

        Никогда не применять встроенные профили – принтер не использует встроенные профили документов, чтобы снизить риск применения нестандартных цветовых схем.

      • Для автоматического применения в документе известных цветовых профилей выберите Автоматически применять соответствующие встроенные профили:. При выборе данной опции становится доступны меню «Текст и графика» и «Изображения». Выбираются нужные опции. Для внесения других изменений выберите Дополнительные параметры.
        1. Выберите опцию в меню «Текст и графика»:
          • Автонастройка цвета - для текста, рисунков и фотографий применяется наиболее подходящая цветовая коррекция. Xerox рекомендует использовать данную настройку.
          • Яркий цвет – получение ярких, более насыщенных цветов.
          • Черно-белый - копирование только в черно-белом режиме. Цвета оригинала преобразуются в оттенки серого.
          • Цветометрия – осуществляется попытка добиться точного соответствия цвета. В данном режиме имитируется регулируемые условия освещения.
          • Фото – обеспечивается оптимальная визуализация фотографических изображений. Это сбалансированная визуализация с использованием субъективной коррекции для пользовательских цветов.
          • Насыщенность – обеспечивается оптимальная визуализация графики. Цвета получаются более яркими и броскими, но менее точными, чем при выборе настройки «Фото».
        2. Выберите опцию в меню «Изображения». Список опций приведен в предыдущем пункте.
        3. Для дополнительной коррекции настройки «Изображения» нажмите кнопку Дополнительные параметры.
      • Выберите опции в меню «Профиль изображения» и «Профиль текста и графики».
      • Для настройки яркости, контраста, насыщенности и уровней цвета выберите Дополнительные параметры.
    • Для настройки «Шкала серого» выполните одно из следующих действий:
      • Выберите Исп. настр. цвета RGB для черного.

        Использовать настройки цвета RGB для черного – использование серых линий для соответствия настройкам красного, зеленого, синего (RGB) и черного. При выборе данной опции в полях «Текст и графика» и «Изображения» устанавливаются текущие настройки, заданные на вкладке «Офисный цвет».

      • Выберите Исп. настр. цвета CMYK для черного.

        Использовать настройки цвета CMYK для черного – использование серых каналов для соответствия настройкам голубого, пурпурного, желтого и черного. Данная настройка наиболее близка к монохромной. При выборе данной опции в полях «Текст и графика» и «Изображения» устанавливаются текущие настройки, заданные на вкладке «Соответствие печати».

      • Для установки настроек черного цвета выберите опцию Настройка черного вручную. При выборе данной опции становится доступны меню «Текст и графика» и «Изображения». Выбираются нужные опции. Для внесения других изменений выберите Дополнительные параметры.
        1. Для дополнительной коррекции настройки «Текст и графика» нажмите кнопку Дополнительные параметры. Выберите опцию в меню «Текст и графика»:
          • Монохромная графика – оптимизированная печать с плавными линейными переходами от белого к черному.
          • Монохромные изображения – использование гамма-кривой, оптимизированной для получения более резких визуальных эффектов в изображениях (например, для повышения контрастности).
          • Коммерческая – применение цветов для печати на промышленной печатной машине.
          • Печать SWOP (Specifications for Web Offset Printing Press) – представляет собой проффессиональный стандарт печати CMYK, широко распространенный в Северной Америке.

            Голубой, пурпурный, желтый и черный (CMYK) цвета, используемые принтером для получения всех цветов.

          • Печать Euroscale – соответствует европейскому стандарту FOGRA для глянцевой бумаги.

            FOGRA — это стандарт, стандарт предназначенный для печати в цвете при использовании офсетных и обычных цифровых технологий.

          • Japan Color – это настройка цветовой коррекции для Японии. Она соответствует спецификации Japan Color 2001.
          • Покрытие ISO – эмуляция стандарта FOGRA 27 для печати на бумаге с покрытием. При использовании данной настройки печать на бумаге с покрытием необязательна.

            FOGRA — это стандарт, стандарт предназначенный для печати в цвете при использовании офсетных и обычных цифровых технологий.

          • Без покрытия ISO – эмуляция стандарта FOGRA 29 для печати на бумаге без покрытия.

            FOGRA — это стандарт, стандарт предназначенный для печати в цвете при использовании офсетных и обычных цифровых технологий.

          • Печать SNAP (Specifications for Non-Heatset Advertising Printing Press) – стандарт для полиграфической печати.
          • Дисплей sRGB – эмуляция принтром цветов калиброванного дисплея RGB, такого как ЭЛТ-дисплей или профессиональный ЖК-дисплей.

            Настройка красного, зеленого и синего (RGB) цветов.

          • Жидкокристаллический дисплей – приближение цветов в работе печати к цветам монитора компьютера.
          • Яркий sRGB – получение ярких, более насыщенных цветов с использованием стандартного цветового пространства RGB.
          • Цветовая заливка – смешивание голубого, пурпурного, желтого и черного для получения цветов красителей, используемых в печатной машине. Для того чтобы получить самую широкую гамму и наилучшее соответствие цветовому пространству Pantone, выберите опцию «Цветовая заливка».
          • Нет – цветовая коррекция не используется.
        2. Выберите опцию в меню «Изображения». Список опций приведен в предыдущем пункте. Для дополнительной коррекции настройки «Изображения» нажмите кнопку Дополнительные параметры.
      • Выберите опции в меню «Профиль изображения» и «Профиль текста и графики».
      • Для настройки яркости, контраста, насыщенности и уровней цвета выберите Дополнительные параметры.
  2. Для восстановления первоначальных настроек системы нажмите кнопку По умолч..
  3. Нажмите кнопку OK.
