Print With

Print With allows you to print stored private and public saved jobs with the current print job. For example, print a cover letter with a set of saved documents such as brochures or price lists.

Note: This job type is only available on printers that are connected to a network with bi-directional communication enabled.

To print using Print With:

  1. In the Paper/Output window, next to Job Type, click the arrows, then select Print With.
  2. When prompted, type the passcode that you assigned to the saved jobs on the printer, then click OK.

    If you did not assign a passcode, click Cancel.

  3. From the Saved Jobs list, select an option.
    • Personal and Shared Saved Jobs
    • Shared Saved Jobs

      Shared Saved Jobs displays shared jobs stored on the printer hard drive. Shared jobs are available for selection by any user.

    • Personal Saved Jobs

      Personal Saved Jobs displays jobs saved under your user name and current passcode so that only you can print them. Click Passcode to type the same passcode that you used when you saved Private Saved jobs on the printer hard drive.

      Note: Personal Saved Jobs require a password.

  4. If you select Personal and Shared Saved Jobs or Personal Saved Jobs, type the passcode for the jobs. If your jobs do not appear, click Passcode, then retype a valid passcode for the jobs.
  5. To add a job to the Final list, select a job from the list, then click the down arrow.
  6. To change the order of the jobs in the Final Job list, select a job, then use the arrows.
  7. To delete a job from the list, select it, then click Delete.
  8. Click OK.
  9. Click Print.
