命名紙張

「命名紙張」是已命名和儲存的一組紙張特性，以供重複使用。 您可以從共用的印表機紙張清單或個人化的我的紙張清單選擇。

如要選擇命名紙張：

  1. 在「紙張/輸出」視窗中，在「紙張」旁，按一下箭頭，然後選擇命名紙張。
  2. 選擇一個選項。
    • 如要從預先載入印表機的清單中選擇紙張，請按一下印表機紙張清單。
    • 如要從您的個人化命名紙張清單選取紙張，請選擇我的紙張清單。

      「命名紙張」是已命名和儲存的一組紙張特性，以供重複使用。 您只能修改或刪除「我的紙張清單」中的命名紙張。

      新增命名紙張

      1. 在「命名紙張」視窗中，按一下最上方功能表，選擇我的紙張清單，然後選擇新增按鈕 。
      2. 在「紙張名稱」中，選擇或輸入名稱。
      3. 如要僅使用紙張名稱，請選擇僅使用者名稱（專家），然後輸入名稱或從清單中選取。

        注意： 如果您選擇僅使用者名稱（專家），屬性欄位將無法使用。

      4. 請執行以下任一個動作：
        • 選擇所需的紙張屬性和選項。
        • 如要使用印表機預設值，請按一下功能表，然後選擇印表機預設值。
      5. 如要新增有關使用此命名紙張的其他資訊，請在備註（選填）中輸入資訊。
      6. 按一下確定。

      複製命名紙張

      1. 在「命名紙張」視窗中，按一下最上方功能表，選擇我的紙張清單。
      2. 從表格中，選擇要複製的命名紙張，然後按一下複製按鈕 。

        注意： 在您從清單中選擇命名紙張後，此按鈕會啟用。

      3. 在「紙張名稱」欄位中，選擇或輸入名稱。
      4. 如要僅使用紙張名稱，請選擇僅使用者名稱（專家），然後輸入名稱或從清單中選取。
      5. 如果您選擇僅使用者名稱（專家），屬性欄位將無法使用。
      6. 請執行以下任一個動作：
        • 保留所需的紙張屬性和選項。
        • 選擇所需的紙張屬性和選項。
        • 如要使用印表機預設值，請按一下功能表，然後選擇印表機預設值。
      7. 如要新增有關使用此命名紙張的其他資訊，請在備註（選填）中輸入資訊。
      8. 按一下確定。

      編輯命名紙張

      1. 在「命名紙張」視窗中，按一下最上方功能表，選擇我的紙張清單。
      2. 從表格中，選擇一個命名紙張，然後按一下編輯按鈕 。

        注意： 在您從清單中選擇命名紙張後，此按鈕會啟用。

      3. 如要僅使用紙張名稱，請選擇僅使用者名稱（專家），然後輸入名稱或從清單中選取。

        注意： 如果您選擇僅使用者名稱（專家），屬性欄位將無法使用。

      4. 請執行以下任一個動作：
        • 選擇所需的紙張屬性和選項。
        • 如要使用印表機預設值，請按一下功能表，然後選擇印表機預設值。
      5. 如要新增有關使用此命名紙張的其他資訊，請在備註（選填）中輸入資訊。
      6. 按一下確定。

      刪除命名紙張

      1. 在「命名紙張」視窗中，按一下最上方功能表，選擇我的紙張清單。
      2. 選擇您要移除的命名紙張，然後按一下刪除按鈕 。
      3. 按一下關閉。

      注意： 由於此清單與您的使用者名稱關聯，您可以從其他 Xerox® 生產印表機在您的網路上使用。

  3. 選擇所需紙張的表格列。
  4. 按一下用於工作中。
