Avdelare för stordior

Med Avdelare för stordior kan du infoga ett tomt ark efter varje utskriven stordia. Avdelarnas format och orientering måste vara samma som till stordiorna.

Obs!

  • Alla alternativ som visas stöds inte på alla skrivare. Vissa alternativ finns bara i en viss skrivarmodell, en viss konfiguration, ett visst operativsystem eller en viss drivrutinstyp.
  • Avdelare för stordior kan inte användas med vissa efterbehandlingsalternativ, t.ex. Falsning eller Häften.

Så här skriver du ut med Avdelare för stordior:

  1. Klicka på pilarna bredvid Papper i fönstret Papper/Utmatning och välj Avdelare för stordior.
  2. Välj ett alternativ för Avdelare för stordior.
    • Av eller Inga avdelare: Avdelare infogas inte.
    • Tomma avdelare: Ett tomt pappersark infogas mellan stordiorna.
    • Avdelare med text: Ett pappersark med text infogas mellan stordiorna. Detta ark innehåller samma bild som framförvarande stordia.
  3. Välj ett alternativ för Färg.
  4. Klicka på OK.
