Options de couleur personnalisée

Propose des options supplémentaires pour personnaliser les couleurs de votre travail d'impression.

Remarque : Toutes les imprimantes ne prennent pas en charge la totalité des options mentionnées. Certaines options s'appliquent uniquement à un modèle d'imprimante, un système d'exploitation ou un type de pilote spécifique.

Description des couleurs

Permet de régler les couleurs en construisant des phrases utilisant des expressions courantes décrivant le changement de couleurs.

Remarque : La fonction Paramètres couleur permet de modifier les couleurs à l'aide de curseurs traditionnels.

Pour utiliser la description des couleurs :

  1. Dans la fenêtre Options de couleur personnalisée, cliquez sur Description des couleurs.
  2. Pour sélectionner la couleur à modifier, en regard de Modifier quelle couleur ?, cliquez sur les flèches et sélectionnez une couleur.
  3. Pour sélectionner l'importance du changement de couleur, en regard de Combien de changement ?, cliquez sur les flèches et sélectionnez une option.
  4. Pour sélectionner la façon dont la couleur va être changée, en regard de Quel genre de changement ?, cliquez sur les flèches et sélectionnez une option.
  5. Pour modifier une phrase de réglage de couleur existante, sélectionnez-la puis sélectionnez de nouvelles phrases dans les menus.
  6. Pour créer une nouvelle phrase, cliquez sur Construire une nouvelle phrase.
  7. Pour changer l'ordre des phrases de réglage de couleur, effectuez votre sélection et utilisez les flèches.
  8. Pour supprimer une phrase de réglage de couleur, sélectionnez-la, puis cliquez sur le bouton X.

    Remarque : L'ordre des phrases détermine l'application de chaque réglage couleur.

  9. Pour afficher un échantillon d'image illustrant les changements des couleurs actuelles, sélectionnez Aperçu. Pour afficher l'image d'origine, désélectionnez la case à cocher Aperçu. Pour modifier l'image d'aperçu, sous l'image actuelle, utilisez les flèches et sélectionnez une option.

Correction des couleurs

La fonction de correction des couleurs propose des modes de correction des couleurs applicables aux travaux d'impression. La correction des couleurs permet de reproduire le plus fidèlement possible les couleurs d'autres périphériques (écran d'ordinateur ou presse d'impression, par exemple).

Pour utiliser la correction des couleurs :

  1. Dans la fenêtre Correction des couleurs , cliquez sur les flèches et sélectionnez une correction couleur.
  2. Pour créer un format personnalisé de correction des couleurs, sélectionnez Couleur automatique personnalisée.
  3. Cliquez sur OK.

Paramètres couleur

Utilisez les paramètres couleur pour régler des caractéristiques spécifiques de la couleur imprimée. La luminosité, le contraste, la saturation et la nuance figurent parmi ces caractéristiques.

Pour définir les paramètres couleur :

  1. Pour spécifier la luminosité, le contraste, la saturation et la nuance, sélectionnez les options à l'aide des curseurs.
  2. Pour afficher un échantillon d'image illustrant les changements des couleurs actuelles, sélectionnez Aperçu.
  3. Pour modifier l'aperçu d'image, cliquez sur les flèches.
  4. Cliquez sur OK.
