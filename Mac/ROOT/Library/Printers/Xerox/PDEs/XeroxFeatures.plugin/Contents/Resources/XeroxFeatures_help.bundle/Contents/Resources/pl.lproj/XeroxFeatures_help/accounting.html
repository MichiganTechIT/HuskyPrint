Zliczanie

Funkcja zliczania wiąże każdy drukowany dokument z nazwą użytkownika i identyfikatorem konta. Dane zliczania są zbierane i zapisywane po odebraniu prac przez drukarkę.

Uwagi:

  • Nie wszystkie drukarki obsługują wymienione opcje. Część z nich ma zastosowanie tylko w przypadku konkretnych modeli, konfiguracji, systemów operacyjnych czy typów sterownika.
  • W zależności od wyboru systemu zliczania i przypomnienia o zliczaniu mogą nie być dostępne wszystkie opcje.

Aby skonfigurować zliczanie:

  1. W oknie Zaawansowane kliknij strzałki obok obszaru System zliczania, następnie wybierz opcję.
    • Standardowe zliczanie Xerox®: Śledzi liczbę kopii, wydruków, skanów i faksów dla każdego użytkownika. Korzysta z identyfikatorów użytkownika i konta, które są zdefiniowane w drukarce.
    • Xerox® Zliczanie Sieciowe: Uzyskuje identyfikatory użytkownika i konta z serwera uwierzytelniania sieciowego.
    • Zliczanie lokalne: Wymaga kodu zliczania podsieci. Ten kod zliczania może być działowym kodem fakturowania, np. 0001. Inne kody są opcjonalne.
    • Obciąż kosztem wydruku: Wymaga identyfikatora fakturowania dla prac drukowania, taki jak identyfikator fakturowania danego działu. Inne identyfikatory są opcjonalne.
    • Wyślij ident. użytkownika i ident. zliczania: Do przesłania pracy drukowania wymaga wpisania identyfikatora użytkownika, identyfikatora zliczania i hasła.
    • Wyślij tylko identyfikator użytkownika: Do przesłania pracy drukowania wymaga wpisania identyfikatora użytkownika i hasła.
    • Wyślij tylko ident. zliczania: Do przesłania pracy drukowania wymaga wpisania identyfikatora zliczania i hasła.
    • Wyślij tylko ident. fakturowania: Do przesłania pracy drukowania wymaga wpisania identyfikatora fakturowania i hasła.
    • Wyślij wszystko: Do przesłania pracy drukowania wymaga wpisania identyfikatora użytkownika, zliczania, fakturowania i hasła.
  2. Wybierz opcję obok pozycji Przypomnienie o zliczaniu.
    • Always Prompt: Requires you to enter your user ID and account ID every time you submit a print job.
    • Do Not Prompt: Uses the information that you enter under Account Codes for every print job.
    • Przypominaj tylko dla faksu: Wymaga wprowadzenia identyfikatora użytkownika i identyfikatora konta przy każdym wysyłanym faksie.
    • Only Prompt for Color: Requires you to enter your user ID and account ID only for color print jobs.
    • Only Prompt for Color and Fax: Requires you to type your user ID and account ID only for color print jobs and faxes.
  3. Aby ukryć identyfikator użytkownika, wybierz opcję Maskuj identyfikator użytkownika. Znaki gwiazdki (*) zastępują dowolny ciąg znaków wpisanych w polu.
  4. Aby ukryć identyfikator konta, wybierz opcję Maskuj identyfikator konta. Znaki gwiazdki (*) zastępują dowolny ciąg znaków wpisanych w polu.
  5. W obszarze Domyślne kody zliczania wpisz odpowiednie wartości. Te pola są wymagane po wybraniu opcji Nie przypominaj. Identyfikatorem konta i identyfikatorem fakturowania może być działowy kod zliczania, na przykład 0001.
  6. Kliknij przycisk OK.
