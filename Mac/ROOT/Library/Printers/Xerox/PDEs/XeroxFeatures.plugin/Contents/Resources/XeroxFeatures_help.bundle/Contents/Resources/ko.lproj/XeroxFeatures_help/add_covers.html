표지 추가

표지 추가 창을 사용하여 문서의 각 사본에 표지를 추가하고 표지 용지 속성을 지정합니다. 작업 설정 상자에는 문서의 본문에 대한 용지 설정이 표시됩니다. 표지 속성의 작업 설정을 사용하려면 작업 설정 사용을 선택합니다.

참고: 일부 프린터에서는 여기 나열된 모든 옵션이 지원되지 않습니다. 일부 옵션은 특정 프린터 모델, 구성, 운영 체제 또는 드라이버 유형에만 적용됩니다.

표지를 추가하려면 다음 작업을 수행하십시오.

  1. 특수 페이지 창에서 표지 추가를 클릭합니다.
  2. 모든 설정에 대해 작업 기본값을 그대로 사용하려면 확인을 클릭합니다.
  3. 선택하려면 표지 옵션 목록에서 표지 옵션을 하나 선택합니다.
    • 표지 없음을 선택하면 이전 표지 설정이 취소됩니다.
    • 앞면만을 선택하면 지정된 용지함에서 배출된 용지에 첫 페이지가 인쇄됩니다. 자동 양면 인쇄를 사용하면 표지에 처음 두 페이지가 인쇄됩니다.
    • 뒷면만을 선택하면 지정된 용지함에서 배출된 용지에 뒷면 페이지가 인쇄됩니다. 자동 양면 인쇄를 사용하면 페이지 수가 짝수일 때 표지에 마지막 두 페이지가 인쇄됩니다.
    • 앞뒷면: 동일을 선택하면 앞뒷면 표지가 같은 용지함에서 인쇄됩니다.
    • 앞뒷면: 다름: 여러 용지함의 용지를 사용하여 앞표지와 뒷표지를 인쇄합니다.
  4. 용지의 크기, 컬러 또는 종류를 선택하려면 [앞면 표지 용지] 또는 [뒷면 표지 용지] 옆에 있는 화살표를 클릭한 후 옵션을 선택합니다.
  5. 표지 인쇄 옵션을 선택합니다.
    • 빈 용지 또는 사전 인쇄 용지: 표지에 빈 용지나 사전 인쇄된 용지를 사용합니다.
    • 앞면에 인쇄는 표지의 앞면만 인쇄합니다.
    • 뒷면에 인쇄: 표지 용지의 뒷면을 인쇄합니다.
    • 양면에 인쇄: 표지 용지의 양면을 인쇄합니다.
  6. 확인을 클릭합니다.
