Xerox 작업 티켓

작업 티켓은 Xerox 관련 프린터 기능(예: 스테이플 사용 또는 양면 인쇄)의 인쇄 작업 지침이 포함된 파일입니다. 프린터에서 이 파일을 사용하여 인쇄 작업을 처리합니다. 작업 티켓 지침은 PDL 파일 지침 전에 처리됩니다.

참고:

  • 일부 프린터에서는 여기 나열된 모든 옵션이 지원되지 않습니다. 일부 옵션은 특정 프린터 모델, 구성, 운영 체제 또는 드라이버 유형에만 적용됩니다.
  • 일부 응용 프로그램은 XPIF 작업 티켓과 호환되지 않습니다. 파일(예: .PS 또는 .EPS)에 인쇄하거나 인쇄에 문제가 있는 경우 작업 티켓 옵션을 사용하지 않도록 설정하십시오.
  • 작업 티켓 기능이 해제되면 인쇄 드라이버가 PostScript만 출력합니다. Xerox 작업 티켓 설정 또는 프린터 기능을 사용할 수 없습니다.

작업 티켓 만들기

고급 창에서 [Xerox 작업 티켓] 옆에 있는 화살표를 클릭한 후 다음 옵션 중 하나를 선택합니다.

  • 설정: 문서 설정을 사용하여 작업 티켓을 만듭니다. 인쇄 드라이버는 인쇄 작업과 작업 티켓을 인쇄 서버에 전송합니다.
  • 해제: 작업 티켓 기능을 해제합니다.

작업 티켓 가져오기

작업 티켓을 가져오려면 가져오기를 클릭하고, 작업 티켓을 찾아서 선택한 후 확인을 클릭합니다.

작업 티켓 내보내기

  1. 다른 이름으로 저장에서 작업 티켓의 이름을 입력합니다.
  2. 파일을 저장하려면 위치 필드에서 화살표를 사용하거나, 클릭하여 폴더로 이동합니다.
  3. 저장을 클릭합니다.
