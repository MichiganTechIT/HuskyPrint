Programmazione lavoro Xerox

Una programmazione lavoro è un file contenente istruzioni del lavoro di stampa per funzioni stampante specifiche di Xerox, ad esempio pinzatura o stampa fronte/retro. La stampante utilizza questo file per elaborare il lavoro di stampa. Le istruzioni contenuto in Programmazione lavoro vengono elaborate prima delle istruzioni contenute nel file PDL.

Note:

  • Non tutte le opzioni elencate sono supportate su tutte le stampanti. Alcune opzioni si applicano solo a modelli di stampante, configurazioni, sistemi operativi o tipi di driver specifici.
  • Alcune applicazioni non sono compatibili con le programmazioni lavoro XPIF. Se si desidera stampare un file, ad esempio .PS o .EPS, o si verificano problemi di stampa, disabilitare l'opzione di programmazione lavoro.
  • Quando la funzione Programmazione lavoro è disattivata, vengono forniti solo lavori PostScript. Non sarà disponibile alcuna impostazione programmazione lavoro Xerox o funzioni della stampante.

Crea una programmazione lavoro

Nella finestra Avanzate, accanto a Programmazione lavoro Xerox, fare clic sulle frecce, quindi selezionare un'opzione.

  • Sì: utilizza le impostazioni documento per creare una programmazione lavoro. Il driver di stampa invia la programmazione lavoro con il lavoro di stampa al server di stampa.
  • No: consente di disattivare la funzione Programmazione lavoro.

Importa una programmazione lavoro

Per importare una programmazione lavoro, fare clic su Importa, individuare e selezionare la programmazione lavoro, quindi fare clic su OK.

Esporta una programmazione lavoro

  1. In Salva con nome, digitare un nome per la programmazione lavoro.
  2. Per salvare il file, nel campo Dove, utilizzare le frecce o fare clic per spostarsi in una cartella.
  3. Fare clic su Salva.
