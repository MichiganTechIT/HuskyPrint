Accounting Prompt

If Accounting is enabled and the Accounting Prompt is set to Always Prompt, when you submit a print job, the Accounting Prompt appears.

  1. For User ID, if a default value is specified, you can use it, or type a new value.
  2. For Account ID, if a default value is specified, you can use it, or type a new account ID.
  3. Click OK.
