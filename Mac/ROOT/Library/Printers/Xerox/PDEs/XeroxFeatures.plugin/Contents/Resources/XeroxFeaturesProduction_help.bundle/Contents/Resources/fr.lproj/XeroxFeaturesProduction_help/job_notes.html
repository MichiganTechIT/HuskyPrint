Notes de travail

Notes de travail permet de joindre des informations supplémentaires à un travail d'impression. Vous pouvez enregistrer les paramètres des travaux d'impression au format PDF ou PostScript, puis afficher un aperçu du fichier PDF ou l'envoyer par télécopie ou courrier électronique, selon les besoins.

Remarque : Toutes les imprimantes ne prennent pas en charge la totalité des options mentionnées. Certaines options s'appliquent uniquement à un modèle d'imprimante, un système d'exploitation ou un type de pilote spécifique.

Pour imprimer à l'aide de l'option Notes de travail :

  1. Cliquez sur Notes travail.
  2. Effectuez une ou pusieurs des opérations suivantes :
    • Spécifiez le nom du travail.

      Le nom du travail correspond au nom du document qui apparaît sur la page de garde du travail d'impression. Le nom du document apparaît également dans la file d'impression sur l'imprimante.

      Pour ajouter un nom de travail, effectuez l'une des opérations suivantes :

      • En regard de Nom du travail, cliquez sur la flèche, puis sélectionnez Utiliser le nom du document.
      • Cliquez sur la flèche située à droite, puis sélectionnez un nom de travail précédemment entré.
      • Entrez un nouveau nom.
    • Sélectionnez le nom de l'expéditeur.

      Le nom de l'expéditeur correspond au nom de la personne qui a envoyé le document ; il est imprimé sur la page de garde du travail d'impression. Le nom de l'expéditeur apparaît également dans la file d'impression sur l'imprimante.

      • Cliquez sur la flèche située à droite, puis sélectionnez Utiliser le nom de connexion.
      • Cliquez sur la flèche située à droite, puis sélectionnez un nom de travail précédemment entré.
      • Entrez un nouveau nom.
    • Sélectionnez le nom du destinataire.

      Le nom du destinataire correspond au nom de la personne qui reçoit le travail d'impression ; il est imprimé sur la page de garde.

      • Cliquez sur la flèche située à droite, puis sélectionnez Utiliser le nom de connexion.
      • Cliquez sur la flèche située à droite, puis sélectionnez un nom de travail précédemment entré.

        Remarque : Pour effacer la liste des noms entrés précédemment, cliquez sur Effacer la liste.

      • Entrez un nouveau nom.
    • Tapez le message de la page de garde.

      L'option Message de la page de garde permet de saisir des informations que vous voulez faire figurer sur la page de garde d'un travail d'impression.

    • Tapez un message de début du travail.

      L'option Message de début du travail permet de donner des instructions à l'opérateur de l'imprimante. Exemple : Merci d'imprimer ce fichier d'ici 14h00.

    • Indiquez les informations de compte supplémentaires.

      L'option Informations supplémentaires de compte permet de saisir des informations relatives au travail ou à la facturation que vous voulez inclure dans le travail d'impression.

  3. Cliquez sur Impression.
