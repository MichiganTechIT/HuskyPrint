Job Type

Job Type controls how a job prints. You can require a passcode for print jobs, store a document at the printer, or set a specific time for the print job to print. It can also send a fax.

Note: Not all options listed are supported on all printers. Some options apply only to a specific printer model, configuration, operating system, or driver type.

Normal Print

Normal Print does not save to the printer for later use and does not need a passcode. Normal Print is the default job type. Use Normal Print to print a job immediately.

To print using Normal Print:

  1. In the Paper/Output window, next to Job Type, click the arrows, then select Normal Print.
  2. Make other selections for the print job as needed.
  3. Click Print.
