Выбор расширенных настроек бумаги

Функция «Выбор расширенных настроек бумаги» служит для выбора различных параметров бумаги в одном окне. В разделе «Информация о бумаге» отображается краткий перечень выбранных параметров.

Примечание: Не все перечисленные опции поддерживаются всеми принтерами. Некоторые опции предназначены только для определенных моделей и конфигураций принтера, операционных систем и драйверов.

Выбор параметров бумаги в разделе «Выбор расширенных настроек бумаги»:

  1. Для добавления или изменения параметров бумаги с именем с полем «Бумага» выберите значение Бумага с именем, затем выберите параметр.

    Примечание: В драйвере принтера хранится 10 имен. При добавлении 11-го имени из списка удаляется имя, которое использовалось меньше других. Для удаления из списка всех имен, кроме выделенного, нажмите кнопку Удалить список.

  2. В окне «Бумага/Вывод» нажатием стрелок рядом с полем «Бумага» выберите Выбор расширенных настроек бумаги.
  3. Для установки параметров снимите флажок «Исп. только имя (эксперт)».
    • Выберите значение в поле Формат бумаги или нажмите кнопку Настройка пользователя и создайте настраиваемый формат бумаги.
    • Выберите значение в поле Цвет бумаги или нажмите кнопку Настройка пользователя и создайте настраиваемый цвет бумаги.
    • Выберите значение в поле Тип бумаги или нажмите кнопку Настройка пользователя и создайте настраиваемый тип бумаги.
    • Выберите значение в поле Плотность бумаги (г/кв.м.) или нажмите кнопку Настройка пользователя и создайте пользовательскую плотность бумаги.
    • Выберите один из вариантов в поле «Бумага с отверстиями».
    • Выберите один из вариантов в поле «Стороны с покрытием».

      Покрытие бумаги влияет на вид отпечатков. Настройка «Стороны с покрытием» используется при указании покрытия для отпечатков. Покрытие можно указать для одной или для обеих сторон листа бумаги.

    • Выберите один из вариантов в поле «Тип покрытия».

      Указывается тип покрытия бумаги. Тип покрытия может влиять на внешний вид напечатанного документа.

    • Выберите один из вариантов в поле «Направление волокон».

      Настройка «Направление волокон» служит для указания направления волокон бумаги для работы печати. В некоторых случаях направление волокон может влиять на внешний вид печатаемых документов.

    • Выберите один из вариантов в поле «Финишная обработка материала».

      Настройка «Финишная обработка материала» служит для выбора типа послепечатной обработки для работы печати. Она влияет на внешний вид печатаемых документов.

    • В поле Описание выберите значение «Настройка по умолч.», затем выберите описание в списке или введите новое.
    • Для указания дополнительных сведений о заданных настройках введите их в поле Комментарий (дополнительно).
  4. Нажмите кнопку OK.
