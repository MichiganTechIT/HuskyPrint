Options Image

Les options image permettent de régler le niveau de luminosité global du texte et des images de votre travail d'impression.

Remarque : Toutes les imprimantes ne prennent pas en charge la totalité des options mentionnées. Certaines options s'appliquent uniquement à un modèle d'imprimante, un système d'exploitation ou un type de pilote spécifique.

Pour régler les options image de votre travail d'impression :

  1. Pour régler la luminosité, déplacez le curseur vers la droite ou la gauche pour respectivement éclaircir ou foncer les couleurs du travail d'impression.
  2. Pour régler le contraste, déplacez le curseur vers la droite pour réduire le contraste ou vers la gauche pour l'augmenter.
  3. Pour sélectionner les options de décalage image, cliquez sur Décalage image.
  4. Cliquez sur Impression.
