Ajuste da altura da linha

O recurso Ajuste da gramatura da linha permite selecionar a claridade ou o escurecimento das bordas de linhas de trabalhos de impressão.

Para ajustar a gramatura da linha:

Na janela Avançado, ao lado de Ajustar gramatura da linha, clique nas setas e selecione uma opção.

  • -2 Pixels: Usa a gramatura de linha mais fina para o trabalho de impressão.
  • -1 Pixel: Usa uma gramatura de linha fina para o trabalho de impressão.
  • Nenhuma: Use a gramatura de linha padrão da impressora para o trabalho de impressão.
  • +1 Pixel: usa uma gramatura de linha grossa para o trabalho de impressão.
  • +2 Pixels: usa a gramatura de linha mais grossa para o trabalho de impressão.
