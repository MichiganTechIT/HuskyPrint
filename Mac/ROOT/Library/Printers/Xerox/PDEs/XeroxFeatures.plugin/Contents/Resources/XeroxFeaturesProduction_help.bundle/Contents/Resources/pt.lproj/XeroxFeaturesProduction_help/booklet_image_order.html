Ordem de imagem de livreto

Use as opções de Layout de livreto avançado para controlar a ordem da página de livretos impressos. Use a Ordem de Imagem de Livreto para especificar a dobra de livreto no lado esquerdo (esquerda para direita) ou no lado direito (direita para esquerda).

Nota: Esta função somente estará disponível quando Layout de livreto estiver ativada.

Para selecionar opções da Ordem de imagem de livreto:

Na janela Avançado, selecione uma opção ao lado da Ordem de imagem de livreto.

  • Esquerda para direita: Esta opção define o layout de livreto, de modo que quando ele estiver dobrado, a dobra ficará no lado esquerdo do livreto. Esquerda para direita refere-se à direção de leitura do texto.
  • Direita para esquerda configura o layout de livreto, de modo que quando o livreto for dobrado, a dobra permaneça no lado direito. Selecione esta opção se estiver usando um idioma de leitura da direita para esquerda, como Árabe ou Farsi.
