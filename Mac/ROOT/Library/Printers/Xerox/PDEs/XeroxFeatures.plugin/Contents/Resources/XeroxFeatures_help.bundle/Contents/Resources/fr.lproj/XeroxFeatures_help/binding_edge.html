Bord de reliure

Bord de reliure perfore ou agrafe le travail d'impression sur le grand ou sur le petit côté. Cette option s'applique seulement aux supports qui peuvent accepter la finition sur l'un ou l'autre côté.

Pour sélectionner les options Bord de reliure :

Dans la fenêtre Avancé, en regard de Bord de reliure, cliquez sur les flèches et sélectionnez une option.

  • Automatique (Recommandé) relie seulement le travail d'impression sur le grand côté ou le petit côté, pas les deux.
  • Reliure grand côté relie le travail d'impression sur le bord long du papier.
  • Reliure petit côté relie le travail d'impression sur le bord court du papier.
