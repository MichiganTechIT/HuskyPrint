Hinzufügen oder Bearbeiten eines Faxempfängers

Nach dem Erstellen von Faxempfänger-Datensätzen können diese dem persönlichen Telefonbuch hinzugefügt werden.

Empfängerdatensätze hinzufügen:

  1. Im Fenster "Material/Ausgabe" neben "Auftragsart" auf die Pfeile klicken und dann Fax wählen.
  2. Auf Empfänger klicken.
  3. Auf die Schaltfläche Empfänger hinzufügen klicken.
  4. Die Kontaktinformationen eingeben.
  5. Auf OK klicken.

Faxempfänger ohne Bearbeitung des Telefonbuch-Datensatzes bearbeiten:

  1. Im Fenster "Material/Ausgabe" neben "Auftragsart" auf die Pfeile klicken und dann Fax wählen.
  2. Auf Empfänger klicken.
  3. Den Empfänger in der Tabelle auswählen und auf Bearbeiten klicken.
  4. Die Informationen nach Bedarf ändern.
  5. Auf OK klicken.
