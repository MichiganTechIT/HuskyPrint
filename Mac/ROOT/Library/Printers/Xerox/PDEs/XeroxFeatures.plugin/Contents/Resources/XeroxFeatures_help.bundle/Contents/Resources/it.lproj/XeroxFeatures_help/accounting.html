Contabilità

L'opzione Contabilità consente di associare nome utente e informazioni di identificazione del conto con ogni documento stampato. I dati di contabilità vengono raccolti e memorizzati all'invio dei lavori alla stampante.

Note:

  • Non tutte le opzioni elencate sono supportate su tutte le stampanti. Alcune opzioni si applicano solo a modelli di stampante, configurazioni, sistemi operativi o tipi di driver specifici.
  • In base alle selezioni Sistema di contabilità e Richiesta contabilità, non tutte le opzioni vengono selezionate.

Per impostare la contabilità:

  1. Nella finestra Contabilità, accanto a Sistema di contabilità, fare clic sulle frecce, quindi selezionare un'opzione.
    • Xerox® Contabilità standard: consente di tenere traccia del numero di lavori di copia, stampa, scansione e fax per ciascun utente. Utilizza gli ID utente e gli ID conto predefiniti nella stampante.
    • Xerox® Contabilità di rete: consente di ottenere ID utente e ID conto da un server di autenticazione di rete.
    • Contabilità locale: richiede un codice di contabilità per la sottorete. Questo codice di contabilità può essere un codice di fatturazione specifico del reparto, ad esempio 0001. Altri codici sono opzionali.
    • Addebita stampa: richiede un ID fatturazione per lavori di stampa, ad esempio un ID fatturazione del reparto. Altri ID sono opzionali.
    • Invia ID utente e ID conto: richiede di digitare l'ID utente, l'ID contro e la password ogni volta che si invia un lavoro di stampa.
    • Invia solo ID utente: richiede di digitare l'ID utente e la password per inviare un lavoro di stampa.
    • Invia solo ID conto: richiede di digitare l'ID conto e la password per inviare un lavoro di stampa.
    • Invia solo ID fatturazione: richiede di digitare l'ID fatturazione e la password per inviare un lavoro di stampa.
    • Invia tutto: richiede di digitare l'ID utente, l'ID conto, l'ID fatturazione e la password per inviare un lavoro di stampa.
  2. Accanto a Richiesta contabilità, selezionare un'opzione.
    • Richiedi sempre richiede di inserire l'ID utente e l'ID contro ogni volta che si invia un lavoro di stampa.
    • Nessuna richiesta utilizza le informazioni immesse in ID conto per ogni lavoro di stampa.
    • Richiedi solo per fax: richiede di inserire l'ID utente e l'ID conto ogni volta che si invia un fax.
    • Richiesta solo per colore richiede di inserire l'ID utente e l'ID conto solo per lavori di stampa a colori.
    • Richiesta solo per colore e fax richiedere di inserire l'ID utente e l'ID conto solo per lavori di stampa a colori e fax.
  3. Per nascondere la visualizzazione dell'ID utente, selezionare Nascondi ID utente. Gli asterischi * sostituiscono gli eventuali caratteri digitati nel campo.
  4. Per nascondere la visualizzazione dell'ID conto, selezionare Nascondi ID conto. Gli asterischi * sostituiscono gli eventuali caratteri digitati nel campo.
  5. In Codici di contabilità predef., digitare i valori in base alle esigenze. Questi campi sono obbligatori quando si seleziona Nessuna richiesta. L'ID conto e l'ID fatturazione possono essere un codice di fatturazione specifico del reparto, ad esempio 0001.
  6. Fare clic su OK.
