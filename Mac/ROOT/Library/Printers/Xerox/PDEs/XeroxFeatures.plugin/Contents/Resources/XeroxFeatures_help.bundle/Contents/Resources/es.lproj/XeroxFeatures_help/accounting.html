Contabilidad

Contabilidad asocia la información de identificación de nombre de usuario y cuenta con cada documento que se imprime. Los datos de contabilidad se recopilan y guardan cuando se envían trabajos a la impresora.

Notas:

  • No todas las opciones descritas están disponibles en todas las impresoras. Algunas opciones solo se aplican a un modelo de impresora, configuración, sistema operativo o tipo de controlador específico.
  • Dependiendo de las selecciones en Sistema de contabilidad e Indicador de contabilidad, no todas las opciones aparecen.

Para configurar la contabilidad:

  1. En la ventana Contabilidad, junto a Sistema de contabilidad, haga clic en las flechas y, a continuación, seleccione una opción.
    • Xerox® Contabilidad estándar: supervisa el número de trabajos de copia, impresión, escaneado y fax de cada usuario. Utiliza las ID de usuario y las ID de cuenta predefinidas en la impresora.
    • Xerox® Contabilidad de red: obtiene las ID de usuario y las ID de cuenta de un servidor de autenticación de la red.
    • Contabilidad local: requiere un código de contabilidad de la subred. El código de contabilidad puede ser un código de facturación de departamento, como por ejemplo, 0001. Otros códigos son opcionales.
    • Cobrar impresión: necesita una ID de facturación para los trabajos de impresión, por ejemplo, la ID de facturación de su departamento. Otras ID son opcionales.
    • Enviar ID de usuario e ID de contabilidad: requiere la introducción de la ID de usuario, de la ID de contabilidad y de la clave para enviar un trabajo de impresión.
    • Sólo enviar ID de usuario: requiere la introducción de la ID de usuario y de la clave para enviar un trabajo de impresión.
    • Sólo enviar ID de contabilidad: requiere la introducción de la ID de contabilidad y de la clave para enviar un trabajo de impresión.
    • Sólo enviar ID de facturación: requiere la introducción de la ID de facturación y de la clave para enviar un trabajo de impresión.
    • Enviar todo: requiere la introducción de la ID de usuario, de la ID de contabilidad, de la ID de facturación y de la clave para enviar un trabajo de impresión.
  2. Junto a Indicador de contabilidad, seleccione una opción.
    • Indicar siempre requiere la introducción de las ID de usuario y de cuenta cada vez que se envía un trabajo de impresión.
    • No indicar utiliza la información introducida en Códigos de cuentas para cada trabajo de impresión.
    • Solo indicar para Fax: requiere la introducción de las ID de usuario y de cuenta cada vez que se envía un fax.
    • Solo indicar para Color requiere la introducción de las ID de usuario y de cuenta solo para realizar trabajos de impresión en color.
    • Solo indicar para Color y Fax requiere la introducción de las ID de usuario y de cuenta solo para realizar trabajos de impresión en color y de fax.
  3. Para ocultar la ID de usuario, seleccione Ocultar ID de usuario. Los asteriscos * sustituyen a los caracteres introducidos en este campo.
  4. Para ocultar la ID de cuenta, seleccione Ocultar ID de cuenta. Los asteriscos * sustituyen a los caracteres introducidos en este campo.
  5. En Códigos de contabilidad prefijados, escriba los valores que sean precisos. Estos campos son obligatorios cuando se selecciona No indicar. La ID de cuenta y la ID de facturación pueden ser el código de facturación de un departamento, como por ejemplo, 0001.
  6. Haga clic en Aceptar.
