Comptabilité

La fonction de comptabilisation associe votre nom d'utilisateur et votre ID de compte à chaque document que vous imprimez. Les données de comptabilisation sont rassemblées et enregistrées lorsque les travaux sont soumis à l'imprimante.

  1. Pour accéder aux paramètres de comptabilisation, dans la fenêtre Avancé, en regard de Configuration, cliquez sur Comptabilisation.
  2. Sous Système de comptabilisation, sélectionnez une option.
    • Pour configurer la comptabilisation, cliquez sur Activé(e).
    • Pour désactiver la comptabilisation, cliquez sur Aucune.
  3. En regard d'Invite de comptabilisation, sélectionnez une option.
    • Ne pas demander : ne requiert pas l'entrée de vos ID d'utilisateur ou de compte chaque fois que vous envoyez un travail d'impression.
    • Toujours demander : requiert l'entrée de vos ID d'utilisateur et de compte chaque fois que vous envoyez un travail d'impression.
  4. Si vous sélectionnez Toujours demander :
    1. Pour masquer un ID utilisateur ou de compte lors de l'impression d'un travail, sélectionnez la case en regard de ces options.
    2. Le cas échéant, tapez une ID utilisateur par défaut.
    3. Le cas échéant, tapez une ID de compte par défaut ou un code de facturation.
  5. Cliquez sur OK.
