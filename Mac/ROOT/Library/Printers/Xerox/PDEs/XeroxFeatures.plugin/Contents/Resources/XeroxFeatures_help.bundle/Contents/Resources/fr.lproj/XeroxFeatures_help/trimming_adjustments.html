Réglages rognage :

Pour régler le rognage :

  1. Dans la fenêtre Réglages rognage, réglez le rognage selon vos besoins.
    • Verticale : cette fonction permet de rogner le bloc du cahier verticalement.
    • Rotation supérieure : permet de régler avec précision la tête de rognage supérieure.
    • Rotation supérieure : permet de régler avec précision la tête de rognage avant. Ce réglage permet de veiller à ce que le bord avant du cahier soit parallèle au bord de reliure.
    • Rotation inférieure : permet de régler avec précision la tête de rognage inférieure.
  2. Cliquez sur OK.
