Сообщение системы учета

Если включен учет и для параметра «Сообщение системы учета» установлено значение «Всегда выдавать сообщение», при отправке работы печати выводится сообщение системы учета.

  1. Если в поле «ID пользователя» указано значение по умолчанию, можно использовать его или ввести другое значение.
  2. Если в поле «ID счета» указано значение по умолчанию, можно использовать его или ввести другое значение.
  3. Нажмите кнопку OK.
