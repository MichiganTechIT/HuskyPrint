工作記錄

「工作記錄」會將使用者名稱和帳戶識別資訊與您印的每份文件建立關聯。 然後將工作提交至印表機時，會收集並儲存工作記錄資料。

注意：

  • 並非所有印表機都支援所有列出的選項。 某些選項只適用於特定印表機機型、設定、作業系統或驅動程式類型。
  • 視工作記錄系統和工作記錄提示選擇而定，有些選項可能不會出現。

如要設定工作記錄：

  1. 在「進階」視窗中，在「工作記錄系統」旁，按一下箭頭，然後選擇一個選項。
    • Xerox® 標準工作記錄：記錄每位使用者的影像、列印、掃描和傳真工作數量。 它會使用在印表機中預先定義的使用者 ID 和帳戶 ID。
    • Xerox® 網路工作記錄：從網路驗證伺服器取得使用者 ID 和帳戶 ID。
    • 本機工作記錄：要求輸入您子網路的工作記錄碼。 此工作記錄碼可以是部門的計次代碼，例如 0001。 其他代碼為選用。
    • 收費列印：需要列印工作的計次 ID，例如您部門的計次 ID。 其他 ID 為選用。
    • 傳送使用者 ID 和帳戶 ID： 要求您輸入使用者 ID、帳戶 ID 和密碼，以提交列印工作。
    • 僅傳送使用者 ID： 要求您輸入使用者 ID 和密碼，以提交列印工作。
    • 僅傳送帳戶 ID： 要求您輸入帳戶 ID 和密碼，以提交列印工作。
    • 僅傳送計次 ID： 要求您輸入計次 ID 和密碼，以提交列印工作。
    • 全部傳送： 要求您輸入使用者 ID、帳戶 ID、計次 ID 和密碼，以提交列印工作。
  2. 在「工作記錄提示選」旁，選擇一個選項。
    • 一律出現提示會要求您在每次提交列印工作時，輸入您的使用者 ID 和帳戶 ID。
    • 不出現提示會對每個列印工作使用您在「工作記錄碼」之下輸入的資訊。
    • 僅針對傳真出現提示： 要求您在每次傳送傳真時，輸入您的使用者 ID 和帳戶 ID。
    • 僅針對彩色列印出現提示僅會針對彩色列印要求您輸入使用者 ID 和帳戶 ID。
    • 僅針對彩色列印和傳真出現提示僅會針對彩色列印和傳真要求您輸入使用者 ID 和帳戶 ID。
  3. 如要隱藏使用者 ID 的顯示，請選擇「遮蓋使用者 ID」。 星號 * 會取代欄位中輸入的任何字元。
  4. 如要隱藏帳戶 ID 的顯示，請選擇「遮蓋帳戶 ID」。 星號 * 會取代欄位中輸入的任何字元。
  5. 在「預設工作記錄碼」中，視需要輸入值。 當您選擇不出現提示時，就必須填寫這些欄位。 帳戶 ID 和計次 ID 可以是部門的計次代碼，例如 0001。
  6. 按一下確定。
