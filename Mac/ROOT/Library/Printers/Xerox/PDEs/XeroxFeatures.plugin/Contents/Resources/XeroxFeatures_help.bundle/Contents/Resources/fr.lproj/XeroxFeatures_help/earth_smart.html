Paramètres intelligents

Permettent de réduire la pollution en réduisant l'utilisation de matériaux et d'énergie lors de l'impression. Vous pouvez sélectionner les paramètres intelligents comme paramètres par défaut pour tous les travaux d'impression. Pour accéder aux paramètres intelligents, cliquez sur le bouton Paramètres intelligents.

Le bouton Paramètres intelligents se trouve au bas du pilote d'imprimante. Utilisez ce bouton pour sélectionner les paramètres intelligents pour tous les travaux d'impression.

Remarques :

  • Toutes les imprimantes ne prennent pas en charge la totalité des options mentionnées. Certaines options s'appliquent uniquement à un modèle d'imprimante, un système d'exploitation ou un type de pilote spécifique.
  • Toute modification apportée aux paramètres intelligents d'un travail d'impression envoyé depuis une application logicielle s'applique uniquement au travail en cours. En revanche, lorsqu'ils sont effectués au sein du pilote, les changements sont répercutés sur les paramètres par défaut de l'ensemble des applications.

Pour sélectionner des paramètres intelligents :

  1. Dans la fenêtre Paramètres intelligents, sélectionnez les options requises.
    • Pour imprimer chaque travail sur les deux faces du papier, sélectionnez Recto verso.
      • Recto verso : permet d'imprimer sur les deux faces d'une feuille de papier, les images étant imprimées de sorte que le travail puisse être relié sur le grand côté de la feuille. Les illustrations suivantes montrent les résultats pour les documents avec orientations portrait et paysage, respectivement :

    • Pour réduire la consommation de papier et de toner ou d'encre, sélectionnez l'option Jeu échantillon.

      Le type de travail Jeu échantillon permet d'imprimer un seul exemplaire d'un travail avant de procéder à l'impression des autres exemplaires requis. Après avoir examiné le jeu échantillon, vous pouvez soit imprimer les exemplaires restants, soit les supprimer depuis le panneau de commande de l'imprimante.

    • Pour imprimer un exemplaire d'un travail imprimé en plusieurs exemplaires et mettre les autres exemplaires en attente sur l'imprimante, sélectionnez Définir la fonction intelligente comme mon type de travail.
    • Pour ne pas imprimer de pages de garde avec chaque travail, sélectionnez Pas de page de garde ou Désactiver les pages de garde. Sélectionnez une option :

      Une page de garde est une page de séparation imprimée avec un travail. Une page de garde indique le nom d'utilisateur réseau de la personne qui a envoyé le travail, ainsi que le nom du document. Les pages de garde permettent de séparer les travaux terminés reçus dans les bacs récepteurs. Sur certaines imprimantes, le paramètre défini sur le panneau de commande a priorité sur l'option de page de garde sélectionnée au niveau du pilote d'imprimante.

      • Ne pas imprimer l'ID de travail : aucune information d'identification du travail n'est imprimée.
      • Imprimer ID dans la marge - 1ère page : imprime l'identification du travail sur la première page du document dans la marge, en haut à gauche.
      • Imprimer ID dans les marges – Toutes les pages : imprime l'identification du travail dans la marge, en haut à gauche de chaque page du document.
    • Pour économiser le toner en réduisant la qualité des images produites, sélectionnez l'option Mode brouillon ou Définir Mode Brouillon comme ma qualité d'impression.
  2. Cliquez sur OK.
