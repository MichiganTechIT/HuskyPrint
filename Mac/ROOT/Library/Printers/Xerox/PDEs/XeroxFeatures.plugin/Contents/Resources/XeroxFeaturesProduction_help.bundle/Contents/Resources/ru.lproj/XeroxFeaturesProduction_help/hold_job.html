Задержать работу

Функция «Задержать работу» служит для удержания работы на принтере до тех пор, пока не будет разблокирована с панели управления. После разблокирования для печати работы на сервере печати не сохраняется.

Задержание работы печати:

  1. В окне «Бумага/Вывод» нажатием стрелок рядом с полем «Тип работы» выберите значение Задержать работу.
  2. Выберите остальные настройки для работы печати.
  3. Нажмите кнопку Печать. Работа будет оставаться в списке работ на сервере печати до тех пор, пока не будет разблокирована для печати.
