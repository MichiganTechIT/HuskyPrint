Брошюра/Водяные знаки

В окне «Брошюра/Водяные знаки» выберите макет брошюры страницы и опции водяного знака.

Примечание: Не все перечисленные опции поддерживаются всеми принтерами. Некоторые опции предназначены только для определенных моделей и конфигураций принтера, операционных систем и драйверов.

Печать с использованием функции «Брошюра/Водяные знаки»:

  1. В окне «Брошюра/Водяные знаки» нажатием стрелок рядом с полем «Макет брошюры» выберите один из вариантов.
    • Включено — документ печатается в виде брошюры. Используется для документов, не сформатированных в виде брошюры.
    • Выключено — отключение функции «Макет брошюры». Используется для документов, сформатированных в виде брошюры.
  2. Для указания других настроек брошюры выберите «Макет брошюры» и нажмите Опции брошюры.
  3. Чтобы вокруг каждой страницы печаталась рамка, выберите опцию Печать границ страницы.
  4. Выберите настройки в разделе Водяной знак.
  5. Нажмите кнопку Печать.
