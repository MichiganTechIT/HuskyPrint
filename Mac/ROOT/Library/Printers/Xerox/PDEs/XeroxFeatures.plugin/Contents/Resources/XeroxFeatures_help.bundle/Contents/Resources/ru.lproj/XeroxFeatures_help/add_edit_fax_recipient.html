Добавление в список получателей факса и внесение изменений

После создания записей получателей факса их можно добавить в персональную телефонную книгу.

Добавление получателей:

  1. В окне «Бумага/Вывод» нажатием стрелок рядом с полем «Тип работы» выберите значение Факс.
  2. Нажмите кнопку Получатели.
  3. Нажмите кнопку Добавить получателя.
  4. Введите информацию для контакта.
  5. Нажмите кнопку OK.

Изменение информации о получателе факса без редактирования записи в телефонной книги:

  1. В окне «Бумага/Вывод» нажатием стрелок рядом с полем «Тип работы» выберите значение Факс.
  2. Нажмите кнопку Получатели.
  3. Выберите получателя в таблице и нажмите кнопку Редактировать.
  4. При необходимости измените информацию.
  5. Нажмите кнопку OK.
