Добавление исключений

Функция «Добавить исключения» служит для указания страниц-исключений в работе печати. Настройки страниц-исключений отличаются от настроек остальных страниц в работе. Отличаться могут, например, заданные настройки формата, типа и цвета бумаги. Также можно указывать другую сторону листа бумаги для печати работы. В работе печати могут содержаться разные исключения.

Примечание: Не все перечисленные опции поддерживаются всеми принтерами. Некоторые опции предназначены только для определенных моделей и конфигураций принтера, операционных систем и драйверов.

Добавление страницы-исключения:

  1. В окне «Специальные страницы» выберите Добавить исключения.
  2. В окне «Страницы» укажите страницу или диапазон страниц, к которым применяется исключение. Разделяйте отдельные страницы или диапазоны страниц запятыми. Для указания диапазона страниц используйте дефис. Например, чтобы задать страницы 1, 6, 9, 10 и 11, следует ввести 1, 6, 9-11.
  3. Для указания размера, цвета и типа выберите значения нажатием стрелок в разделе «Бумага».
    • Если в поле «Тип бумаги» выбирается «Бумага с выступами», необходимо ввести количество выступов в поле «Листов в комплекте».
    • Если для типа бумаги выбирается опция «В комплекте», следует указать название типа бумаги и количество листов в последовательности.
  4. Для изменения сторон для печати выберите нужную опцию в разделе «2-сторонняя печать»:
    • 1-сторонняя печать - печать на страницах-исключениях выполняется только с одной стороны.
    • 2-сторонняя печать - печать на страницах-исключениях выполняется с обеих сторон листа, а листы переворачиваются по длинной кромке. Работу затем можно сшивать вдоль длинной кромки листов бумаги.
    • 2-ст. печать, переворот по корот. кромке - печать на страницах-исключениях выполняется с обеих сторон листа, а листы переворачиваются по короткой кромке. Работу затем можно сшивать вдоль короткой кромки листов бумаги.
  5. Для сдвига изображения на странице выберите Сдвиг изображения.
  6. Нажмите кнопку OK.
