工作記錄

「工作記錄」會將使用者名稱和帳戶識別資訊與您印的每份文件建立關聯。 然後將工作提交至印表機時，會收集並儲存工作記錄資料。

  1. 如要存取工作記錄設定，在「進階」視窗中，在「設定」旁，按一下工作記錄。
  2. 針對「工作記錄系統」，選擇一個選項。
    • 如要設定工作記錄，請按一下啟用。
    • 如要停用工作記錄，請按一下無。
  3. 在「工作記錄提示選」旁，選擇一個選項。
    • 不出現提示： 當您提交列印工伯時，不出現提示您輸入使用者 ID 或帳戶 ID。
    • 一律出現提示： 要求您在每次提交列印工作時，輸入您的使用者 ID 和帳戶 ID。
  4. 如果您選擇「一律出現提示」：
    1. 當列印工作時，如要遮蓋使用者 ID 或帳戶 ID，在這些選項旁，選擇此核取方塊。
    2. 如有需要，輸入預設使用者 ID 值。
    3. 如有需要，輸入預設帳戶 ID 或計次代碼值。
  5. 按一下確定。
