목차

다음으로 인쇄

다음으로 인쇄 옵션을 사용하여 저장된 개인 및 공용 작업을 현재 인쇄 작업과 함께 인쇄할 수 있습니다. 예를 들어 브로슈어나 가격표와 같은 저장된 문서 집합과 함께 표지를 인쇄합니다.

참고: 이 작업 유형은 네트워크에 연결되어 있으며 양방향 통신이 설정된 프린터에서만 사용 가능합니다.

다음으로 인쇄 옵션을 사용하여 인쇄:

  1. 용지/출력 창에서 [작업 유형] 옆에 있는 화살표를 클릭한 후 다음으로 인쇄를 선택합니다.
  2. 프롬프트가 표시되면 프린터에 저장된 작업을 할당한 비밀번호를 입력한 후 확인을 클릭합니다.

    비밀번호를 할당하지 않은 경우 취소를 클릭합니다.

  3. 저장된 작업 목록에서, 다음 옵션 중 하나를 선택합니다.
    • 개인 및 공유 저장 작업
    • 공유 저장 작업

      공유 저장 작업 옵션은 프린터 하드 드라이브에 저장된 공유 작업을 표시합니다. 어떤 사용자든 공유 작업을 선택할 수 있습니다.

    • 개인 저장 작업

      개인 저장 작업 옵션은 사용자 이름과 현재 비밀번호로 저장된 작업을 인쇄할 수 있도록 표시합니다. 비밀번호 를 클릭하여 프린터 하드 드라이브에 개인 저장 작업을 저장할 때 사용한 것과 동일한 비밀번호를 입력합니다.

      참고: 개인 저장 작업에는 비밀번호가 필요합니다.

  4. 개인 및 공유 저장 작업 또는 개인 저장 작업을 선택하는 경우 작업의 비밀번호를 입력합니다. 작업이 나타나지 않으면 비밀번호를 클릭한 후 작업의 올바른 비밀번호를 다시 입력합니다.
  5. 최종 작업 목록에 작업을 추가하려면 목록에서 작업을 선택한 후 아래쪽 화살표를 클릭합니다.
  6. 최종 작업 목록의 작업 순서를 변경하려면 작업을 선택한 후 화살표를 사용합니다.
  7. 목록에서 작업을 삭제하려면 작업을 선택한 후 삭제를 클릭합니다.
  8. 확인을 클릭합니다.
  9. 인쇄를 클릭합니다.
