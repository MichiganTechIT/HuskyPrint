Watermark Editor

A watermark is a picture or text printed behind or on top of the text and images on the pages of a print job.

Note: Not all operating systems support watermark printing.

Create a Text Watermark

A Text Watermark uses the characters typed in the Text field as the watermark. You can create a text watermark to print on a document. Select options such as font size, font type, font color, and angle.

To create a text watermark:

  1. In the Booklet/Watermark window, under Watermark, click the arrows, then select New.
  2. In the Watermark Editor window, click Text.
  3. For Text, use the watermark name or type new text.
  4. To format the watermark, click Show Fonts, then select options.
  5. To specify the angle of the watermark on the page, type or select a percentage or use the slider to adjust the angle.
  6. To specify the location of the watermark on the page, select the measurement unit, then enter numbers for the Horizontal Position and Vertical Position.
  7. To center the watermark on the page, click Center.
  8. Click OK.

Create a Picture Watermark

A Picture Watermark uses an image as the watermark. The picture file can be in .bmp, .gif, or .jpg format. You can create a picture watermark to print on a document, and select options including size and position on the page.

To create a picture watermark:

  1. In the Booklet/Watermark window, under Watermark, click the arrows, then select New.
  2. In the Watermark Editor window, click Picture.
  3. Type the path and name of the picture file, or click Choose Picture, then locate and select a picture file. This option supports .bmp, .jpg, and .gif files.
  4. To scale the image, specify the percentage to enlarge or reduce the image.
  5. To shift the watermark from the center of the page, first select the measurement unit, then enter numbers for the horizontal and vertical positions.
  6. To center the watermark on the page, click Center.
  7. Click OK.

Edit a Watermark

You can change a watermark using the Watermark Editor.

To edit a watermark:

  1. In the Bookle/Watermark window, under Watermark, click the arrows, then select Edit.
  2. In the Watermark Editor window, change the selected options as needed.
  3. Click OK.

Delete a Watermark

  1. In the Booklet /Watermark window, under Watermark, select the watermark to delete, then click Delete.
  2. Click Yes to confirm the deletion.
