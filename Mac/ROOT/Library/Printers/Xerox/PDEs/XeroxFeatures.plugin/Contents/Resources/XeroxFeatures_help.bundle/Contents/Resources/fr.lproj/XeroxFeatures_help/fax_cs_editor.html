Créer/Modifier la page de garde

Vous pouvez ajouter un texte ou des images aux pages de garde de vos télécopies ou utiliser un texte et des images enregistrés sur l'imprimante.

  1. Pour ajouter ou modifier le texte sur la page de garde des télécopies.
    1. Dans la fenêtre Créer/Modifier la page de garde, sélectionnez Texte.
    2. Pour mettre en forme le texte, cliquez sur Afficher les polices et sélectionnez vos options de police dans la liste.
    3. Pour spécifier l'angle du texte, tapez une valeur dans le champ Angle. Pour ajuster la valeur, utilisez les flèches. Pour sélectionner une valeur, utilisez le curseur.
    4. Pour déplacer le filigrane qui se trouve au centre de la page, sélectionnez l'unité de mesure puis entrez les valeurs pour la position horizontale et la position verticale.
    5. Pour centrer l'image, cliquez sur Centrer.
    6. Cliquez sur OK.
  2. Pour ajouter ou modifier une image sur la page de garde des télécopies.
    1. Dans la fenêtre Créer/Modifier la page de garde, sélectionnez Image.
    2. Dans le champ Choisir le fichier, indiquez le chemin d'accès à votre fichier image ou cliquez sur Choisir le fichier pour le localiser.

      Remarque : Cette option prend en charge les fichiers .bmp, .jpg et .gif.

    3. Pour redimensionner l'image, sous Échelle, tapez ou sélectionnez un pourcentage ou modifiez l'échelle à l'aide du curseur.
    4. Pour déplacer le filigrane qui se trouve au centre de la page, sélectionnez l'unité de mesure puis entrez les valeurs pour la position horizontale et la position verticale.
    5. Pour centrer l'image, cliquez sur Centrer.
    6. Cliquez sur OK.
