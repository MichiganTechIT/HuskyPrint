Bildverschiebung

„Bildverschiebung“ dient zum Ändern der Position des Druckbilds auf der Seite. Das Druckbild kann auf jedem Blatt entlang der x- und y-Achse verschoben werden.

Optionen für die Bildverschiebung festlegen:

  1. Neben "Bildverschiebungsoptionen" die Pfeile verwenden und dann eine Option wählen.
  2. Wenn im Fenster "Druckmaterial" die Option "1-seitig" ausgewählt wird, eine Option für "Bildverschiebungsoptionen" wählen.
    • Druckereinstellung: Verwendet die Druckerstandardeinstellungen.
    • Unabhängig: Die Einstellung kann manuell angepasst werden.
  3. Wenn im Fenster "Druckmaterial" die Option "2-seitig" ausgewählt wird, eine Option für "Bildverschiebungsoptionen" wählen.
    • Druckereinstellung: Verwendet die Druckerstandardeinstellungen.
    • Unabhängig: Die Einstellung kann manuell angepasst werden.
    • X spiegeln: Bilder können gleichzeitig an beiden Seiten um einen gleichen Abstand, aber in entgegengesetzte Richtungen entlang der x-Achse verschoben werden.
    • Y spiegeln: Bilder können gleichzeitig an beiden Seiten um einen gleichen Abstand, aber in entgegengesetzte Richtungen entlang der y-Achse verschoben werden.
    • Beide spiegeln: Bilder können gleichzeitig an beiden Seiten um einen gleichen Abstand, aber in entgegengesetzte Richtungen entlang der x- und der y-Achse verschoben werden.
    • X spiegeln, Y gleich: Bilder können gleichzeitig an beiden Seiten um einen gleichen Abstand, in entgegengesetzte Richtungen entlang der x-Achse und in die gleiche Richtung entlang der y-Achse verschoben werden.
    • Y spiegeln, X gleich: Bilder können gleichzeitig an beiden Seiten um einen gleichen Abstand, in entgegengesetzte Richtungen entlang der y-Achse und in die gleiche Richtung entlang der x-Achse verschoben werden.
    • X gleich: Bilder können gleichzeitig an beiden Seiten um einen gleichen Abstand und in die gleiche Richtung entlang der x-Achse verschoben werden.
    • Y gleich: Bilder können gleichzeitig an beiden Seiten um einen gleichen Abstand und in die gleiche Richtung entlang der y-Achse verschoben werden.
    • Beide gleich: Bilder können gleichzeitig an beiden Seiten um einen gleichen Abstand und in die gleiche Richtung entlang der x- und der y-Achse verschoben werden.
  4. Für "Unabhängig" die folgenden Schritte ausführen:
    1. Für die Vorderseite einen Wert für die Verschiebung entlang der X und der Y-Achse eingeben.
    2. Wenn "Rückseite" verfügbar ist, einen Betrag eingeben, um die Seite entlang der x- und der y-Achse zu verschieben.

      Hinweis: Die Rückseitenoption ist nur bei Duplexdruck verfügbar.

  5. Auf OK klicken.
