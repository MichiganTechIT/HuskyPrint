Opciones ecológicas

Las opciones ecológicas ayudan a reducir la contaminación medioambiental al imprimir utilizando menos materiales y energía. Si lo desea puede utilizar las Opciones ecológicas como valores prefijados para todos los trabajos de impresión. Para acceder a Opciones ecológicas, haga clic en el botón Opciones ecológicas.

El botón verde de Opciones ecológicas se encuentra en la parte inferior del controlador de impresión. Utilice este botón para seleccionar las opciones ecológicas para los trabajos de impresión.

Notas:

  • No todas las opciones descritas están disponibles en todas las impresoras. Algunas opciones solo se aplican a un modelo de impresora, configuración, sistema operativo o tipo de controlador específico.
  • Si se seleccionan las Opciones ecológicas en la aplicación de software, los cambios solo se aplican a dicho trabajo de impresión. Si se seleccionan las Opciones ecológicas en el controlador de impresión, las opciones prefijadas cambian para todas las aplicaciones.

Para seleccionar opciones ecológicas:

  1. En la ventana Opciones ecológicas, seleccione las opciones deseadas.
    • Para imprimir todos los trabajos utilizando ambas caras del papel, seleccione Configuración de la impresión a 2 caras.
      • Impresión a 2 caras imprime el trabajo por ambas caras del papel para permitir la encuadernación por el borde largo del papel. Las ilustraciones siguientes muestran el resultado para documentos con orientaciones vertical y horizontal respectivamente:

    • Para utilizar menos papel y tóner/tinta, seleccione Juego de muestra.

      Si se selecciona Juego de muestra se imprime una copia de un trabajo de impresión de varias copias antes de imprimir el resto del trabajo. Una vez revisado el juego de muestra, puede imprimir el resto de las copias o eliminarlas desde el panel de impresión de la impresora.

    • Para imprimir una copia de un trabajo de impresión de varias copias y retener las copias restantes en la impresora, seleccione Establecer impresión de prueba ecológica como mi tipo de trabajo.
    • Para evitar que se impriman portadas en cada trabajo de impresión, seleccione Sin portada o Desactivar portadas. Seleccione una opción:

      Una portada es una hoja de separación que se imprime con un trabajo de impresión. Una portada incluye el nombre de usuario de red de la persona que envía el trabajo y el nombre del documento. Cuanto se utiliza una portada, la portada separa los distintos trabajos en la bandeja de salida. En algunas impresoras, las opciones del panel de control tienen preferencia sobre la opción de portada seleccionada en el controlador de impresión.

      • No imprimir ID del trabajo: impide que se impriman los datos de identificación del trabajo de impresión.
      • Imprimir ID en márgenes: sólo primera página: imprime la identificación del trabajo en la parte superior del margen izquierdo de la primera página.
      • Imprimir ID en margen: todas las páginas: imprime la identificación del trabajo en la parte superior del margen izquierdo de todas las páginas del documento.
    • Para ahorrar tóner o tinta y producir imágenes de menor calidad, seleccione Modo borrador o Establecer el modo de borrador como mi calidad de impresión.
  2. Haga clic en Aceptar.
