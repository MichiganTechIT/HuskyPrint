Régler l'épaisseur de trait

La fonction Régler l'épaisseur de trait permet d'éclaircir ou de foncer le trait des travaux d'impression.

Pour régler l'épaisseur de trait :

Dans la fenêtre Avancé, en regard de Régler l'épaisseur de trait, cliquez sur les flèches et sélectionnez une option.

  • -2 Pixels : utilise l'épaisseur de trait la plus mince pour le travail d'impression.
  • -1 Pixel : utilise un trait mince pour le travail d'impression.
  • Aucun : utilise l'épaisseur de trait par défaut pour le travail d'impression.
  • +1 Pixel : utilise un trait épais pour le travail d'impression.
  • +2 Pixels : utilise le trait le plus épais pour le travail d'impression.
