Récapitulatif couleur

L'option Couleur offre des choix supplémentaires pour personnaliser les couleurs de votre travail d'impression.

Remarque : Toutes les imprimantes répertoriées ne prennent pas en charge la totalité des options mentionnées. Certaines options s'appliquent uniquement à un modèle d'imprimante, une configuration, un système d'exploitation ou un type de pilote spécifique.

Paramètres couleur

  1. Dans la fenêtre Options image, cliquez sur Couleur.
  2. Pour régler les couleurs, sous les curseurs, désélectionnez les cases à cocher Défaut Imprimante, puis déplacez les curseurs selon vos besoins.
    • Luminosité : permet de régler le niveau de luminosité global du texte et des images de votre travail d'impression.
    • Contraste : permet de régler la variation entre les zones claires et sombres du travail d'impression.
    • Saturation : permet de régler l'intensité d'une couleur et la quantité de gris qu'elle contient.
    • Nuance : comporte trois paramètres, correspondant respectivement au Cyan/Rouge, Magenta/Vert et Jaune/Bleu. Vous pouvez régler la proportion d'une couleur sélectionnée dans la sortie imprimée sans en altérer la luminosité. Si vous placez tous les curseurs au même niveau, aucune nuance n'est appliquée. Le paramètre Nuance s'applique à toutes les couleurs d'entrée, à l'exception des couleurs natives et CMJN SWOP.

      La nuance est un gris contenant une teinte d'une autre couleur, par exemple marron, rose ou jaune.

  3. Cliquez sur OK.

Options couleur

  1. Dans la fenêtre Couleur, cliquez sur les flèches et sélectionnez Options couleur.
  2. Sélectionnez les options de Surimpression du noir et Recouvrement. Pour fournir des paramètres spécifiques sur le sens du départ, sous Recouvrement, sélectionnez Activé(e).

    Le recouvrement des couleurs permet de faire en sorte que les bords des zones colorées, à l'exception des zones blanches, se chevauchent et recouvrent aussi légèrement les bords des zones noires. Le recouvrement aide à compenser les éventuelles irrégularités d'impression liées aux décalages papier ou à d'autres problèmes.

    Utilisez la surimpression du noir pour empêcher le fond d'un objet noir d'être supprimé lorsqu'il est imprimée sur de la couleur. Cette fonction est importante en cas de mauvais cadrage. Le fond supprimé fait apparaître une bande blanche sur les bords de l'objet noir. Sélectionnez Surimpression du noir pour éliminer cette bande blanche.

  3. Désélectionnez les cases à cocher Défaut imprimante avant de régler la hauteur et la largeur.
  4. Cliquez sur OK.

Couleur source

  1. Dans la fenêtre Couleur, cliquez sur les flèches et sélectionnez Couleur de la source.
  2. Pour Modèle de couleur RVB, Modèle de couleur CMJN et Modèle de couleur gris, effectuez l'une des opérations suivantes.
    1. Sous Profil des images, sélectionnez Défaut Imprimante, puis choisissez une option ou sélectionnez Personnalisé(e).
    2. Pour profil textes et graphiques, sélectionnez Défaut Imprimante, puis choisissez une option ou sélectionnez Personnalisé(e).
  3. Cliquez sur OK.

Mode couleur

Utilisez les paramètres du Mode couleur pour contrôler le rendu de la couleur dans le document imprimé. Le rendu spécifie une méthode prédéfinie pour le réglage des couleurs pendant le processus d'impression, afin d'obtenir l'aspect voulu. L'imprimante utilise le profil de destination pour faire correspondre les données couleur au modèle de couleur du périphérique approprié.

Pour sélectionner le mode couleur :

  1. Dans la fenêtre Couleurs , cliquez sur les flèches et sélectionnez Mode couleur.
  2. Sous Profil de destination, sélectionnez une option.
    • Val. impr. par défaut : utilise les paramètres actuels de l'imprimante.
    • Déterminé par le système : utilise les paramètres configurés par l'administrateur pour l'imprimante.
    • Personnalisé : ouvre la fenêtre de Sélection personnalisée, dans laquelle vous pouvez entrer une valeur.
  3. Pour les options Rendu objet image, Rendu objet texte et Rendu objet graphique, effectuez une des opérations suivantes :
    • Pour appliquer le rendu d'un objet à tous les objets, cochez la case Rendu identique pour tous les objets. Les deux autres objets sont automatiquement ajustés.
    • Pour appliquer un rendu différent à chaque objet, désélectionnez Rendu identique pour tous les objets, puis sélectionnez une option pour chaque objet.
  4. Cliquez sur OK.

Extended Gamut (Gamme de couleurs élargie)

Extended Gamut (Gamme de couleurs élargie) est une fonction de traitement de l'image qui permet d'utiliser les couleurs prises en charge par le périphérique. En plus du cyan, du magenta, du jaune et du noir, vous pouvez sélectionner d'autres couleurs de toner si elles sont disponibles sur le périphérique. Cette fonction permet d'améliorer les impressions couleur.

Remarque : Le nombre de couleurs et les couleurs spécifiques que vous pouvez ajouter dépendent du périphérique.
