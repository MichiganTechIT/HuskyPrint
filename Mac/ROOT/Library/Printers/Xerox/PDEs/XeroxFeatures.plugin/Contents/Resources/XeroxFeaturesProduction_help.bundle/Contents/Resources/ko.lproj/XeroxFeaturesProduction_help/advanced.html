고급

고급 창을 사용하여 고급 인쇄 옵션을 설정합니다. 이 창의 선택 항목은 인쇄 드라이버를 사용하여 인쇄하는 모든 작업에 적용됩니다.

설치된 인쇄 드라이버의 버전을 보려면 창 하단으로 스크롤합니다.

소책자 이미지 순서

고급 창의 [소책자 이미지 순서] 옆에 있는 옵션 중에서 하나를 선택합니다.

  • 왼쪽에서 오른쪽으로: 이 옵션은 소책자를 접을 때 소책자의 왼쪽에 접기가 오도록 소책자를 설정합니다. 왼쪽에서 오른쪽은 텍스트의 읽는 방향을 나타냅니다.
  • 오른쪽에서 왼쪽으로를 선택하면 소책자를 접을 때 소책자 레이아웃이 오른쪽으로 제본되도록 설정됩니다. 아랍어나 페르시아어처럼 오른쪽에서 왼쪽으로 읽는 언어를 사용하는 경우 이 옵션을 선택합니다.

이미지 180도 회전

고급 창에서 [이미지 180도 회전] 옆에 있는 화살표를 클릭한 후 다음 옵션 중 하나를 선택합니다.

  • 프린터 기본값: 현재 프린터 설정을 사용합니다.
  • 설정 또는 사용: 프린터에서 이미지를 자동으로 180도 회전합니다. 이미지를 회전할 때 프린터에서 용지 크기, 방향, 용지 공급 방향 및 선택된 마감 옵션을 고려합니다.
  • 해제 또는 사용 안 함: 기능을 해제합니다.

작업 ID

출력 옵션

고급 창에서 [출력 옵션] 옆에 있는 화살표를 클릭한 후 다음 옵션 중 하나를 선택합니다.

  • 프린터 기본값: 시스템 관리자가 프린터 제어판에서 설정한 기본값을 사용하여 인쇄합니다.
  • 앞면을 위로 역순 배출: 내림차순(3, 2, 1)으로 페이지 앞면이 위를 향하도록 제공합니다.
  • 앞면을 아래로(권장함)는 내림차순(3, 2, 1)으로 페이지 앞면이 아래를 향하도록 제공합니다.
  • 앞면을 위로: 오름차순(1, 2, 3)으로 페이지 앞면이 위를 향하도록 제공합니다.
  • 앞면을 아래로 역순 배출: 오름차순(1, 2, 3)으로 페이지 앞면이 아래를 향하도록 제공합니다.

거울 이미지/좌우대칭 출력

고급 창에서 [좌우대칭 출력] 또는 [거울 이미지] 옆에 있는 옵션 중에서 하나를 선택합니다.

  • 설정 또는 예: 왼쪽에서 오른쪽으로 페이지 이미지를 반전시킵니다.
  • 사용 안 함 또는 아니요: 좌우대칭 출력을 해제합니다.

Xerox 작업 티켓

반전 출력

고급 창에서 [반전 출력] 옆에 있는 화살표를 클릭한 후 다음 옵션 중 하나를 선택합니다.

  • 사용: 밝은 영역과 어두운 영역을 반전시킵니다.
  • 사용 안 함: 반전 출력을 해제합니다.

선 두께 조절

선 두께 조절 기능을 통해 인쇄 작업의 선 가장자리의 밝기를 선택할 수 있습니다.

선 두께 조절:

고급 창에서 [선 두께 조절] 옆에 있는 화살표를 클릭한 후 다음 옵션 중 하나를 선택합니다.

  • -2픽셀: 인쇄 작업의 가장 얇은 선 두께를 사용합니다.
  • -1픽셀: 인쇄 작업의 얇은 선 두께를 사용합니다.
  • 없음: 인쇄 작업의 프린터 기본 선 두께를 사용합니다.
  • +1픽셀: 인쇄 작업의 두꺼운 선 두께를 사용합니다.
  • +2픽셀: 인쇄 작업의 가장 두꺼운 선 두께를 사용합니다.

OPI 대체

고급 창에서 [OPI 대체] 옆에 있는 화살표를 클릭한 후 다음 옵션 중 하나를 선택합니다.

  • 프린터 기본값: 현재 프린터 설정을 사용합니다.
  • 사용: OPI 주석을 처리하고 인쇄 문서에서 저해상도 이미지를 고해상도 이미지로 바꾸도록 인쇄 서버에 지시합니다.
  • 사용 안 함: OPI 대체를 해제합니다.

OPI 사전 스캔

고급 창에서 [OPI 사전 스캔] 옆에 있는 화살표를 클릭한 후 다음 옵션 중 하나를 선택합니다.

  • 프린터 기본값: 현재 프린터 설정을 사용합니다.
  • 없음: 기능을 해제합니다.
  • 스캔: 고해상도 파일을 찾아서 파일 경로를 사용하여 해당 파일을 문서에 연결합니다.
  • 스캔 및 수집: 고해상도 파일을 찾아서 문서에 가져옵니다.

새 페이지 크기에 맞추기

  1. 고급 창에서 [새 페이지 크기에 맞추기] 옆에 있는 화살표를 클릭한 후 옵션을 하나 선택합니다.
  2. 사용자 지정 용지 크기를 지정하려면 사용자 정의를 클릭합니다.

    참고: 원고 크기가 출력 크기보다 큰 경우 페이지 이미지가 잘립니다.
