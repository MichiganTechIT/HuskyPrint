Support nommé

Un support nommé désigne un ensemble de caractéristiques du papier qui ont été nommées et enregistrées en vue d'une réutilisation ultérieure. Vous pouvez effectuer votre sélection dans la liste partagée des supports pour l'imprimante ou dans la liste personnalisée Ma liste de supports.

Pour sélectionner un support nommé :

  1. Dans la fenêtre Support/Réception, en regard de Support, cliquez sur les flèches et sélectionnez Support nommé.
  2. Sélectionnez une option.
    • Pour sélectionner un support dans une liste parmi ceux chargés dans l'imprimante, cliquez sur Liste de supports pour l'imprimante.
    • Pour sélectionner un support dans une liste personnelle de supports nommés, cliquez sur Ma liste de supports.

      Un support nommé désigne un ensemble de caractéristiques du papier qui ont été nommées et enregistrées en vue d'une réutilisation ultérieure. Vous pouvez modifier ou supprimer des supports nommés uniquement dans votre liste de supports (Ma liste de supports).

      Ajouter un support nommé

      1. Dans la fenêtre Support nommé, cliquez sur le menu supérieur, sélectionnez Ma liste de supports puis sélectionnez le bouton Nouveau.
      2. Dans le champ Nom du support, sélectionnez ou entrez un nom.
      3. Pour utiliser uniquement le nom du support, sélectionnez Utiliser le nom uniquement (avancé), puis entrez le nom ou sélectionnez-le dans la liste.

        Remarque : Si vous sélectionnez Utiliser le nom uniquement (avancé), les champs d'attribut ne sont pas disponibles.

      4. Effectuez l'une des opérations suivantes :
        • Sélectionnez les attributs et options de support voulus.
        • Pour utiliser les valeurs par défaut de l'imprimante, cliquez sur un menu et sélectionnez Défaut imprimante.
      5. Vous pouvez ajouter des informations supplémentaires sur l'utilisation du support nommé dans le champ Commentaire (facultatif).
      6. Cliquez sur OK.

      Copier un support nommé

      1. Dans la fenêtre Support nommé, cliquez sur le menu supérieur, sélectionnez Ma liste de supports.
      2. Dans le tableau, sélectionnez un support nommé que vous voulez copier et cliquez sur le bouton Copier .

        Remarque : Le bouton est activé après la sélection d'un nom de support dans la liste.

      3. Dans le champ Nom du support, sélectionnez ou entrez un nom.
      4. Pour utiliser uniquement le nom du support, sélectionnez Utiliser le nom uniquement (avancé), puis entrez le nom ou sélectionnez-le dans la liste.
      5. Si vous sélectionnez Utiliser le nom uniquement (avancé), les champs d'attribut ne sont pas disponibles.
      6. Effectuez l'une des opérations suivantes :
        • Conserver les attributs et options de support.
        • Sélectionnez les attributs et options de support voulus.
        • Pour utiliser les valeurs par défaut de l'imprimante, cliquez sur un menu et sélectionnez Défaut imprimante.
      7. Vous pouvez ajouter des informations supplémentaires sur l'utilisation du support nommé dans le champ Commentaire (facultatif).
      8. Cliquez sur OK.

      Modifier un support nommé

      1. Dans la fenêtre Support nommé, cliquez sur le menu supérieur, sélectionnez Ma liste de supports.
      2. Dans le tableau, sélectionnez un support nommé et cliquez sur Modifier .

        Remarque : Le bouton est activé après la sélection d'un nom de support dans la liste.

      3. Pour utiliser uniquement le nom du support, sélectionnez Utiliser le nom uniquement (avancé), puis entrez le nom ou sélectionnez-le dans la liste.

        Remarque : Si vous sélectionnez Utiliser le nom uniquement (avancé), les champs d'attribut ne sont pas disponibles.

      4. Effectuez l'une des opérations suivantes :
        • Sélectionnez les attributs et options de support voulus.
        • Pour utiliser les valeurs par défaut de l'imprimante, cliquez sur un menu et sélectionnez Défaut imprimante.
      5. Vous pouvez ajouter des informations supplémentaires sur l'utilisation du support nommé dans le champ Commentaire (facultatif).
      6. Cliquez sur OK.

      Supprimer un support nommé

      1. Dans la fenêtre Support nommé, cliquez sur le menu supérieur, sélectionnez Ma liste de supports.
      2. Sélectionnez le support nommé que vous voulez supprimer et cliquez sur le bouton Supprimer .
      3. Cliquez sur Fermer.
