예외 추가

예외 추가를 사용하여 인쇄 작업에서 예외 페이지를 지정합니다. 예외 페이지에 작업의 나머지 페이지와 다른 설정이 있습니다. 페이지 크기, 페이지 종류, 페이지 컬러와 같은 차이를 지정할 수 있습니다. 작업의 요구 사항에 따라 용지에서 인쇄할 면을 변경할 수도 있습니다. 인쇄 작업에 여러 개의 예외를 포함할 수 있습니다.

참고: 일부 프린터에서는 여기 나열된 모든 옵션이 지원되지 않습니다. 일부 옵션은 특정 프린터 모델, 구성, 운영 체제 또는 드라이버 유형에만 적용됩니다.

예외 페이지를 추가하려면 다음 작업을 수행하십시오.

  1. 특수 페이지 창에서 예외 추가를 클릭합니다.
  2. 페이지 수에는 예외를 적용할 페이지 수를 지정합니다. 개별 페이지나 페이지 범위를 쉼표로 구분합니다. 일정 범위 내의 페이지는 하이픈으로 지정합니다. 예를 들어 페이지 1, 6, 9, 10, 11을 지정하려면 1,6,9-11과 같이 입력합니다.
  3. 크기, 컬러 또는 종류를 선택하려면 [용지] 옆에 있는 화살표를 클릭한 후 옵션을 선택합니다.
    • 용지 종류의 프리 컷 탭을 선택하는 경우 [연속된 수]에 탭 수를 입력합니다.
    • 연속된 용지 종류를 선택하면 용지 종류 이름과 연속된 수를 지정합니다.
  4. 인쇄할 면을 변경하려면 양면 인쇄 아래에서 다음 옵션 중에서 선택하십시오.
    • 단면 인쇄는 한 면에만 예외 페이지를 인쇄합니다.
    • 양면 인쇄는 용지의 양면에 예외 페이지를 인쇄하고 페이지를 장변으로 넘깁니다. 그러면 작업을 페이지의 장변으로 제본할 수 있습니다.
    • 양면 인쇄, 단변으로 넘김은 용지의 양면에 예외 페이지를 인쇄하고 페이지를 단변으로 넘깁니다. 그러면 작업을 페이지의 단변으로 제본할 수 있습니다.
  5. 용지에서 이미지를 이동하려면 이미지 이동을 선택합니다.
  6. 확인을 클릭합니다.
