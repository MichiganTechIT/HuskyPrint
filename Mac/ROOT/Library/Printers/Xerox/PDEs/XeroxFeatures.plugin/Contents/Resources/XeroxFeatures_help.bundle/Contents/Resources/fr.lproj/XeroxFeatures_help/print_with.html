Imprimer avec

Cette option permet d'imprimer des travaux enregistrés privés et publics avec le travail d'impression en cours. Exemple : Imprimer une lettre de couverture avec un ensemble de documents enregistrés tels que des brochures ou des listes de prix.

Remarque : Ce type de travail n'est disponible que sur les imprimantes connectées en réseau pour lesquelles la communication bidirectionnelle est activée.

Pour imprimer avec l'option Imprimer avec :

  1. Dans la fenêtre Avancé, en regard de Type de travail, cliquez sur les flèches et sélectionnez l'option Imprimer avec.
  2. Lorsque vous y êtes invité, tapez le code d'accès que vous avez affecté aux travaux enregistrés sur le disque dur de l'imprimante, puis cliquez sur OK.

    Si vous n'avez pas affecté de code d'accès, cliquez sur Annuler.

  3. Dans la liste des travaux enregistrés, sélectionnez une option.
    • Travaux enregistrés personnels et partagés
    • Travaux enregistrés partagés

      Cette option affiche les travaux partagés enregistrés sur le disque dur de l'imprimante. Les travaux partagés peuvent être sélectionnés par n'importe quel utilisateur.

    • Travaux enregistrés personnels

      Cette option affiche les travaux enregistrés personnels sauvegardés sous vos nom d'utilisateur et code d'accès actuels pour que vous soyez seul à pouvoir les imprimer. Cliquez sur Code d'accès pour taper le même code d'accès utilisé lors de l'enregistrement des travaux privés sur le disque dur de l'imprimante.

      Remarque : Les travaux enregistrés personnels requièrent un mot de passe.

  4. Si vous avez sélectionné Travaux enregistrés personnels et partagés ou Travaux enregistrés partagés, entrez le code d'accès correspondant aux travaux. Si vos travaux ne s'affichent pas, cliquez sur Code d'accès et tapez à nouveau un code d'accès valide pour ces travaux.
  5. Pour ajouter un travail à la liste Travail final, sélectionnez un travail dans la liste et cliquez sur la flèche pointant vers le bas.
  6. Pour changer l'ordre des travaux dans la liste Travail final, sélectionnez un travail puis utilisez les flèches.
  7. Pour supprimer un travail de la liste, sélectionnez-le et cliquez sur Supprimer.
  8. Cliquez sur OK.
  9. Cliquez sur Impression.
