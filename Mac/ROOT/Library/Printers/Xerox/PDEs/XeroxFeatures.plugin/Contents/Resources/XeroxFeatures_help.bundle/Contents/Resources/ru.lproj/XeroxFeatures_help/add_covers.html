Добавление обложек

В окне «Добавить обложки» можно добавить обложки к каждой копии документа и указать параметры бумаги обложек. В разделе «Настройка работы» отображаются параметры бумаги основной части документа. Чтобы использовать настройку работы для обложек, выберите опцию «Использовать настройку работы».

Примечание: Не все перечисленные опции поддерживаются всеми принтерами. Некоторые опции предназначены только для определенных моделей и конфигураций принтера, операционных систем и драйверов.

Добавление обложек:

  1. В окне «Специальные страницы» выберите Добавить обложки.
  2. Чтобы подтвердить для работы все настройки по умолчанию, нажмите кнопку OK.
  3. В списке «Опции обложек» выберите один из вариантов.
    • Без обложек – отмена всех предыдущих настроек для обложек.
    • Только передняя - печать первой страницы на бумаге из заданного лотка. При автоматической двусторонней печати на обложке печатаются две первые страницы.
    • Только задняя - печать последней страницы на бумаге из заданного лотка. При автоматической двусторонней печати на обложке печатаются две последние страницы, когда число страниц четное.
    • Передняя и задняя: одинаковые - печать первой и последней обложек на бумаге из одного лотка.
    • Передняя и задняя: разные — передняя и задняя обложки печатаются на бумаге из разных лотков.
  4. Для указания размера, цвета и типа бумаги выберите значения нажатием стрелок в разделе «Бумага для передней обложки» или «Бумага для задней обложки».
  5. Выберите параметры печати обложек.
    • Пустые или с печатью — для обложек используются чистые листы или листы с печатью.
    • Печать на стороне 1 - печать на передней стороне обложки.
    • Печать на стороне 2 — печать только на второй стороне обложки.
    • Печать на обеих сторонах — печать на обеих сторонах обложки.
  6. Нажмите кнопку OK.
