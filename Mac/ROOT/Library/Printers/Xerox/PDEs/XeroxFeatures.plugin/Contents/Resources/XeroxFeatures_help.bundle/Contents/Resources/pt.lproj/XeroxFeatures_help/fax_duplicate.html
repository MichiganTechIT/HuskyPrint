Duplicar Destinatários de fax

O recurso Fax permite a criação de registros múltiplos para o mesmo destinatário e número de fax. Caso deseje incluir informações diferentes de caixa de correio, endereços de e-mail ou números telefônicos, é possível adicionar um registro duplicado.

Quando se adicionar um registro duplicado, é exibida uma janela de confirmação.

  1. Para adicionar um destinatário duplicado, clique em Sim.
  2. Para retornar à janela anterior, clique em Não.
