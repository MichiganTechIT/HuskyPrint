Senden eines Faxes an einen im Telefonbuch enthaltenen Empfänger

Faxnachrichten können Faxempfänger aus dem Telefonbuch hinzugefügt werden.

Faxnachrichten einen Empfänger aus dem Telefonbuch hinzufügen:

  1. Im Fenster "Material/Ausgabe" neben "Auftragsart" auf die Pfeile klicken und dann Fax wählen.
  2. Auf Empfänger klicken.
  3. Auf das Adressbuchsymbol klicken und dann einen der folgenden Schritte ausführen:
    • Eine Gruppe und dann einen Empfänger auswählen.
    • Zum Suchen eines Empfängers den Namen des Empfängers in das Suchfeld eingeben und dann auf OK klicken.
  4. Auf OK klicken.
