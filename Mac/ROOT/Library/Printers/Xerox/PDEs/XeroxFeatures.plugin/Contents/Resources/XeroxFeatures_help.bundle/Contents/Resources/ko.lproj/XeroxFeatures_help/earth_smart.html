스마트 환경 설정

스마트 환경 설정을 사용하면 자료 및 에너지 사용을 줄여서 인쇄로 인한 환경 오염을 줄일 수 있습니다. 스마트 환경 설정을 모든 인쇄 작업의 기본값으로 선택할 수 있습니다. 스마트 환경 설정에 액세스하려면 스마트 환경 설정 버튼을 클릭합니다.

녹색 스마트 환경 설정 버튼은 인쇄 드라이버의 맨 아래에 있습니다. 이 버튼을 사용하여 모든 인쇄 작업에 대한 스마트 환경 설정을 선택할 수 있습니다.

참고:

  • 일부 프린터에서는 여기 나열된 모든 옵션이 지원되지 않습니다. 일부 옵션은 특정 프린터 모델, 구성, 운영 체제 또는 드라이버 유형에만 적용됩니다.
  • 소프트웨어 응용 프로그램 내에서 전송된 인쇄 작업에 대한 스마트 환경 설정을 변경하면 현재 작업에 대한 설정만 변경됩니다. 그러나 프린터 드라이버 내에서 스마트 환경 설정을 변경하면 모든 응용 프로그램에 대한 기본 설정이 변경됩니다.

스마트 환경 설정 선택:

  1. 스마트 환경 설정 창에서 필요에 따라 옵션을 선택합니다.
    • 용지의 양면을 사용하여 모든 작업을 인쇄하려면 양면 인쇄 설정을 선택합니다.
      • 양면 인쇄는 작업을 용지의 양면에 인쇄하여 용지의 장변으로 제본할 수 있도록 합니다. 다음 그림은 각각 세로 방향 및 가로 방향 문서의 결과를 나타냅니다.

    • 용지 및 토너나 잉크를 덜 사용하려면 샘플 세트를 선택합니다.

      샘플 세트 작업 유형을 선택하면 여러 부 인쇄 작업 중 한 부를 먼저 인쇄한 후에 나머지를 인쇄합니다. 샘플 세트를 검토한 후에 나머지 매수를 인쇄하거나 프린터 제어판에서 삭제할 수 있습니다.

    • 프린터에서 여러 부 인쇄 작업 중 한 부를 인쇄하고 나머지 부수를 보류하려면 내 작업 유형으로 스마트 환경 교정 인쇄 설정을 선택합니다.
    • 배너 페이지가 모든 인쇄 작업에 인쇄되지 않도록 하려면 배너 페이지 없음 또는 배너 페이지 끄기를 선택합니다. 다음 옵션 중에서 선택하십시오.

      배너 페이지는 인쇄 작업과 함께 인쇄되는 별도의 페이지입니다. 배너 페이지에는 작업 전송자의 네트워크 사용자 이름과 문서 이름이 포함됩니다. 배너 페이지를 사용하면 이 페이지가 출력함에서 완료된 인쇄 작업을 각각 구분해 줍니다. 일부 프린터에서는 제어판 설정이 인쇄 드라이버에서 선택한 배너 페이지 옵션을 무시합니다.

      • 작업 ID 인쇄 안 함: 작업 ID 정보가 인쇄되지 않습니다.
      • 여백에 ID 인쇄 - 첫 페이지만: 문서의 첫 번째 페이지 상단 왼쪽 여백에 작업 ID를 인쇄합니다.
      • 여백에 ID 인쇄 - 모든 페이지: 문서의 모든 페이지 상단 왼쪽 여백에 작업 ID를 인쇄합니다.
    • 저품질 이미지를 생성하여 토너나 잉크를 절약하려면 간단하게 인쇄 모드 또는 내 인쇄 품질로 간단하게 인쇄 모드 설정을 선택합니다.
  2. 확인을 클릭합니다.
