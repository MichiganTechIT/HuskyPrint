소책자 만들기

소책자 기능은 용지의 각 면에 두 페이지씩 인쇄합니다. 인쇄 드라이버에서 페이지의 순서를 다시 매기고 방향을 정하여 접기 작업 시 번호 순으로 인쇄할 수 있도록 합니다. 그러면 소책자를 제작할 수 있는 상태로 페이지를 스테이플 처리할 수 있습니다. 소책자의 레이아웃은 용지 각 면에 페이지 이미지가 하나인 문서의 레이아웃과 다르다는 점을 기억해 두십시오. 그에 따라 줄바꿈 및 페이지 나누기가 달라집니다.

참고:

  • 일부 프린터에서는 여기 나열된 모든 옵션이 지원되지 않습니다. 일부 옵션은 특정 프린터 모델, 구성, 운영 체제 또는 드라이버 유형에만 적용됩니다.
  • 소책자 기능을 사용하면 전체 페이지 이미지가 줄어들어 요청한 용지 크기에 맞출 수 있습니다.

소책자 만들기:

  1. 소책자 만들기 패널에서 [소책자 마감 작업] 옆에 있는 화살표를 클릭한 후 다음 옵션 중 하나를 선택합니다.
    • 소책자 마감 작업 안 함 또는 접기 안 함: 이 기능은 인쇄 작업을 스테이플 처리하거나 접지 않습니다.
    • 소책자 접기 또는 접기: 중앙 선을 따라 인쇄 작업을 접습니다.
    • 소책자 접기 및 스테이플 또는 접지선 내기 및 스테이플: 이 기능은 중앙 선을 따라 인쇄 작업을 접고 접힌 부분을 따라 두 개의 스테이플을 적용하여 페이지를 제자리에 고정합니다.
    • 소책자 레이아웃 사용 안 함: 인쇄 작업에 소책자 형식 옵션을 적용하지 않습니다. 인쇄 작업이 이미 소책자 형식으로 지정된 경우 이 옵션을 사용하십시오.
  2. 새 용지 크기에 이미지를 맞추도록 프린터를 설정하려면 [새 용지 크기에 맞추기] 옆의 화살표를 클릭합니다. 필요한 용지 크기를 선택합니다. 사용자 지정 용지 크기를 지정하려면 사용자 정의를 클릭합니다.
  3. 여백 크기를 설정하려면 [소책자 여백] 옆에 있는 화살표를 클릭한 후 다음 옵션 중 하나를 선택합니다.
    • 표준을 선택하면 이미지 손실 없이 표준 용지 크기의 소책자로 문서를 인쇄할 수 있도록 페이지 이미지 크기가 축소됩니다. 프린터에서 용지의 가장자리까지 인쇄할 수 없는 경우 표준을 선택합니다.
    • 없음을 선택하면 페이지 이미지 크기를 축소하지 않은 상태로 문서를 소책자로 인쇄합니다. 프린터에서 용지 가장자리에 인쇄할 수 없는 경우 이 옵션을 선택하면 이미지가 잘릴 수 있습니다.

      참고: 이 예는 타블로이드 크기의 용지에 레터 크기의 페이지 2개가 놓인 모습을 보여 줍니다. 왼쪽의 이미지는 표준 여백을 사용합니다. 오른쪽의 이미지에 대해서는 없음 옵션이 선택되어 있습니다.

  4. 필요에 따라 수평 여백과 위치 조정을 지정합니다.
    • 수평 여백: 페이지 이미지 간 중앙 여백의 너비를 포인트 단위로 지정합니다. 값을 입력하거나 화살표를 사용하여 포인트 수를 선택합니다. 1 포인트는 1/72 인치 또는 0.35mm입니다.
    • 위치 조정: 페이지 이미지를 밖으로 이동할 크기를 포인트의 1/10 단위로 지정합니다. 페이지 이미지의 이동량을 지정하려면 값을 입력하거나 화살표를 사용하여 값을 선택합니다.
  5. 확인을 클릭합니다.
