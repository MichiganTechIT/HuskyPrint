Mode couleur

Dans la fenêtre Support/Réception, en regard de Mode couleur, cliquez sur les flèches et sélectionnez une option.

  • Valeur imprimante par défaut : utilise les paramètres par défaut de l'imprimante.
  • Nuances de gris : permet d'imprimer les couleurs sous la forme de variations de gris.
  • Couleur : imprime en quadrichromie.
