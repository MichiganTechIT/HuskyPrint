Solicitação de contabilidade

Se Contabilidade estiver ativada e a Solicitação de contabilidade estiver configurada para Sempre solicitar, a Solicitação de contabilidade é exibida quando se envia um trabalho de impressão.

  1. Para ID de usuário, se um valor padrão estiver especificado, é possível usá-lo ou digitar um novo valor.
  2. Para ID de contabilidade, se um valor padrão estiver especificado, é possível usá-lo ou digitar uma nova ID de contabilidade.
  3. Clique em OK.
