Информация о цвете

В разделе «Цвет» содержатся дополнительные параметры для настройки цвета в работе печати.

Примечание: Перечисленные опции некоторыми принтерами не поддерживаются. Отдельные опции предназначены только для определенных моделей и конфигураций принтера, операционных систем и драйверов.

Настройка цветов

  1. В окне «Параметры изображения» нажмите Цвет.
  2. Для настройки цветов снимите флажки Настройка по умолч. под ползунками и установите их в нужную позицию.
    • Яркость — возможность сделать светлее или темнее текст и изображения в работе печати.
    • Контраст — настройка разницы между светлыми и темными участками в работе печати.
    • Насыщение — настройка плотности цвета и количества серого в цвете.
    • Оттенок — предусмотрено три настройки для каждой пары цветов: голубой/красный, пурпурный/зеленый и желтый/синий. Настраивается количество выбранного цвета в изображении при сохранении яркости. Если все ползунки находятся в одной позиции, оттенок отсутствует. Настройка Оттенок применяется ко всем исходным цветам, кроме собственных и цветов SWOP CMYK.

      Оттенком считается серый цвет с окрашиванием другим цветом (например, коричневым, розовым или желтым).

  3. Нажмите кнопку OK.

Опции цвета

  1. В окне «Цвет» нажатием стрелок выберите Опции цвета.
  2. Выберите значения для настроек Надпечатывание черным и Треппинг. Для указания настроек направления подачи выберите для опции «Треппинг» значение Вкл..

    Треппинг обеспечивает перекрывание краев цветных участков (за исключением белых) с краями черных участков. Наложение краев позволяет компенсировать неравномерность печати, которая может возникать из-за смещения бумаги или других проблем.

    Применение функции «Надпечатывание черным» предотвращает удаление фона черного объекта при печати поверх цветного. Данная функция необходима при любом смещении цветов. При удалении фона по краям черного объекта появляется белая полоска. Для предотвращения появления белой полоски следует использовать надпечатывание черным.

  3. Для настройки высоты и ширины снимите флажки Настройка по умолч..
  4. Нажмите кнопку OK.

Цвет источника

  1. В окне «Цвет» нажатием стрелок выберите Цвет источника.
  2. Для режимов «Цветовое пространство RGB», «Цветовое пространство CMYK» и «Шкала серого» установите следующие настройки.
    1. В поле «Профиль изображений» выберите Настройка по умолч., один из вариантов или значение Настройка пользователя.
    2. В поле «Профиль текста и графики» выберите Настройка по умолч., один из вариантов или значение Настройка пользователя.
  3. Нажмите кнопку OK.

Цвет отпечатков

Настройки «Цвет копии» служат для коррекции цветопередачи печатаемого документа. Настройка цветопередачи указывает метод настройки цветов в процессе печати для получения требуемого внешнего вида отпечатков. Профиль выходного устройства используется для преобразования цветовых данных в соответствующее цветовое пространство выходного устройства.

Выбор цвета копий:

  1. В окне «Цвет» нажатием стрелок выберите Цвет копии.
  2. Выберите один из вариантов в поле «Профиль выходного устройства».
    • Настройка принтера по умолчанию — используются действующие настройки принтера.
    • Определяется системой — использование настроек, установленных для принтера системным администратором.
    • Настройка пользователя — при выборе данной опции открывается окно «Настройка пользователя» для ввода значения.
  3. Для настроек «Цветопередача для изображений», «Цветопередача для текста» и «Цветопередача для графики» выберите один из следующих вариантов:
    • Чтобы для всех объектов использовалась одинаковая цветопередача, выбранная на основе одного объекта, установите флажок Одинак. цветопередача для всех объектов. Для остальных двух объектов опция устанавливается автоматически.
    • Чтобы для каждого объекта использовалась отдельная цветопередача, снимите флажок Одинак. цветопередача для всех объектов и выберите вариант для каждого объекта.
  4. Нажмите OK.

Расширенная гамма

Функция «Расширенная гамма» позволяет использовать дополнительные красители, поддерживаемые устройством. Если данная функция поддерживается, то в дополнение к голубому, пурпурному, желтому и черному цвету можно выбирать другие цвета. Эта функция позволяет улучшать качество цветных отпечатков.

Примечание. Количество и доступность цветов зависит от аппарата.

Выбор дополнительных цветов:

  1. В окне «Цвет» нажатием стрелок выберите Цвет.
  2. В списке «Расширенная гамма» выберите один из вариантов.
  3. Нажмите OK.

Для восстановления первоначальных настроек системы нажмите кнопку По умолч..
