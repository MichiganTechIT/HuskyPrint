Impression standard

Dans le cadre d'une impression standard, le travail n'est pas enregistré sur l'imprimante pour être réimprimé ultérieurement et aucun code d'accès n'est requis. L'impression standard correspond au type de travail par défaut. Sélectionnez Impression standard pour imprimer un travail immédiatement.

Pour imprimer à l'aide de l'option Impression standard :

  1. Dans la fenêtre Support/Réception, en regard de Type de travail, cliquez sur les flèches et sélectionnez l'option Impression standard.
  2. Faites les autres sélections nécessaires pour votre travail d'impression.
  3. Cliquez sur Impression.
