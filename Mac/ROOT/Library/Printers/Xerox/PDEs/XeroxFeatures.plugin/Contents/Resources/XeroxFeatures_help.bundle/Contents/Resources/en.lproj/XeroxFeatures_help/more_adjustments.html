More Adjustments

To make further adjustments:

  1. In the Custom Color Correction window, for any selection, click More Adjustments.
  2. To use the color adjustments from the previous window, select Use Global Adjustments.
  3. To specify Lightness, Contrast, Saturation, and Color Cast, use the sliders to select options.
  4. To return to the original system settings, click Defaults.
  5. Click OK.
