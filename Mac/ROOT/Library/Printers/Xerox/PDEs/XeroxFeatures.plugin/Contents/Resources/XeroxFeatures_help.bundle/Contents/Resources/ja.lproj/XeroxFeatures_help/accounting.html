集計管理

アカウンティングは、自分のユーザ名とアカウントID情報を、印刷する各ドキュメントと関連付ける機能です。 アカウンティングデータがジョブとして収集、保存され、プリンタに送信されます。

注:

  • 表示されるオプションがどのプリンタでも使用できるわけではありません。 一部のオプションは、特定のモデル、構成、オペレーティングシステム、ドライバタイプのプリンタでしか使用できません。
  • [集計管理システム]および[集計管理要求]の選択肢によっては、すべてのオプションが表示されないことがあります。

集計管理を設定するには:

  1. [集計管理]ウィンドウの[集計管理システム]の隣にある矢印をクリックし、次のいずれかのオプションを選択します。
    • Xerox® 標準集計管理: コピー、印刷、スキャン、およびファクスジョブの数をユーザーごとに追跡します。 プリンタに事前定義されているユーザーIDとアカウントIDが使用されます。
    • Xerox® ネットワーク集計管理: ネットワーク認証サーバーからユーザーIDとアカウントIDを取得します。
    • ローカル集計管理: サブネットの集計管理コードを入力する必要があります。 この集計管理コードには、部署の請求コード(例: 0001)などを割り当てることができます。他のコードの入力は任意です。
    • 課金印刷: 印刷ジョブの課金ID（例: 部署の課金ID）を入力する必要があります。 その他のIDの入力は任意です。
    • ユーザー ID と集計管理 ID を送信する: 印刷ジョブを送信するためにユーザーにユーザーID、集計管理ID、およびパスワードを入力するよう要求します。
    • ユーザー ID のみ送信する: 印刷ジョブを送信するためにユーザーにユーザーIDとパスワードを入力するよう要求します。
    • 集計管理 ID のみ送信する: 印刷ジョブを送信するためにユーザーに集計管理IDとパスワードを入力するよう要求します。
    • 課金 ID のみ送信する: 印刷ジョブを送信するためにユーザーに課金IDとパスワードを入力するよう要求します。
    • すべて送信する: 印刷ジョブを送信するためにユーザーにユーザーID、集計管理ID、課金ID、およびパスワードを入力するよう要求します。
  2. [集計管理要求]の隣で、次のいずれかのオプションを選択します。
    • [常に要求]：印刷ジョブを送信するたびにユーザーにユーザーIDとアカウントIDを入力するよう要求します。
    • [要求しない]：すべての印刷ジョブで、[アカウンティングコード]に入力した情報が使用されます。
    • ファクスのみ要求: ファクスを送信するたびにユーザーにユーザーIDとアカウントIDを入力するよう要求します。
    • [カラーの場合のみ要求]：カラー印刷ジョブに対してのみユーザーIDとアカウントIDを入力するよう要求します。
    • [カラー印刷とFaxの場合のみメッセージを表示]：カラー印刷ジョブおよびファクスに対してのみユーザーIDとアカウントIDを入力するよう要求します。
  3. ビューにユーザIDを表示しないようにするには、[ユーザ ID をマスクする]を選択します。 アスタリスク(*)は、フィールドに入力する任意の文字の代用です。
  4. ビューにアカウントIDを表示しないようにするには、[アカウント ID をマスクする]を選択します。 アスタリスク(*)は、フィールドに入力する任意の文字の代用です。
  5. 必要に応じて[デフォルト集計管理コード]に値を入力します。 これらのフィールドの情報は、[要求しない]を選択した場合に必要になります。 アカウントIDとビリングIDには、部署の請求コード(例: 0001)などを割り当てることができます。
  6. [OK]をクリックします。
