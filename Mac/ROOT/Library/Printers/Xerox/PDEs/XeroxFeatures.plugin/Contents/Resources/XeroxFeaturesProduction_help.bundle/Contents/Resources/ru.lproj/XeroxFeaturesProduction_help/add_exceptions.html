Содержание

Добавление исключений

Функция «Добавить исключения» служит для указания страниц-исключений в работе печати. Настройки страниц-исключений отличаются от настроек остальных страниц в работе. Отличаться могут, например, заданные настройки формата, типа и цвета бумаги. Также можно указывать другую сторону листа бумаги для печати работы. В работе печати могут содержаться разные исключения.

Примечание: Не все перечисленные опции поддерживаются всеми принтерами. Некоторые опции предназначены только для определенных моделей и конфигураций принтера, операционных систем и драйверов.

Исключения в работе печати

  1. В окне «Специальные страницы» выберите Добавить исключения.
  2. В окне «Страницы» укажите страницу или диапазон страниц, к которым применяется исключение. Разделяйте отдельные страницы или диапазоны страниц запятыми. Для указания диапазона страниц используйте дефис. Например, чтобы задать страницы 1, 6, 9, 10 и 11, следует ввести 1, 6, 9-11.
  3. Чтобы подтвердить для работы все настройки по умолчанию, нажмите кнопку OK.
  4. Для указания размера, цвета, типа и плотности выберите значения нажатием стрелок в разделе «Бумага».
    • Если в поле «Тип бумаги» выбирается «Бумага с выступами», необходимо ввести количество выступов в поле «Листов в комплекте».
    • Если для типа бумаги выбирается опция «В комплекте», следует указать название типа бумаги и количество листов в последовательности.
  5. Для изменения сторон для печати выберите нужную опцию в разделе «2-сторонняя печать»:
    • 1-сторонняя печать — печать на страницах-исключениях выполняется только на одной стороне листа.
    • 2-сторонняя печать — печать на страницах-исключениях выполняется с обеих сторон листа, а листы переворачиваются по длинной кромке. Работу затем можно сшивать вдоль длинной кромки листов.
    • 2-ст. печать, перев. по кор. кр. — печать на страницах-исключениях выполняется с обеих сторон листа, а листы переворачиваются по короткой кромке. Работу затем можно сшивать вдоль короткой кромки листов.
  6. В поле «Цвет отпечатков» выберите один из вариантов.
  7. Нажмите кнопку OK.

Исключения для параметров изображения

В окне «Параметры изображения» можно изменять внешний вид печатаемых документов.

Примечание: Не все перечисленные опции поддерживаются всеми принтерами. Некоторые опции предназначены только для определенных моделей и конфигураций принтера, операционных систем и драйверов.

Цвет

Бесцветные сухие чернила/Прозрачный по объекту

Применение функции «Бесцветные сухие чернила» позволяет придавать распечатанным изображениям вид фотографий или выделять изображения, текст и графику.

Примечание: Не все перечисленные опции поддерживаются всеми принтерами. Некоторые опции предназначены только для определенных моделей и конфигураций принтера, операционных систем и драйверов.

Указание опции «Прозрачный»:

  1. В окне «Прозрачный» выберите значение в поле «Clear Options» (Опции прозрачного покрытия).
    • Настройка по умолч. — используются действующие настройки принтера.
    • Бесцветная заливка — покрытие всей страницы прозрачным тонером.
    • Только изображения — покрытие на странице только тех объектов, что идентифицированы как изображения.
    • Только текст — покрытие текста прозрачным тонером.
    • Только графика — покрытие графики прозрачным тонером.
    • Искл.изобр.(тол.текст и граф.) — покрытие всей страницы, текста и графики прозрачным тонером, кроме изображений.
    • Иск.текст(тол.изобр.и граф.) — покрытие всей страницы, изображений и графики прозрачным тонером, кроме текста.
    • Иск.граф.(тол.изобр.и текст) — покрытие всей страницы, изображений и текста прозрачным тонером, кроме графики.
    • Подавление прозрачного — запрет покрытия прозрачным тонером на странице.
    • Использовать встроенный прозрачный — применение инструкций для прозрачного тонера, указанных в разделе PDL.

      Для создания печатаемых страниц в принтере используется язык описания страниц (Page Description Language, PDL). К языкам PDL относятся PostScript (PS) и язык управления принтером (Printer Command Language, PCL).

  2. Для указания количества прозрачного тонера нажатием стрелок выберите значение рядом с полем «Кол-во прозр. покр.».
  3. Нажмите кнопку OK.

Сглаживание

Функция «Сглаживание» служит для сглаживания неровных краев текста и графики.

Выбор опции сглаживания:

В окне «Параметры изображения» выберите один из вариантов в поле «Сглаживание».

  • Настройка по умолч. — используются действующие настройки принтера.
  • Вкл. — включение функции сглаживания.
  • Не горит — отключение функции сглаживания.
  • K100% высокое разрешение — создание яркого, чистого черного цвета. Данную функцию можно использовать для мелкого текста или графики со сплошными участками черного цвета.

TRC пользователя

Кривые тоновоспроизведения (Tone Reproduction Curves, TRC) обеспечивают согласованность цвета печатаемых изображений. Функция «TRC пользователя» служит для точной настройки цветов путем коррекции кривых тоновоспроизведения.

Печать с использованием кривых TRC:

В окне «Параметры изображения» выберите один из вариантов в поле «TRC пользователя».

  • Настройка по умолч. — используются настройки принтера по умолчанию.
  • Выберите имя в списке последних имен TRC (до 10 имен).
  • Введите новое имя.

Разрешение

Разрешение определяет качество графических изображений. Разрешение изображения измеряется в точках на дюйм (dpi). Например, настройка 600 dpi указывает на то, что изображение содержит 600 точек (пикселей) на каждый дюйм. Чем выше разрешение, тем лучше качество печати.

Для указания разрешения выберите Разрешение, выберите значение или установите Настройка по умолч..

Полутоновой растр

Использование полутонов при печати позволяет получать изменения в цвете и яркости за счет варьирования затененности, плотности и размеров точек изображения. Детализация полутонового изображения зависит от количества точек на дюйма – чем больше точек, тем лучше детализация отпечатка. Полутона иногда называют экраном.

Для стохастического полутонового растра применяется случайное распределение точек, для которого указывается плотность точек в зависимости от требуемого уровня серого. Стохастический растр предназначен для предотвращения появления муара. Благодаря случайному характеру стохастический полутоновый растр оптимален для воспроизведения фотографий.

Выбор опций полутонового растра:

В окне «Параметры изображения» выберите один из вариантов в поле «Полутоновой растр».

  • Настройка по умолч. — используются настройки принтера по умолчанию.
  • Выберите значение. Чем больше точек, линий или линий на дюйм, тем более детальным будет изображение в работе печати.
  • Выберите Стохастический.

Экономия тонера

Функция «Экономия тонера» обеспечивает сокращение количества тонера, используемого в работах печати, и применяется для отпечатков чернового качества. Качество печати несколько ниже, чем в режиме «Стандартный».

Для включения данной функции в окне «Параметры изображения» нажатием стрелок рядом с полем «Экономия тонера» выберите значение Вкл..

Сдвиг изображения

Функция «Сдвиг изображения» служит для изменения положения изображения на странице. Изображение может сдвигаться по осям X и Y на каждом листе бумаги.

Настройка сдвига изображения:

  1. В поле «Опции сдвига изображения» выберите один из вариантов нажатием стрелок.
  2. Если в окне «Вывод бумаги» выбирается опция «1-сторонняя печать», следует выбрать один из вариантов в разделе «Опции сдвига изображения».
    • Настройка по умолч. — используются настройки принтера по умолчанию.
    • Независимо — установка настроек вручную.
  3. Если в окне «Вывод бумаги» выбирается опция «2-сторонняя печать», следует выбрать один из вариантов в разделе «Опции сдвига изображения».
    • Настройка по умолч. — используются настройки принтера по умолчанию.
    • Независимо — установка настроек вручную.
    • Зеркально по X — одновременный сдвиг изображений на одинаковое расстояние в обе стороны по оси X.
    • Зеркально по Y — одновременный сдвиг изображений на одинаковое расстояние в обе стороны по оси Y.
    • Зеркально по обеим осям — одновременный сдвиг изображений на одинаковое расстояние в обе стороны по осям X и Y.
    • Зеркально по X, вместе по Y — одновременный сдвиг изображений на одинаковое расстояние в обе стороны по оси X и в одном направлении по оси Y.
    • Зеркально по Y, вместе по X — одновременный сдвиг изображений на одинаковое расстояние в обе стороны по оси Y и в одном направлении по оси X.
    • Вместе по X — одновременный сдвиг изображений на одинаковое расстояние по обеим сторонам в одном направлении по оси X.
    • Вместе по Y — одновременный сдвиг изображений на одинаковое расстояние по обеим сторонам в одном направлении по оси Y.
    • Вместе по обеим осям — одновременный сдвиг изображений на одинаковое расстояние по обеим сторонам в одном направлении по осям X и Y.
  4. Для настройки «Независимо» выполните следующее:
    1. В поле «Сторона 1» введите величину сдвига изображения страницы по осям X и Y.
    2. В используется сторона 2, введите величину сдвига изображения страницы по осям X и Y.

      Примечание: Опция «Сторона 2» доступна только в режиме 2-сторонняя печать.

  5. Нажмите кнопку OK.

Выделение цветом

Функция «Выделение цветом» служит для выделения текста и важных фрагментов на отпечатках.

Выбор способов выделения цветом:

  1. В окне «Параметры изображения» выберите Выделение цветом.
  2. Выберите нужную опцию.
    • Настройка по умолч. — используются настройки принтера по умолчанию.
    • Алгоритм отображения — определяется способ печати всех цветов, кроме выбранного в поле «Выделение цветом отображения». Применение разных алгоритмов в зависимости от выбора дает разные результаты.
    • Выделение цветом отображения — выбор определенного цвета выделения.
    • Выделение цветом печати — печать документа в выбранном цвете.
    • Правила при несовпадении — указывается действие в случае несоответствия между выбранным цветом и цветовыми характеристиками принтера.
