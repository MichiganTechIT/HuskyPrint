집계 관리

집계 관리는 인쇄하는 각 문서와 사용자 이름 및 계정 ID 정보를 연결시킵니다. 집계 관리 데이터는 작업이 프린터에 전송될 때 수집되고 저장됩니다.

참고:

  • 일부 프린터에서는 여기 나열된 모든 옵션이 지원되지 않습니다. 일부 옵션은 특정 프린터 모델, 구성, 운영 체제 또는 드라이버 유형에만 적용됩니다.
  • 집계 관리 시스템 및 집계 관리 프롬프트 선택에 따라 일부 옵션이 나타나지 않습니다.

집계 관리 설정:

  1. 고급 창에서 [집계 관리 시스템] 옆에 있는 화살표를 클릭한 후 다음 옵션 중 하나를 선택합니다.
    • Xerox® 표준 집계 관리: 각 사용자의 복사, 인쇄, 스캔 및 팩스 작업 수를 추적합니다. 프린터에 사전 정의된 사용자 ID 및 계정 ID를 사용합니다.
    • Xerox® 네트워크 집계 관리: 네트워크 인증 서버에서 사용자 ID 및 계정 ID를 가져옵니다.
    • 로컬 집계 관리: 서브넷의 집계 관리 코드가 필요합니다. 이 집계 관리 코드는 0001과 같은 부서별 과금 코드일 수 있습니다. 기타 코드는 옵션입니다.
    • 인쇄 비용 청구: 부서의 과금 ID와 같은 인쇄 작업의 과금 ID가 필요합니다. 기타 ID는 옵션입니다.
    • 사용자 ID 및 계정 ID 전송: 인쇄 작업을 전송하려면 사용자 ID, 집계 관리 ID 및 비밀번호를 입력해야 합니다.
    • 사용자 ID만 전송: 인쇄 작업을 전송하려면 사용자 ID 및 비밀번호를 입력해야 합니다.
    • 계정 ID만 전송: 인쇄 작업을 전송하려면 집계 관리 ID 및 비밀번호를 입력해야 합니다.
    • 과금 ID만 전송: 인쇄 작업을 전송하려면 과금 ID 및 비밀번호를 입력해야 합니다.
    • 모두 전송: 인쇄 작업을 전송하려면 사용자 ID, 집계 관리 ID, 과금 ID 및 비밀번호를 입력해야 합니다.
  2. [집계 관리 프롬프트] 옆에 있는 다음 옵션 중 하나를 선택합니다.
    • 항상 알림을 설정하려면 인쇄 작업을 제출할 때마다 사용자 ID와 계정 ID를 입력해야 합니다.
    • 표시 안 함을 선택하면 모든 인쇄 작업에 대해 계정 코드 아래에 입력한 정보가 사용됩니다.
    • 팩스에 대해서만 확인: 팩스를 전송할 때마다 사용자 ID와 계정 ID를 입력해야 합니다.
    • 컬러에 대해서만 확인을 설정하려면 컬러 인쇄 작업의 사용자 ID와 계정 ID를 입력해야 합니다.
    • 컬러 및 팩스에 대해서만 확인을 설정하려면 컬러 인쇄 작업 및 팩스의 사용자 ID와 계정 ID를 입력해야 합니다.
  3. 보기에서 사용자 ID를 숨기려면 [사용자 ID 가림]을 선택합니다. 별표 *는 필드에 입력된 모든 문자를 대체합니다.
  4. 보기에서 계정 ID를 숨기려면 [계정 ID 가림]을 선택합니다. 별표 *는 필드에 입력된 모든 문자를 대체합니다.
  5. [기본 집계 관리 코드]에서 필요에 따라 값을 입력합니다. [표시 안 함]을 선택하면 이러한 필드는 필수입니다. 계정 ID와 과금 ID는 0001과 같이 부서별 과금 코드일 수 있습니다.
  6. 확인을 클릭합니다.
