Multiple Finishing

Multiple Finishing allows you to select several finishing options for the same job from a single window.

To print using Multiple Finishing:

  1. In the Paper/Output window, next to Finishing, click the arrows, then select Multiple Finishing.
  2. Select one or more options.
    • Stapling
    • Hole Punching
    • Folding

      Notes:

      • Depending on the option you select first, the Stapling and Hole Punch location options change.
      • The Z-fold Half Sheet option is not available for all configurations.
  3. Click OK.
