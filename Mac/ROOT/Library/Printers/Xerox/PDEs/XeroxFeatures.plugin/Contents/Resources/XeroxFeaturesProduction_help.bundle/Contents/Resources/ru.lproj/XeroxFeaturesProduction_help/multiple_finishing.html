Финишная обработка нескольких листов

Функция «Финишная обработка нескольких листов» служит для выбора в одном окне нескольких опций послепечатной обработки для работы.

Печать с использованием опции «Финишная обработка нескольких листов»:

  1. В окне «Бумага/Вывод» нажатием стрелок рядом с полем «Послепечатная обработка» выберите значение Финишная обработка нескольких листов.
  2. Выберите настройки:
    • Сшивание
    • Перфорация
    • Фальцовка

      Примечания:

      • В зависимости от того, какая опция выбирается первой, изменяются опции сшивания и перфорации.
      • Z-фальцовка в половину листа недоступна для всех конфигураций.
  3. Нажмите кнопку OK.
