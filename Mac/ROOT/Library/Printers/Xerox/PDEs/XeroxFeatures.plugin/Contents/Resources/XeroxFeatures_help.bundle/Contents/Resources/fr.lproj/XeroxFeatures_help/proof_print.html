Impression d'épreuves

Cette option vous permet d'imprimer un exemplaire d'un travail qui en comporte plusieurs pour vous permettre de vérifier le résultat avant d'imprimer le reste. Après avoir examiné l'épreuve, vous pouvez soit imprimer les exemplaires restants, soit les supprimer depuis le panneau de commande de l'imprimante.

Remarque : Toutes les imprimantes ne prennent pas en charge la totalité des options mentionnées. Certaines options s'appliquent uniquement à un modèle d'imprimante, un système d'exploitation ou un type de pilote spécifique.

Pour imprimer avec l'option Impression d'épreuves :

  1. Dans la fenêtre Support/Réception, en regard de Type de travail, cliquez sur les flèches et sélectionnez Impression d'épreuves.
  2. Indiquez le Nom du travail ou sélectionnez Utiliser le nom du document pour utiliser le nom de fichier du document envoyé avec le travail.
  3. Cliquez sur OK.
  4. Cliquez sur Impression. Une seule épreuve s'imprime.
  5. Si l'épreuve vous satisfait, imprimez les exemplaires restants à partir du panneau de commande de l'imprimante.
