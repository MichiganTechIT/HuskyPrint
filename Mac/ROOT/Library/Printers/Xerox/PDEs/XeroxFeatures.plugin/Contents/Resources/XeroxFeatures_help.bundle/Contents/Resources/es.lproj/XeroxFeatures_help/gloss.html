Nivel de brillo

Utilice Nivel de brillo para imprimir una imagen mate o brillante.

Nota: No todas las opciones descritas están disponibles en todas las impresoras. Algunas opciones solo se aplican a un modelo de impresora, configuración, sistema operativo o tipo de controlador específico.

Para seleccionar las opciones de Nivel de brillo:

En la ventana Avanzadas, junto a Nivel de brillo, haga clic en las flechas y, a continuación, seleccione una opción.

  • Normal imprime una versión mate de una imagen original brillante.
  • Mejorada duplica la imagen brillante original. La impresora imprime más lentamente al utilizar esta opción.
