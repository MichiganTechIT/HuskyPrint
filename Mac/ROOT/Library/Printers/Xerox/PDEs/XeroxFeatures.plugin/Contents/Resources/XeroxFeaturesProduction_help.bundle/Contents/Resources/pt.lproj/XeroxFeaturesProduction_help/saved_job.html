Trabalho salvo

Trabalhos salvos são documentos enviados para a impressora e armazenados nela para uso futuro.

Nota: Nem todas as opções listadas são suportadas em todas as impressoras. Algumas opções são aplicáveis apenas a um modelo de impressora, configuração, sistema operacional ou tipo de driver específico.

Para imprimir um trabalho salvo:

  1. Na janela Papel/Saída, ao lado de Tipo de trabalho, clique nas setas e selecione Trabalho salvo.
  2. Para selecionar um nome para o trabalho, adotar uma das seguintes opções:
    • Digite um nome para o trabalho.
    • Clique na seta à direita do campo e selecione um nome na lista.
    • Clique na seta à direita do campo e selecione Usar nome do documento para usar o nome do documento.
  3. Selecione uma opção. É possível salvar um trabalho ou imprimir e salvar um trabalho. Depois que o trabalho for salvo, você poderá imprimir conforme necessário.
  4. Ao lado de Formato, selecione uma opção de formato para o trabalho salvo.
    • Padrão da Impressora usa o valor padrão configurado para formatos de arquivo.
    • TIFF de Uma Página salva o trabalho impresso em formato .tif. Cada página do trabalho de impressão é salva como arquivo .tif separado.
    • TIFF com Várias Páginas salva o trabalho de impressão em formato .tif. Todas as páginas do trabalho de impressão são salvas em um arquivo .tif.
    • PDF salva o trabalho como um arquivo .pdf.
    • Reimpressão rápida salva o trabalho de impressão em Fast Reprint Format (FRF).
    • Reimpressão rápida com DRI salva o trabalho de impressão em Fast Reprint Format (FRF) para permitir reimpressão rápida. A configuração Display Resolution Image (DRI) permite que o trabalho seja visualizado ou editado no servidor de impressão.
  5. Clique em OK.
  6. Clique em Imprimir.
