Кромка переплета

Выбирается перфорация или сшивание работы печати принтером по длинной или короткой кромке листов. Настройка «Кромка переплета» применяется только для бумаги, для которой может применяться финишная обработка по всем кромкам.

Выбор опций кромки переплета:

В окне «Дополнительно» выберите значение нажатием стрелок рядом с полем «Кромка переплета».

  • Автоматический (рекомендуется) - переплет работы печати по длинной или короткой кромке.
  • Брошюровка по длине - переплет работы печати по длинной кромке листов.
  • Брошюровка по ширине - переплет работы печати по короткой кромке листов.
