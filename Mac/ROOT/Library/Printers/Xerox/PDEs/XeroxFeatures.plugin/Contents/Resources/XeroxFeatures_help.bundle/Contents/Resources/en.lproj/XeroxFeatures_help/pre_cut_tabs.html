Pre-Cut Tabs

  1. In Number in Sequence, type the number of tabs. Number in Sequence defines the number of unique sheets within an ordered set before the set repeats. For example, if a pre-cut tab set has five different tab placements to the set, type 5 in the Number in Sequence field.
  2. In Select a Tab Margin, select an option. If you select Custom, enter the amount of the Tab Margin Shift, then select an option for Units.
  3. Click OK.
