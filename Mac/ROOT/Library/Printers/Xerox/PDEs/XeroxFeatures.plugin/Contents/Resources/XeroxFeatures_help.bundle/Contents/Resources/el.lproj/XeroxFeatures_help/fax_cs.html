Συνοδευτικό φύλλο φαξ

Για να συμπεριλάβετε ένα συνοδευτικό φύλλο στο φαξ σας:

  1. Στο παράθυρο Χαρτί/Έξοδος, δίπλα στην επιλογή Είδος εργασίας, κάντε κλικ στα βέλη και μετά επιλέξτε Φαξ.
  2. Κάντε κλικ στην επιλογή Συνοδευτική σελίδα.
  3. Δίπλα στην επιλογή Συνοδευτική σελίδα, κάντε κλικ στα βέλη και μετά επιλέξτε Εκτύπωση συνοδευτικού φύλλου.
  4. Για να προσθέσετε ένα μήνυμα στους παραλήπτες, κάντε κλικ στο πλήκτρο δίπλα στην επιλογή Σημειώσεις συνοδευτικού φύλλου.
  5. Για να προσθέσετε τα στοιχεία επικοινωνίας, κάντε κλικ στο πλήκτρο δίπλα στην επιλογή Στοιχεία αποστολέα.
  6. Για να επεξεργαστείτε τα στοιχεία του παραλήπτη, κάντε κλικ στο πλήκτρο δίπλα στην επιλογή Στοιχεία παραλήπτη.
  7. Για τον καθορισμό ενός ειδώλου συνοδευτικού φύλλου, κάντε κλικ στα βέλη και μετά ορίστε μια επιλογή.
    • Για την απενεργοποίηση της επιλογής Είδωλα συνοδευτικού φύλλου, επιλέξτε Κανένα.
    • Για να ορίσετε μια επιλογή ειδώλου συνοδευτικής σελίδας που είναι ήδη αποθηκευμένη στον εκτυπωτή, επιλέξτε τη από τη λίστα.
    • Για να επεξεργαστείτε ένα ήδη αποθηκευμένο είδωλο συνοδευτικής σελίδας, επιλέξτε το από τη λίστα και μετά κάντε κλικ στην επιλογή Επεξεργασία.
    • Για να δημιουργήσετε νέο είδωλο συνοδευτικής σελίδας, κάντε κλικ στην επιλογή Νέο.
  8. Κάντε κλικ στην επιλογή Μέγεθος χαρτιού συνοδευτ. φύλλου και μετά ορίστε μια επιλογή.
  9. Κάντε κλικ στο πλήκτρο OK.
