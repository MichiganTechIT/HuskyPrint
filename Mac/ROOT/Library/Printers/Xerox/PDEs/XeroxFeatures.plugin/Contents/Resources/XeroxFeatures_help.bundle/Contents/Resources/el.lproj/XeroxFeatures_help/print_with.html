Εκτύπωση με

Με τη λειτουργία Εκτύπωση με επιτρέπεται η εκτύπωση αποθηκευμένων ιδιωτικών και δημόσιων εργασιών με την τρέχουσα εργασία εκτύπωσης. Για παράδειγμα, εκτυπώστε μια συνοδευτική επιστολή με ένα σύνολο αποθηκευμένων εγγράφων, όπως φυλλάδια ή τιμοκαταλόγους.

Σημείωση: Αυτό το είδος εργασίας είναι διαθέσιμο μόνο σε εκτυπωτές οι οποίοι είναι συνδεδεμένοι σε δίκτυο με ενεργοποιημένη την αμφίδρομη επικοινωνία.

Για εκτύπωση με χρήση της λειτουργίας Εκτύπωση με:

  1. Στο παράθυρο Χαρτί/Έξοδος, δίπλα στην επιλογή Είδος εργασίας, κάντε κλικ στα βέλη και μετά επιλέξτε Εκτύπωση με.
  2. Όταν σας ζητηθεί, πληκτρολογήστε τον κωδικό πρόσβασης που αντιστοιχίσατε στις αποθηκευμένες εργασίες στον εκτυπωτή και κάντε κλικ στο πλήκτρο OK.

    Εάν δεν αντιστοιχίσατε κωδικό πρόσβασης, κάντε κλικ στο πλήκτρο Ακύρωση.

  3. Ορίστε μια επιλογή στη λίστα Αποθηκευμένες εργασίες.
    • Προσωπικές και κοινόχρηστες αποθηκευμένες εργασίες
    • Κοινόχρηστες αποθηκευμένες εργασίες

      Με την επιλογή Κοινόχρηστες αποθηκευμένες εργασίες εμφανίζονται κοινόχρηστες εργασίες που έχουν αποθηκευτεί στο σκληρό δίσκο του εκτυπωτή. Οι κοινόχρηστες εργασίες είναι διαθέσιμες για επιλογή από οποιονδήποτε χρήστη.

    • Προσωπικές αποθηκευμένες εργασίες

      Με την επιλογή Προσωπικές αποθηκευμένες εργασίες εμφανίζονται εργασίες που έχουν αποθηκευτεί με το όνομα χρήστη σας και τον τρέχοντα κωδικό πρόσβασής σας για να μπορείτε μόνο εσείς να τις εκτυπώνετε. Κάντε κλικ στην επιλογή Κωδικός πρόσβασης για να πληκτρολογήσετε τον ίδιο κωδικό πρόσβασης που χρησιμοποιήσατε όταν αποθηκεύσατε τις αποθηκευμένες ιδιωτικές εργασίες στο σκληρό δίσκο του εκτυπωτή.

      Σημείωση: Οι προσωπικές αποθηκευμένες εργασίες απαιτούν έναν κωδικό πρόσβασης.

  4. Αν επιλέξετε Προσωπικές και Κοινόχρηστες αποθηκευμένες εργασίες ή Προσωπικές αποθηκευμένες εργασίες, πληκτρολογήστε τον κωδικό πρόσβασης για τις εργασίες. Αν οι εργασίες σας δεν εμφανιστούν, κάντε κλικ στο στοιχείο Κωδικός πρόσβασης και μετά πληκτρολογήστε ξανά έναν έγκυρο κωδικό πρόσβασης για τις εργασίες.
  5. Για να προσθέσετε μια εργασία στη λίστα Τελική εργασία, επιλέξτε μια εργασία από τη λίστα και, στη συνέχεια, κάντε κλικ στο κάτω βέλος.
  6. Για αλλαγή της σειράς των εργασιών στη λίστα Τελική εργασία, επιλέξτε μια εργασία και μετά χρησιμοποιήστε τα βέλη.
  7. Για τη διαγραφή μιας εργασίας από τη λίστα, επιλέξτε την εργασία και μετά κάντε κλικ στο πλήκτρο Διαγραφή.
  8. Κάντε κλικ στο πλήκτρο OK.
  9. Κάντε κλικ στην επιλογή Εκτύπωση.
