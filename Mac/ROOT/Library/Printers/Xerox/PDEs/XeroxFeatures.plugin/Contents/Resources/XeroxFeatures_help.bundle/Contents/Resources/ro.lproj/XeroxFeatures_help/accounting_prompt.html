Solicitare Contorizare

În cazul în care Contorizarea este activată şi funcţia Solicitare Contorizare este configurată să se declanşeze pentru lucrările de imprimare, fax, imprimare color, Solicitare Contorizare se afişează atunci când trimiteţi o asemenea lucrare.

Notă: În funcţie de selecţiile de Sistem de Contorizare şi Solicitare Contorizare, unele opţiuni nu apar.

  1. Dacă a fost setat un ID Utilizator implicit, îl puteţi utiliza pe acesta sau puteţi tasta unul nou.
  2. Dacă a fost setat un Cod Parolă implicit, îl puteţi utiliza pe acesta sau puteţi tasta unul nou.
  3. Dacă este cazul, selectaţi un tip de cont.
  4. Dacă a fost setat un ID Cont implicit, îl puteţi utiliza pe acesta sau puteţi tasta unul nou.
  5. Dacă a fost setat un Cod Parolă implicit pentru ID Cont, îl puteţi utiliza pe acesta sau puteţi tasta unul nou.
