Pages to Print

Pages to Print allows you to specify a page range for the print job. Use this option to ensure that Special Pages refer to the correct pages in the document.

To select a page range:

  1. In the Advanced window, next to Pages to Print, click Setup.
  2. In the From field, type the first page number of the print range.
  3. In the To field, type the last page number of the print range.
  4. Click OK.
