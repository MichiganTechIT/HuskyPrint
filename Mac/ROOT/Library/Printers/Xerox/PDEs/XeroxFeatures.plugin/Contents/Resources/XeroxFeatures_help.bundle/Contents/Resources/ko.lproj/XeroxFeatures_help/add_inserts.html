삽지 추가

문서의 첫 번째 페이지 앞이나 문서의 지정된 페이지 뒤에 빈 삽지나 사전 인쇄 삽지를 추가합니다. 예를 들어 문서에서 각 섹션 뒤에 빈 컬러 용지를 삽입하려면 각 섹션이 끝나는 페이지 번호를 기록합니다. 이러한 페이지 번호를 사용하여 지정된 각 페이지 뒤에 삽지를 넣을 페이지를 지정합니다.

참고: 일부 프린터에서는 여기 나열된 모든 옵션이 지원되지 않습니다. 일부 옵션은 특정 프린터 모델, 구성, 운영 체제 또는 드라이버 유형에만 적용됩니다.

삽지를 추가하려면 다음 작업을 수행하십시오.

  1. 특수 페이지 창에서 간지 추가를 클릭합니다.
  2. 다음 중에서 삽지 옵션 유형을 선택하십시오.
    • 페이지 1 앞을 선택하면 페이지 1 앞에 삽지가 삽입됩니다.
    • 다음 페이지 뒤: 지정한 페이지 또는 페이지 범위 뒤에 간지가 삽입됩니다.
  3. [다음 페이지 뒤]를 선택한 경우 [페이지 수] 필드에 각 간지 앞에 오는 페이지 수를 입력합니다.
  4. 수량의 경우 필요한 간지 수를 선택합니다.
  5. 모든 설정에 대해 작업 기본값을 그대로 사용하려면 확인을 클릭합니다.
  6. 크기, 컬러 또는 종류를 선택하려면 [용지] 옆에 있는 화살표를 클릭한 후 옵션을 선택합니다.
  7. 확인을 클릭합니다.
