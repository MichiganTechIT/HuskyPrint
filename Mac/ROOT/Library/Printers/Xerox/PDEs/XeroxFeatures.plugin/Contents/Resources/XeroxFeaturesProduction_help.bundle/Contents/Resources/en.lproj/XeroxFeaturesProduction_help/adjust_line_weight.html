Adjust Line Weight

The Adjust Line Weight feature allows you to select the lightness or darkness of line edges for print jobs.

To adjust line weight:

In the Advanced window, next to Adjust Line Weight, click the arrows, then select an option.

  • -2 Pixels: Uses the thinnest line weight for the print job.
  • -1 Pixel: Uses a thin line weight for the print job.
  • None: Uses the printer default line weight for the print job.
  • +1 Pixel: Uses a thick line weight for the print job.
  • +2 Pixels: Uses the thickest line weight for the print job.
