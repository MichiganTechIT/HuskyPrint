Работа с факсом

Тип работы «Факс» служит для отправки факсов и управления телефонными книгами и настройками факса.

Для доступа к меню факса выберите Факс рядом с полем «Тип работы».

  • Для добавления или изменения получателей факса нажмите кнопку Получатели.
  • Для добавления или изменения титульный листов выберите Титульный лист.
  • Для установки настроек факса выберите Параметры.
  • Дублирование получателя факса
