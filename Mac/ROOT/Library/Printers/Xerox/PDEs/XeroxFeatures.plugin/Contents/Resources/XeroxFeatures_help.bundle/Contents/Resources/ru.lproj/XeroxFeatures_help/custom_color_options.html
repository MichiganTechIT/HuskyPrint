Содержание

Пользовательские параметры цвета

В разделе «Пользовательские параметры цвета» содержатся дополнительные параметры для настройки цвета в работе печати.

Примечание: Не все перечисленные опции поддерживаются всеми принтерами. Некоторые опции предназначены только для определенных моделей и конфигураций принтера, операционных систем и драйверов.

Описание цвета

Функция «Описание цвета» позволяет настраивать цвета с помощью составления предложений из обычных фраз, описывающих изменение цветов.

Примечание: В разделе «Настройка цветов» настройка выполняется с помощью обычных ползунков для изменения цветов.

Порядок использования функции «Описание цвета»:

  1. В окне «Пользовательские параметры цвета» выберите один из вариантов в поле «Описание цвета».
  2. Цвет для изменения выбирается нажатием стрелок рядом с полем «Какой цвет изменить?».
  3. Величина изменения цвета выбирается нажатием стрелок рядом с полем «Изменить насколько?».
  4. Способ изменения цвета выбирается нажатием стрелок рядом с полем «Тип изменения?».
  5. Для изменения фразы для настройки цвета выберите эту фразу, затем выберите в меню новые фразы.
  6. Для создания другой фразы для настройки цвета выберите Составить новое предложение.
  7. Для изменения последовательности фраз для настройки цвета выберите их нажатием стрелок.
  8. Для удаления фразы для настройки цвета выберите ее и нажмите кнопку X.

    Примечание: Последовательность предложений определяет применение каждой настройки цвета.

  9. Для просмотра эскиза изображения с выбранными изменениями цвета выберите Просмотр. Для просмотра оригинального изображения снимите флажок Просмотр. Для изменения эскиза изображения сделайте выбор нажатием стрелок под текущим изображением.

Цветовая коррекция

Функция «Цветовая коррекция» позволяет применять к работам печати предварительно настроенные режимы коррекции цвета. С помощью цветовой коррекции осуществляется подбор цветов принтера к цветам других устройств, таких как монитор компьютера или печатная машина.

Использование цветовой коррекции:

  1. В окне «Цветовая коррекция» нажатием стрелок выберите вариант цветовой коррекции.
  2. Чтобы создать вариант цветовой коррекции, выберите опцию Пользовательская автонастройка цвета.
  3. Нажмите кнопку OK.

Настройка цветов

Функция «Настройка цветов» служит для настройки отдельных характеристик цвета. Характеристики включают яркость, контраст, насыщенность и оттенок.

Настройка цветов:

  1. Установите настройки плотности, контраста, насыщенности и оттенка с помощью ползунков.
  2. Для просмотра эскиза изображения с выбранными изменениями цвета выберите Просмотр.
  3. Отображение эскиза изображения можно изменять кнопками со стрелками.
  4. Нажмите кнопку OK.
