Sélection avancée du support

Cette option permet de sélectionner divers attributs de support dans une même fenêtre. Le récapitulatif support affiche le récapitulatif de vos sélections.

Remarque : Toutes les imprimantes ne prennent pas en charge la totalité des options mentionnées. Certaines options s'appliquent uniquement à un modèle d'imprimante, un système d'exploitation ou un type de pilote spécifique.

Pour sélectionner les attributs papier à l'aide de l'option Sélection avancée du support :

  1. Pour ajouter ou mettre à jour les attributs d'un Support nommé, sélectionnez Support nommé puis l'option désirée.

    Remarque : Le pilote d'imprimante conserve 10 noms. Si vous en ajoutez un 11e, le nom le moins utilisé est automatiquement supprimé de la liste. Pour supprimer de la liste des noms autres que celui choisi, sélectionnez Effacer la liste.

  2. Dans la fenêtre Support/Réception, en regard de Support, cliquez sur les flèches et sélectionnez Sélection avancée du support.
  3. Pour activer les attributs individuels, désélectionnez la case Utiliser le nom uniquement (avancé).
    • Sélectionnez un Format de support ou pour créer un format de support personnalisé, cliquez sur Personnalisé(e).
    • Sélectionnez une Couleur de support ou pour créer une couleur de support personnalisée, cliquez sur Personnalisé(e).
    • Sélectionnez un Type de support ou pour créer un type de support personnalisé, cliquez sur Personnalisé(e).
    • Sélectionnez un Grammage (g/m²) ou pour créer un grammage personnalisé, cliquez sur Personnalisé(e).
    • Sous Support perforé, sélectionnez une option :
    • Sous Faces couchées, sélectionnez une option.

      Le couchage du papier modifie l'aspect d'une impression. L'option Faces couchées permet d'appliquer un couchage au travail imprimé. Il est également possible d'appliquer un couchage sur une seule face ou sur les deux faces du papier.

    • Sous Type de couchage, sélectionnez une option.

      Précisez le type de couchage utilisé sur le papier couché. Le type de couchage peut modifier l'aspect du document imprimé.

    • Sous Sens de fabrication, sélectionnez une option.

      L'option Sens de fabrication permet de spécifier le sens de fabrication du support utilisé pour le travail d'impression. Dans certains cas, le sens de fabrication peut modifier l'aspect du document imprimé.

    • Pour Finition du support, sélectionnez une option.

      Cette option permet de sélectionner le type de support pour votre travail d'impression. Elle modifie l'aspect d'une impression.

    • Sous Description, sélectionnez Défaut imprimante, puis choisissez une description dans la liste ou tapez une nouvelle description.
    • Vous pouvez ajouter des informations supplémentaires sur vos sélections dans le champ Commentaire (facultatif).
  4. Cliquez sur OK.
