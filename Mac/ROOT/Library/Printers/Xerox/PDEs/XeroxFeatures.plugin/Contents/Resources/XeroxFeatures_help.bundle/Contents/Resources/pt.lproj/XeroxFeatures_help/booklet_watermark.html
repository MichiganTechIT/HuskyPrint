Livreto/Marca d'água

Use a janela Livreto/Marca d'água para selecionar as opções de layout de livreto e marca d'água.

Nota: Nem todas as opções listadas são suportadas em todas as impressoras. Algumas opções são aplicáveis apenas a um modelo de impressora, configuração, sistema operacional ou tipo de driver específico.

Para imprimir usando o Livreto/marca d'água:

  1. Na janela Livreto/Marca d'água, sob Layout de livreto, clique nas setas e selecione uma opção.
    • Ativada: Essa opção imprime o documento como um livreto. É possível utilizar essa opção para um documento que não está formatado como um livreto.
    • Desativada: Essa opção desativa o Layout de livreto. É possível utilizar essa opção para um documento que está formatado como um livreto.
  2. Para especificar recursos adicionais de livreto, ative Layout de livreto e clique em Opções de livreto.
  3. Para imprimir uma borda em volta de cada página, selecione Desenho bordas da página.
  4. Selecione opções para a Marca d'água.
  5. Clique em Imprimir.
