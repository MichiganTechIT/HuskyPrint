Opciones de color personalizados

Opciones de color personalizados proporciona más opciones para personalizar los colores del trabajo de impresión.

Nota: No todas las opciones descritas están disponibles en todas las impresoras. Algunas opciones solo se aplican a un modelo de impresora, configuración, sistema operativo o tipo de controlador específico.

Color en palabras

Color por palabras permite ajustar los colores creando frases comunes, que forman sentencias que describen cómo modificar los colores.

Nota: La función Ajustes de color permite la modificación de colores mediante los controles deslizantes de ajuste tradicionales.

Para utilizar Color en palabras:

  1. En la ventana Opciones de color personalizado, haga clic en Color por palabras.
  2. Para seleccionar el color que desea modificar, al lado de ¿Qué color cambiar?, haga clic en las flechas y, a continuación, seleccione un color.
  3. Para seleccionar la cantidad de cambio del color, al lado de ¿Cuánto cambio?, haga clic en las flechas y, a continuación, seleccione una opción.
  4. Para seleccionar cómo cambiar el color, al lado de ¿Qué tipo de cambio?, haga clic en las flechas y, a continuación, seleccione una opción.
  5. Para modificar una frase de ajuste de color existente, seleccione la frase y, a continuación, seleccione frases nuevas en los menús.
  6. Haga clic en Crear sentencia nueva para crear otra frase de ajuste de color.
  7. Para cambiar el orden de las frases de ajuste de color, selecciónelas y, a continuación, utilice las flechas.
  8. Para eliminar una frase de ajuste de color, selecciónela y, a continuación, haga clic en el botón X.

    Nota: El orden de la frase determina la aplicación de cada ajuste de color.

  9. Para ver una imagen de ejemplo con los cambios de color, seleccione Vista preliminar. Para ver la imagen original, quite la marca de la casilla de verificación Vista preliminar. Para cambiar la imagen de muestra, en la imagen actual, utilice las flechas y, a continuación, seleccione una opción.

Corrección del color

Esta opción proporciona modos de corrección de color prefijados que se pueden aplicar a los trabajos de impresión. Corrección del color hace que los colores de la impresora coincidan al máximo con los colores de otros dispositivos, por ejemplo, la pantalla del PC o una prensa.

Para utilizar Corrección del color:

  1. En la ventana Corrección del color, haga clic en las flechas y, a continuación, seleccione una opción de corrección del color.
  2. Para crear una opción de corrección del color personalizado, seleccione Color automático personalizado.
  3. Haga clic en Aceptar.

Ajustes de color

Utilice Ajustes de color para ajustar características específicas del color impreso. Algunas de las características son: claridad, saturación y variación de color.

Para realizar ajustes de color:

  1. Para especificar la claridad, contraste, saturación y variación de color, utilice los controles deslizantes para seleccionar las opciones.
  2. Para ver una imagen de ejemplo con los cambios de color, seleccione Vista preliminar.
  3. Para cambiar la imagen de muestra, haga clic en las flechas.
  4. Haga clic en Aceptar.
