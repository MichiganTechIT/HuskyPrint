표지 이미지 편집기

팩스 표지에 텍스트 또는 이미지를 추가하거나 프린터에 저장된 텍스트 또는 이미지를 사용할 수 있습니다.

  1. 표지 이미지 텍스트를 추가 또는 편집합니다.
    1. 표지 이미지 편집기 창에서 텍스트를 선택합니다.
    2. 텍스트의 서식을 지정하려면 글꼴 표시를 클릭한 후 목록에서 필요한 글꼴 옵션을 선택합니다.
    3. 텍스트 각도를 지정하려면 [각도] 아래에 값을 입력합니다. 값을 조정하려면 화살표를 사용합니다. 값을 선택하려면 슬라이더를 사용합니다.
    4. 페이지의 가운데에서 워터마크를 이동하려면 먼저 측정 단위를 선택한 후 가로 위치와 세로 위치의 숫자를 입력합니다.
    5. 이미지를 가운데에 배치하려면 중앙을 클릭합니다.
    6. 확인을 클릭합니다.
  2. 표지 이미지 그림을 추가 또는 편집합니다.
    1. 표지 이미지 편집기 창에서 그림을 선택합니다.
    2. [파일 선택] 옆에서 그림 파일의 경로를 입력하거나, 파일을 찾으려면 파일 선택을 클릭합니다.

      참고: 이 옵션은 .bmp, .jpg, .gif 파일을 지원합니다.

    3. 그림의 크기를 조절하려면 [배율] 아래에서 백분율을 입력 또는 선택하거나 슬라이더를 사용하여 배율을 조정합니다.
    4. 페이지의 가운데에서 워터마크를 이동하려면 먼저 측정 단위를 선택한 후 가로 위치와 세로 위치의 숫자를 입력합니다.
    5. 이미지를 가운데에 배치하려면 중앙을 클릭합니다.
    6. 확인을 클릭합니다.
