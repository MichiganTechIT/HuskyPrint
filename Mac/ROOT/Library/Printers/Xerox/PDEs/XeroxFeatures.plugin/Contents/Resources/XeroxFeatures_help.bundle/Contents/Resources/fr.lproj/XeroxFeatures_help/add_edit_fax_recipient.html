Ajouter ou modifier un destinataire de télécopie

Après avoir créé des enregistrements de destinataire de télécopie, vous pouvez ajouter des enregistrements à votre répertoire personnel.

Pour ajouter des enregistrements de destinataires :

  1. Dans la fenêtre Support/Réception, en regard de Type de travail, cliquez sur les flèches et sélectionnez l'option Fax.
  2. Cliquez sur Destinataires.
  3. Cliquez sur Ajouter un destinataire.
  4. Indiquez les informations de contact.
  5. Cliquez sur OK.

Pour modifier les détails d'un destinataire de télécopie sans modifier l'enregistrement dans le répertoire :

  1. Dans la fenêtre Support/Réception, en regard de Type de travail, cliquez sur les flèches et sélectionnez l'option Fax.
  2. Cliquez sur Destinataires.
  3. Sélectionnez le destinataire dans le tableau et cliquez sur Modifier.
  4. Modifiez les informations selon vos besoins.
  5. Cliquez sur OK.
