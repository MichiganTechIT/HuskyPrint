Бумага

Выбор лотка для работы печати зависит от настроек, выбранных в меню «Бумага». Настройки, выбранные для текущей работы печати, отображаются в разделе «Информация о бумаге». Раздел «Информация о бумаге» находится рядом с раскрывающимся меню «Бумага».

Примечание: Не все перечисленные опции поддерживаются всеми принтерами. Некоторые опции предназначены только для определенных моделей и конфигураций принтера, операционных систем и драйверов.

Тип бумаги

Настройка «Другой тип» служит для выбора типа бумаги для работы печати.

Примечания:

  • Не все перечисленные опции поддерживаются всеми принтерами. Некоторые опции предназначены только для определенных моделей и конфигураций принтера, операционных систем и драйверов.
  • Если выбирается бумага, не загруженная в принтер, работа задерживается в принтере до тех пор, пока не будет загружена требуемая бумага.

Выбор типа бумаги:

  1. В окне «Бумага/Вывод» нажатием стрелок рядом с полем «Бумага» выберите значение Другой тип.
  2. Выполните одно из следующих действий.
    • Для использования в принтере типа бумаги по умолчанию выберите значение Настройка по умолч.
    • Выберите в списке тип бумаги. Когда администратор создает для принтера настраиваемые типы бумаги, они отображаются в списке.
    • Для выбора лотка с бумагой любого цвета выберите опцию Автовыбор.
    • Выберите настройки в разделе Бумага с выступами/Картотечная с выступами.

Действия при несоответствии бумаги

  1. В окне «Бумага/Вывод» нажатием стрелок рядом с полем «Бумага» выберите значение Действия при несоответствии бумаги.
  2. Выберите нужную опцию.
    • Запрос на загрузку бумаги нужного форм. — обеспечение соответствия форматов бумаги перед печатью.
    • Авто масштаб и печать на загруж. бумаге — автоматическое масштабирование документа по формату бумаги в лотке.
    • Без масштаба, но печать на загруж.бумаге — печать на бумаге несоответствующего формата.
    • Выдать запрос бумаги подходящего типа — обеспечение соответствия типов бумаги перед печатью.
    • Игнорировать несоответствие и начать — печать на бумаге несоответствующего типа.

Разделители прозрачных пленок

Функция «Разделители прозрачных пленок» служит для вставки чистых листов бумаги между листами прозрачной пленки с печатью. Размер и ориентация таких разделителей должны совпадать с размером и ориентацией прозрачной пленки.

Примечания:

  • Не все перечисленные опции поддерживаются всеми принтерами. Некоторые опции предназначены только для определенных моделей и конфигураций принтера, операционных систем и драйверов.
  • Разделители прозрачных пленок нельзя указывать при использовании некоторых функций, таких как «Фальцовка» и «Брошюра».

Печать с использованием разделителей прозрачных пленок:

  1. В окне «Бумага/Вывод» нажатием стрелок рядом с полем «Бумага» выберите значение Разделители прозрачных пленок.
  2. Выберите один из вариантов в поле «Разделители прозрачных пленок».
    • Выкл. или Без разделителей — разделители не вставляются
    • Разделители без печати — между листами пленки вставляется чистый лист бумаги.
    • Разделители с печатью — между листами пленки вставляется лист бумаги с печатью. На листе печатается то же изображение, что и на пленке.
  3. Выберите один из вариантов в поле «Цвет».
  4. Нажмите кнопку OK.
