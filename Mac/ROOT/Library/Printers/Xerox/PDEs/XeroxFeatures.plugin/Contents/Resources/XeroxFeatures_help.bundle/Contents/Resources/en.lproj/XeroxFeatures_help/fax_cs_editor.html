Cover Sheet Image Editor

You can add text or images to your fax cover sheets or you can use-text or images saved on the printer.

  1. To add or edit Cover Sheet Image Text.
    1. In the Cover Sheet Image Editor window, select Text.
    2. To format the text, click Show Fonts, then from the list, select the desired font options.
    3. To specify the angle of the text, under Angle, type a value. To adjust the value, use the arrows. To select a value, use the slider.
    4. To shift the watermark from the center of the page, first select the measurement unit, then enter numbers for the horizontal and vertical positions.
    5. To center the image, click Center.
    6. Click OK.
  2. To add or edit a Cover Sheet Image Picture.
    1. In the Cover Sheet Image Editor window, select Picture.
    2. Next to Choose File, type the path to the picture file or to locate the file, click Choose File.

      Note: This option supports .bmp, .jpg, and .gif files.

    3. To resize the picture, under Scale, type or select a percentage or use the slider to adjust the scale.
    4. To shift the watermark from the center of the page, first select the measurement unit, then enter numbers for the horizontal and vertical positions.
    5. To center the image, click Center.
    6. Click OK.
