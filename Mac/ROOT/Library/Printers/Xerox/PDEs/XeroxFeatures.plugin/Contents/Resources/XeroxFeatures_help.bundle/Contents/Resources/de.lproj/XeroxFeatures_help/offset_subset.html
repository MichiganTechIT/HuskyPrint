Versatz/Teilsatz

"Versatzausgabe" steuert, wie die einzelnen Exemplare von aus mehreren Exemplaren bestehenden Aufträgen versetzt werden. Ist die Option aktiviert, werden die einzelnen Exemplare ganzer Aufträge im Ausgabefach zueinander versetzt ausgegeben. Diese Option ist verfügbar, wenn das Ausgabefach über eine Versatzfunktion verfügt.

"Versatz/Teilsatz" verwenden:

  1. Im Fenster "Material/Ausgabe" neben "Endverarbeitung" auf die Pfeile klicken und dann Versatz/Teilsatz wählen.
  2. Die gewünschte Option auswählen.
    • Aus: Versatzfunktion ist deaktiviert. Teilsätze können dennoch gewählt werden.
    • Satzweise: Die einzelnen Exemplare eines aus mehreren Exemplaren bestehenden Druckauftrags werden versetzt ausgegeben.
    • Teilsatzweise: Teilsätze eines Satzes mit einer bestimmten Anzahl Seiten werden versetzt ausgegeben.
    • Satzweise und teilsatzweise: Aus einer bestimmten Anzahl Seiten bestehende Teilsätze eines Auftrags und ganze Sätze von Druckaufträgen mit mehreren Exemplaren werden versetzt ausgegeben.
  3. Zum Drucken von Teilsätzen innerhalb eines einzelnen Druckauftrags Teilsätze aktivieren wählen.
  4. In Seiten pro Teilsatz die Anzahl der Seiten pro Teilsatz eingeben. Jeder Teilsatz muss die gleiche Anzahl Seiten enthalten.
  5. Auf OK klicken.
