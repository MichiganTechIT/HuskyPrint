Создание брошюры

Функция брошюры позволяет печатать по две страницы на каждой стороне листа бумаги. Порядок и ориентация страниц устанавливаются таким образом, чтобы при складывании листов страницы расположились в порядке нумерации. Страницы можно сшивать в брошюру. Следует помнить, что макет брошюры отличается от макета страницы документа с одним изображением страницы на каждой стороне листа. Поэтому соответственно отличается разделение на строки и страницы.

Примечания:

  • Не все перечисленные опции поддерживаются всеми принтерами. Некоторые опции предназначены только для определенных моделей и конфигураций принтера, операционных систем и драйверов.
  • При использовании функции «Брошюра» общий размер изображения страницы также может уменьшаться для размещения страницы на бумаге требуемого формата.

Порядок создания брошюры:

  1. В окне «Создание брошюры» нажатием стрелок выберите выберите значение рядом с полем «Финишная обработка брошюры».
    • Без финишной обработки брошюры или Без фальцовки — сшивание или фальцовка работы печати не выполняется.
    • Фальцовка брошюры или Складывание брошюры— выполняется фальцовка работы печати по центральной линии.
    • Фальцовка и сшивание брошюры — по центральной линии выполняется фальцовка работы печати, а по линии сгиба устанавливаются две скрепки, чтобы сшить страницы между собой.
    • Отключить макет брошюры — опции макета брошюры к работе печати не применяются. Данная настройка используется в тех случаях, когда работа печати уже выполнена в формате брошюры.
  2. Для масштабирования изображения по новому размеру бумаги нажатием стрелок выберите значение рядом с полем «По новому размеру бумаги». Выберите размер бумаги. Для указания настраиваемого формата бумаги выберите опцию Настройка пользователя.
  3. Для установки размеров полей нажатием стрелок выберите значение рядом с полем «Поля брошюры».
    • Стандартный: уменьшение размера изображения страницы для печати документа в виде брошюры на бумаге стандартного формата без потери изображения. Если принтер не поддерживает печать «в край» бумаги, выберите Стандартный.
    • Нет – печать документа в виде брошюры без уменьшения размера изображений страницы. Если принтер не печатает по всей ширине страницы, возможная обрезка изображения.

      Примечание: В приведенном примере показано размещение двух страниц формата Letter на листе формата Tabloid. На изображении слева заданы стандартные поля. На изображении справа поля отключены.

  4. Укажите сдвиг изображения и поле переплета.
    • Поле переплета — указывается центральное поле между изображениями страниц (в пунктах). Введите значение в пунктах или выберите его с помощью стрелок. Пункт равен 0,35 мм или 1/72 дюйма.
    • Сдвиг изображения — указывается сдвиг изображения наружу относительно переплета (в десятых долях пункта). Для указания сдвига изображений страниц введите значение или выберите его с помощью стрелок.
  5. Нажмите кнопку OK.
