Desplazamiento/Subjuegos

Desplazar salida controla cómo se desplazan las copias en los trabajos de impresión de varias copias. Cuando está activada, los juegos de copias se desplazan ligeramente entre sí en la bandeja de salida. Esta opción está disponible si la bandeja de salida tiene capacidad de desplazamiento.

Para utilizar Desplazamiento/Subjuegos:

  1. En la ventana Papel/Salida, en Acabado, haga clic en las flechas y, a continuación, seleccione Desplazamiento/Subjuegos.
  2. Seleccione una opción.
    • No desplazar desactiva la función de desplazamiento. No obstante, se pueden seleccionar subjuegos.
    • Cada juego desplaza cada uno de los juegos de un trabajo de impresión de varias copias.
    • Cada subjuego desplaza un número especificado de páginas de un juego.
    • Cada juego y cada subjuego desplaza un número especificado de páginas de un juego, además de desplazar cada uno de los juegos de un trabajo de impresión de varias copias.
  3. Para imprimir subjuegos de un solo trabajo de impresión, seleccione Activar subjuegos.
  4. En Páginas por subjuego, escriba el número de páginas que se incluirán en cada subjuego. Cada subjuego debe contener el mismo número de páginas.
  5. Haga clic en Aceptar.
