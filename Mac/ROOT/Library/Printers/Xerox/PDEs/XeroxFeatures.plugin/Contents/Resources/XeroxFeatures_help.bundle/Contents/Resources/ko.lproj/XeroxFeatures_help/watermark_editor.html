워터마크 편집기

워터마크는 인쇄 작업 페이지의 텍스트 또는 이미지 뒤에 또는 그 위에 인쇄되는 그림 또는 텍스트입니다.

참고: 일부 운영 체제에서는 워터마크 인쇄를 지원하지 않습니다.

텍스트 워터마크 생성

텍스트 워터마크는 텍스트 필드에 입력한 문자를 워터마크로 사용합니다. 문서에 인쇄할 텍스트 워터마크를 생성할 수 있습니다. 글꼴 크기, 글꼴 형식, 글꼴 컬러 및 각도와 같은 옵션을 선택합니다.

텍스트 워터마크 생성:

  1. 소책자/워터마크 창에서 [워터마크] 아래에 있는 화살표를 클릭한 후 새로 만들기를 선택합니다.
  2. 워터마크 편집기 창에서 텍스트를 클릭합니다.
  3. 텍스트에 대해 워터마크 이름을 사용하거나 새 텍스트를 입력합니다.
  4. 워터마크의 서식을 지정하려면 글꼴 표시를 클릭한 후 옵션을 선택합니다.
  5. 페이지의 워터마크 각도를 지정하려면 백분율을 입력 또는 선택하거나 슬라이더를 사용하여 각도를 조정합니다.
  6. 페이지에서 워터마크의 위치를 지정하려면 측정 단위를 선택한 후 가로 위치와 세로 위치의 숫자를 입력합니다.
  7. 워터마크를 가운데에 배치하려면 중앙을 클릭합니다.
  8. 확인을 클릭합니다.

그림 워터마크 생성

그림 워터마크는 이미지를 워터마크로 사용합니다. 그림 파일은 .bmp, .gif 또는 .jpg 형식이어야 합니다. 문서에서 인쇄할 그림 워터마크를 생성하고 크기 및 페이지상의 위치를 포함한 옵션을 선택할 수 있습니다.

그림 워터마크 생성:

  1. 소책자/워터마크 창에서 [워터마크] 아래에 있는 화살표를 클릭한 후 새로 만들기를 선택합니다.
  2. 워터마크 편집기 창에서 그림을 클릭합니다.
  3. 그림 파일의 경로와 이름을 입력하거나, 그림 선택을 클릭한 후 그림 파일을 찾아서 선택합니다. 이 옵션은 .bmp, .jpg, .gif 파일을 지원합니다.
  4. 이미지 배율을 지정하려면 이미지를 확대 또는 축소하는 비율을 지정합니다.
  5. 페이지의 가운데에서 워터마크를 이동하려면 먼저 측정 단위를 선택한 후 가로 위치와 세로 위치의 숫자를 입력합니다.
  6. 워터마크를 가운데에 배치하려면 중앙을 클릭합니다.
  7. 확인을 클릭합니다.

워터마크 편집

워터마크 편집기를 사용하여 워터마크를 변경할 수 있습니다.

워터마크 편집:

  1. 소책자/워터마크 창에서 [워터마크] 아래에 있는 화살표를 클릭한 후 편집을 선택합니다.
  2. 워터마크 편집기 창에서 선택한 옵션을 필요에 따라 변경합니다.
  3. 확인을 클릭합니다.

워터마크 삭제

  1. 소책자/워터마크 창에서 [워터마크] 아래에서 삭제할 워터마크를 선택한 후 삭제를 클릭합니다.
  2. 예를 클릭하여 삭제를 확인합니다.
