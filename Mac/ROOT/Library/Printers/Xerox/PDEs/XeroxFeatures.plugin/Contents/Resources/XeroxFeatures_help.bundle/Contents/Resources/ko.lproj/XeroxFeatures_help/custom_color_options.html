사용자 정의 컬러 옵션

사용자 정의 컬러 옵션은 인쇄 작업의 색상을 사용자 정의할 수 있는 여러 옵션을 제공합니다.

참고: 일부 프린터에서는 여기 나열된 모든 옵션이 지원되지 않습니다. 일부 옵션은 특정 프린터 모델, 구성, 운영 체제 또는 드라이버 유형에만 적용됩니다.

단어별 컬러 지정

단어별 컬러 지정에서 색상 변경 방법을 기술하는 일반적인 단계에 따라 문장을 구성하여 색상을 조정할 수 있습니다.

참고: 컬러 조정 기능을 통해 기존의 조정 슬라이더로 색상을 변경할 수 있습니다.

단어별 컬러 지정 사용:

  1. 사용자 정의 컬러 옵션 창에서 단어별 컬러 지정을 클릭합니다.
  2. 변경할 색상을 선택하려면 변경할 컬러 옆에 있는 화살표를 클릭한 후 색상을 선택합니다.
  3. 색상의 변경 양을 선택하려면 변경량 옆에 있는 화살표를 클릭한 후 옵션을 하나 선택합니다.
  4. 색상의 변경 방법을 선택하려면 변경 유형 옆에 있는 화살표를 클릭한 후 옵션을 하나 선택합니다.
  5. 기존 컬러 조정 단계를 수정하려면 단계를 선택한 후 메뉴에서 새 단계를 선택합니다.
  6. 다른 색상 조정 단계를 구성하려면 새로운 문장 구성을 클릭합니다.
  7. 컬러 조정 단계의 순서를 변경하려면 해당 단계를 선택한 후 화살표를 사용합니다.
  8. 컬러 조정 단계를 삭제하려면 단계를 선택한 후 X 버튼을 클릭합니다.

    참고: 문장 순서가 각 색상 조정의 적용을 결정합니다.

  9. 현재 색상 변경이 적용된 샘플 이미지를 보려면 미리 보기를 선택합니다. 원래 이미지를 보려면 미리 보기 확인란의 선택을 취소합니다. 샘플 이미지를 변경하려면 현재 이미지에서 화살표를 사용하여 옵션을 하나 선택합니다.

색 보정

컬러 보정은 인쇄 작업에 적용할 수 있는 컬러 보정 모드를 사전 설정합니다. 컬러 보정은 인쇄된 컬러를 다른 장치(예: 컴퓨터 모니터 디스플레이 화면 또는 인쇄기)의 컬러와 일치시킵니다.

컬러 보정을 사용하려면 다음 작업을 수행하십시오.

  1. 컬러 보정 창에서 화살표를 클릭한 후 컬러 보정 옵션을 선택합니다.
  2. 사용자 정의 컬러 보정 옵션을 만들려면 사용자 정의 자동 컬러를 선택합니다.
  3. 확인을 클릭합니다.

컬러 조정

인쇄된 컬러의 특정 특성을 조정하려면 컬러 조정을 사용합니다. 특성에는 밝기, 콘트라스트, 채도 및 색조가 포함됩니다.

컬러 조정:

  1. 밝기, 콘트라스트, 채도 및 색조를 지정하려면 슬라이더를 사용하여 옵션을 선택합니다.
  2. 현재 색상 변경이 적용된 샘플 이미지를 보려면 미리 보기를 선택합니다.
  3. 샘플 이미지를 변경하려면 화살표를 클릭합니다.
  4. 확인을 클릭합니다.
