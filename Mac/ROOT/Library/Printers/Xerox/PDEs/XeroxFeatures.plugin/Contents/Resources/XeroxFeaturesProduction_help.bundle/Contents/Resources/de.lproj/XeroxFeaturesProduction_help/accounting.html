Kostenzählung

Bei der Kostenzählung werden ein Benutzername und Kostenstellenangaben mit jedem gedruckten Dokument verbunden. Die Kostenzählungsdaten werden gespeichert, wenn die Aufträge an den Drucker übermittelt werden.

  1. Zum Aufrufen der Kostenzählungseinstellungen Im Fenster "Erweitert" neben "Konfiguration" auf Kostenzählung klicken.
  2. Für "Kostenzählungssystem" eine Option wählen.
    • Zum Einrichten der Kostenzählung auf Ein klicken.
    • Zum Deaktivieren der Kostenzählung auf Aus klicken.
  3. Neben "Kostenzählungsaufforderung" eine Option wählen.
    • Nie: Beim Übermitteln eines Druckauftrags erfolgt keine Aufforderung zur Eingabe einer Benutzerkennung oder einer Kostenstellennummer.
    • Immer: Benutzerkennung und Kostenstellenummer müssen jedes Mal beim Übermitteln eines Druckauftrags eingegeben werden.
  4. Wenn "Immer" gewählt wird:
    1. Zum Verbergen einer Benutzerkennung oder Kostenstellennummer beim Drucken des Auftrags neben diesen Optionen das Kontrollkästchen aktivieren.
    2. Falls erforderlich, eine Standardbenutzerkennung eingeben.
    3. Falls erforderlich, eine Standardkostenstellennummer oder eine Standardkontonummer eingeben.
  5. Auf OK klicken.
