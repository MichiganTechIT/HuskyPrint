Προσθήκη διαφορετικών σελίδων

Χρησιμοποιήστε τη λειτουργία Προσθήκη διαφορετικών σελίδων για τον καθορισμό των διαφορετικών σελίδων στην εργασία εκτύπωσης. Οι ρυθμίσεις για τις διαφορετικές σελίδες διαφέρουν από τις ρυθμίσεις για τις υπόλοιπες σελίδες της εργασίας. Μπορείτε να ορίσετε διαφορές όπως μέγεθος, είδος και χρώμα χαρτιού. Επίσης, μπορείτε να αλλάξετε την όψη χαρτιού στην οποία θα γίνει η εκτύπωση, ανάλογα με τις ανάγκες της εργασίας σας. Μια εργασία εκτύπωσης μπορεί να περιλαμβάνει πολλές διαφορετικές σελίδες.

Σημείωση: Δεν υποστηρίζονται όλες οι επιλογές που παρατίθενται από όλους τους εκτυπωτές. Ορισμένες επιλογές ισχύουν μόνο σε συγκεκριμένα μοντέλα ή διαμορφώσεις εκτυπωτών, λειτουργικά συστήματα ή τύπους οδηγών.

Για την προσθήκη μιας διαφορετικής σελίδας:

  1. Στο παράθυρο Ειδικές σελίδες, κάντε κλικ στο στοιχείο Προσθήκη διαφορετικών σελίδων.
  2. Για το πεδίο Σελίδα(ες), καθορίστε τη σελίδα ή τις σελίδες που είναι διαφορετικές. Διαχωρίστε τις μεμονωμένες σελίδες ή το εύρος σελίδων με κόμματα. Καθορίστε τις σελίδες ενός εύρους σελίδων με ένα ενωτικό (παύλα). Για παράδειγμα, για να καθορίσετε τις σελίδες 1, 6, 9, 10 και 11, πληκτρολογήστε: 1, 6, 9-11.
  3. Για να επιλέξετε το μέγεθος, το χρώμα ή το είδος, δίπλα στο στοιχείο Χαρτί, κάντε κλικ στα βέλη και μετά ορίστε επιλογές.
    • Αν επιλέξετε Διαχωρ. καρτέλες κομμένες από πριν για το Είδος χαρτιού, τότε στο πεδίο Αριθμός στη σειρά, εισαγάγετε τον αριθμό καρτελών.
    • Αν επιλέξετε Ακουλουθιακό για το Είδος χαρτιού, καθορίστε την ονομασία του είδους χαρτιού και τον αριθμό στην ακολουθία.
  4. Για την αλλαγή των όψεων χαρτιού που θα εκτυπωθούν, ορίστε μια επιλογή στη λειτουργία Εκτύπωση διπλής όψης:
    • Εκτύπωση μονής όψης για την εκτύπωση των διαφορετικών σελίδων στη μία μόνο όψη.
    • Εκτύπωση διπλής όψης για την εκτύπωση των διαφορετικών σελίδων και στις δύο όψεις του φύλλου χαρτιού και την αναστροφή των σελίδων στη μεγάλη πλευρά. Στη συνέχεια, η εργασία μπορεί να δεθεί στη μεγάλη πλευρά των σελίδων.
    • Εκτύπωση διπλής όψης, αναστροφή στη μικρή πλευρά για την εκτύπωση των διαφορετικών σελίδων και στις δύο όψεις του φύλλου χαρτιού και την αναστροφή των σελίδων στη μικρή πλευρά. Στη συνέχεια, η εργασία μπορεί να δεθεί στη μικρή πλευρά των σελίδων.
  5. Για μετατόπιση του ειδώλου στο χαρτί, επιλέξτε Μετατόπιση ειδώλου.
  6. Κάντε κλικ στο πλήκτρο OK.
