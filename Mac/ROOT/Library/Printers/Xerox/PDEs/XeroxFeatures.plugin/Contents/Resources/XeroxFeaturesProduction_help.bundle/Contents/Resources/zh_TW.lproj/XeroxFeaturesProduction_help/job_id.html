工作 ID

工作 ID 會將您工作的詳細資料列印在文件的邊界中。

注意： 並非所有印表機都支援所有列出的選項。 某些選項只適用於特定印表機機型、設定、作業系統或驅動程式類型。

如要選擇工作 ID 選項：

在「進階」視窗中，在「工作 ID」旁，按一下箭頭，然後選擇一個選項。

  • 停用工作 ID： 將不會列印任何工作 ID 資訊。
  • 在邊界列印 ID - 僅第一頁： 會在文件第一頁的左上角邊界列印工作 ID。
  • 在邊界列印 ID - 所有頁面： 會在文件每一頁的左上角邊界列印工作 ID。
  • 使用開頭空白頁：可讓您輸入工作 ID 中包含的頁數。頁數總計會從第一個列印頁開始計算。
