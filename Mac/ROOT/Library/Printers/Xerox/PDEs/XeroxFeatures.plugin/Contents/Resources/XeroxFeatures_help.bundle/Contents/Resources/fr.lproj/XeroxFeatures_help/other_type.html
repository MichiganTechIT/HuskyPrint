Type de support

Cette option permet de sélectionner le type de support pour votre travail d'impression.

Remarques :

  • Toutes les imprimantes ne prennent pas en charge la totalité des options mentionnées. Certaines options s'appliquent uniquement à un modèle d'imprimante, un système d'exploitation ou un type de pilote spécifique.
  • Si vous sélectionnez un type de support qui n'est pas chargé dans l'imprimante, celle-ci suspend le travail jusqu'à ce que le papier requis soit mis en place.

Pour sélectionner un type de support :

  1. Dans la fenêtre Support/Réception, en regard de Support, cliquez sur les flèches et sélectionnez Autre type.
  2. Effectuez l'une des opérations suivantes :
    • Pour utiliser le support par défaut pour l'imprimante, sélectionnez Défaut imprimante.
    • Sélectionnez un type de support dans la liste. Si l'administrateur a défini des types de support personnalisés pour l'imprimante, ceux-ci apparaissent dans la liste.
    • Pour sélectionner un magasin contenant n'importe quelle couleur de papier, choisissez Sélection automatique.
    • En cas d'utilisation d'intercalaires répertoire ou de carte intercalaire répertoire, sélectionnez les options requises.
