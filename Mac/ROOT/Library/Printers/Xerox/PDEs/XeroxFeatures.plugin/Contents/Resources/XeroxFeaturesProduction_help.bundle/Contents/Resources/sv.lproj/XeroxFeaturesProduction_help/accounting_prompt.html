Fråga vid redovisning

Om Redovisning har aktiverats och Fråga vid redovisning är inställt på Fråga alltid visas Fråga vid redovisning.

  1. Om ett grundinställt värde har angetts för användar-ID kan du använda det. Skriv annars ett nytt värde.
  2. Om ett grundinställt värde har angetts för konto-ID kan du använda det. Skriv annars ett nytt värde.
  3. Klicka på OK.
