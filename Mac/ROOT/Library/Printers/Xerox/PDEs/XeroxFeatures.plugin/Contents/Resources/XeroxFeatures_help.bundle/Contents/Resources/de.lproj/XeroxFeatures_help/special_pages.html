Sonderseiten

Als Sonderseiten werden Deckblätter, Leerblätter oder Seiten bezeichnet, deren Einstellungen sich von den anderen Dokumentseiten unterscheiden. Die Einstellungen von Sonderseiten unterscheiden sich von denen für den Hauptteil des Dokuments.

Verwalten von Sonderseiten

Die für einen Druckauftrag eingerichteten Sonderseiten werden in der Übersichtstabelle "Sonderseiten" angezeigt. Aus dieser Tabelle können Seiten zum Bearbeiten oder Löschen ausgewählt werden. Auch die Reihenfolge der Leerblätter innerhalb einer Leerblattgruppe kann geändert werden.

Einträge bearbeiten, löschen oder verschieben:

  1. Aus der Liste der gespeicherten Seiten eine Option auswählen. Die Schaltflächen werden aktiviert.
  2. Zum Auswählen der gewünschten Aktion auf die Schaltflächen klicken.
