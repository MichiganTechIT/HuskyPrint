특수 페이지

특수 페이지는 인쇄 작업에 추가되는 표지, 삽지, 예외 페이지입니다. 예외 페이지는 인쇄 작업의 나머지 페이지와 설정이 다릅니다.

특수 페이지 관리

이전에 인쇄 작업에 대해 설정한 특수 페이지는 특수 페이지 요약 표에 나타납니다. 이 표에서 편집하거나 삭제할 페이지를 선택할 수 있습니다. 삽지 그룹 내에서 삽입할 순서를 변경할 수도 있습니다.

항목 편집, 삭제 또는 이동:

  1. 저장된 페이지의 목록에서 옵션을 하나 선택합니다. 버튼이 활성화됩니다.
  2. 필요한 작업을 선택하려면 버튼을 클릭합니다.
