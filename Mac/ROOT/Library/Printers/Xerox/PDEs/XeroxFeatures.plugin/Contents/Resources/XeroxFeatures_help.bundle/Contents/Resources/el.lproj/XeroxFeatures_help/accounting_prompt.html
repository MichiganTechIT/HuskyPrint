Προτροπή λογιστικών

Αν τα Λογιστικά είναι ενεργοποιημένα και η Προτροπή λογιστικών έχει ρυθμιστεί να εμφανίζει ένα μήνυμα για εργασίες εκτύπωσης, φαξ και έγχρωμης εκτύπωσης όταν αποστέλλετε μία από αυτές τις εργασίες, τότε εμφανίζεται η Προτροπή λογιστικών.

Σημείωση: Ανάλογα με το σύστημα λογιστικών και τις επιλογές για την Προτροπή λογιστικών, δεν εμφανίζονται όλες οι επιλογές.

  1. Για τον Κωδικό χρήστη, αν έχει καθοριστεί μια προεπιλεγμένη τιμή, μπορείτε να τη χρησιμοποιήσετε ή να πληκτρολογήσετε μια νέα τιμή.
  2. Για τον Κωδικό πρόσβασης, αν έχει καθοριστεί μια προεπιλεγμένη τιμή, μπορείτε να τη χρησιμοποιήσετε ή να πληκτρολογήσετε ένα νέο κωδικό πρόσβασης.
  3. Αν χρειάζεται, επιλέξτε έναν τύπο λογαριασμού.
  4. Για τον Κωδικό λογαριασμού, αν έχει καθοριστεί μια προεπιλεγμένη τιμή, μπορείτε να τη χρησιμοποιήσετε ή να πληκτρολογήσετε ένα νέο κωδικό λογαριασμού.
  5. Για τον Κωδικό πρόσβασης κωδικού λογαριασμού, αν έχει καθοριστεί μια προεπιλεγμένη τιμή, μπορείτε να τη χρησιμοποιήσετε ή να πληκτρολογήσετε ένα νέο κωδικό πρόσβασης.
