Дополнительные параметры

Настройка дополнительных параметров:

  1. В окне «Пользовательская цветовая коррекция» выберите Дополнительные параметры.
  2. Для использования настроек цветов, заданных в предыдущем окне, выберите Использовать общие настройки.
  3. Установите настройки плотности, контраста, насыщенности и оттенка с помощью ползунков.
  4. Для восстановления первоначальных настроек системы нажмите кнопку По умолч..
  5. Нажмите кнопку OK.
