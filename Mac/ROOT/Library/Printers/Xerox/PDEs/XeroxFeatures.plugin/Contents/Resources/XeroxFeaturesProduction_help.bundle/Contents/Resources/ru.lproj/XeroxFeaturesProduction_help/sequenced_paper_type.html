Тип бумаги «В комплекте»

  1. В окне «В комплекте» выберите значение нажатием стрелок рядом с полем «Тип бумаги».
    • Введите новое имя для типа бумаги.
    • Выберите в списке тип бумаги.
  2. В поле «Листов в комплекте» введите значение или выберите его с помощью стрелок.
  3. Нажмите кнопку OK.
