Finition pliage dos carré

Pliage dos carré permet de définir la quantité de pliage pour le dos des cahiers à dos carré.

Pour sélectionner des options de pliage dos carré :

  1. Dans la fenêtre Pliage dos carré, sous Dos carré, sélectionnez une option.
    • Pour spécifier un pliage plus carré, sélectionnez une valeur entre Normal et +2.
    • Pour spécifier un pliage moins carré, sélectionnez une valeur entre Normal et -2.
    • Pour désactiver le pliage dos carré, sélectionnez Désactivé(e).
  2. Cliquez sur OK.
