Informations sur le destinataire

Pour ajouter ou modifier les informations sur le destinataire :

Dans la fenêtre informations sur le destinataire, sélectionnez une option :

  • L'option Montrer les destinataires permet d'afficher le nom des destinataires sur la page de garde d'une télécopie. Ces informations sont également visibles dans l'aperçu de page de garde.
  • L'option Ne pas afficher les destinataires permet de masquer le nom des destinataires sur la page de garde d'une télécopie.
  • Afficher le texte personnalisé : permet d'entrer un texte personnalisé dans la zone de texte qui s'imprime sur la page de garde d'une télécopie.
