오프셋/서브세트

오프셋 출력은 여러 부 인쇄 작업에서 복사본의 오프셋 방법을 제어합니다. 이 옵션을 사용하면 매수가 출력함에서 서로 약간 오프셋 상태가 됩니다. 출력함에 오프셋 기능이 있는 경우 이 옵션을 사용할 수 있습니다.

오프셋/서브세트 사용:

  1. 용지/출력 창에서 [마감 작업] 옆에 있는 화살표를 클릭한 후 오프셋/서브세트를 선택합니다.
  2. 옵션을 하나 선택하십시오.
    • 없음은 오프셋을 해제합니다. 서브세트는 여전히 선택할 수 있습니다.
    • 모든 세트를 선택하면 여러 부 인쇄 작업에서 각 세트의 오프셋이 지정됩니다.
    • 모든 서브세트를 선택하면 단일 세트에 지정된 페이지 수의 오프셋이 지정됩니다.
    • 모든 세트 및 모든 서브세트를 선택하면 단일 세트에 지정된 페이지 수의 오프셋이 지정되고, 여러 부 인쇄 작업에서 각 세트의 오프셋이 지정됩니다.
  3. 단일 인쇄 작업 내의 서브세트를 인쇄하려면 서브세트 사용을 선택합니다.
  4. 서브세트당 페이지 수에, 각 문서 서브세트에 포함할 페이지 수를 입력합니다. 각 서브세트는 동일한 페이지 수를 포함해야 합니다.
  5. 확인을 클릭합니다.
