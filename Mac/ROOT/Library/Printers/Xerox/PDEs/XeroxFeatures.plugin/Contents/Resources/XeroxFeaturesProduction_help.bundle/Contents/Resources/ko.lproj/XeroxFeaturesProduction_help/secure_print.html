보안 인쇄 설정

민감한 정보나 기밀 정보를 인쇄하려면 보안 인쇄를 사용합니다. 작업을 전송한 후 프린터 제어판에서 비밀번호를 입력할 때까지 작업이 프린터에 보류되어 있습니다.

참고: 일부 프린터에서는 여기 나열된 모든 옵션이 지원되지 않습니다. 일부 옵션은 특정 프린터 모델, 구성, 운영 체제 또는 드라이버 유형에만 적용됩니다.

보안 인쇄를 사용하여 인쇄하려면 다음 작업을 수행하십시오.

  1. 용지/출력 창에서 [작업 종류] 옆에 있는 화살표를 클릭한 후 보안 인쇄를 선택합니다.
  2. 비밀번호 필드에 4 – 10자의 비밀번호를 입력합니다. 보안 인쇄 작업을 보낼 때 제어판에서 작업을 해제할 비밀번호를 입력할 때까지 작업이 보류됩니다.
  3. 비밀번호 확인 필드에 비밀번호를 다시 입력합니다.
  4. 확인을 클릭합니다.
  5. 인쇄를 클릭합니다.
  6. 작업을 인쇄하려면 프린터 제어판에 비밀번호를 입력합니다.
