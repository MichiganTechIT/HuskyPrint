Intercalaires répertoire

  1. Dans le champ Nombre d'onglets, indiquez le nombre d'intercalaires. Ce champ définit le nombre de feuilles uniques contenues dans un jeu paginé avant que celui-ci ne se répète. Exemple : si un jeu d'intercalaires répertoire comprend cinq positions d'intercalaire, tapez 5 dans le champ Nombre d'onglets.
  2. Dans le champ Select a Tab Margin (Sélectionner une marge intercalaire), sélectionnez une option. Si vous optez pour Personnalisé(e), entrez la valeur de décalage dans le champ Décalage marge intercalaire, et définissez le paramètre Unités.
  3. Cliquez sur OK.
