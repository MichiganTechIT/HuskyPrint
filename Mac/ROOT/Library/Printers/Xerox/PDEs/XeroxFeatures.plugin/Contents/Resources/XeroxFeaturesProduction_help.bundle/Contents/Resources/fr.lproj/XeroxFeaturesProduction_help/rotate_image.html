Rotation image de 180 degrés

Lorsqu'elle est activée, l'option Rotation image de 180 degrés fait pivoter l'image sur la page de sorte que le bord supérieur soit tourné vers le bas. L'orientation peut être portrait ou paysage.

Pour sélectionner les options Rotation image de 180 degrés :

Dans la fenêtre Avancé, en regard de Rotation image de 180 degrés, cliquez sur les flèches et sélectionnez une option.

  • Valeur imprimante par défaut : utilise les paramètres actuels de l'imprimante.
  • Oui ou Activée : permet de faire pivoter automatiquement les images de 180 degrés. L'imprimante prend en considération les formats de support, l'orientation, le sens d'alimentation du papier et les options de finition sélectionnées lors de la rotation des images.
  • Non ou Désactivé(e) : annule la fonction.
