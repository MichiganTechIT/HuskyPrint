Tóner transparente

Tóner transparente puede hacer que la imagen impresa parezca una fotografía; también puede utilizarse para resaltar una imagen, texto o gráfico.

Nota: No todas las opciones descritas están disponibles en todas las impresoras. Algunas opciones solo se aplican a un modelo de impresora, configuración, sistema operativo o tipo de controlador específico.

Para utilizar Tóner transparente:

  1. En la ventana Tóner transparente, para Opciones de tóner transparente, seleccione una opción.
    • Prefijado de impresora: utiliza las opciones actuales de la impresora.
    • Saturación transparente: recubre toda la página con tóner transparente.
    • Solo imágenes: Solo recubre objetos de página que se identifican como imágenes.
    • Sólo texto: recubre solo el texto con tóner transparente.
    • Sólo gráficos: recubre solo los gráficos con tóner transparente.
    • Sin imágenes (solo texto, gráficos): Recubre toda la página, texto y gráficos con tóner transparente, omitiendo los gráficos.
    • Sin texto (solo imágenes, gráficos): Recubre toda la página, imágenes y gráficos con tóner transparente, omitiendo el texto.
    • Sin gráficos (solo imágenes, texto): Recubre toda la página, imágenes y texto con tóner transparente, omitiendo los gráficos.
    • Suprimir transparente: evita que se recubra con tóner transparente cualquier parte de la página.
    • Usar Transparente incorporado: La impresora utiliza las instrucciones que se encuentran en PDL para aplicar tóner transparente.

      La impresora utiliza Lenguajes de descripción de página (PDL) para preparar una página impresa. Los PDL incluyen PostScript (PS) y Lenguaje de descripción de página (PCL).

  2. Dependiendo del dispositivo, para especificar la cantidad de tóner transparente, seleccione una de las opciones.
    • Haga clic de nuevo en Tóner transparente y, a continuación, seleccione la cantidad de tóner transparente que desea aplicar a la selección.
    • En la lista Nivel de transparencia, seleccione el nivel de tóner transparente.
  3. Haga clic en Aceptar.
