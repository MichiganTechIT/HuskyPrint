Agregar inserciones

Agregue una inserción en blanco o preimpresa antes de la primera página del documento o después de páginas determinadas de un documento. Por ejemplo, para insertar una hoja de papel en color vacía después de cada sección de un documento, anote el número de página en que termina cada sección. Utilice estos números de página para indicar que se coloque una inserción después de cada página especificada.

Nota: No todas las opciones descritas están disponibles en todas las impresoras. Algunas opciones solo se aplican a un modelo de impresora, configuración, sistema operativo o tipo de controlador específico.

Para agregar una inserción:

  1. En el ventana Páginas especiales, haga clic en Agregar inserciones.
  2. Seleccione el tipo de opción de inserción:
    • Antes de la página 1 coloca una inserción delante de la página 1.
    • Después de página(s): coloca una inserción después de las páginas que especifique.
  3. Si selecciona Después de página(s), escriba el número de la página que precede a cada inserción en el campo Página(s).
  4. En Cantidad, seleccione el número de inserciones deseadas.
  5. Para aceptar los valores prefijados del trabajo, haga clic en Aceptar.
  6. Para seleccionar el tamaño, color o tipo, junto a Papel, haga clic en las flechas y, a continuación, seleccione las opciones.
  7. Haga clic en Aceptar.
