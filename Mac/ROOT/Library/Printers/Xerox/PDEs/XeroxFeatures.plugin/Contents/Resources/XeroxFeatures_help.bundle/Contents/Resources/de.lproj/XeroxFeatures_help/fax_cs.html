Faxdeckblatt

Einem Fax ein Deckblatt hinzufügen:

  1. Im Fenster "Material/Ausgabe" neben "Auftragsart" auf die Pfeile klicken und dann Fax wählen.
  2. Auf Deckblatt klicken.
  3. Neben "Deckblatt" auf die Pfeile klicken und dann Deckblatt drucken wählen.
  4. Zum Hinzufügen einer Nachricht an Empfänger auf die Schaltfläche neben Deckblatt-Anmerkungen klicken.
  5. Zum Hinzufügen Ihrer Kontaktdaten auf die Schaltfläche neben Absenderangaben klicken.
  6. Zum Bearbeiten von Empfängerdaten auf die Schaltfläche neben Empfängerangaben klicken.
  7. Zum Festlegen eines Deckblattbilds auf die Pfeile klicken und dann eine Option wählen.
    • Zum Deaktivieren von Deckblattbildern Aus wählen.
    • Zum Auswählen einer Deckblattbildoption, die zuvor auf dem Drucker gespeichert wurde, die entsprechende Deckblattbildoption aus der Liste auswählen.
    • Zum Bearbeiten eines zuvor gespeicherten Deckblattbilds das Deckblattbild aus der Liste auswählen und dann auf Bearbeiten klicken.
    • Zum Erstellen eines neuen Deckblattbilds auf Neu klicken.
  8. Auf Deckblattformat klicken und dann eine Option wählen.
  9. Auf OK klicken.
