Páginas a imprimir

Páginas a imprimir permite que se especifique um intervalo de páginas para o trabalho de impressão. Utilize essa opção para assegurar que as Páginas especiais referem-se às páginas corretas no documento.

Para selecionar um intervalo de páginas:

  1. Na janela Avançado, ao lado de Páginas a imprimir, clique em Configuração.
  2. No campo De, digite o primeiro número de página do intervalo de impressão.
  3. No campo Para, digite o último número de página do intervalo de impressão.
  4. Clique em OK.
