Fond de page enregistré

Les fonds de page permettent de fusionner les données du travail avec des formulaires stockés sur le disque dur de l'imprimante. Lorsque vous imprimez un travail avec un fond de page, le document et le fond de page sont fusionnés. Le document résultant semble imprimé sur un formulaire pré-imprimé.

Avant d'enregistrer un fond de page, il convient de le créer dans une autre application telle que Microsoft Word. Vous pouvez ensuite enregistrer le fond de page sur le disque dur de l'imprimante.

Remarque : Toutes les imprimantes ne prennent pas en charge la totalité des options mentionnées. Certaines options s'appliquent uniquement à un modèle d'imprimante, un système d'exploitation ou un type de pilote spécifique.

Pour imprimer en utilisant un fond de page :

  1. Dans la fenêtre Support/Réception, en regard de Type de travail, cliquez sur les flèches et sélectionnez Fond de page enregistré.
  2. Pour sélectionner un nom pour votre travail, effectuez l'une des opérations suivantes :
    • Saisissez le nom du travail.
    • Cliquez sur la flèche à droite du champ et sélectionnez un nom dans la liste.
    • Cliquez sur la flèche à droite du champ et sélectionnez Utiliser le nom du document pour utiliser le nom du document.
  3. Cliquez sur OK.
