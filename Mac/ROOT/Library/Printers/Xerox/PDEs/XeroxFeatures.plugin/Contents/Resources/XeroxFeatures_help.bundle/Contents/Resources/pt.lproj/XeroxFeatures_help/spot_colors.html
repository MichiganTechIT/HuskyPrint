Cores sólidas

Cor sólida mistura o ciano, magenta, amarelo e preto para se aproximar das cores de tinta sólida em uma impressora gráfica.

Nota: Nem todas as opções listadas são suportadas em todas as impressoras. Algumas opções são aplicáveis apenas a um modelo de impressora, configuração, sistema operacional ou tipo de driver específico.

Para selecionar as opções de Cores sólidas:

Na janela Avançado, ao lado de Cores sólidas, clique nas setas e selecione uma opção.

  • Permite que o software gerencie cores sólidas permite ao driver PostScript, aplicativo da impressora ou ambos controlar a interpretação de cores sólidas em CMYK para impressão. Use esta configuração quando uma cor sólida especial não imprimir a sombra esperada.

    As cores ciano, magenta, amarelo e preto (CMYK) que a impressora usa para fazer todas as cores.

  • Permitir que a impressora gerencie cores sólidas permite à impressora otimizar a interpretação de cores sólidas para CMYK para impressão.

    As cores ciano, magenta, amarelo e preto (CMYK) que a impressora usa para fazer todas as cores.
