Настройка защищенной печати

Защищенная печать используется для печати конфиденциальных или секретных документов. После отправки работа удерживается в принтере, пока на панели управления не будет введен пароль владельца работы.

Примечание: Не все перечисленные опции поддерживаются всеми принтерами. Некоторые опции предназначены только для определенных моделей и конфигураций принтера, операционных систем и драйверов.

Использование защищенной печати:

  1. В окне «Бумага/Вывод» нажатием стрелок рядом с полем «Тип работы» выберите значение Защищенная печать.
  2. Введите пароль (4–10 цифр) и подтвердите его в предусмотренных полях. Когда отправляется работа защищенной печати, она задерживается, пока отправитель не введет с панели управления код доступа для ее разблокирования.
  3. В поле «Подтвердить код доступа» введите пароль еще раз.
  4. Нажмите кнопку OK.
  5. Нажмите кнопку Печать.
  6. Для печати работы введите пароль с панели управления принтера.
