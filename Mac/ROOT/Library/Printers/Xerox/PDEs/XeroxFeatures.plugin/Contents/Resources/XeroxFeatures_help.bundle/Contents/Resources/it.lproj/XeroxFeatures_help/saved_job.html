Lavori salvati/Memorizza file nella cartella

I lavori salvati sono documenti inviati alla stampante e memorizzati per l'utilizzo futuro.

Nota: Non tutte le opzioni elencate sono supportate su tutte le stampanti. Alcune opzioni si applicano solo a modelli di stampante, configurazioni, sistemi operativi o tipi di driver specifici.

Per stampare un lavoro salvato:

  1. Nella finestra Carta/Copie, accanto a Tipo di lavoro, fare clic sulle frecce, quindi selezionare Lavori salvati o Memorizza file nella cartella.
  2. Per selezionare un nome per il lavoro, eseguire una delle operazioni seguenti:
    • Digitare un nome per il lavoro.
    • Fare clic sulla freccia a destra del campo e poi selezionare un nome dall'elenco.
    • Fare clic sulla freccia a destra del campo e poi selezionare Usa nome documento per utilizzare questo valore.
  3. Per la cartella, selezionare una delle opzioni seguenti:
    • Digitare il numero della cartella di destinazione.
    • Fare clic sulla freccia, quindi selezionare un numero di cartella dall'elenco.
  4. Selezionare un'opzione. Se si seleziona Privato, digitare quindi confermare un codice di accesso.
  5. Fare clic su OK.
