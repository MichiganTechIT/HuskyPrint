Редактор изображения титульного листа

В титульные листы факса можно добавлять текст или изображения или использовать для этого текст и изображения, хранящиеся на принтере.

  1. Добавление или изменение текста для титульных листов:
    1. В окне «Редактор изображения титульного листа» выберите Текст.
    2. Для форматирования текста нажмите Показать шрифты и выберите настройки шрифта в списке.
    3. Для указания угла поворота текста на странице введите значение в поле «Угол». Для изменения значения воспользуйтесь стрелками. Для выбора значения воспользуйтесь ползунком.
    4. Чтобы сдвинуть водяной знак относительно центра страницы, выберите единицу измерения, затем введите значения для положений по горизонтали и вертикали.
    5. Чтобы центрировать изображение, нажмите Центр.
    6. Нажмите кнопку OK.
  2. Добавление или изменение изображения для титульных листов:
    1. В окне «Редактор изображения титульного листа» выберите Изображение.
    2. В поле «Выбрать файл» введите путь к файлу изображения или нажмите кнопку Выбрать файл и укажите файл.

      Примечание: Поддерживаются форматы .bmp, .jpg и .gif.

    3. Для изменения размера изображения в поле «Масштаб» введите или выберите значение в процентах или используйте ползунок.
    4. Чтобы сдвинуть водяной знак относительно центра страницы, выберите единицу измерения, затем введите значения для положений по горизонтали и вертикали.
    5. Чтобы центрировать изображение, нажмите Центр.
    6. Нажмите кнопку OK.
