Примечания к заданию

Для указания дополнительных сведений о задании печати. Настройки работы печати можно сохранять в формате PDF или PostScript, а затем просматривать эти файлы или отправлять по факсу или электронной почте.

Примечание: Не все перечисленные опции поддерживаются всеми принтерами. Некоторые опции предназначены только для определенных моделей и конфигураций принтера, операционных систем и драйверов.

Печать и использованием примечаний к работе:

  1. Нажмите кнопку Примеч..
  2. Выполните одно или несколько из следующих действий:
    • Выберите Имя работы.

      Имя работы включает имя документа, которое печатается на титульном листе работы печати. Имя документа также отображается в очереди печати на принтере.

      Чтобы добавить имя работы, выполните одно из следующих действий:

      • Рядом с полем «Имя работы» нажатием стрелки выберите значение Использовать имя документа.
      • Нажмите стрелка «вправо» и выберите ранее введенное имя работы.
      • Введите новое имя.
    • Выберите Имя отправителя.

      Имя отправителя документа печатается на титульном листе работы печати. Имя отправителя также отображается в очереди печати на принтере.

      • Нажмите стрелку «вправо» и выберите Использовать имя для входа.
      • Нажмите стрелка «вправо» и выберите ранее введенное имя работы.
      • Введите новое имя.
    • Выберите Имя получателя.

      Имя получателя работы печати печатается на титульном листе.

      • Нажмите стрелку «вправо» и выберите Использовать имя для входа.
      • Нажмите стрелка «вправо» и выберите ранее введенное имя работы.

        Примечание: Для удаления списка ранее введенных имен выберите опцию Удалить список.

      • Введите новое имя.
    • Введите сообщение в поле Информационное сообщение.

      Функция «Информационное сообщение» служит для ввода информации, которая печатается на титульном листе работы печати.

    • Введите сообщение в поле Сообщение о начале работы.

      Функция «Сообщение о начале работы» служит для выдачи указаний оператору принтера. Пример: «Распечатать этот файл до 14-00».

    • Введите дополнительные учетные сведения.

      Функция «Дополнительная информация об учетной записи» служит для ввода информации о работе или оплате, которую необходимо включить в работу печати.

  3. Нажмите кнопку Печать.
