Επιλογές ειδώλου

Η λειτουργία Επιλογές ειδώλου επιτρέπει τη ρύθμιση της συνολικής φωτεινότητας ή σκουρότητας του κειμένου και των ειδώλων στην εργασία εκτύπωσης.

Σημείωση: Δεν υποστηρίζονται όλες οι επιλογές που παρατίθενται από όλους τους εκτυπωτές. Ορισμένες επιλογές ισχύουν μόνο σε συγκεκριμένα μοντέλα ή διαμορφώσεις εκτυπωτών, λειτουργικά συστήματα ή τύπους οδηγών.

Για τη ρύθμιση των επιλογών ειδώλου για την εργασία εκτύπωσης:

  1. Για τη ρύθμιση της φωτεινότητας, μετακινήστε το ρυθμιστικό προς τα δεξιά για φωτεινότερη απόδοση ή προς τα αριστερά για πιο σκούρα απόδοση των χρωμάτων στην εργασία εκτύπωσης.
  2. Για να ρυθμίσετε την αντίθεση, μετακινήστε το ρυθμιστικό προς τα δεξιά για τη μείωση της αντίθεσης ή προς τα αριστερά για την αύξηση της αντίθεσης.
  3. Για να ορίσετε επιλογές για τη Μετατόπιση, κάντε κλικ στην επιλογή Μετατόπιση ειδώλου.
  4. Κάντε κλικ στην επιλογή Εκτύπωση.
