Gloss Level

Use Gloss Level to print a matte or glossy image.

Note: Not all options listed are supported on all printers. Some options apply only to a specific printer model, configuration, operating system, or driver type.

To select Gloss Level options:

In the Advanced window, next to Gloss Level, click the arrows, then select an option.

  • Normal prints a matte version of the original glossy image.
  • Enhanced duplicates the original glossy image. The printer prints more slowly using this option.
