User TRCs

Tone reproduction curves (TRCs) maintain color consistency of the printed image. User TRCs allow you to fine-tune printed colors by adjusting the tone reproduction curves.

To print using TRCs:

In the Image Options window, for User TRCs, select an option.

  • Printer Default: Uses the default printer settings.
  • From the list of up to 10 of the last TRC names entered, select a name.
  • Enter a new name.
