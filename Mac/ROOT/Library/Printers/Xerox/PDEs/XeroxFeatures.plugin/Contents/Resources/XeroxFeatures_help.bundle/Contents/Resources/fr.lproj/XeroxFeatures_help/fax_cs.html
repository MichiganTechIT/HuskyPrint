Page de garde de télécopie

Pour ajouter une page de garde à une télécopie :

  1. Dans la fenêtre Support/Réception, en regard de Type de travail, cliquez sur les flèches et sélectionnez l'option Fax.
  2. Cliquez sur Page de garde.
  3. En regard de Page de garde, cliquez sur les flèches et sélectionnez Imprimer une page de garde.
  4. Pour ajouter un message aux destinataires, en regard de Notes de page de garde, cliquez sur le bouton.
  5. Pour ajouter vos informations de contact, en regard de Informations sur l'expéditeur, cliquez sur le bouton.
  6. Pour modifier les informations sur l'expéditeur, en regard de Informations sur l'expéditeur, cliquez sur le bouton.
  7. Pour spécifier une page de garde, cliquez sur les flèches et sélectionnez une option.
    • Pour désactiver les pages de garde, sélectionnez Aucun(e).
    • Pour choisir une option de page de garde qui a été enregistrée dans l'imprimante, sélectionnez-la dans la liste.
    • Pour modifier une page de garde qui a été enregistrée, sélectionnez-la dans la liste puis cliquez sur Modifier.
    • Pour créer une nouvelle page de garde, cliquez sur Nouveau.
  8. Cliquez sur Format page de garde, puis sélectionnez une option.
  9. Cliquez sur OK.
