Rückenpressung

Mit der Funktion "Rückenpressung" kann der Rücken von Broschüren mit Rückenpressung gefalzt werden.

Rückenpressungsoptionen wählen:

  1. Im Fenster "Rückenpressung" neben "Rückenpressung" eine Option wählen.
    • Zum Festlegen einer eckigeren Rückenpressung von Normal bis +2 wählen.
    • Zum Festlegen einer runderen Rückenpressung von Normal bis -2 wählen.
    • Zum Deaktivieren der Rückenpressung Aus wählen.
  2. Auf OK klicken.
