Klebebinder

„Klebebinder“ dient zur Ausgabe eines gebundenen Taschenbuchs. Um den Buchblock wird ein Umschlag gelegt und festgeklebt. Das Buch kann auch auf die gewünschte Größe zugeschnitten werden.

Hinweise:

  • Diese Option erscheint nur, wenn der entsprechende Finisher an den Drucker angeschlossen ist. Wenn Deckblätter, Heften, Lochen, Binden, Falzen und/oder die Broschürenausgabe aktiviert sind, wird sie nicht angezeigt.
  • Nicht alle aufgelisteten Optionen werden von allen Druckern unterstützt. Einige Optionen stehen nur für bestimmte Druckermodelle, Konfigurationen, Betriebssysteme oder Treibertypen zur Verfügung.

Klebebinder verwenden:

  1. Im Fenster "Material/Ausgabe" neben "Endverarbeitung" auf die Pfeile klicken und dann Klebebindung wählen.
  2. Neben "Bindung" eine Option wählen.
    • Hochformat - links: Hochformat wird links gebunden.
    • Hochformat - rechts: Hochformat wird rechts gebunden.
    • Querformat - oben: Querformat wird oben gebunden.
    • Querformat - unten: Setzt die Ausrichtung auf "Querformat" und bindet das Buch unten.
  3. Auf Beschneiden klicken und eine Option auswählen.
    • Nicht beschneiden: Seiten nicht beschneiden.
    • Druckbild oder Schriftbild: Die der Bindekante gegenüber liegende Seite beschneiden.
    • Randlos: Über „Beschnittene Größe“ eine Beschnittgröße auswählen. Mit der Funktion „Randlos“ kann das Druckbild bis zur Materialkante reichen.
  4. Zur Auswahl der Beschnittgröße auf Beschnittene Größe klicken und die Größe aus der Liste auswählen oder Benutzerdefiniert auswählen.
  5. Zur Auswahl des Deckblattmaterials aus der Liste „Deckblatt“ unter den folgenden Optionen auswählen.
    • Eigenes Material: Ermöglicht die Auswahl eines eigenen Materials aus einer Liste bzw. das Erstellen eines neuen eigenen Materials.
    • Anderes Format: Ermöglicht die Auswahl eines vorhandenen oder Erstellung eines neuen benutzerdefinierten Materialformats.
    • Andere Farbe: Ermöglicht die Auswahl aus einer Liste verfügbarer Farben bzw. das Erstellen einer benutzerdefinierten Materialfarbe.
    • Andere Art: Ermöglicht die Auswahl der Materialart bzw. das Erstellen benutzerdefinierter Materialarten.
    • Anderes Gewicht: Ermöglicht die Auswahl des Materialgewichts bzw. das Erstellen eines benutzerdefinierten Materialgewichts.
    • Erweiterte Materialoptionen: Ermöglicht eine Auswahl erweiterter Materialattribute für das Deckblatt.
  6. Auf OK klicken.
