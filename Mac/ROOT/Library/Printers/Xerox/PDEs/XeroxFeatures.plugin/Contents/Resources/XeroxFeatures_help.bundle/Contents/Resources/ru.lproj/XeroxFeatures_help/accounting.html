Учет

Функция учета обеспечивает привязку каждого печатаемого документа к имени пользователя и идентификационным учетным данным. Данные учета работ собираются и сохранятся по мере поступления работ на принтер.

Примечания:

  • Не все перечисленные опции поддерживаются всеми принтерами. Некоторые опции предназначены только для определенных моделей и конфигураций принтера, операционных систем и драйверов.
  • В зависимости от системы учета и настройки запроса системы учета некоторые опции могут не отображаться.

Настройка учета:

  1. В окне «Учет» выберите значение нажатием стрелок рядом с полем «Система учета».
    • Xerox® Стандартный учет — служит для отслеживания количества копий, отпечатков, изображений сканирования и факсов для каждого пользователя. Для этого используются идентификаторы пользователя и счета, предварительно заданные на принтере.
    • Xerox® Сетевой учет — идентификаторы пользователя и счета выдает сервер сетевой идентификации.
    • Локальный учет — требуется код учета для вашей подсети. Может использоваться код оплаты подразделения, например, 0001. Дополнительно могут использоваться и другие коды.
    • Платная печать — для работ требуется указывать идентификатор оплаты, например ID оплаты для подразделения. Дополнительно могут использоваться и другие идентификаторы.
    • Отправить ID пользователя и ID учета — для отправки работы печати необходимо ввести идентификаторы пользователя и учета и пароль.
    • Отправить только ID пользователя — для отправки работы печати необходимо ввести идентификатор пользователя и пароль.
    • Отправить только ID учета — для отправки работы печати необходимо ввести идентификатор учета и пароль.
    • Отправить только ID оплаты — для отправки работы печати необходимо ввести идентификатор оплаты и пароль.
    • Отправить все — для отправки работы печати необходимо ввести идентификаторы пользователя, учета, оплаты и пароль.
  2. Выберите один из вариантов в поле «Сообщение системы учета».
    • Всегда выдавать сообщение - при отправке каждой работы печати необходимо вводить ID пользователя и ID счета.
    • Не выдавать сообщение - используется информация, введенная в поле «Коды учета» для каждой работы печати.
    • Выдавать сообщение только для факса — при отправке каждого факса необходимо вводить идентификаторы пользователя и счета.
    • Выдавать сообщение только для цветного режима - для работ печати в цветном режиме требуется вводить ID пользователя или ID счета.
    • Выдавать сообщение только для цветной печати и факса - для факсов и работ печати в цветном режиме требуется вводить ID пользователя или ID счета.
  3. Чтобы идентификатор пользователя скрывался, выберите «Скрыть ID пользователя». Вместо вводимых в поле символов будут отображаться звездочки (*).
  4. Чтобы идентификатор счета скрывался, выберите «Скрыть ID счета». Вместо вводимых в поле символов будут отображаться звездочки (*).
  5. Введите коды в разделе «Коды учета по умолчанию». Данные поля требуются при выборе опции "Не выдавать сообщение". В качестве ID счета и ID оплаты может использоваться кода оплаты подразделения, например 0001.
  6. Нажмите кнопку OK.
