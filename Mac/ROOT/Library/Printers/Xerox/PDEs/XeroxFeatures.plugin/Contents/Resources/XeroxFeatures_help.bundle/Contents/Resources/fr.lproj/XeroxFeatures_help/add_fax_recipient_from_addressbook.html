Envoyer une télécopie à un destinataire dans le répertoire

Vous pouvez ajouter un destinataire de télécopie à partir du répertoire.

Pour ajouter un destinataire à partir de votre répertoire :

  1. Dans la fenêtre Support/Réception, en regard de Type de travail, cliquez sur les flèches et sélectionnez l'option Fax.
  2. Cliquez sur Destinataires.
  3. Cliquez sur l'icône du carnet d'adresses, puis effectuez l'une des opérations suivantes :
    • Sélectionnez un groupe puis un destinataire.
    • Pour trouver un destinataire, entrez son nom dans le champ de recherche, puis cliquez sur OK.
  4. Cliquez sur OK.
