Anti-Aliasing

L'opzione Anti-aliasing consente di smussare l'aspetto di bordi frastagliati in immagini e testo.

Per selezionare un'opzione anti-aliasing:

Nella finestra Opzioni immagine, per Anti-aliasing, selezionare un'opzione.

  • Default stampante: utilizza le impostazioni stampante correnti.
  • Sì: consente di attivare l'anti-aliasing.
  • No: consente di disattivare l'anti-aliasing.
  • K 100% Alta risoluzione: consente di utilizzare colore nero nitido, pieno. È possibile utilizzare questa funzione per testo minuto o grafica contenente blocchi omogenei di nero.
