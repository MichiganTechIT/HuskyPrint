Gris pálido RGB

Gris pálido RGB permite definir cómo se imprimirán las áreas grises para producir grises sin variaciones de color significativas.

Para utilizar Gris pálido RGB.

En la ventana Avanzadas, junto a Gris pálido RGB, haga clic en las flechas y, a continuación, seleccione una opción.

  • Prefijado de impresora: utiliza las opciones prefijadas de la impresora.
  • Negro automático utiliza tóner negro para texto negro y gris y dibujos lineales. Esta opción también utiliza negro compuesto para las partes en negro y gris de fotos y gráficos.

    Negro compuesto es una mezcla de tóner cian, magenta y amarillo con tóner negro.

  • Utilizar negro puro utiliza tóner negro para todos los elementos negros y grises de la página, incluido el texto.
  • Utilizar negro compuesto utiliza el negro compuesto para las partes negras y grises del texto, como dibujos lineales, fotos y gráficos.
