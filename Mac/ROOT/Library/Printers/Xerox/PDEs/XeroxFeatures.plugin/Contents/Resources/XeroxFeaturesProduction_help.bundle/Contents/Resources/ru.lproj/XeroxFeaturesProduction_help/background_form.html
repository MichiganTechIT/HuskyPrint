Фоновая форма

Функция «Фоновые формы» служит для объединения данных работы печати с формами, сохраненными на жестким диске принтера. При печати работы с фоновой формой документ и форма объединяются. Готовый документ выглядит так, будто он напечатан на бланке.

Фоновые формы создаются в таких приложениях, как Microsoft Word, и сохраняются на жестком диске принтера.

  2. Работа печати переднего плана
  3. Готовый документ
  1. Создание фоновой формы:
  1. Выберите Фоновые формы.
  2. Для параметра «Имя формы» выберите один следующих вариантов:
    • Для отображения списка существующих фоновых форм нажатием стрелок выберите значение рядом с полем «Имя формы».
    • Для создания формы введите имя.
  3. Для применения фоновой формы к работе печати переднего плана выберите один из вариантов в поле «Применить страницы форм».
    • Для всей работы — применение страниц фоновой формы ко всей работе печати переднего плана. Это повторяется для всех страниц работы печати переднего плана.

    • Один раз — однократное применение страниц фоновой формы к работе печати переднего плана. Остальные страницы работы печати переднего плана печатаются без фоновой формы.

    • Один раз, затем повторить — сначала однократное применение страниц фоновой формы к работе печати переднего плана. Затем это повторяется, начиная с указанной страницы фоновой формы, для всех страниц работы печати переднего плана.

  4. Для сдвига изображения страницы по осям X и Y вводятся значения в поле «Сдвиг изображения».
  5. Для выбора единиц измерения нажатием стрелок внизу окна выберите дюймы или миллиметры.
  6. Нажмите кнопку OK.
