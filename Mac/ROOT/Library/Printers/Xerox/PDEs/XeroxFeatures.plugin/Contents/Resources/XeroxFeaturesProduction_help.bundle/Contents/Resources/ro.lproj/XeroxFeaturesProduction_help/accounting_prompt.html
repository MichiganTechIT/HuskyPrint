Solicitare Contorizare

Dacă funcţia de Contorizare este activată şi opţiunea Solicitare Contorizare este setată la valoarea Solicită întotdeauna, în momentul în care trimiteţi o lucrare de imprimare, apare Solicitare Contorizare.

  1. Dacă a fost setat un ID Utilizator implicit, îl puteţi utiliza pe acesta sau puteţi tasta unul nou.
  2. Dacă a fost setat un ID Cont implicit, îl puteţi utiliza pe acesta sau puteţi tasta unul nou.
  3. Faceţi clic pe OK.
