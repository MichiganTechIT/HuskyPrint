Подрезание наружной кромки

Настройка «Подрезание наружной кромки» служит для указания ширины подрезания наружной кромки брошюры.

Выбор подрезания наружной кромки:

  1. Для включения функции «Подрезание наружной кромки» установите данный флажок.
  2. В поле «Подрезание наружной кромки» введите значение подрезания кромки или выберите его с помощью стрелок.
  3. Для указания единиц измерения нажатием стрелок выберите мм или дюйм в поле «Единицы».
  4. Нажмите кнопку OK.
