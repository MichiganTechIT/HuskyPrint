Genomskinligt färgpulver

Med Genomskinligt färgpulver kan utskriften fås att se ut som ett fotografi. Det kan också användas för att framhäva en viss bild, text eller grafik.

Obs! Alla alternativ som visas stöds inte på alla skrivare. Vissa alternativ finns bara i en viss skrivarmodell, en viss konfiguration, ett visst operativsystem eller en viss drivrutinstyp.

Så här använder du genomskinligt färgpulver:

  1. Välj ett alternativ vid Alternativ för genomskinligt i fönstret Genomskinligt färgpulver.
    • Skrivarens grundinställda typ: Aktuella skrivarinställningar används.
    • Flöda genomskinligt: Hela sidan täcks med genomskinligt färgpulver.
    • Endast bilder: Endast sidobjekt som identifieras som bilder täcks med färgpulver.
    • Endast text: Bara själva texten täcks med genomskinligt färgpulver.
    • Endast grafik: Bara grafik täcks med genomskinligt färgpulver.
    • Inte bilder (endast text och grafik): Hela sidan – text och grafik – täcks med genomskinligt färgpulver men inga bilder.
    • Inte text (endast bilder och grafik): Hela sidan – bilder och grafik – täcks med genomskinligt färgpulver men ingen text.
    • Inte grafik (endast bilder och text): Hela sidan – bilder och text – täcks med genomskinligt färgpulver men ingen grafik.
    • Använd inte genomskinligt: Inget genomskinligt färgpulver används över huvud taget.
    • Använd inbäddat genomskinligt: Genomskinligt färgpulver tillämpas enligt instruktionerna i PDL (sidbeskrivningsspråket).

      Den utskrivna sidan skapas enligt instruktionerna i PDL (sidbeskrivningsspråket). PDL inbegriper PostScript (PS) och Printer Command Language (PCL).

  2. Ange mängden genomskinligt färgpulver genom att välja ett alternativ som är lämpligt för enheten.
    • Klicka på Genomskinligt färgpulver igen och välj den mängd genomskinligt färgpulver som ska användas på objektet.
    • Välj mängden genomskinligt färgpulver i listan Mängd genomskinligt.
  3. Klicka på OK.
