Опции факса

Эта страница служит для настройки отправки факсов. Например, можно задать печать отчета-подтверждения или отправку факса в указанное время.

Примечание: Не все перечисленные опции поддерживаются всеми принтерами. Некоторые опции предназначены только для определенных моделей и конфигураций принтера, операционных систем и драйверов.

Использование опций факса:

  1. В окне «Настройка факса» выберите один из вариантов нажатием стрелок рядом с полем «Отчет-подтверждение».
    • Не печатать отчет-подтверждение — отчет-подтверждение не печатается.
    • Печатать отчет-подтверждение — отчет-подтверждение печатается.
    • Отправить по эл. почте — отправка отчета-подтверждения по указанному адресу эл. почты.
  2. Выберите скорость в поле «Скорость передачи».

    Настройка «Скорость передачи» служит для выбора скорости передачи факса в килобитах в секунду (Кбит/с) или в битах в секунду (бит/с). Если в данном регионе телефонные линии не обеспечивают высокоскоростную передачу факса и данных, выберите низкую скорость передачи. При низкой скорости передачи снижается риск возможных ошибок передачи.

  3. Выберите один из вариантов в поле «Разрешение факса».

    Разрешение факса определяет уровень воспроизведения деталей изображений при передаче факса. Передача факсов с высоким разрешением занимает больше памяти и требует больше времени.

    Примечание: Если принимающий факсимильный аппарат не поддерживает выбранное разрешение, отправляющее устройство изменяет разрешение на поддерживаемое значение.

    • Стандартное — используется для текста, набранного на компьютере.
    • Высокое — используется для штриховых рисунков, мелкого и рукописного текста.
    • Сверхвыс. — используется для очень мелкого текста, рисунков и изображений с низким разрешением.
    • Сверхвысокое — используется для фотографий и штриховой графики.
  4. В поле «Время передачи» выберите опцию Отправить сейчас или Отправить в.

    Выберите время отправки факса. Отправку факса можно отложить на срок до 24 часов.

    При выборе опции «Отправить сейчас» факс отправляется сразу.

  5. В поле «Префикс набора» введите префикс (до четырех цифр).
  6. В поле «Кредитная карта» введите номер карты (до 16 цифр).

    Примечание: Номер кредитной карты в отчетах-подтверждениях не указывается.

  7. Нажмите кнопку OK.
