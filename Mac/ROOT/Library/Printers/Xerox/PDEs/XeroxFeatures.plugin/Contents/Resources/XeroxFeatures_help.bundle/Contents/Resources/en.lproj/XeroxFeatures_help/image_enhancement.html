Image Enhancement

Image Enhancement allows the printer to determine when to apply Edge Enhancement, Thin Line Enhancement, or Trapping to an image.

To select Image Enhancement options:

In the Advanced window, next to Image Enhancement, click the arrows, then select an option.

  • Automatic applies a collection of settings with the best speed and quality trade-off for each print mode.
  • On uses all available enhancements to improve print quality. Jobs using this option take longer to print.
  • Off does not apply any image enhancements.
