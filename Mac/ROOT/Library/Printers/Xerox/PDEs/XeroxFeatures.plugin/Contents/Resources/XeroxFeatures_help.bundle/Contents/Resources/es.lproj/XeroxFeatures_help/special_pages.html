Páginas especiales

Las páginas especiales son cubiertas, inserciones y excepciones que se agregan a un trabajo de impresión. Las excepciones tienen opciones diferentes a las del resto del trabajo de impresión.

Administración de Páginas especiales

Las páginas especiales configuradas previamente para sus trabajos de impresión aparecen en la tabla de resumen de Páginas especiales. Desde esta tabla puede seleccionar una página para editarla o eliminarla. También puede cambiar el orden de las inserciones dentro de un grupo de inserciones.

Para editar, eliminar o mover las entradas:

  1. En la lista de páginas guardadas, seleccione una opción. Los botones se activan.
  2. Para seleccionar la acción deseada, haga clic en los botones.
