Recobriment

Recobriment permet que les vores de les àrees negres i amb color, llevat de les àrees blanques, se superposin. La superposició de les vores contribueix a compensar les irregularitats d'impressió que es poden produir a causa de desplaçaments del paper o altres circumstàncies.

Nota: No totes les opcions indicades són compatibles amb totes les impressores. Algunes opcions només són compatibles amb determinats models d'impressores, configuracions, sistemes operatius o tipus de controladors.

Per utilitzar el recobriment:

A la finestra Avançada, al costat de Recobriment, feu clic a les fletxes i seleccioneu una opció.

  • Actiu o Activat: fa que les vores de les àrees amb color se superposin a les vores de les àrees negres. Aquesta opció no afecta les àrees blanques.
  • Inactiu o Desactivat: desactiva la funció.
  • Només negre: fa que les àrees amb color se superposin només a les vores de les àrees negres. Aquesta opció no afecta les àrees blanques.
