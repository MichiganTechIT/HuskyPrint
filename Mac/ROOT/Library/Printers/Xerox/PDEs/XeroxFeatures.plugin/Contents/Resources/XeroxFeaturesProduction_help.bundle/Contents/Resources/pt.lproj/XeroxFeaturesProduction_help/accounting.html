Contabilidade

Contabilidade associa seu nome de usuário e as informações de identificação da conta a cada documento que imprimir. Os dados de contabilidade são coletados e armazenados à medida que os trabalhos são enviados à impressora.

  1. Para acessar as configurações de Contabilidade, clique em Contabilidade na janela Avançada, ao lado de Configuração.
  2. Para o Sistema de contabilidade, selecione uma opção.
    • Para configurar uma contabilidade, clique em Ativada.
    • Para desabilitar a contabilidade, clique em Nenhuma.
  3. Selecione uma opção ao lado de Solicitação de contabilidade.
    • Não solicitar: não solicita a digitação de uma ID de usuário ou de uma ID de conta quando o trabalho de impressão é enviado.
    • Sempre solicitar: requer que você digite a sua ID de usuário e ID de conta toda vez que enviar um trabalho de impressão.
  4. Caso seja selecionado Sempre solicitar:
    1. Para mascarar uma ID de usuário ou ID de conta quando o trabalho é impresso, selecione a caixa de seleção ao lado dessas opções.
    2. Caso necessário, digite um valor padrão de ID de usuário.
    3. Caso necessário, digite uma ID padrão de conta ou um valor de código de faturamento.
  5. Clique em OK.
