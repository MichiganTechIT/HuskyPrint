Materialwahl

Mit „Erweiterte Materialwahl“ können verschiedene Materialattribute in einem Fenster ausgewählt werden. Die Zusammenfassung zeigt die gewählten Optionen.

Hinweis: Nicht alle aufgelisteten Optionen werden von allen Druckern unterstützt. Einige Optionen stehen nur für bestimmte Druckermodelle, Konfigurationen, Betriebssysteme oder Treibertypen zur Verfügung.

Materialattribute über „Materialwahl“ wählen:

  1. Zum Hinzufügen oder Aktualisieren von Attributen für ein eigenes Material neben "Material" die Option Eigenes Material wählen und dann die gewünschte Option auswählen.

    Hinweis: Der Druckertreiber speichert 10 Namen. Beim Hinzufügen eines 11. Namens wird der am wenigsten verwendete Name automatisch aus der Liste gelöscht. Zum Entfernen von anderen als dem markierten Namen Liste löschen wählen.

  2. Im Fenster "Material/Ausgabe" neben "Material" auf die Pfeile klicken und dann Materialwahl wählen.
  3. Zum Festlegen einzelner Attribute das Kontrollkästchen neben "Nur nach Namen" deaktivieren.
    • Ein Materialformat wählen oder zum Erstellen eines benutzerdefinierten Materialformats auf Benutzerdefiniert klicken.
    • Eine Materialfarbe wählen oder zum Erstellen einer benutzerdefinierten Materialfarbe auf Benutzerdefiniert klicken.
    • Eine Materialart wählen oder zum Erstellen einer benutzerdefinierten Materialart auf Benutzerdefiniert klicken.
    • Ein Materialgewicht (g/m²) wählen oder zum Erstellen eines benutzerdefinierten Materialgewichts auf Benutzerdefiniert klicken.
    • Für Gelochtes Material eine Option wählen.
    • Für Strich eine Option wählen.

      Der Strich des Papiers wirkt sich auf das Aussehen des gedruckten Dokuments aus. "Strich" wählen, um die Druckausgabe mit einem Strich zu versehen. Strich kann für eine oder beide Seiten des Papiers angegeben werden.

    • Für Strichart eine Option wählen.

      Art des auf gestrichenem Papier verwendeten Strichs wählen. Die Art des Strichs kann sich auf das Aussehen des gedruckten Dokuments auswirken.

    • Für Faserlauf eine Option wählen.

      "Faserlauf" spezifiziert den Faserlauf des Materials für den Druckauftrag. In einigen Fällen kann sich der Faserlauf das Aussehen des gedruckten Dokuments auswirken.

    • Für Oberfläche eine Option wählen.

      Mit "Oberfläche" kann die Art der Endverarbeitung für den Druckauftrag gewählt werden. Sie wirkt sich auf das Aussehen des gedruckten Dokuments aus.

    • Für Beschreibung die Druckereinstellung wählen, eine Beschreibung aus der Liste auswählen oder eine neue Beschreibung eingeben.
    • Zur Erfassung optionaler Informationen zur Auswahl in das Feld Anmerkung (optional) Details eingeben.
  4. Auf OK klicken.
