Colore di uscita

Nella finestra Carta/Copie, accanto a Colore di uscita, fare clic sulle frecce, quindi selezionare un'opzione.

  • Default stampante: utilizza le impostazioni stampante predefinite.
  • Scala dei grigi: consente di stampare i colori come variazioni di grigio.
  • Colore: consente di stampare in quadricromia.
