Mode esborrany

El mode esborrany produeix imatges de menys qualitat, però permet estalviar tòner o tinta.

Nota: El mode esborrany forma part dels paràmetres d'intel·ligència mediambiental.

Per seleccionar opcions de mode esborrany:

A la finestra Avançada, al costat d'Imatge esborrany, feu clic a les fletxes i seleccioneu una opció.

  • Actiu o Activat: imprimeix esborranys.
  • Inactiu o Desactivat: desactiva el mode esborrany.
