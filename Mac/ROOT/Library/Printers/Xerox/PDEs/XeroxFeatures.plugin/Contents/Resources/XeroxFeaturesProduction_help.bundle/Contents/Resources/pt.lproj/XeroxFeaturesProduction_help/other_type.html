Tipo de Papel

Outro tipo permite selecionar o tipo de papel para o trabalho de impressão.

Notas:

  • Nem todas as opções listadas são suportadas em todas as impressoras. Algumas opções são aplicáveis apenas a um modelo de impressora, configuração, sistema operacional ou tipo de driver específico.
  • Se você selecionar um tipo de papel que não está carregado na impressora, a impressora reterá o trabalho até que o papel desejado seja carregado.

Para selecionar um tipo do papel:

  1. Na janela Papel/Saída, ao lado de Papel, clique nas setas e selecione Outro tipo.
  2. Faça da seguinte maneira:
    • Para usar o tipo de papel padrão para a impressora, selecione Padrão da impressora.
    • Para selecionar uma bandeja que tenha alguma cor de papel, selecione Selecionar automaticamente.
    • Selecione um Tipo de papel ou, para criar um tipo de papel personalizado, clique em Personalizado.
    • Para Divisórias com índice, especifique o número em sequência.
    • Caso tenha selecionado Sequenciado, especifique o número na sequência.
