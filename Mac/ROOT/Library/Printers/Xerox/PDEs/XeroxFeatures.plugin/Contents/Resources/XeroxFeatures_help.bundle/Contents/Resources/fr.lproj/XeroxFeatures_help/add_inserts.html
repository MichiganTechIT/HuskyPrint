Ajouter des insertions

Cette option permet d'ajouter une insertion vierge ou préimprimée avant la première page de chaque document ou après les pages désignées d'un document. Par exemple, pour insérer une feuille de papier couleur vierge après chaque chapitre d'un document, notez le numéro de la page à laquelle se termine chaque chapitre. Servez-vous de ces numéros pour désigner une insertion à placer après chaque page spécifiée.

Remarque : Toutes les imprimantes ne prennent pas en charge la totalité des options mentionnées. Certaines options s'appliquent uniquement à un modèle d'imprimante, un système d'exploitation ou un type de pilote spécifique.

Pour ajouter une insertion :

  1. Dans la fenêtre Pages spéciales, cliquez sur Ajouter des insertions.
  2. Sélectionnez le type d'option d'insertion :
    • Avant la page 1 : place une insertion avant la page 1.
    • Après la ou les pages : place une insertion après les pages ou les plages de pages spécifiées.
  3. Si vous sélectionnez Après la ou les pages, dans le champ Pages(s), tapez le numéro de la page qui précède chaque insertion.
  4. Pour Quantité, sélectionnez le nombre de feuilles à insérer requis.
  5. Pour accepter les valeurs par défaut du travail pour tous les paramètres, cliquez sur OK.
  6. Pour sélectionner le format, la couleur ou le type, en regard de Support, cliquez sur les flèches et sélectionnez une option.
  7. Cliquez sur OK.
