Travaux mémorisés

Les travaux mémorisés sont des documents envoyés à l'imprimante et qui y sont conservés pour une utilisation ultérieure.

Remarque : Toutes les imprimantes ne prennent pas en charge la totalité des options mentionnées. Certaines options s'appliquent uniquement à un modèle d'imprimante, un système d'exploitation ou un type de pilote spécifique.

Pour imprimer un travail enregistré :

  1. Dans la fenêtre Support/Réception, en regard de Type de travail, cliquez sur les flèches et sélectionnez Travail enregistré.
  2. Pour sélectionner un nom pour votre travail, effectuez l'une des opérations suivantes :
    • Saisissez le nom du travail.
    • Cliquez sur la flèche à droite du champ et sélectionnez un nom dans la liste.
    • Cliquez sur la flèche à droite du champ et sélectionnez Utiliser le nom du document pour utiliser le nom du document.
  3. Sélectionnez une option. Vous pouvez enregistrer un travail ou l'imprimer avant de l'enregistrer. Une fois le travail enregistré, vous pouvez l'imprimer lorsque vous le souhaitez.
  4. En regard de Format, sélectionnez une option de format pour le travail enregistré.
    • Valeur imprimante par défaut : permet d'utiliser la valeur par défaut définie pour les formats de fichier.
    • TIFF une page : permet d'enregistrer le travail d'impression au format .tif. Chaque page du travail d'impression est enregistrée dans un fichier .tif distinct.
    • TIFF multipage : permet d'enregistrer le travail d'impression au format .tif. Toutes les pages du travail d'impression sont enregistrées dans un seul fichier .tif.
    • PDF : permet d'enregistrer le travail d'impression au format .pdf.
    • Réimpression rapide : permet d'enregistrer le travail d'impression au format FRF (Fast Reprint Format).
    • Réimpression rapide avec DRI : permet d'enregistrer le travail d'impression au format FRF (Fast Reprint Format) afin qu'il puisse être réimprimé rapidement. Le paramètre DRI (Display Resolution Image) permet d'afficher ou de modifier le travail sur le serveur d'impression.
  5. Cliquez sur OK.
  6. Cliquez sur Impression.
