Relieuse à collage

Utilisez cette option pour imprimer et relier un livre broché. Elle encolle et applique une couverture autour du corps du livre. Vous pouvez également rogner le livre au format voulu.

Remarques :

  • Cette option n'apparaît que si le module de finition approprié est installé sur l'imprimante et qu'aucune des options suivantes n'est sélectionnée : couvertures, agrafage, perforation, reliure, pliage ou format cahier.
  • Toutes les imprimantes ne prennent pas en charge la totalité des options mentionnées. Certaines options s'appliquent uniquement à un modèle d'imprimante, un système d'exploitation ou un type de pilote spécifique.

Pour utiliser la relieuse à collage :

  1. Dans la fenêtre Support/Réception, en regard de Finition, cliquez sur les flèches et sélectionnez Reliure par collage.
  2. Dans l'onglet Reliure par collage, sélectionnez une option :
    • Portrait - À gauche : règle l'orientation sur portrait et la reliure sur la gauche.
    • Portrait - À droite : règle l'orientation sur portrait et la reliure sur la droite.
    • Paysage - En haut : règle l'orientation sur paysage et la reliure en haut.
  3. Pour choisir le papier utilisé pour la couverture, sélectionnez dans la liste de couvertures les options suivantes :
    • Autre format : permet de sélectionner le format papier ou de créer un format personnalisé.
    • Autre couleur : permet de faire une sélection dans la liste des couleurs disponibles ou une liste de couleurs personnalisées définies par l'administrateur.
    • Autre type : permet de sélectionner le type de support ou de sélectionner un type personnalisé défini par l'administrateur.
    • Sélectionner par magasin  : permet de sélectionner un magasin pour vos travaux d'impression.
  4. Sélectionnez une Position de la couverture.
  5. Pour indiquer comment la couverture est imprimée, cliquez sur le menu sous Couverture et sélectionnez une option :
    • Vierge ou préimprimée : utilise du papier vierge ou préimprimé pour la couverture.
    • Imprimé : imprime la couverture sur le papier de votre choix.
  6. Cliquez sur Rognage, puis effectuez votre sélection.
    • Aucun rognage : les pages ne sont pas rognées.
    • Avant : rogne le bord opposé au bord de reliure.
    • Fond perdu : active le format de rognage et vous permet de sélectionner un format de rognage. Cette fonction permet à l'image de s'étendre jusqu'au bord du papier.
  7. Pour sélectionner un format de rognage, cliquez sur Format de rognage et effectuez une sélection dans la liste des formats ou sélectionnez Personnalisé pour un format personnalisé.
  8. Pour régler le rognage sur un bord des copies, cliquez sur Réglages rognages.
  9. Cliquez sur OK.
