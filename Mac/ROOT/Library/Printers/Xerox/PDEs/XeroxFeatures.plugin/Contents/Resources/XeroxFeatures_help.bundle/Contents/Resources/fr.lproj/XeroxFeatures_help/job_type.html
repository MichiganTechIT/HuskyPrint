Type de travail

Cette option contrôle la façon dont le travail est imprimé. Vous pouvez définir un code d'accès pour les travaux d'impression, stocker un document sur l'imprimante ou définir une heure d'impression pour le travail. Il est également possible d'envoyer un fax.

Remarque : Toutes les imprimantes ne prennent pas en charge la totalité des options mentionnées. Certaines options s'appliquent uniquement à un modèle d'imprimante, un système d'exploitation ou un type de pilote spécifique.

Impression standard

Dans le cadre d'une impression standard, le travail n'est pas enregistré sur l'imprimante pour être réimprimé ultérieurement et aucun code d'accès n'est requis. L'impression standard correspond au type de travail par défaut. Sélectionnez Impression standard pour imprimer un travail immédiatement.

Pour imprimer à l'aide de l'option Impression standard :

  1. Dans la fenêtre Support/Réception, en regard de Type de travail, cliquez sur les flèches et sélectionnez l'option Impression standard.
  2. Faites les autres sélections nécessaires pour votre travail d'impression.
  3. Cliquez sur Impression.

Jeu épreuve

Le type de travail Jeu échantillon permet d'imprimer un seul exemplaire d'un travail avant de procéder à l'impression des autres exemplaires requis. Après avoir examiné le jeu échantillon, vous pouvez soit imprimer les exemplaires restants, soit les supprimer depuis le panneau de commande de l'imprimante.

Pour imprimer à l'aide de l'option Jeu échantillon :

  1. Dans la fenêtre Support/Réception, en regard de Type de travail, cliquez sur les flèches et sélectionnez Jeu échantillon.
  2. Cliquez sur Impression. Une seule épreuve s'imprime.
  3. Si l'épreuve vous satisfait, imprimez les exemplaires restants à partir du panneau de commande de l'imprimante.
