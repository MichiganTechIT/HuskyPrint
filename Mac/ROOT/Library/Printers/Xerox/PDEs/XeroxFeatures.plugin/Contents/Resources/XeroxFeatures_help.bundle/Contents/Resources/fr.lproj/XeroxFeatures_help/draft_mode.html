Mode brouillon

Le mode brouillon permet d'imprimer des images présentant une qualité inférieure et d'économiser ainsi le toner ou l'encre.

Remarque : Le mode brouillon fait partie des paramètres intelligents.

Pour sélectionner les options du mode brouillon :

Dans la fenêtre Avancé, en regard de Image brouillon, cliquez sur les flèches et sélectionnez une option.

  • Oui ou Activé(e) : imprime des brouillons.
  • Non ou Désactivé(e) : annule le mode brouillon.
