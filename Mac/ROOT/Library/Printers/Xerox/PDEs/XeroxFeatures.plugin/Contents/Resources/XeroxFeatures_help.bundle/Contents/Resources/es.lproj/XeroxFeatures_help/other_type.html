Tipo de papel

La opción Otro tipo permite seleccionar el tipo de papel para el trabajo de impresión.

Notas:

  • No todas las opciones descritas están disponibles en todas las impresoras. Algunas opciones solo se aplican a un modelo de impresora, configuración, sistema operativo o tipo de controlador específico.
  • Si selecciona un tipo de papel diferente al cargado en la impresora, la impresora retendrá el trabajo hasta que se coloque el tipo de papel seleccionado.

Para seleccionar un tipo de papel:

  1. En la ventana Papel/Salida, en Papel, haga clic en las flechas y, a continuación, seleccione Otro tipo.
  2. Siga uno de estos pasos:
    • Para utilizar el tipo de papel prefijado en la impresora, seleccione Prefijado de impresora.
    • En la lista, seleccione un tipo de papel. Si el administrador ha definido tipos de papel personalizados para la impresora, aparecen en la lista.
    • Para seleccionar una bandeja de papel que contiene cualquier color de papel, seleccione Seleccionar automáticamente.
    • Para Separador con lengüeta/Separador con lengüeta de cartulina, seleccione las opciones.
