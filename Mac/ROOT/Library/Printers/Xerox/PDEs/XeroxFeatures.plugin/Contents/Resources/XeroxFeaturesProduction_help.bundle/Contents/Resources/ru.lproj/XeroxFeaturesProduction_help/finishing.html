Послепечатная обработка

Опции в разделе «Финишная обработка» позволяют выбрать способ вывода распечатанного документа. Например, можно выбрать перфорацию, сшивание, фальцовку или печать в формате брошюры.

Примечание: Не все перечисленные опции поддерживаются всеми принтерами. Некоторые опции предназначены только для определенных моделей и конфигураций принтера, операционных систем и драйверов.

Сшивание

  1. В окне «Бумага/Вывод» нажатием стрелок рядом с полем «Послепечатная обработка» выберите Сшивание.
  2. Выберите количество и расположение скрепок.

Перфорация

  1. В окне «Бумага/Вывод» нажатием стрелок рядом с полем «Послепечатная обработка» выберите значение Перфорирование.
  2. Выберите количество и расположение отверстий.

Фальцовка

  1. В окне «Бумага/Вывод» нажатием стрелок рядом с полем «Послепечатная обработка» выберите значение Фальцовка.
  2. Выберите один из вариантов:
    • Фальцовка отдельных листов в два сложения (Печать с внутренней стороны) — первая страница печатается на внутренней стороне сложенного листа. Печать производится на одной стороне, а каждый лист складывается отдельно.
    • Фальцовка отдельных листов в два сложения (Печать с внешней стороны) — первая страница печатается на наружной стороне сложенного листа. Печать производится на одной стороне, а каждый лист складывается отдельно.
    • Фальцовка нескольких листов в два сложения (фальцовка брошюры) — на нечетных страницах печать производится на наружной стороне сложенного листа, а на четных страницах – на внутренней стороне. Листы складываются вместе в виде брошюры.
    • C-фальцовка (Печать с внутренней стороны) — печать начинается на внутренней стороне лицевой части. Наружная сторона лицевой части сложенного документа остается пустой.
    • C-фальцовка (Печать с внешней стороны) — печать на лицевой части сложенного документа.
    • Z-фальцовка в половину листа (Переплет слева) — складывание с помощью Z-фальцовки, переплет с левой стороны.
    • Z-фальцовка в половину листа (Переплет справа) — складывание с помощью Z-фальцовки, переплет с правой стороны.
    • Z-фальцовка (Печать с внутренней стороны) — печать начинается на внутренней стороне лицевой части. Наружная сторона лицевой части сложенного документа остается пустой.
    • Z-фальцовка (Печать с внешней стороны) — печать на лицевой части сложенного документа.
  3. При выборе опции «Фальцовка нескольких листов в два сложения (фальцовка брошюры)» можно указать дополнительные настройки.
    • Прямоугольный корешок
    • Подрезание наружной кромки

Финишная обработка нескольких листов

Функция «Финишная обработка нескольких листов» служит для выбора в одном окне нескольких опций послепечатной обработки для работы.

Выбор опций финишной обработки нескольких листов:

В окне «Бумага/Вывод» нажатием стрелок рядом с полем «Послепечатная обработка» выберите значение Финишная обработка нескольких листов.

  1. Выберите нужную опцию.
    • Сшивание
      1. В окне «Бумага/Вывод» нажатием стрелок рядом с полем «Послепечатная обработка» выберите Сшивание.
      2. Выберите количество и расположение скрепок.
    • Перфорация
      1. В окне «Бумага/Вывод» нажатием стрелок рядом с полем «Послепечатная обработка» выберите значение Перфорирование.
      2. Выберите количество и расположение отверстий.
    • Фальцовка
      1. В окне «Бумага/Вывод» нажатием стрелок рядом с полем «Послепечатная обработка» выберите значение Фальцовка.
      2. Выберите один из вариантов:
        • Фальцовка отдельных листов в два сложения (Печать с внутренней стороны) — первая страница печатается на внутренней стороне сложенного листа. Печать производится на одной стороне, а каждый лист складывается отдельно.
        • Фальцовка отдельных листов в два сложения (Печать с внешней стороны) — первая страница печатается на наружной стороне сложенного листа. Печать производится на одной стороне, а каждый лист складывается отдельно.
        • Фальцовка нескольких листов в два сложения (фальцовка брошюры) — на нечетных страницах печать производится на наружной стороне сложенного листа, а на четных страницах – на внутренней стороне. Листы складываются вместе в виде брошюры.
        • C-фальцовка (Печать с внутренней стороны) — печать начинается на внутренней стороне лицевой части. Наружная сторона лицевой части сложенного документа остается пустой.
        • C-фальцовка (Печать с внешней стороны) — печать на лицевой части сложенного документа.
        • Z-фальцовка в половину листа (Переплет слева) — складывание с помощью Z-фальцовки, переплет с левой стороны.
        • Z-фальцовка в половину листа (Переплет справа) — складывание с помощью Z-фальцовки, переплет с правой стороны.
        • Z-фальцовка (Печать с внутренней стороны) — печать начинается на внутренней стороне лицевой части. Наружная сторона лицевой части сложенного документа остается пустой.
        • Z-фальцовка (Печать с внешней стороны) — печать на лицевой части сложенного документа.
      3. При выборе опции «Фальцовка нескольких листов в два сложения (фальцовка брошюры)» можно указать дополнительные настройки.
        • Прямоугольный корешок
        • Подрезание наружной кромки

      Примечание: В окне «Финишная обработка нескольких листов» для режима фальцовки доступны только опции «Нет» и «Z-фальцовка половины листа».

  2. Нажмите кнопку OK.

Настройка пользователя

Функция «Профили финишера DFA» служит для указания профиля послепечатной обработки, настроенного на принтере. После назначения имени финишеру оно отображается в драйвере принтера.

Ввод нового имени:

  1. В окне «Бумага/Вывод» нажатием стрелок рядом с полем «Послепечатная обработка» выберите значение Настройка пользователя.
  2. Введите имя финишера DFA в поле «Настройка пользователя».
  3. Нажмите кнопку OK.

Разделители прозрачных пленок

  1. В окне «Бумага/Вывод» нажатием стрелок рядом с полем «Послепечатная обработка» выберите значение Разделители прозрачных пленок.
  2. Выберите один из вариантов:
    • Вкл. или Включено — вставка разделителей.
    • Выкл. или Включено – отключение функции вставки разделителей.
  3. Нажмите кнопку OK.
