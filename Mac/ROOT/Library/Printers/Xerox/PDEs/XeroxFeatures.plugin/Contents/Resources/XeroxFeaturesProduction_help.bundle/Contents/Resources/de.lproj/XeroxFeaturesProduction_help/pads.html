Blockerstellung

Mit „Blöcke“ können Blöcke aus Formularen, die aus einem Blatt bestehen, gebildet werden, von denen die Formulare nach Bedarf abgerissen werden. Ein Block hat in der Regel in hinteres Deckblatt aus Karton und eine Klebebindung oben.

Blöcke drucken:

  1. Im Fenster "Material/Ausgabe" neben "Endverarbeitung" auf die Pfeile klicken und dann Blöcke wählen.
  2. Neben "Blöcke" eine Option wählen.
    • Aus: Deaktiviert die Funktion "Blockerstellung".
    • Ein: Aktiviert den Blockdruck.
  3. Für „Anzahl Blöcke“ einen der folgenden Schritte ausführen:
    • De Anzahl der zu druckenden Blöcke eingeben.
    • Zum Verwenden der Standardeinstellung des Druckers Druckereinstellung wählen.
  4. Für "Blätter pro Block" einen der folgenden Schritte ausführen:
    • Die Anzahl der Blätter pro Block eingeben.
    • Zum Verwenden der Standardeinstellung des Druckers Druckereinstellung wählen.
  5. Unter „Hinteres Deckblatt“ die Materialart für das hintere Deckblatt auswählen.
  6. Optionen für "Material" wählen.
    • Eigenes Material: Ermöglicht die Auswahl eines eigenen Materials aus einer Liste bzw. das Erstellen eines neuen eigenen Materials.
    • Anderes Format: Ermöglicht die Auswahl eines vorhandenen oder Erstellung eines neuen benutzerdefinierten Materialformats.
    • Andere Farbe: Ermöglicht die Auswahl aus einer Liste verfügbarer Farben bzw. das Erstellen einer benutzerdefinierten Materialfarbe.
    • Andere Art: Ermöglicht die Auswahl der Materialart bzw. das Erstellen benutzerdefinierter Materialarten.
    • Anderes Gewicht: Ermöglicht die Auswahl des Materialgewichts bzw. das Erstellen eines benutzerdefinierten Materialgewichts.
    • Erweiterte Materialoptionen: Ermöglicht eine Auswahl erweiterter Materialattribute für das Deckblatt.
  7. Auf OK klicken.
