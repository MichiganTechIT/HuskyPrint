Účtování

Funkce účtování přidruží ke každému dokumentu, který vytisknete, vaše uživatelské jméno a identifikační informace o účtu. Účtovací data jsou shromažďována a ukládána současně s tím, jak jsou na tiskárnu odesílány úlohy.

  1. Chcete-li přejít k nastavení účtování, v okně Rozšířené vedle položky Konfigurace klikněte na položku Účtování.
  2. V části Systém účtování vyberte požadovanou volbu.
    • Chcete-li nastavit účtování, klikněte na položku Zapnuto.
    • Chcete-li účtování vypnout, klikněte na položku Žádné.
  3. V části Výzva účtování vyberte požadovanou možnost.
    • Nezobrazovat výzvu: Při odesílání tiskové úlohy nezobrazí výzvu k zadání ID uživatele nebo ID účtu.
    • Vždy zobrazit výzvu: Požaduje zadání ID uživatele a ID účtu při každém odeslání tiskové úlohy.
  4. Pokud vyberete možnost Vždy zobrazit výzvu:
    1. Pokud chcete, aby bylo při tisku úlohy maskováno ID uživatele nebo ID účtu, zaškrtněte políčko vedle těchto voleb.
    2. V případě potřeby zadejte hodnotu Výchozí ID uživatele.
    3. V případě potřeby zadejte hodnotu Výchozí ID účtu nebo hodnotu kódu pro účtování.
  5. Klikněte na tlačítko OK.
