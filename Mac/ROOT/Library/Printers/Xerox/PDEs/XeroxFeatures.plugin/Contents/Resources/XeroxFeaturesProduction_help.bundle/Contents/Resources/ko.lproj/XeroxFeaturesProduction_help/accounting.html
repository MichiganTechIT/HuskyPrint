집계 관리

집계 관리는 인쇄하는 각 문서와 사용자 이름 및 계정 ID 정보를 연결시킵니다. 집계 관리 데이터는 작업이 프린터에 전송될 때 수집되고 저장됩니다.

  1. 집계 관리 설정에 액세스하려면 고급 창의 [구성] 옆에서 집계 관리를 클릭합니다.
  2. 집계 관리 시스템에 대해 다음 옵션 중 하나를 선택합니다.
    • 집계 관리를 설정하려면 사용을 클릭합니다.
    • 집계 관리를 사용하지 않으려면 없음을 클릭합니다.
  3. [집계 관리 프롬프트] 옆에 있는 다음 옵션 중 하나를 선택합니다.
    • 묻지 않음: 인쇄 작업을 전송할 때 사용자 ID 또는 계정 ID를 입력하라는 메시지를 표시하지 않습니다.
    • 항상 알림: 인쇄 작업을 전송할 때마다 사용자 ID 및 계정 ID를 입력해야 합니다.
  4. 항상 알림을 선택할 경우:
    1. 작업이 인쇄될 때 사용자 ID 또는 계정 ID를 가리려면 해당 옵션 옆에 있는 확인란을 선택합니다.
    2. 필요한 경우 기본 사용자 ID 값을 입력합니다.
    3. 필요한 경우 기본 계정 ID 또는 과금 코드 값을 입력합니다.
  5. 확인을 클릭합니다.
