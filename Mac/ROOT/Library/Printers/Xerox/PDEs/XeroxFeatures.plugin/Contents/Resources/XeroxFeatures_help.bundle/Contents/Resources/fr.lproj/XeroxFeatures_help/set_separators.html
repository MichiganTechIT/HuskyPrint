Définir les séparateurs

Cette option permet d'ajouter une feuille vierge entre les copies, les pages individuelles ou le travail d'impression même.

Remarque : Toutes les imprimantes ne prennent pas en charge la totalité des options mentionnées. Certaines options s'appliquent uniquement à un modèle d'imprimante, un système d'exploitation ou un type de pilote spécifique.

Pour imprimer à l'aide de séparateurs :

  1. Dans la fenêtre Définir les séparateurs, cliquez sur Configurer.
  2. Sous Séparateurs, sélectionnez une option.
    • Désactivé(e) ou Pas de séparateurs : aucune feuille de séparation n'est insérée.
    • Après chaque jeu : insère un séparateur après chaque copie du travail d'impression. Cette option nécessite que l'imprimante soit équipée d'un disque dur.
  3. Sous Support, sélectionnez le magasin des séparateurs. Les séparateurs doivent utiliser un magasin différent de celui qui contient les transparents.
  4. Cliquez sur OK.
