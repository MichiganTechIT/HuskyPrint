Разделители прозрачных пленок

Функция «Разделители прозрачных пленок» служит для вставки чистых листов бумаги между листами прозрачной пленки с печатью. Размер и ориентация таких разделителей должны совпадать с размером и ориентацией прозрачной пленки.

Примечания:

  • Не все перечисленные опции поддерживаются всеми принтерами. Некоторые опции предназначены только для определенных моделей и конфигураций принтера, операционных систем и драйверов.
  • Разделители прозрачных пленок нельзя указывать при использовании некоторых функций, таких как «Фальцовка» и «Брошюра».

Печать с использованием разделителей прозрачных пленок:

  1. В окне «Бумага/Вывод» нажатием стрелок рядом с полем «Бумага» выберите значение Разделители прозрачных пленок.
  2. Выберите один из вариантов в поле «Разделители прозрачных пленок».
    • Выкл. или Без разделителей — разделители не вставляются
    • Разделители без печати — между листами пленки вставляется чистый лист бумаги.
    • Разделители с печатью — между листами пленки вставляется лист бумаги с печатью. На листе печатается то же изображение, что и на пленке.
  3. Выберите один из вариантов в поле «Цвет».
  4. Нажмите кнопку OK.
