Fond de page

Les fonds de page permettent de fusionner les données du travail avec des formulaires stockés sur le disque dur de l'imprimante. Lorsque vous imprimez un travail avec un fond de page, le document et le fond de page sont fusionnés. Le document résultant semble imprimé sur un formulaire pré-imprimé.

Avant d'enregistrer un fond de page, il convient de le créer dans une autre application telle que Microsoft Word. Vous pouvez ensuite enregistrer le fond de page sur le disque dur de l'imprimante.
  1. Fond de page
  2. Travail d'impression
  3. Document imprimé résultant
  1. Pour créer un fond de page :
  1. Cliquez sur Fonds de page.
  2. Pour Nom du fond de page, effectuez l'une des opérations suivantes :
    • En regard de Nom du fond de page, pour afficher la liste des fonds de page existants, cliquez sur les flèches et sélectionnez une option.
    • Pour créer un fond de page, entrez un nom.
  3. Pour appliquer le fond de page au travail d'impression, sélectionnez une option dans Application du fond de page.
    • À tout le travail : applique les pages du fond de page au travail d'impression. Ce processus est répété jusqu'à ce que toutes les pages du travail soient imprimées.

    • Une fois : applique les pages du fond de page une fois au travail. Le reste des pages du travail d'impression est imprimé sans fond de page.

    • Une fois, puis répéter : applique les pages du fond de page une fois au travail. Ce processus est répété, à partir de la page spécifiée du fond de page jusqu'à ce que toutes les pages du travail soient imprimées.

  4. Pour décaler l'image de page le long des axes X et Y, entrez les valeurs dans Décalage image.
  5. Pour choisir entre les pouces et les millimètres, en bas de la fenêtre, cliquez sur les flèches puis sélectionnez une unité de mesure.
  6. Cliquez sur OK.
