Image miroir

L'option Image miroir inverse l'image et le texte sur la page. Le résultat est le même que si vous retourniez un transparent imprimé ; la vue correspondant à celle du verso.

Pour sélectionner les options Image miroir :

Dans la fenêtre Avancé, en regard de Image miroir, sélectionnez une option.

  • Activé(e) ou Oui inverse l'image sur la page (de gauche à droite) .
  • Désactivé(e) ou Non annule l'option image miroir.
