Indicador de contabilidad

Si se ha activado Contabilidad y el Indicador de contabilidad se ha activado para los trabajos de impresión, fax, impresión en color, el Indicador de contabilidad aparece al enviar estos tipos de trabajos.

Nota: Dependiendo de las selecciones en Sistema de contabilidad e Indicador de contabilidad, no todas las opciones aparecen.

  1. En ID de usuario, si se ha especificado el valor prefijado, puede utilizarlo o especificar un valor nuevo.
  2. En Clave, si se ha especificado el valor prefijado, puede utilizarlo o especificar una clave nueva.
  3. Si es preciso, seleccione un tipo de cuenta.
  4. En ID de cuenta, si se ha especificado el valor prefijado, puede utilizarlo o especificar una ID de cuenta nueva.
  5. En Clave de ID de cuenta, si se ha especificado el valor prefijado, puede utilizarlo o especificar una clave nueva.
