Cahier/Filigrane

Utilisez la fenêtre Cahier/Filigrane pour sélectionner les options de format cahier et de filigrane.

Remarque : Toutes les imprimantes ne prennent pas en charge la totalité des options mentionnées. Certaines options s'appliquent uniquement à un modèle d'imprimante, un système d'exploitation ou un type de pilote spécifique.

Pour imprimer en utilisant Cahier/Filigrane :

  1. Dans la fenêtre Cahier/Filigrane, sous Format cahier, cliquez sur les flèches et sélectionnez une option.
    • Activé(e) : cette option permet d'imprimer le document en tant que cahier. Vous pouvez l'utiliser dans le cas d'un document qui n'est pas au format cahier.
    • Désactivé(e) : cette option désactive le format cahier. Vous pouvez l'utiliser dans le cas d'un document au format cahier.
  2. Pour définir des paramètres image supplémentaires, activez Format cahier puis cliquez sur Options cahier.
  3. Pour imprimer une bordure autour de chaque page, sélectionnez Imprimer les bordures
  4. Sélectionnez des options pour le filigrane.
  5. Cliquez sur Impression.
