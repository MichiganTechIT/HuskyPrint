Seleção avançada do papel

A Seleção avançada de papel permite selecionar vários atributos de papel em uma única janela. O Resumo de papel exibe um resumo das seleções.

Nota: Nem todas as opções listadas são suportadas em todas as impressoras. Algumas opções são aplicáveis apenas a um modelo de impressora, configuração, sistema operacional ou tipo de driver específico.

Selecionar os atributos de papel utilizando a Seleção avançada de papel:

  1. Para acrescentar ou atualizar atributos de um papel nomeado, ao lado de Papel, selecione Papel nomeado e selecione a opção desejada.

    Nota: O driver de impressão retém 10 nomes. Ao adicionar um décimo-primeiro nome, o nome menos usado será automaticamente excluído da lista. Para remover os nomes da lista, exceto o nome especificado, selecione Limpar lista.

  2. Na janela Papel/Saída, ao lado de Papel, clique nas setas e selecione Seleção avançada de papel.
  3. Para configurar atributos individuais, limpe a caixa de seleção ao lado de Usar nome somente (Profissional).
    • Selecione um Tamanho de papel ou, para criar um tamanho de papel personalizado, clique em Personalizado.
    • Selecione uma Cor de papel ou, para criar uma cor de papel personalizada, clique em Personalizado.
    • Selecione um Tipo de papel ou, para criar um tipo de papel personalizado, clique em Personalizado.
    • Selecione uma Gramatura de papel (gsm) ou, para criar uma gramatura de papel personalizada, clique em Personalizado.
    • Para Papel perfurado, selecione uma opção.
    • Para Faces revestidas, selecione uma opção.

      O revestimento do papel afeta a aparência do trabalho impresso. Use Faces revestidas para revestir o trabalho impresso. Você também pode especificar para revestir uma ou ambas as faces do papel.

    • Para Tipo de revestimento, selecione uma opção.

      Especifique o tipo de revestimento usado em papel revestido. O tipo de revestimento pode afetar a aparência do documento impresso.

    • Para Grão, selecione uma opção.

      O grão especifica o grão do papel para o trabalho de impressão. Em alguns casos, o grão pode afetar a aparência do documento impresso.

    • Para Acabamento do material, selecione uma opção.

      O Acabamento do material permite selecionar o tipo de acabamento para o trabalho de impressão. O mesmo afeta a aparência do documento impresso.

    • Para Descrição, selecione Padrão da impressora, selecione uma descrição da lista ou digite uma descrição nova.
    • Para capturar informações opcionais sobre as seleções, para Comentário (Opcional), digite os detalhes.
  4. Clique em OK.
