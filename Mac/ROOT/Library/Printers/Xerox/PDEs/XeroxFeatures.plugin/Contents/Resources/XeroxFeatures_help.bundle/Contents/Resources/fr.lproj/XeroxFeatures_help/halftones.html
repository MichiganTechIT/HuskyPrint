Demi-teintes/Demi-teintes de l'imprimante

L'impression en demi-teinte est une méthode dans le cadre de laquelle la teinte foncée, la densité et la taille relatives de points minuscules varient pour refléter des changements de couleur et de luminosité. Le niveau de détail d'une demi-teinte est proportionnel au nombre de points par pouce ; plus les points sont nombreux, plus le niveau de détail est élevé sur les impressions. Le mot « trame » est parfois utilisé pour désigner une demi-teinte.

Remarque : Toutes les imprimantes ne prennent pas en charge la totalité des options mentionnées. Certaines options s'appliquent uniquement à un modèle d'imprimante, un système d'exploitation ou un type de pilote spécifique.

Pour sélectionner les options de demi-teintes :

Dans la fenêtre Avancé, en regard d'Utiliser les demi-teintes de l'imprimante ou de Demi-teintes, cliquez sur les flèches et sélectionnez une option.

  • Laisser le logiciel gérer les demi-teintes : permet de régler le motif de points de demi-teinte dans l'application. Utilisez ce paramètre pour simuler des conditions telles que celles rencontrées lors de l'utilisation d'une presse à faible résolution. Ce paramètre utilise les demi-teintes du pilote PostScript, les demi-teintes de l'application de l'imprimante ou les deux, lors du téléchargement de demi-teintes vers l'imprimante avec le travail d'impression.
  • Laisser l'imprimante gérer les demi-teintes : utilise les motifs de points de demi-teinte stockés dans l'imprimante.
  • Choisissez une valeur dans le menu.
