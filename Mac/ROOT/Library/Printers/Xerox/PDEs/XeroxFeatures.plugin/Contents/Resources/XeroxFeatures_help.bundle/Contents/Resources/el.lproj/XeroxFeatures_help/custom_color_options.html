Επιλογές ειδικών χρωμάτων

Η λειτουργία Επιλογές ειδικών χρωμάτων παρέχει περισσότερες επιλογές για την προσαρμογή των χρωμάτων στην εργασία εκτύπωσης.

Σημείωση: Δεν υποστηρίζονται όλες οι επιλογές που παρατίθενται από όλους τους εκτυπωτές. Ορισμένες επιλογές ισχύουν μόνο σε συγκεκριμένα μοντέλα ή διαμορφώσεις εκτυπωτών, λειτουργικά συστήματα ή τύπους οδηγών.

Χρώμα ανά περιγραφή

Η λειτουργία Χρώμα ανά περιγραφή επιτρέπει τη ρύθμιση των χρωμάτων με τη δημιουργία προτάσεων μέσω συνηθισμένων εκφράσεων περιγραφής του τρόπου αλλαγής των χρωμάτων.

Σημείωση: Η λειτουργία Ρυθμίσεις χρωμάτων ενεργοποιεί τους παραδοσιακούς ρυθμιστές για την αλλαγή χρωμάτων.

Για τη χρήση της λειτουργίας Χρώμα ανά περιγραφή:

  1. Στο παράθυρο Επιλογές ειδικών χρωμάτων, κάντε κλικ στην επιλογή Χρώμα ανά περιγραφή.
  2. Για να επιλέξετε το χρώμα που θα αλλάξει, κάντε κλικ στα βέλη δίπλα στην επιλογή Ποιο χρώμα θέλετε να αλλάξει; και μετά επιλέξτε ένα χρώμα.
  3. Για να επιλέξετε πόσο θα αλλάξει το χρώμα, κάντε κλικ στα βέλη δίπλα στην επιλογή Πόσο θέλετε να αλλάξει; και μετά ορίστε μια επιλογή.
  4. Για να επιλέξετε με ποιον τρόπο θα αλλάξει το χρώμα, κάντε κλικ στα βέλη δίπλα στην επιλογή Τι είδους αλλαγή; και μετά ορίστε μια επιλογή.
  5. Για να τροποποιήσετε μια υπάρχουσα φράση που αφορά σε ρύθμιση χρώματος, επιλέξτε τη φράση και μετά προβείτε στον νέων φράσεων από τα μενού.
  6. Για τη δημιουργία μιας άλλης φράσης που αφορά σε ρύθμιση χρώματος, κάντε κλικ στο στοιχείο Δημιουργία νέας πρότασης.
  7. Για την αλλαγή της σειράς των φράσεων που αφορούν σε ρύθμιση χρώματος, επιλέξτε τις και μετά κάντε κλικ στα βέλη.
  8. Για να διαγράψετε μια φράση που αφορά σε ρύθμιση χρώματος, επιλέξτε τη φράση και μετά κάντε κλικ στο πλήκτρο X.

    Σημείωση: Η σειρά των προτάσεων καθορίζει την εφαρμογή της κάθε ρύθμισης χρώματος.

  9. Για την προβολή ενός δείγματος ειδώλου με τις τρέχουσες αλλαγές χρωμάτων, επιλέξτε Προεπισκόπηση. Για την προβολή του αρχικού ειδώλου, ακυρώστε την επιλογή στο πλαίσιο ελέγχου Προεπισκόπηση. Για να αλλάξετε το δείγμα ειδώλου, κάτω από το τρέχον είδωλο, χρησιμοποιήστε τα βέλη και μετά ορίστε μια επιλογή.

Διόρθωση χρωμάτων

Η λειτουργία Διόρθωση χρωμάτων παρέχει προεπιλεγμένες λειτουργίες διόρθωσης χρωμάτων που μπορείτε να εφαρμόσετε στις εργασίες εκτύπωσης. Η λειτουργία αυτή αντιστοιχίζει τα εκτυπωμένα χρώματα με τα χρώματα άλλων συσκευών, όπως η οθόνη του υπολογιστή σας ή ένα τυπογραφικό πιεστήριο.

Για τη χρήση της λειτουργίας Διόρθωση χρωμάτων:

  1. Στο παράθυρο Διόρθωση χρωμάτων, κάντε κλικ στα βέλη και μετά ορίστε μια επιλογή διόρθωσης χρωμάτων.
  2. Για τη δημιουργία μιας μιας επιλογής διόρθωσης εξατομικευμένου χρώματος, επιλέξτε Εξατομικευμένο αυτόματο χρώμα.
  3. Κάντε κλικ στο πλήκτρο OK.

Ρυθμίσεις χρωμάτων

Χρησιμοποιήστε τη λειτουργία Ρυθμίσεις χρωμάτων για τη ρύθμιση συγκεκριμένων χαρακτηριστικών του εκτυπωμένου χρώματος. Στα χαρακτηριστικά περιλαμβάνονται η φωτεινότητα, η αντίθεση, ο κορεσμός και το κυρίαρχο χρώμα.

Για να κάνετε ρυθμίσεις χρωμάτων:

  1. Για τον καθορισμό των τιμών Φωτεινότητα, Κορεσμός και Κυρίαρχο χρώμα, χρησιμοποιήστε τα ρυθμιστικά για να ορίσετε επιλογές.
  2. Για την προβολή ενός δείγματος ειδώλου με τις τρέχουσες αλλαγές χρωμάτων, επιλέξτε Προεπισκόπηση.
  3. Για την αλλαγή του δείγματος ειδώλου, κάντε κλικ στα βέλη.
  4. Κάντε κλικ στο πλήκτρο OK.
