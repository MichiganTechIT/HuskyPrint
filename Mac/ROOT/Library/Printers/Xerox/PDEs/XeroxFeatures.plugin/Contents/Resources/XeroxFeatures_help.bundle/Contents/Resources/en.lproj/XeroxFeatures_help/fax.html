Faxing

Use the Fax job type to send a fax and manage fax phonebooks and settings.

To access the fax menus, next to Job Type, select Fax.

  • To add or edit Fax Recipients, select Recipients.
  • To add or edit Cover Sheets, select Cover Sheet.
  • To set fax options, select Options.
  • To duplicate a Fax Recipient
