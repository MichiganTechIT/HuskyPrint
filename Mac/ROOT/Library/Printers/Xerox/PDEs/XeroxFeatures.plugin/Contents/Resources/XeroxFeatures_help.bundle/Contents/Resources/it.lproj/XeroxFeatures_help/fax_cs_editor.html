Editor immagini foglio intestazione

È possibile aggiungere testo o immagini ai fogli intestazione o utilizzare testo o immagini salvate nella stampante.

  1. Per aggiungere o modificare il testo immagini foglio intestazione.
    1. Nella finestra Editor immagini foglio intestazione, selezionare Testo.
    2. Per formattare il testo, fare clic su Mostra font dall'elenco, quindi selezionare le opzioni.
    3. Per specificare l'inclinazione del testo, in Inclinazione specificare un valore. Per regolare il valore, utilizzare le frecce. Per selezionare un valore, utilizzare un cursore.
    4. Per sfalsare la filigrana rispetto al centro della pagina, selezionare innanzitutto l'unità di misura, quindi immettere i numeri per le posizioni orizzontale e verticale.
    5. Per centrare l'immagine sulla pagina, fare clic su Centrale.
    6. Fare clic su OK.
  2. Per aggiungere o modificare le immagini foglio intestazione.
    1. Nella finestra Editor immagini foglio di intestazione, selezionare Immagine.
    2. Accanto a Scegli file, digitare il percorso al file dell'immagine o individuare il file e fare clic su Scegli file.

      Nota: Questa opzione supporta i file .bmp, .jpg e .gif.

    3. Per ridimensionare l'immagine, in Proporzioni immettere o selezionare una percentuale oppure utilizzare il cursore per regolare le proporzioni.
    4. Per sfalsare la filigrana rispetto al centro della pagina, selezionare innanzitutto l'unità di misura, quindi immettere i numeri per le posizioni orizzontale e verticale.
    5. Per centrare l'immagine sulla pagina, fare clic su Centrale.
    6. Fare clic su OK.
