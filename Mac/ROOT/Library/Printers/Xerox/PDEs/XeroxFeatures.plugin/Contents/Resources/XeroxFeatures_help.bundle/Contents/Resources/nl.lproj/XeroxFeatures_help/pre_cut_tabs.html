Voorgesneden tabbladen

  1. Typ bij Aantal in reeks het aantal tabbladen. Aantal in reeks definieert het aantal unieke vellen binnen een geordende set voordat de set zich herhaalt. Als een set voorgesneden tabbladen bijvoorbeeld vijf verschillende tabplaatsingen per set heeft, typt u 5 in het veld Aantal in reeks.
  2. Selecteer een optie bij Verschuiven op tabblad. Als u Aangepast selecteert, voert u de hoeveelheid verschuiving op het tabblad in en selecteert u vervolgens een optie bij Eenheden.
  3. Klik op OK.
