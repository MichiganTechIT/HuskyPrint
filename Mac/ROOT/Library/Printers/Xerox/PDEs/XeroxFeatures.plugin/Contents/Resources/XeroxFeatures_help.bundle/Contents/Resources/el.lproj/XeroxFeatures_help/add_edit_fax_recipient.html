Προσθήκη ή Επεξεργασία παραλήπτη φαξ

Αφού δημιουργήσετε εγγραφές παραληπτών φαξ, μπορείτε να προσθέσετε τις εγγραφές στον προσωπικό σας τηλεφωνικό κατάλογο.

Για να προσθέσετε εγγραφές παραληπτών:

  1. Στο παράθυρο Χαρτί/Έξοδος, δίπλα στην επιλογή Είδος εργασίας, κάντε κλικ στα βέλη και μετά επιλέξτε Φαξ.
  2. Κάντε κλικ στο Παραλήπτες.
  3. Κάντε κλικ στο πλήκτρο Προσθήκη παραλήπτη.
  4. Πληκτρολογήστε τα στοιχεία επικοινωνίας.
  5. Κάντε κλικ στο πλήκτρο OK.

Για να επεξεργαστείτε έναν παραλήπτη φαξ χωρίς επεξεργασία της εγγραφής τηλεφωνικού καταλόγου:

  1. Στο παράθυρο Χαρτί/Έξοδος, δίπλα στην επιλογή Είδος εργασίας, κάντε κλικ στα βέλη και μετά επιλέξτε Φαξ.
  2. Κάντε κλικ στο Παραλήπτες.
  3. Επιλέξτε τον παραλήπτη από τον πίνακα και κάντε κλικ στο Επεξεργασία.
  4. Αλλάξτε τα στοιχεία, όπως απαιτείται.
  5. Κάντε κλικ στο πλήκτρο OK.
