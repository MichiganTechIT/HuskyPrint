Ordre des images cahier

Utilisez les options avancées de format cahier pour contrôler l'ordre des pages des cahiers imprimés. Utilisez l'option Ordre des images cahier pour indiquer si le cahier doit être plié sur le côté gauche (De gauche à droite) ou sur le côté droit (De droite à gauche).

Remarque : Cette fonction est disponible uniquement lorsque l'option Format cahier est activée.

Pour sélectionner les options Ordre des images cahier :

Dans la fenêtre Avancé, en regard de Ordre des images cahier, sélectionnez une option.

  • Gauche à droite : Cette option règle le format cahier de sorte que le pli se trouve sur le côté gauche du cahier lorsque celui-ci est plié. De gauche à droite désigne le sens de lecture du texte.
  • De droite à gauche : définit le format cahier de sorte que le pli se trouve sur le côté droit une fois le cahier plié. Sélectionnez cette option pour les langues qui se lisent de droite à gauche, notamment l'arabe ou le farsi.
