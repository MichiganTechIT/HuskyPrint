Redovisning

Med Redovisning kopplas ditt användarnamn och din kontoinformation till varje dokument som du skriver ut. Redovisningsdata samlas in och lagras när jobben skickas till skrivaren.

  1. Du hämtar Redovisningsinställningar genom att klicka på Redovisning bredvid Konfiguration i fönstret Avancerat.
  2. Välj ett alternativ för Redovisningssystem.
    • Du konfigurerar Redovisning genom att klicka på Har aktiverats.
    • Du stänger av redovisning genom att klicka på Ingen.
  3. Välj ett alternativ bredvid Fråga vid redovisning.
    • Fråga inte: Du uppmanas inte att ange användar- eller konto-ID när du skickar utskriften.
    • Fråga alltid: Kräver att du anger ditt användar-ID och konto-ID varje gång du skickar ett utskriftsjobb.
  4. Om du väljer Fråga alltid:
    1. Om du vill dölja användar-ID eller konto-ID vid utskrift markerar du kryssrutan bredvid dessa alternativ.
    2. Ange ett Grundinställt användar-ID vid behov.
    3. Ange Grundinställt redovisnings-ID eller Grundinställd faktureringskod vid behov.
  5. Klicka på OK.
