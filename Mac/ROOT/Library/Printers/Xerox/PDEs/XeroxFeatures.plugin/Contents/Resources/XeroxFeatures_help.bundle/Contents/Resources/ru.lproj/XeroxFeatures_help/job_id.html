Идентификация работы

Когда включена функция «Идентиф. работы», сведения о работе печатаются на титульном листе или на полях документа. Выбранные здесь настройки действуют для всех работ печати, пока не будут изменены настройками рационального режима.

Титульный лист представляет собой страницу-разделитель, которая печатается вместе с работой. На титульном листе указывается сетевое имя пользователя, отправившего работу, и имя документа. Если используются титульные листы, они разделяют полные комплекты напечатанных работ в выходном лотке. На некоторых принтерах с панели управления можно изменить опцию титульного листа, выбранную в настройках драйвера принтера.

Примечания:

  • Возможность отключения печати титульных листов и включения печати сведений для идентификации документов на полях страниц входит в настройки рационального режима.
  • Не все перечисленные опции поддерживаются всеми принтерами. Некоторые опции предназначены только для определенных моделей и конфигураций принтера, операционных систем и драйверов.

Выбор опций идентификации работы:

В окне «Дополнительно» выберите значение нажатием стрелок рядом с полем «Идентификация работы».

  • Отключить ID работы — идентификационная информация для работы не печатается.
  • Печать ID на титульном листе — идентификационная информация для работы печатается на отдельном титульном листе.
  • Печать ID на полях - только на первой стр. — идентификационная информация для работы печатается на первой странице документа, в верхней части левого поля.
  • Печать ID на полях - на всех стр. – идентификационная информация для работы печатается на всех страницах документа, в верхней части левого поля.
