Тип бумаги

Настройка «Другой тип» служит для выбора типа бумаги для работы печати.

Примечания:

  • Не все перечисленные опции поддерживаются всеми принтерами. Некоторые опции предназначены только для определенных моделей и конфигураций принтера, операционных систем и драйверов.
  • Если выбирается бумага, не загруженная в принтер, работа задерживается в принтере до тех пор, пока не будет загружена требуемая бумага.

Выбор типа бумаги:

  1. В окне «Бумага/Вывод» нажатием стрелок рядом с полем «Бумага» выберите значение Другой тип.
  2. Выполните одно из следующих действий.
    • Для использования в принтере типа бумаги по умолчанию выберите значение Настройка по умолч.
    • Для выбора лотка с бумагой любого цвета выберите опцию Автовыбор.
    • Выберите значение в поле Тип бумаги или нажмите кнопку Настройка пользователя и создайте настраиваемый тип бумаги.
    • В поле Бумага с выступами укажите количество выступов в последовательности.
    • При выборе значения В комплекте укажите количество листов в последовательности.
