Indicador de contabilidad

Si se ha activado Contabilidad y el Indicador de contabilidad se ha activado para Indicar siempre, el Indicador de contabilidad aparece al enviar un trabajo de impresión.

  1. En ID de usuario, si se ha especificado el valor prefijado, puede utilizarlo o especificar un valor nuevo.
  2. En ID de cuenta, si se ha especificado el valor prefijado, puede utilizarlo o especificar una ID de cuenta nueva.
  3. Haga clic en Aceptar.
