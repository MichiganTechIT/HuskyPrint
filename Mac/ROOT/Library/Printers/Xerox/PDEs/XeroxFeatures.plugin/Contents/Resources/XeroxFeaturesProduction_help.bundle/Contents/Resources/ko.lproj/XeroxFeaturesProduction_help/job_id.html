작업 ID

작업 ID는 문서의 여백에 작업 세부 정보를 인쇄합니다.

참고: 일부 프린터에서는 여기 나열된 모든 옵션이 지원되지 않습니다. 일부 옵션은 특정 프린터 모델, 구성, 운영 체제 또는 드라이버 유형에만 적용됩니다.

작업 ID 옵션 선택:

고급 창에서 [작업 ID] 옆에 있는 화살표를 클릭한 후 다음 옵션 중 하나를 선택합니다.

  • 작업 ID 사용 안 함: 작업 ID 정보가 인쇄되지 않습니다.
  • 여백에 ID 인쇄 - 첫 페이지만: 문서의 첫 번째 페이지 상단 왼쪽 여백에 작업 ID를 인쇄합니다.
  • 여백에 ID 인쇄 - 모든 페이지: 문서의 모든 페이지 상단 왼쪽 여백에 작업 ID를 인쇄합니다.
  • 선행 페이지 사용: 작업 ID를 포함하는 페이지 수를 입력할 수 있습니다. 페이지 수는 첫 번째 인쇄 페이지부터 시작합니다.
