Περιεχόμενα

Δημιουργία ειδικού μεγέθους χαρτιού

  1. Στο παράθυρο Ειδικό μέγεθος, δίπλα στην επιλογή Πλάτος, εισαγάγετε μια τιμή για το ειδικό πλάτος ή χρησιμοποιήστε τα βέλη για να επιλέξετε μια τιμή.
  2. Δίπλα στην επιλογή Ύψος, εισαγάγετε μια τιμή για το ειδικό ύψος ή χρησιμοποιήστε τα βέλη για να επιλέξετε μια τιμή.
  3. Για τον καθορισμό των μονάδων μέτρησης, στο πεδίο μονάδων, κάντε κλικ στα βέλη και, στη συνέχεια, επιλέξτε ίντσες ή χιλιοστά.
  4. Κάντε κλικ στο πλήκτρο OK.
