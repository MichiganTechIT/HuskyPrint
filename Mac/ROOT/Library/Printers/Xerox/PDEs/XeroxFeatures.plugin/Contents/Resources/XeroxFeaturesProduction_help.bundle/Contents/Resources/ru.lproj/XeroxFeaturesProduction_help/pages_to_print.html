Страниц для печати

Настройка «Страниц для печати» служит для указания диапазона страниц в работе печати. Она позволяет указать специальные страницы в документе.

Выбор диапазона страниц:

  1. В окне «Дополнительно» рядом с полем «Страниц для печати» нажмите кнопку Настройка.
  2. В поле «От» введите номер первой страницы диапазона для печати.
  3. В поле «Кому» введите номер последней страницы диапазона для печати.
  4. Нажмите кнопку OK.
