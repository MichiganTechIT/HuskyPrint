Печать с

Функция «Печать» служит для печати сохраненных личных и общих работ вместе с текущей работой. Например, можно печатать титульный лист вместе с комплектом сохраненных документов, таких как брошюры или прайс-листы.

Примечание: Данный тип работы доступен только на принтерах, подключенных к компьютерной сети с включенной двунаправленной связью.

Печать с помощью функции «Печать с»:

  1. В окне «Бумага/Вывод» нажатием стрелок рядом с полем «Тип работы» выберите значение Печать с.
  2. По запросу введите пароль, назначенный сохраненным работам на принтере, и нажмите кнопку OK.

    Если пароль не назначен, нажмите кнопку Отменить.

  3. Выберите один из вариантов в списке «Сохраненные работы».
    • Личные и общие сохраненные работы
    • Общие сохраненные работы

      В разделе «Общие сохраненные работы» отображаются работы, сохраненные на жестком диске принтера. Общие работы доступны для выбора всем пользователям.

    • Личные сохраненные работы

      В разделе «Личные сохраненные работы» отображаются работы, сохраненные для конкретного имени пользователя с указанием пароля и доступные для печати только данному пользователю. Нажмите кнопку Пароль для ввода пароля, указанного при сохранении личных работ на жестком диске принтера.

      Примечание: Для личных сохраненных работ требуется пароль.

  4. При выборе опции «Личные и общие сохраненные задания» или «Личные сохраненные работы» требуется вводить пароль для работ. Если ваши работы не отображаются, нажмите кнопку Пароль и введите правильный пароль для работ.
  5. Для добавления работы в списке «Готовая работа» выберите ее и нажмите кнопку со стрелкой «вниз».
  6. Чтобы изменить порядок работ в списке «Готовая работа», выберите одну из работ и нажмите соответствующую кнопку со стрелкой.
  7. Чтобы удалить работу из списка, выберите ее и нажмите кнопку Удалить.
  8. Нажмите кнопку OK.
  9. Нажмите кнопку Печать.
