Sostituzione OPI

Sostituzione OPI consente di sostituire le immagini bozza a bassa risoluzione o i segnaposto di immagini con immagini ad alta risoluzione quando si stampa il documento. Questa sostituzione consente di risparmiare spazio di memorizzazione nel server di stampa e risorse di rete.

Per selezionare le opzioni di Sostituzione OPI:

Nella finestra Avanzate, accanto a Sostituzione OPI, fare clic sulle frecce, quindi selezionare un'opzione.

  • Default stampante: utilizza le impostazioni stampante correnti.
  • Abilitato: indica al server di stampa di elaborare i commenti OPI e di sostituire le immagini a bassa risoluzione con immagini ad alta risoluzione nel documento di stampa.
  • Disabilitato: consente di disattivare la funzione Sostituzione OPI.
