Richiesta contabilità

Se la contabilità è abilitata e Richiesta contabilità è impostata per richiedere lavori di stampa, fax, a colori quando si invia uno di questi lavori, viene visualizzato Richiesta contabilità.

Nota: In base alle selezioni Sistema di contabilità e Richiesta contabilità, non tutte le opzioni vengono selezionate.

  1. Per ID utente, se si specifica un valore predefinito, è possibile utilizzarlo o digitare un nuovo valore.
  2. Per Codice, se si specifica un valore predefinito, è possibile utilizzarlo o digitare un nuovo codice di accesso.
  3. Se necessario, selezionare un tipo di conto.
  4. Per ID conto, se si specifica un valore predefinito, è possibile utilizzarlo o digitare un nuovo ID conto.
  5. Per Codice ID conto, se si specifica un valore predefinito, è possibile utilizzarlo o digitare un nuovo codice di accesso.
