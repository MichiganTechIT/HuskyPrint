Сообщение системы учета

Если включен учет и для параметра «Сообщение системы учета» установлена выдача сообщения для работ печати, факса и цветной печати, при отправке работы выводится сообщение системы учета.

Примечание: В зависимости от системы учета и настройки запроса системы учета некоторые опции могут не отображаться.

  1. Если в поле «ID пользователя» указано значение по умолчанию, можно использовать его или ввести другое значение.
  2. Если в поле «Пароль» указано значение по умолчанию, можно использовать его или ввести другое значение.
  3. При необходимости выберите тип учета.
  4. Если в поле «ID счета» указано значение по умолчанию, можно использовать его или ввести другое значение.
  5. Если в поле «ID / пароль оплаты» указано значение по умолчанию, можно использовать его или ввести другое значение.
