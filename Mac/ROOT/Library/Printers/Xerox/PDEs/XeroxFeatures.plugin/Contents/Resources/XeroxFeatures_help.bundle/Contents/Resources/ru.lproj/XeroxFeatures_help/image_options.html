Параметры изображения

Настройка «Параметры изображения» позволяет делать светлее или темнее весь текст и изображения в печатаемых работах.

Примечание: Не все перечисленные опции поддерживаются всеми принтерами. Некоторые опции предназначены только для определенных моделей и конфигураций принтера, операционных систем и драйверов.

Установка параметров изображения для работы печати:

  1. Настройка яркости: если цвета в работе печати нужно сделать светлее, передвиньте ползунок вправо, если темнее — влево.
  2. Для уменьшения контрастности передвиньте ползунок вправо, для увеличения — влево.
  3. Для выбора опций сдвига изображения выберите Сдвиг изображения.
  4. Нажмите кнопку Печать.
