Opções de saída

Entrega da saída controla como os documentos são entregues à bandeja de saída da impressora. Durante a impressão normal, a impressora envia os documentos à bandeja de saída com a face para baixo. Entretanto, alguns tamanhos e tipos de papel não podem ser processados normalmente e devem ser enviados à bandeja de saída com a face para cima. Consequentemente, algumas opções de acabamento e de impressão em 2 faces não estão disponíveis para estes tipos de papel.

Para selecionar as Opções de saída:

Na janela Avançado, ao lado de Opções de saída, clique nas setas e selecione uma opção.

  • Face para baixo (Recomendado) entrega as páginas com a face para baixo em ordem decrescente (3, 2, 1).
  • Face para cima entrega as páginas com a face para cima em ordem crescente (1, 2, 3).
  • Face para baixo e Ordem inversa entrega as páginas com a face para baixo em ordem decrescente (3, 2, 1).
    • Face para cima e Ordem inversa entrega as páginas com a face para cima em ordem decrescente (3, 2, 1).
    • Face para baixo e Ordem inversa entrega as páginas com a face para baixo em ordem decrescente (3, 2, 1).
