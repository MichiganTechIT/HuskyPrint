Сглаживание

Функция «Сглаживание» служит для сглаживания неровных краев текста и графики.

Выбор опции сглаживания:

В окне «Параметры изображения» выберите один из вариантов в поле «Сглаживание».

  • Настройка по умолч. — используются действующие настройки принтера.
  • Вкл. — включение функции сглаживания.
  • Не горит — отключение функции сглаживания.
  • K100% высокое разрешение — создание яркого, чистого черного цвета. Данную функцию можно использовать для мелкого текста или графики со сплошными участками черного цвета.
