Opcions de color personalitzat

Les Opcions de color personalitzat proporcionen més opcions per personalitzar els colors del treball d'impressió.

Nota: No totes les opcions indicades són compatibles amb totes les impressores. Algunes opcions només són compatibles amb determinats models d'impressores, configuracions, sistemes operatius o tipus de controladors.

Color per paraules

Color per paraules permet ajustar els colors mitjançant frases que descriuen com canviar els colors.

Nota: La funció Ajusts de color activa controls lliscants d'ajust tradicionals per canviar els colors.

Per utilitzar Color per paraules:

  1. A la finestra Opcions de color personalitzat, feu clic a Color per paraules.
  2. Per seleccionar el color que cal canviar, al costat de Quins colors voleu canviar?, feu clic a les fletxes i seleccioneu un color.
  3. Per seleccionar el grau de canvi del color, al costat de Quant canvi?, feu clic a les fletxes i seleccioneu una opció.
  4. Per seleccionar la manera com canvia el color, al costat de Quin tipus de canvi?, feu clic a les fletxes i seleccioneu una opció.
  5. Per modificar una frase d'ajust de color existent, seleccioneu-la i trieu noves frases als menús.
  6. Per crear una altra frase d'ajust de color, feu clic a Genera una nova frase.
  7. Per canviar l'ordre de les frases d'ajust de color, seleccioneu-les i utilitzeu les fletxes.
  8. Per eliminar una frase d'ajust de color, seleccioneu-la i feu clic al botó X.

    Nota: L'ordre de la frase determina l'aplicació de cada ajust de color.

  9. Per veure una imatge de mostra amb els canvis de color actuals, seleccioneu Visualització prèvia. Per veure la imatge original, desactiveu la casella Visualització prèvia. Per canviar la imatge de mostra, utilitzeu les fletxes de sota de la imatge actual i seleccioneu una opció.

Correcció del color

Correcció del color proporciona modes de correcció de color predefinits que podeu aplicar als treballs d'impressió. Correcció del color ajusta els colors impresos als colors d'altres dispositius, com ara pantalles d'ordinador o impremtes.

Per utilitzar Correcció del color:

  1. A la finestra Correcció de color, feu clic a les fletxes i seleccioneu una opció de correcció de color.
  2. Per crear una opció de correcció de color personalitzada, seleccioneu Color automàtic personalitzat.
  3. Feu clic a D'acord.

Ajusts de color

Utilitzeu Ajusts de color per ajustar determinades característiques del color imprès. Les característiques inclouen la claror, el contrast, la saturació i la variació de color.

Per realitzar ajusts de color:

  1. Per especificar la claror, el contrast, la saturació i la variació de color, utilitzeu els controls lliscants per seleccionar opcions.
  2. Per veure una imatge de mostra amb els canvis de color actuals, seleccioneu Visualització prèvia.
  3. Per canviar la imatge de mostra, feu clic a les fletxes.
  4. Feu clic a D'acord.
