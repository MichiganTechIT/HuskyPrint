Broschürenoptionen

  1. Eine Option für die Broschürenendverarbeitung wählen.
  2. Zum Skalieren der Broschüre auf ein neues Format neben "In neues Format einpassen" auf die Pfeile klicken und dann die gewünschte Größe auswählen. Zum Erstellen eines benutzerdefinierten Formats Benutzerdefiniert wählen.
  3. Zum Festlegen der Randgrößen neben "Broschürenränder" auf die Pfeile klicken und dann eine Option wählen.
    • Standard bewirkt, dass die Seite so verkleinert wird, dass das Dokument ohne Bildverlust als Broschüre gedruckt wird.
    • Aus: Dokument wird ohne Verkleinerung der Seiten als Broschüre gedruckt. Auf Druckern, auf denen randloser Druck nicht möglich ist, kann es am Rand zu Bildverlusten kommen.

  4. Bundsteg und Falzausgleich wie gewünscht festlegen.
    • Bundsteg: Legt die Breite des Rands zwischen den Druckseiten in Punkt fest. Einen Wert in Punkt eingeben oder mithilfe der Pfeile auswählen. Ein Punkt entspricht 0,35 mm bzw. 1/72 Zoll
    • Falzausgleich: Legt in 1/10 Punkt fest, wie weit die Druckbilder je Seite nach außen verschoben werden sollen. Zum Festlegen, um wie viel die Seitenbilder verschoben werden sollen, einen Wert eingeben oder mithilfe der Pfeile auswählen.
  5. Auf OK klicken.
