Seitenaufdruck

Verfügt der Drucker über eine Duplexdruckfunktion, können die Druckeinstellungen so gewählt werden, dass Aufträge automatisch auf beide Blattseiten gedruckt werden.

Hinweise:

  • Nicht alle aufgelisteten Optionen werden von allen Druckern unterstützt. Einige Optionen stehen nur für bestimmte Druckermodelle, Konfigurationen, Betriebssysteme oder Treibertypen zur Verfügung.
  • Bei einigen Materialarten, -formaten und -gewichten ist keine automatische Duplexausgabe möglich. Weitere Informationen siehe Dokumentation zum Drucker.

Beidseitig drucken:

Im Fenster "Material/Ausgabe" neben "Seitenaufdruck" auf die Pfeile klicken und dann eine Option wählen.

  • 1-seitig: Das Druckmaterial wird nur auf einer Seite bedruckt. Diese Option beim Bedrucken von Klarsichtfolien, Umschlägen, Etiketten und anderen Druckmaterialien wählen, die nicht beidseitig bedruckt werden können.
  • 2-seitig: Die Blätter werden so beidseitig bedruckt, dass sie an der langen Kante gebunden werden können.
  • 2-seitig, wenden Schmalseite: Die Blätter werden so beidseitig bedruckt, dass sie an der kurzen Kante gebunden werden können.
