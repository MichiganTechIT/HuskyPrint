Opções de cor personalizada

Opções de cor personalizada fornece mais opções de personalização de cores para o trabalho de impressão.

Nota: Nem todas as opções listadas são suportadas em todas as impressoras. Algumas opções são aplicáveis apenas a um modelo de impressora, configuração, sistema operacional ou tipo de driver específico.

Cores por textos

Cores por textos permite a você ajustar as cores criando sentenças usando frases comuns para descrever como alterar as cores.

Nota: O recurso Ajustes de cores permite que controles deslizantes de ajuste tradicionais modifiquem cores.

Para usar a opção Cores por textos:

  1. Na janela Opções de cor personalizada, clique em Cores por textos.
  2. Para selecionar a cor a modificar, clique nas setas ao lado de Qual cor a alterar? e selecione uma cor.
  3. Para selecionar a quantia de alteração para a cor, clique nas setas ao lado de Quanto alterar? e selecione uma opção.
  4. Para selecionar como alterar a cor, clique nas setas ao lado de Que tipo de alteração? e selecione uma opção.
  5. Para modificar uma frase de ajuste de cores existente, selecione a frase e então selecione novas frases a partir dos menus.
  6. Para criar outra frase de ajuste de cor, clique em Criar nova sentença.
  7. Para alterar a ordem das frases de ajuste de cores, selecione-as e utilize as setas.
  8. Para excluir uma frase de ajuste de cores, selecione a frase e clique no botão X.

    Nota: A ordem da sentença determina a aplicação de cada ajuste de cor.

  9. Para exibir uma imagem de amostra com as alterações de cor atuais, selecione Visualizar. Para exibir a imagem original, limpe a caixa de seleção Visualizar. Para alterar a imagem de amostra, utilize as setas sob a imagem atual e selecione uma opção.

Correção de Cor

Correção de Cor fornece os modos de correção da cor predefinidos, que você pode aplicar aos trabalhos de impressão. Correção de Cor faz a correspondência das cores impressas com as cores em outros dispositivos, como uma tela do seu computador ou em uma impressora gráfica.

Para usar a Correção de cor:

  1. Na janela Correção de cores, clique nas setas e selecione uma opção de correção de cores.
  2. Para criar uma opção de correção de cores personalizadas, selecione Cor personalizada automática.
  3. Clique em OK.

Ajustes de Cor

Use Ajustes de Cor para ajustar características específicas da cor impressa. As características incluem claridade, contraste, saturação e variação de cor.

Para efetuar ajustes de cores:

  1. Para especificar clareza, contraste, saturação e variação de cor, utilize os controles deslizantes para selecionar opções.
  2. Para exibir uma imagem de amostra com as alterações de cor atuais, selecione Visualizar.
  3. Para alterar a imagem de amostra, clique nas setas.
  4. Clique em OK.
