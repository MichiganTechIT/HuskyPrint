Обложки

Обложки можно использовать для каждой копии работы. Можно использовать только передние обложки или передние и задние. Обложки можно печатать на бумаге другого цвета или типа по сравнению с остальной частью работы.

Печать с использованием обложек:

  1. В окне «Бумага/Вывод» нажатием стрелок рядом с полем «Бумага» выберите значение Обложки.
  2. В разделе «Опции обложек» выберите один из вариантов.
    • Без обложек – отмена всех предыдущих настроек для обложек.
    • Только передняя — вставка обложки перед первой страницей работы печати.
    • Только задняя — вставка обложки после последней страницы работы печати.
    • Передняя и задняя: одинаковые - печать первой и последней обложек на бумаге из одного лотка.
    • Передняя и задняя: разные — передняя и задняя обложки печатаются на бумаге из разных лотков.
  3. В поле «Источник» выберите лоток с бумагой для обложек.
  4. Нажмите кнопку OK.
