Enviar fax

Use o tipo de trabalho Fax para enviar um fax e gerenciar catálogos de endereços de fax e configurações.

Para acessar os menus de fax, selecione Fax ao lado de Tipo de trabalho.

  • Para adicionar ou editar Destinatários de fax, selecione Destinatários.
  • Para adicionar ou editar Folhas de rosto, selecione Folha de rosto.
  • Para configurar opções de fax, selecione Opções.
  • Para duplicar um destinatário de fax
