Fråga vid redovisning

Om Redovisning har aktiverats och Fråga vid redovisning är inställt på Fråga vid utskrifter, fax, färgutskrifter visas Fråga vid redovisning när du skickar ett sådant jobb.

Obs! Beroende på dina inställningar för Redovisningssystem och Fråga vid redovisning kanske inte alla alternativ visas.

  1. Om ett grundinställt värde har angetts för användar-ID kan du använda det. Skriv annars ett nytt värde.
  2. Om ett grundinställt värde har angetts för lösenord kan du använda det. Skriv annars ett nytt värde.
  3. Välj kontotyp vid behov.
  4. Om ett grundinställt värde har angetts för konto-ID kan du använda det. Skriv annars ett nytt värde.
  5. Om ett grundinställt värde har angetts för lösenord för konto-ID kan du använda det. Skriv annars ett nytt värde.
