Vorsortiertes Material

  1. Im Fenster "Vorsortiertes Material" neben "Materialart" auf den Pfeil klicken und dann eine Option wählen.
    • Einen neuen Namen für die Materialart eingeben.
    • Aus der Liste eine Materialart auswählen.
  2. Neben "Anzahl im Satz" einen Wert eingeben oder die Pfeile verwenden, um einen Wert auszuwählen.
  3. Auf OK klicken.
