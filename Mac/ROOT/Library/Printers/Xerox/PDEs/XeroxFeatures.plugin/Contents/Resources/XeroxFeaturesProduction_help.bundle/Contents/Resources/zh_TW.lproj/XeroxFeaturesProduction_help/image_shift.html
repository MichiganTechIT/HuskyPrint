圖像位移

「圖像位移」可讓您修改頁面上的圖像位置。 您可以沿著每頁的 X 和 Y 軸移動列印的圖像。

如要設定圖像位移選項：

  1. 在「圖像位移選項」旁，使用箭頭，然後選擇一個選項。
  2. 如果您在「輸出紙張」視窗中選擇「單面列印」，請針對「圖像位移選項」選擇一個選項。
    • 印表機預設值：使用印表機預設值。
    • 獨立： 可讓您手動調整設定。
  3. 如果您在「輸出紙張」視窗中選擇「雙面列印」，請針對「圖像位移選項」選擇一個選項。
    • 印表機預設值：使用印表機預設值。
    • 獨立： 可讓您手動調整設定。
    • 鏡射 X 軸： 可讓您在從兩側沿著 X 軸同時朝相反方向等距移動影像。
    • 鏡射 Y 軸： 可讓您在從兩側沿著 Y 軸同時朝相反方向等距移動影像。
    • 鏡射雙軸： 可讓您在從兩側沿著 X 軸和 Y 軸同時朝相反方向等距移動影像。
    • 鏡射 X 軸，Tandem Y 軸： 可讓您在從兩側沿著 X 軸朝相反方向、同時沿著 Y 軸朝相同方向等距移動影像。
    • 鏡射 Y 軸，Tandem X 軸： 可讓您在從兩側沿著 Y 軸朝相反方向、同時沿著 X 軸朝相同方向等距移動影像。
    • Tandem X 軸： 可讓您在從兩側沿著 X 軸同時朝相同方向等距移動影像。
    • Tandem Y 軸： 可讓您在從兩側沿著 Y 軸同時朝相同方向等距移動影像。
    • Tandem 雙軸： 可讓您在從兩側沿著 X 軸和 Y 軸同時朝相同方向等距移動影像。
  4. 針對「獨立」，執行下列動作：
    1. 針對「第 1 面」，輸入沿著 X 軸和 Y軸的頁面圖像位移量。
    2. 如果「第 2 面」可用的話，輸入沿著 X 軸和 Y軸的頁面圖像位移量。

      注意： 「第 2 面」選項僅適用於當您選擇雙面列印時。

  5. 按一下確定。
