Паспорт работы Xerox

Паспорт работы — это файл с инструкциями к работе печати для таких функций принтеров Xerox, как сшивание и двусторонняя печать. Принтер использует данный файл для обработки работы печати. Инструкции паспорта работы обрабатываются раньше инструкций файла PDL.

Примечания:

  • Не все перечисленные опции поддерживаются всеми принтерами. Некоторые опции предназначены только для определенных моделей и конфигураций принтера, операционных систем и драйверов.
  • Некоторые приложения могут быть несовместимы с паспортами работ XPIF. Для печати в файл (например, .PS или .EPS) или в случае возникновения трудностей при печати отключите опцию «Паспорт работы Xerox».
  • Когда опция «Паспорт работы Xerox» отключена, выводится только файл PostScript. При этом настройки паспорта работы Xerox и функции принтера становятся недоступны.

Создание паспорта работы

В окне «Дополнительно» выберите значение нажатием стрелок рядом с полем «Паспорт работы Xerox».

  • Вкл. — для создания паспорта работы используются настройки документа. Паспорт работы отправляется на сервер печати вместе с работой.
  • Выкл. — отключение функции паспорта работы.

Импортирование паспорта работы

Для импортирования паспорта работы нажмите Импорт, найдите и выберите паспорт работы и нажмите кнопку OK.

Экспортирование паспорта работы

  1. В поле «Сохранить как» введите имя паспорта работы.
  2. Выберите место сохранения файла с помощью стрелок или перейдите к папке для сохранения.
  3. Нажмите кнопку Сохранить.
