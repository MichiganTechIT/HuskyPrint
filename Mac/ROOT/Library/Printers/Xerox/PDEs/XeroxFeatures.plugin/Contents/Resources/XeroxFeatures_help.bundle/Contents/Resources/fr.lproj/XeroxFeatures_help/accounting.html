Comptabilité

La fonction de comptabilisation associe votre nom d'utilisateur et votre ID de compte à chaque document que vous imprimez. Les données de comptabilisation sont rassemblées et enregistrées lorsque les travaux sont soumis à l'imprimante.

Remarques :

  • Toutes les imprimantes ne prennent pas en charge la totalité des options mentionnées. Certaines options s'appliquent uniquement à un modèle d'imprimante, un système d'exploitation ou un type de pilote spécifique.
  • Toutes les options peuvent ne pas apparaître, selon votre système de comptabilisation et vos sélections d'invite de comptabilisation.

Pour configurer la comptabilisation :

  1. Dans la fenêtre Comptabilisation, en regard de Système de comptabilisation, cliquez sur les flèches et sélectionnez une option.
    • Xerox® La fonction Comptabilisation standard  : assure le suivi du nombre de travaux de copie, d'impression, de numérisation et de télécopie pour chaque utilisateur. Elle utilise les ID utilisateur et de compte pré-définis dans l'imprimante.
    • Xerox® Comptabilité réseau : se procure les ID utilisateur et de compte auprès d'un serveur d'authentification réseau.
    • Comptabilisation locale : requiert un code de comptabilisation pour votre sous-réseau. Il peut s'agir d'un code de facturation d'un service (0001, par exemple). Les autres codes sont facultatifs.
    • Impression facturée : requiert un ID de facturation pour les travaux d'impression, tels que l'ID de facturation associé à votre service. Les autres ID sont facultatifs.
    • Envoyer les ID utilisateur et de comptabilisation : requiert l'entrée de vos ID d'utilisateur et de compte ainsi que de votre mot de passe chaque fois que vous envoyez un travail d'impression.
    • Envoyer l'ID utilisateur seulement : requiert l'entrée de votre ID d'utilisateur et de votre mot de passe chaque fois que vous envoyez un travail d'impression.
    • Envoyer l'ID de comptabilisation seulement : requiert l'entrée de votre ID de comptabilisation et de votre mot de passe chaque fois que vous envoyez un travail d'impression.
    • Envoyer l'ID de facturation seulement : requiert l'entrée de votre ID de facturation et de votre mot de passe chaque fois que vous envoyez un travail d'impression.
    • Envoyer tout : requiert l'entrée de vos ID d'utilisateur, de compte et de facturation ainsi que de votre mot de passe chaque fois que vous envoyez un travail d'impression.
  2. En regard d'Invite de comptabilisation, sélectionnez une option.
    • Toujours demander : requiert l'entrée de vos ID d'utilisateur et de compte chaque fois que vous envoyez un travail d'impression.
    • Ne pas demander : utilise les informations entrées sous Codes de comptabilité pour chaque travail d'impression.
    • Demander pour la télécopie uniquement : requiert l'entrée de vos ID d'utilisateur et de compte chaque fois que vous envoyez une télécopie.
    • Demander uniquement pour l'impression couleur : requiert l'entrée de vos ID d'utilisateur et de compte uniquement dans le cadre d'impressions couleur.
    • Demander uniquement pour la couleur et la télécopie : requiert l'entrée de vos ID d'utilisateur et de compte uniquement dans le cadre d'impressions couleur et de télécopies.
  3. Sélectionnez Masquer ID utilisateur pour que l'ID utilisateur ne soit pas visible. Tous les caractères entrés dans le champ sont remplacés par des astérisques *.
  4. Sélectionnez Masquer ID de compte pour que l'ID de compte ne soit pas visible. Tous les caractères entrés dans le champ sont remplacés par des astérisques *.
  5. Dans le champ Codes de comptabilisation par défaut, saisissez les valeurs, le cas échéant Il est impératif de renseigner ces champs lorsque l'option Ne pas demander est sélectionnée. Les ID de compte et de facturation peuvent correspondre au code de facturation d'un département (0001, par exemple).
  6. Cliquez sur OK.
