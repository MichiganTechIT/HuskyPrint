Support

Les sélections disponibles dans le menu Support déterminent le magasin sélectionné par l'imprimante pour l'impression d'un travail. Les paramètres du travail d'impression actuel s'affichent dans la zone Récapitulatif support. La zone Récapitulatif support se trouve en regard du menu déroulant Support.

Remarque : Toutes les imprimantes ne prennent pas en charge la totalité des options mentionnées. Certaines options s'appliquent uniquement à un modèle d'imprimante, un système d'exploitation ou un type de pilote spécifique.

Couleur du support

Type de support

Cette option permet de sélectionner le type de support pour votre travail d'impression.

Remarques :

  • Toutes les imprimantes ne prennent pas en charge la totalité des options mentionnées. Certaines options s'appliquent uniquement à un modèle d'imprimante, un système d'exploitation ou un type de pilote spécifique.
  • Si vous sélectionnez un type de support qui n'est pas chargé dans l'imprimante, celle-ci suspend le travail jusqu'à ce que le papier requis soit mis en place.

Pour sélectionner un type de support :

  1. Dans la fenêtre Support/Réception, en regard de Support, cliquez sur les flèches et sélectionnez Autre type.
  2. Effectuez l'une des opérations suivantes :
    • Pour utiliser le support par défaut pour l'imprimante, sélectionnez Défaut imprimante.
    • Sélectionnez un type de support dans la liste. Si l'administrateur a défini des types de support personnalisés pour l'imprimante, ceux-ci apparaissent dans la liste.
    • Pour sélectionner un magasin contenant n'importe quelle couleur de papier, choisissez Sélection automatique.
    • En cas d'utilisation d'intercalaires répertoire ou de carte intercalaire répertoire, sélectionnez les options requises.

Procédure de format papier inadapté

  1. Dans la fenêtre Support/Réception, en regard de Support, cliquez sur les flèches et sélectionnez Procédure de format de papier inadapté.
  2. Sélectionnez une option.
    • Afficher sur l'imprimante une invite à charger le format support approprié : assure la correspondance des supports avant l'impression.
    • Mise à l'échelle automatique, puis impression sur les supports chargés : adapte le document au format du support chargé dans le magasin.
    • Pas de mise à l'échelle, mais impression sur les supports chargés : imprime sur des supports de formats différents.
    • Afficher sur l'imprimante une invite à charger le format support approprié : assure la correspondance des supports avant l'impression.
    • Ignorer la non-correspondance et imprimer sur les supports chargés : imprime sur des types de support qui sont différents.

Séparateurs de transparents

Il est possible d'ajouter des feuilles de papier vierges entre chaque transparent imprimé. Les séparateurs doivent être de format et d'orientation identiques aux transparents.

Remarques :

  • Toutes les imprimantes ne prennent pas en charge la totalité des options mentionnées. Certaines options s'appliquent uniquement à un modèle d'imprimante, un système d'exploitation ou un type de pilote spécifique.
  • Les séparateurs de transparents ne peuvent pas être spécifiés avec certaines options de finition telles que le pliage ou les cahiers.

Pour imprimer à l'aide de séparateurs de transparents :

  1. Dans la fenêtre Support/Réception, en regard de Support, cliquez sur les flèches et sélectionnez Séparateurs de transparents.
  2. Sous Séparateurs de transparents, sélectionnez une option.
    • Désactivé(e) ou Pas de séparateurs : aucune feuille de séparation n'est insérée.
    • Séparateurs vierges : insère une feuille vierge entre les transparents.
    • Séparateurs imprimés : insère une feuille imprimée entre les transparents. La feuille imprimée contient la même image que le transparent.
  3. Sous Couleur, sélectionnez une option.
  4. Cliquez sur OK.
