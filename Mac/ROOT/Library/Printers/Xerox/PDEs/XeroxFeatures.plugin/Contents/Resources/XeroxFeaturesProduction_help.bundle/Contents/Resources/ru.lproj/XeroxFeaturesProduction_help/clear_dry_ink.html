Бесцветные сухие чернила

Применение функции «Бесцветные сухие чернила» позволяет придавать распечатанным изображениям вид фотографий или выделять изображения, текст и графику.

Примечание: Не все перечисленные опции поддерживаются всеми принтерами. Некоторые опции предназначены только для определенных моделей и конфигураций принтера, операционных систем и драйверов.

Указание опции «Прозрачный»:

  1. В окне «Прозрачный» выберите значение в поле «Clear Options» (Опции прозрачного покрытия).
    • Настройка по умолч. — используются действующие настройки принтера.
    • Бесцветная заливка — покрытие всей страницы прозрачным тонером.
    • Только изображения — покрытие на странице только тех объектов, что идентифицированы как изображения.
    • Только текст — покрытие текста прозрачным тонером.
    • Только графика — покрытие графики прозрачным тонером.
    • Искл.изобр.(тол.текст и граф.) — покрытие всей страницы, текста и графики прозрачным тонером, кроме изображений.
    • Иск.текст(тол.изобр.и граф.) — покрытие всей страницы, изображений и графики прозрачным тонером, кроме текста.
    • Иск.граф.(тол.изобр.и текст) — покрытие всей страницы, изображений и текста прозрачным тонером, кроме графики.
    • Подавление прозрачного — запрет покрытия прозрачным тонером на странице.
    • Использовать встроенный прозрачный — применение инструкций для прозрачного тонера, указанных в разделе PDL.

      Для создания печатаемых страниц в принтере используется язык описания страниц (Page Description Language, PDL). К языкам PDL относятся PostScript (PS) и язык управления принтером (Printer Command Language, PCL).

  2. Укажите количество прозрачного тонера в зависимости от аппарата, используя один из следующих вариантов.
    • Снова нажмите кнопку Прозрачный и выберите количество прозрачного тонера.
    • Выберите количество прозрачного тонера в списке «Кол-во прозр. покр.».
  3. Нажмите кнопку OK.
