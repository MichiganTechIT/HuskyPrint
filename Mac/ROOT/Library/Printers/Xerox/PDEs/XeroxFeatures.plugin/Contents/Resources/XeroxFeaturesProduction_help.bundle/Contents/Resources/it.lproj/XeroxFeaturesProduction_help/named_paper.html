Carta con nome

Carta con nome è un insieme di caratteristiche della carta che sono state denominate e salvate per il riutilizzo. È possibile scegliere da un Elenco carta stampante condiviso o da un Elenco carta personale.

Per selezionare una carta con nome:

  1. Nella finestra Carta/Copie, accanto a Carta, fare clic sulle frecce, quindi selezionare Carta con nome.
  2. Selezionare un'opzione.
    • Per selezionare un tipo di carta da un elenco precaricato nella stampante, fare clic su Elenco carta stampante.
    • Per selezionare un tipo di carta da un elenco personale di carte con nome, selezionare Elenco carta personale.

      Carta con nome è un insieme di caratteristiche della carta che sono state denominate e salvate per il riutilizzo. È possibile modificare o cancellare carta con nome solo in Elenco carta personale.

      Aggiunta di una carta con nome

      1. Nella finestra Carta con nome, fare clic sul menu in alto, selezionare Elenco carta personale, quindi selezionare il pulsante Nuovo.
      2. In Nome carta, selezionare o inserire un nome.
      3. Per utilizzare solo il nome carta, selezionare Usa solo nome (imp. avanzata) e immettere il nome o selezionarlo dall'elenco.

        Nota: Se si sceglie Usa solo nome (imp. avanzata), i campi attributo non sono disponibili.

      4. Procedere con una delle opzioni seguenti:
        • Selezionare gli attributi e le opzioni carta desiderati.
        • Per utilizzare i default stampante, fare clic su un menu e selezionare Default stampante.
      5. Per aggiungere ulteriori informazioni utilizzando questa carta con nome, in Commento (facoltativo), immettere le informazioni.
      6. Fare clic su OK.

      Copia di una carta con nome

      1. Nella finestra Carta con nome, fare clic sul menu in alto, quindi selezionare Elenco carta personale.
      2. Dalla tabella, selezionare una carta con nome da copiare, quindi fare clic sul pulsante Copia .

        Nota: Il pulsante diventa attivo dopo che si seleziona una carta con nome dall'elenco.

      3. Nel campo Nome carta, selezionare o inserire un nome.
      4. Per utilizzare solo il nome carta, selezionare Usa solo nome (imp. avanzata) e immettere il nome o selezionarlo dall'elenco.
      5. Se si sceglie Usa solo nome (imp. avanzata), i campi attributo non sono disponibili.
      6. Procedere con una delle opzioni seguenti:
        • Trattenere gli attributi e le opzioni carta.
        • Selezionare gli attributi e le opzioni carta desiderati.
        • Per utilizzare i default stampante, fare clic su un menu e selezionare Default stampante.
      7. Per aggiungere ulteriori informazioni utilizzando questa carta con nome, in Commento (facoltativo), immettere le informazioni.
      8. Fare clic su OK.

      Modifica di una carta con nome

      1. Nella finestra Carta con nome, fare clic sul menu in alto, quindi selezionare Elenco carta personale.
      2. Dalla tabella, selezionare una carta con nome, quindi fare clic sul pulsante Modifica .

        Nota: Il pulsante diventa attivo dopo che si seleziona una carta con nome dall'elenco.

      3. Per utilizzare solo il nome carta, selezionare Usa solo nome (imp. avanzata) e immettere il nome o selezionarlo dall'elenco.

        Nota: Se si sceglie Usa solo nome (imp. avanzata), i campi attributo non sono disponibili.

      4. Procedere con una delle opzioni seguenti:
        • Selezionare gli attributi e le opzioni carta desiderati.
        • Per utilizzare i default stampante, fare clic su un menu e selezionare Default stampante.
      5. Per aggiungere ulteriori informazioni utilizzando questa carta con nome, in Commento (facoltativo), immettere le informazioni.
      6. Fare clic su OK.

      Cancellazione di una carta con nome

      1. Nella finestra Carta con nome, fare clic sul menu in alto, quindi selezionare Elenco carta personale.
      2. Selezionare la carta con nome da rimuovere, quindi fare clic sul pulsante Cancella .
      3. Fare clic su Cancella.

      Nota: Poiché l'elenco in uso è associato al nome utente, è disponibile da altre stampanti di produzione Xerox® sulla propria rete.

  3. Selezionare la riga tabella della carta desiderata.
  4. Fare clic su Usa nel lavoro.
