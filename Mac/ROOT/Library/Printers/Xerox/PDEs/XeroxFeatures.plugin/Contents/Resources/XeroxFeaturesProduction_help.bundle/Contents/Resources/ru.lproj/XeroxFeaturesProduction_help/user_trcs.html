TRC пользователя

Кривые тоновоспроизведения (Tone Reproduction Curves, TRC) обеспечивают согласованность цвета печатаемых изображений. Функция «TRC пользователя» служит для точной настройки цветов путем коррекции кривых тоновоспроизведения.

Печать с использованием кривых TRC:

В окне «Параметры изображения» выберите один из вариантов в поле «TRC пользователя».

  • Настройка по умолч. — используются настройки принтера по умолчанию.
  • Выберите имя в списке последних имен TRC (до 10 имен).
  • Введите новое имя.
