Richiesta contabilità

Se la contabilità è abilitata e Richiesta contabilità è impostata su Richiedi sempre, quando si invia un lavoro di stampa, viene visualizzato Richiesta contabilità.

  1. Per ID utente, se si specifica un valore predefinito, è possibile utilizzarlo o digitare un nuovo valore.
  2. Per ID conto, se si specifica un valore predefinito, è possibile utilizzarlo o digitare un nuovo ID conto.
  3. Fare clic su OK.
