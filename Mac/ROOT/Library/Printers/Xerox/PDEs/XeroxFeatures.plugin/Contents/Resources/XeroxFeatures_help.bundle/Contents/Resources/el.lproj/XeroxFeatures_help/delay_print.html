Ρύθμιση μεταχρονολογημένης εκτύπωσης

Χρησιμοποιήστε τη Μεταχρονολογημένη εκτύπωση για να εκτυπώσετε την εργασία σας σε κάποια συγκεκριμένη ώρα. Για παράδειγμα, εάν έχετε μια μεγάλη εργασία εκτύπωσης, μπορείτε να ρυθμίσετε την εργασία για εκτύπωση μετά από τις ώρες εργασίας.

Η λειτουργία Μεταχρονολογημένη εκτύπωση μπορεί να προγραμματιστεί για χρονικό διάστημα έως 24 ώρες από τη στιγμή αποστολής της εργασίας στον εκτυπωτή. Εάν στείλετε την εργασία εκτύπωσης μετά από τη ρυθμισμένη ώρα εκτύπωσης, η εργασία εκτυπώνεται όταν φτάσει η ώρα της ρύθμισης μέσα στις επόμενες 24 ώρες. Για παράδειγμα, εάν ρυθμίσετε την ώρα εκτύπωσης στις 4:30μμ, αλλά στείλετε την εργασία στις 4:31μμ, η εργασία θα εκτυπωθεί στις 4:30μμ την επόμενη ημέρα.

Η εργασία εκτύπωσης ενδέχεται να φτάσει στον εκτυπωτή μετά από την ώρα ρύθμισης εξαιτίας καθυστερήσεων στο δίκτυο ή στην ουρά εκτύπωσης. Επίσης μπορεί να παρουσιαστεί καθυστέρηση όταν υπάρχει διαφορά μεταξύ της ρύθμισης του ρολογιού στον εκτυπωτή και αυτής στον υπολογιστή σας. Σε αυτή την περίπτωση, η εργασία εκτυπώνεται αμέσως.

Σημείωση: Δεν υποστηρίζονται όλες οι επιλογές που παρατίθενται από όλους τους εκτυπωτές. Ορισμένες επιλογές ισχύουν μόνο σε συγκεκριμένα μοντέλα ή διαμορφώσεις εκτυπωτών, λειτουργικά συστήματα ή τύπους οδηγών.

Για την εκτύπωση μέσω της λειτουργίας Μεταχρονολογημένη εκτύπωση:

  1. Στο παράθυρο Χαρτί/Έξοδος, δίπλα στην επιλογή Είδος εργασίας, κάντε κλικ στα βέλη και μετά επιλέξτε Μεταχρονολογημένη εκτύπωση.
  2. Για τη ρύθμιση της ώρας, κάντε κλικ στα βέλη και μετά επιλέξτε την ώρα και τα λεπτά.
  3. Κάντε κλικ στο πλήκτρο OK.
  4. Κάντε κλικ στην επιλογή Εκτύπωση.
