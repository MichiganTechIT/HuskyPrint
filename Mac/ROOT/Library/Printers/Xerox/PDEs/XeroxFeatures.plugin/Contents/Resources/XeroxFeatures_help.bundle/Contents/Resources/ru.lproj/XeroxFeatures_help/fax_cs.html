Титульный лист факса

Включение в факс титульного листа:

  1. В окне «Бумага/Вывод» нажатием стрелок рядом с полем «Тип работы» выберите значение Факс.
  2. Выберите Титульный лист.
  3. Рядом с полем «Титульный лист» нажатием стрелок выберите Печать титульного листа.
  4. Для добавления сообщения получателям нажмите кнопку рядом с полем Комментарий на титульном листе.
  5. Для добавления контактной информации нажмите кнопку рядом с полем Информация об отправителе.
  6. Для изменения информации о получателе нажмите кнопку рядом с полем Информация о получателе.
  7. Чтобы указать изображение для титульного листа, выбери его нажатием стрелок.
    • Для отключения функции «Изображение для титульного листа» выберите опцию Нет.
    • Для выбора изображения для титульного листа, ранее сохраненного на принтере, выберите его с списке.
    • Для изменения ранее сохраненного изображения для титульного листа выберите его в списке и нажмите кнопку Изменить.
    • Для создания изображения для титульного листа нажмите кнопку Создать.
  8. Выберите один из вариантов в поле Формат бумаги титульного листа.
  9. Нажмите кнопку OK.
