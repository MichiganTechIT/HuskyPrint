Création de cahier

Cette option permet de créer des cahiers de feuilles simples, qui peuvent être détachées du cahier selon les besoins. Un cahier présente en général une couverture fin cartonnée et une reliure par collage sur le bord supérieur.

Pour imprimer des cahiers :

  1. Dans la fenêtre Support/Réception, en regard de Finition, cliquez sur les flèches et sélectionnez Cahiers.
  2. En regard de Cahiers, sélectionnez une option.
    • Désactivé(e) : désactive la fonction Création de cahier.
    • Activé(e) : active l'impression de cahiers.
  3. Pour Nombre de cahiers, effectuez l'une des opérations suivantes :
    • Indiquez le nombre de cahiers à imprimer.
    • Pour utiliser la valeur par défaut de l'imprimante, sélectionnez Défaut imprimante.
  4. Pour Feuilles par cahier, effectuez l'une des opérations suivantes :
    • Indiquez le nombre de pages de chaque cahier.
    • Pour utiliser la valeur par défaut de l'imprimante, sélectionnez Défaut imprimante.
  5. Pour Couverture fin, sélectionnez le type de couverture fin des cahiers.
  6. Sous Support, sélectionnez des options :
    • Support nommé : permet de sélectionner un support nommé dans une liste ou de créer un Support nommé.
    • Autre format : permet de sélectionner le format papier ou de créer un format personnalisé.
    • Autre couleur :permet de faire une sélection dans une liste de couleurs disponibles ou de créer des couleurs personnalisées.
    • Autre type : permet de sélectionner le type de support ou de créer des types de support personnalisés.
    • Autre grammage : permet de sélectionner le grammage du support ou de créer un grammage de support personnalisé.
    • Options de support avancées : permet de sélectionner divers attributs de support avancés pour votre couverture.
  7. Cliquez sur OK.
