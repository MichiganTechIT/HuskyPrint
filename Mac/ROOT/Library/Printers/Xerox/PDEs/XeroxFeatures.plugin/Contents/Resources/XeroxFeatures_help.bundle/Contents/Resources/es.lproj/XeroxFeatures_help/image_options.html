Opciones de imagen

Opciones de imagen permite ajustar la claridad u oscuridad general del texto y de las imágenes del trabajo de impresión.

Nota: No todas las opciones descritas están disponibles en todas las impresoras. Algunas opciones solo se aplican a un modelo de impresora, configuración, sistema operativo o tipo de controlador específico.

Para configurar las opciones de imagen del trabajo de impresión:

  1. Para ajustar la claridad, mueva el control deslizante hacia la derecha para aclarar los colores del trabajo de impresión o hacia la izquierda para oscurecerlos.
  2. Para ajustar el contraste, mueva el control hacia la derecha para reducir el contraste o hacia la izquierda para aumentarlo.
  3. Para seleccionar las opciones de Desplazar imagen, haga clic en Desplazar imagen.
  4. Haga clic en Impresión.
