Сохраненная работа

Сохраненными называются работы, отправленные на принтер и сохраненные для последующего использования.

Примечание: Не все перечисленные опции поддерживаются всеми принтерами. Некоторые опции предназначены только для определенных моделей и конфигураций принтера, операционных систем и драйверов.

Печать сохраненной работы:

  1. В окне «Бумага/Вывод» нажатием стрелок рядом с полем «Тип работы» выберите значение Сохраненная работа.
  2. Для выбора имени работы выполните одно из следующих действий:
    • Введите имя работы.
    • Нажмите стрелку справа от поля и выберите имя в списке.
    • Нажмите стрелку справа от поля и выберите опцию Использовать имя документа, чтобы использовать имя документа.
  3. Выберите нужную опцию. Работу можно сохранить или напечатать и сохранить. После сохранения работы ее можно печатать по мере необходимости.
  4. В поле «Формат» выберите формат для сохраняемой работы.
    • Настройка по умолч. — использование для форматов файлов настройки принтера по умолчанию.
    • Одностраничный TIFF — сохранение работы печати в формате TIFF. Каждая страница работы печати сохраняется в отдельном файле формата TIFF.
    • Многостраничный TIFF — сохранение работы печати в формате TIFF. Все страницы работы печати сохраняются в одном файле формата TIFF.
    • PDF — сохранение работы печати в файле формата PDF.
    • Быстрая повторная печать — сохранение работы печати в формате FRF (Fast Reprint Format).
    • Быстрая повторная печать с DRI — сохранение работы печати в формате FRF для ускорения повторной печати. Настройка DRI (Display Resolution Image - изображение с разрешением дисплея) позволяет просматривать и редактировать работу на сервере печати.
  5. Нажмите кнопку OK.
  6. Нажмите кнопку Печать.
