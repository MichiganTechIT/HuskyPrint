Добавление вставок

Можно вставлять листы без печати или с печатью перед первой страницей документа или после указанных страниц. Например, если нужно вставить чистый лист цветной бумаги после каждого раздела документа, необходимо указать номера страниц, где разделы заканчиваются. Эти номера страниц указываются для вставок.

Примечание: Не все перечисленные опции поддерживаются всеми принтерами. Некоторые опции предназначены только для определенных моделей и конфигураций принтера, операционных систем и драйверов.

Добавление вставки:

  1. В окне «Специальные страницы» выберите Добавить вставки.
  2. Выберите тип вставок:
    • Перед страницей 1: лист вставляется перед страницей 1.
    • После страницы — лист вставляется после указанных страниц или диапазона страниц.
  3. При выборе опции «После страницы» введите количество страниц перед каждой вставкой в поле «Страницы».
  4. В поле «Количество» выберите количество листов вставок.
  5. Чтобы подтвердить для работы все настройки по умолчанию, нажмите кнопку OK.
  6. Для указания размера, цвета и типа выберите значения нажатием стрелок в разделе «Бумага».
  7. Нажмите кнопку OK.
