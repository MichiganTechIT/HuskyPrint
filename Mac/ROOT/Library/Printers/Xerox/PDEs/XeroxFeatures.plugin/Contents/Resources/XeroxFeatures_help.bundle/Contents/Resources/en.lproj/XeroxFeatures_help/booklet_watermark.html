Booklet/Watermark

Use the Booklet/Watermark window to select booklet layout and watermark options.

Note: Not all options listed are supported on all printers. Some options apply only to a specific printer model, configuration, operating system, or driver type.

To print using Booklet/Watermark:

  1. In the Booklet/Watermark window, under Booklet Layout, click the arrows, then select an option.
    • Enabled: This option prints the document as a booklet. You can use this option for a document that is not formatted as a booklet.
    • Disabled: This option turns off Booklet Layout. You can use this option for a document that is formatted as a booklet.
  2. To specify additional booklet features, enable Booklet Layout, then click Booklet Options.
  3. To print a border around each page, select Draw Page Borders.
  4. Select options for the Watermark.
  5. Click Print.
