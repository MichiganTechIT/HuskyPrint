Περισσότερες ρυθμίσεις

Για να κάνετε περισσότερες ρυθμίσεις:

  1. Στο παράθυρο Διόρθωση ειδικών χρωμάτων, για οποιαδήποτε επιλογή, κάντε κλικ στο στοιχείο Περισσότερες ρυθμίσεις.
  2. Για να χρησιμοποιήστε τις ρυθμίσεις χρωμάτων από το προηγούμενο παράθυρο, επιλέξτε Χρήση ενιαίων ρυθμίσεων.
  3. Για τον καθορισμό των τιμών Φωτεινότητα, Κορεσμός και Κυρίαρχο χρώμα, χρησιμοποιήστε τα ρυθμιστικά για να ορίσετε επιλογές.
  4. Για επιστροφή στις αρχικές ρυθμίσεις του συστήματος, κάντε κλικ στο στοιχείο Προεπιλ..
  5. Κάντε κλικ στο πλήκτρο OK.
