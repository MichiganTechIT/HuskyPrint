Цвет бумаги

Функция «Цвет бумаги» служит для выбора цвета бумаги для работы печати.

Выбор цвета бумаги:

  1. В окне «Бумага/Вывод» нажатием стрелок рядом с полем «Бумага» выберите значение Другой цвет.
  2. Выполните одно из следующих действий.
    • Для использования в принтере цвета бумаги по умолчанию выберите значение Цвет по умолчанию для принтера.
    • Выберите в списке цвет бумаги. Когда администратор создает для принтера настраиваемые цвета бумаги, они отображаются в списке.
    • Чтобы цвет выбирался автоматически, выберите Автовыбор.

      Примечание: Если выбирается бумага, отсутствующая в принтере, работа задерживается в принтере до тех пор, пока не будет вложена требуемая бумага.
