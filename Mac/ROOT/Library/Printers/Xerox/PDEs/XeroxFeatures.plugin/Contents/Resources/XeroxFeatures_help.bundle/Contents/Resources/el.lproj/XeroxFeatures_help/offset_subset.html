Μετατόπιση/Υποσύνολο

Η έξοδος μετατόπισης ελέγχει τον τρόπο μετατόπισης των αντιγράφων σε εργασίας εκτύπωσης πολλαπλών αντιγράφων. Όταν ενεργοποιηθεί, τα σετ αντιγράφων μετατοπίζονται ελαφρώς στο δίσκο εξόδου. Η επιλογή αυτή είναι διαθέσιμη εάν ο δίσκος εξόδου έχει δυνατότητα μετατόπισης.

Για χρήση της επιλογής Μετατόπιση/Υποσύνολο:

  1. Στο παράθυρο Χαρτί/Έξοδος, δίπλα στην επιλογή Τελική επεξεργασία, κάντε κλικ στα βέλη και μετά επιλέξτε Μετατόπιση/Υποσύνολο.
  2. Ορίστε μια επιλογή.
    • Η ρύθμιση Καμία απενεργοποιεί τη μετατόπιση. Μπορείτε ακόμη να επιλέγετε υποσύνολα.
    • Η επιλογή Κάθε σύνολο μετατοπίζει το κάθε σύνολο μιας εργασίας εκτύπωσης πολλαπλών αντιγράφων.
    • Η επιλογή Κάθε υποσύνολο μετατοπίζει έναν καθορισμένο αριθμό σελίδων ενός ενιαίου συνόλου.
    • Η επιλογή Κάθε σύνολο και κάθε υποσύνολο μετατοπίζει έναν καθορισμένο αριθμό σελίδων ενός συνόλου και επίσης μετατοπίζει κάθε σύνολο μιας εργασίας εκτύπωση πολλαπλών αντιγράφων.
  3. Για να εκτυπώσετε υποσύνολα σε μια ενιαία εργασία εκτύπωσης, επιλέξτε Ενεργοποίηση υποσυνόλων.
  4. Στο πεδίο Σελίδες ανά υποσύνολο, πληκτρολογήστε τον αριθμό των σελίδων που θα περιλαμβάνονται σε κάθε υποσύνολο εγγράφου. Κάθε υποσύνολο πρέπει να περιέχει τον ίδιο αριθμό σελίδων.
  5. Κάντε κλικ στο πλήκτρο OK.
