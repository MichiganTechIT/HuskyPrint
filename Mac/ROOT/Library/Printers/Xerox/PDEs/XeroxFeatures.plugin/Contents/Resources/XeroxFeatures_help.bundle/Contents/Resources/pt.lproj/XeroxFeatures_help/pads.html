Criação de painel

Almofadas permitem a criação de almofadas de formulários de folhas únicas de modo que cada formulário é retirado da almofada conforme necessário. Tipicamente uma almofada tem uma capa traseira de cartão e uma encadernação colada na borda superior.

Imprimir almofadas:

  1. Na janela Papel/Saída, ao lado de Acabamento, clique nas setas e selecione Painéis.
  2. Selecione uma opção ao lado de Almofadas:
    • Desativada: desliga a função de Criação de painel.
    • Ativado: liga a impressão de painel.
  3. Para Quantidade de almofadas, siga um destes procedimentos:
    • Especifique o número de almofadas a imprimir.
    • Para usar o padrão para a impressora, selecione Padrão da impressora.
  4. Para Folhas por painéis, siga um destes procedimentos:
    • Insira a quantidade de páginas para cada almofada.
    • Para usar o padrão para a impressora, selecione Padrão da impressora.
  5. Para Capa traseira selecione o tipo de capa traseira para as almofadas.
  6. Para Papel, selecione opções.
    • Outro tamanho: Permite selecionar o tamanho do papel ou criar um tamanho personalizado.
    • Outra cor: permite selecionar de uma lista de cores disponíveis ou criar cores personalizadas definidas pelo administrador.
    • Outro tipo: permite selecionar o tipo de papel ou selecionar um tipo personalizado definido pelo administrador.
  7. Clique em OK.
