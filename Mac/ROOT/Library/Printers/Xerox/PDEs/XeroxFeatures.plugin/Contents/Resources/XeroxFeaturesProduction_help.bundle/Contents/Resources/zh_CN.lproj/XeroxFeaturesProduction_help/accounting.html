帐目

帐目将您的用户名和帐户标识信息与您打印的每份文档相关联。 将作业提交到打印机时，将收集和存储帐目数据。

  1. 要访问帐目设置，请在“高级”窗口中“配置”的旁边单击帐目。
  2. 对于“帐目系统”，选择一个选项。
    • 要设置帐目，请单击已启用。
    • 要禁用帐目，请单击无。
  3. 在“帐目提示”旁边，选择一个选项。
    • 不提示：在您提交打印作业时不提示您键入用户 ID 或帐户 ID。
    • 总是提示：在您每次提交打印作业时要求您输入用户 ID 和帐户 ID。
  4. 如果选择“总是提示”：
    1. 要在作业打印时屏蔽用户 ID 或帐户 ID，请选中这些选项旁边的复选框。
    2. 如果需要，键入默认用户 ID 值。
    3. 如果需要，键入默认帐户 ID 或开单代码值。
  5. 单击确定。
