Couleur du support

Couleur du support permet de sélectionner la couleur du papier de votre travail d'impression.

Pour sélectionner une couleur de support :

  1. Dans la fenêtre Support/Réception, en regard de Support, cliquez sur les flèches et sélectionnez Autre couleur.
  2. Effectuez l'une des opérations suivantes :
    • Pour utiliser la couleur du support par défaut pour l'imprimante, sélectionnez Couleur par défaut de l'imprimante.
    • Sélectionnez une couleur de support dans la liste.
    • Pour que l'imprimante sélectionne la couleur, cliquez sur Sélection auto.
    • Pour utiliser une couleur qui ne figure pas dans la liste, sélectionnez Personnalisé(e).

      Remarque : Si vous sélectionnez une couleur de support qui n'est pas chargée dans l'imprimante, celle-ci suspend le travail jusqu'à ce que le papier requis soit mis en place.
