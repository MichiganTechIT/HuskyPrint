소책자 옵션

  1. 소책자 마감 작업 옵션을 선택합니다.
  2. 소책자를 새로운 크기에 맞추려면 [새 용지 크기에 맞추기] 옆에 있는 화살표를 클릭한 후 필요한 크기를 선택합니다. 사용자 정의 크기를 만들려면 사용자 정의를 선택합니다.
  3. 여백 크기를 설정하려면 [소책자 여백] 옆에 있는 화살표를 클릭한 후 다음 옵션 중 하나를 선택합니다.
    • 표준을 선택하면 이미지 손실 없이 표준 용지 크기의 소책자로 문서를 인쇄할 수 있도록 페이지 이미지 크기가 축소됩니다.
    • 없음을 선택하면 페이지 이미지 크기를 축소하지 않은 상태로 문서를 소책자로 인쇄합니다. 프린터에서 용지 가장자리에 인쇄할 수 없는 경우 이 옵션을 선택하면 이미지가 잘릴 수 있습니다.

  4. 필요에 따라 수평 여백과 위치 조정을 지정합니다.
    • 수평 여백: 페이지 이미지 간 중앙 여백의 너비를 포인트 단위로 지정합니다. 값을 입력하거나 화살표를 사용하여 포인트 수를 선택합니다. 1 포인트는 1/72 인치 또는 0.35mm입니다.
    • 위치 조정: 페이지 이미지를 밖으로 이동할 크기를 포인트의 1/10 단위로 지정합니다. 페이지 이미지의 이동량을 지정하려면 값을 입력하거나 화살표를 사용하여 값을 선택합니다.
  5. 확인을 클릭합니다.
