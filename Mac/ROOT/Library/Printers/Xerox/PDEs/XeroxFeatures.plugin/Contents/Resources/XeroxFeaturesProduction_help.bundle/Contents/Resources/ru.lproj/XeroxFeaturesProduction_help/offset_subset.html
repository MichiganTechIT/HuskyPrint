Сдвиг/Подкомплект

Настройка «Вывод со сдвигом» служит для указания сдвига копий в многокомплектной работе печати. Когда она включена, комплекты копий в выходном лотке слегка сдвигаются относительно друг друга. Данная опция доступна, если выходной лоток снабжен функцией сдвига.

Использование функции «Сдвиг/Подкомплект»:

  1. В окне «Бумага/Вывод» нажатием стрелок рядом с полем «Послепечатная обработка» выберите Сдвиг/Подкомплект.
  2. Выберите нужную опцию.
    • Настройка по умолч. — используются настройки принтера по умолчанию.
    • Нет — отключение сдвига. При этом можно выбирать подкомплекты.
    • Каждый комплект — сдвиг каждого комплекта многокомплектной работы печати.
    • Каждый подкомплект - сдвиг указанного количества страниц в одном комплекте.
    • Каждый комплект и каждый подкомплект - сдвиг указанного количества страниц в одном комплекте и сдвиг каждого комплекта работы печати.
  3. Для печати комплектов одной работы выберите Разрешить подкомплекты.
  4. В поле Страниц в подкомплекте введите число страниц в каждом подкомплекте документов. В каждом подкомплекте должно быть одинаковое число страниц.
  5. Нажмите кнопку OK.
