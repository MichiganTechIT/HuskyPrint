Profils de module de finition DFA

Profils de module de finition DFA vous connecte à un profil de finition configuré pour l'imprimante. Le nom personnalisé que vous avez créé pour un module de finition apparaît dans le pilote d'imprimante.

Pour entrer un nouveau nom :

  1. Dans la fenêtre Support/Réception, en regard de Finition, cliquez sur les flèches et sélectionnez Personnalisé(e).
  2. Dans le champ Personnalisé(e), tapez le nom du DFA.
  3. Cliquez sur OK.
