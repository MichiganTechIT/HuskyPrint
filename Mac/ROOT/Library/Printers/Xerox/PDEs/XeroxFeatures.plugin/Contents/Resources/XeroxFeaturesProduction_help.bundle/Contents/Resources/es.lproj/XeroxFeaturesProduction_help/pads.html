Creación de bloc

La opción Blocs permite crear blocs de formularios de una hoja; de este modo que cada formulario se arranca del bloc según se necesita. Por lo general, un bloc tiene una cubierta posterior de cartón y un encuadernado con pegamento en la parte superior.

Para imprimir blocs:

  1. En la ventana Papel/Salida, en Acabado, haga clic en las flechas y, a continuación, seleccione Blocs.
  2. Junto a Blocs, seleccione una opción.
    • Desactivado: desactiva la función Creación de bloc.
    • Activado: activa la impresión en bloc.
  3. En Número de blocs, realice una de las acciones siguientes:
    • Introduzca el número de blocs que desea imprimir.
    • Para utilizar el valor prefijado en la impresora, seleccione Prefijado de impresora.
  4. En Páginas por bloc, realice una de las acciones siguientes:
    • Introduzca el número de páginas para cada bloc.
    • Para utilizar el valor prefijado en la impresora, seleccione Prefijado de impresora.
  5. En Cubierta posterior, seleccione el tipo de cubierta posterior que desea para los blocs.
  6. En Papel, seleccione las opciones.
    • Papel con nombre: permite seleccionar un Papel con nombre en la lista o crear un Papel con nombre nuevo.
    • Otros tamaños: Esta opción permite seleccionar el tamaño de papel o crear un tamaño personalizado.
    • Otro color: permite seleccionar un color en la lista de colores disponibles o crear un color de papel personalizado.
    • Otro tipo: permite seleccionar el tipo de papel o crear tipos de papel personalizados.
    • Otro peso: permite seleccionar el peso del papel o crear un peso de papel personalizado.
    • Opciones avanzadas de papel: permite seleccionar atributos de papel avanzados para la cubierta.
  7. Haga clic en Aceptar.
