Ajouter des pages spéciales

Utilisez l'option Ajouter des pages spéciales pour spécifier des pages spéciales à ajouter dans un travail d'impression. Les paramètres appliqués aux pages spéciales sont différents de ceux qui sont appliqués aux autres pages du travail. Vous pouvez, par exemple, appliquer aux pages spéciales des paramètres de format, de type et de couleur différents. Vous pouvez également sélectionner un mode d'impression (recto ou recto verso) différent. Un travail d'impression peut contenir plusieurs pages spéciales.

Remarque : Toutes les imprimantes ne prennent pas en charge la totalité des options mentionnées. Certaines options s'appliquent uniquement à un modèle d'imprimante, un système d'exploitation ou un type de pilote spécifique.

Pages spéciales - réception

  1. Dans la fenêtre Pages spéciales, cliquez sur Ajouter des pages spéciales.
  2. Dans Page(s), spécifiez la ou les pages auxquelles appliquer les pages spéciales. Séparez les pages isolées ou les séries de pages par des virgules. Utilisez un tiret pour indiquer une série de pages. Par exemple, pour spécifier les pages 1, 6, 9, 10 et 11, vous devez entrer 1,6,9-11.
  3. Pour accepter les valeurs par défaut du travail pour tous les paramètres, cliquez sur OK.
  4. Pour sélectionner le format, la couleur, le type ou le grammage, en regard de Support, cliquez sur les flèches et sélectionnez des options.
    • Dans la fenêtre Intercalaires répertoire pour Type de support, dans Nombre d'onglets dans la séquence, entrez le nombre d'intercalaires.
    • Si avez sélectionné Sequenced for Paper type (Type de support paginé), spécifiez le nom du support et le nombre d'onglets dans la séquence.
  5. Pour modifier le mode d'impression, sélectionnez une option sous Mode d'impression :
    • Recto : permet d'imprimer les pages spéciales sur une face du papier uniquement.
    • Impression recto verso : imprime les pages spéciales sur les deux côtés du papier et retourne les pages sur le grand côté. Le travail peut ensuite être relié sur le grand côté des pages.
    • Recto verso, reliure petit côté : imprime les pages spéciales sur les deux côtés du papier et retourne les pages sur le petit côté. Le travail peut ensuite être relié sur le petit côté des pages.
  6. Sous Mode couleur, sélectionnez une option.
  7. Cliquez sur OK.

Pages spéciales - options image

La fenêtre Options Image permet de modifier l'apparence des documents imprimés.

Remarque : Toutes les imprimantes ne prennent pas en charge la totalité des options mentionnées. Certaines options s'appliquent uniquement à un modèle d'imprimante, un système d'exploitation ou un type de pilote spécifique.

Toner transparent ou Encre transparente sur objets

Toner transparent permet de conférer à votre image imprimé l'aspect d'une photographie ou de mettre en évidence une image, du texte ou un graphique.

Remarque : Toutes les imprimantes ne prennent pas en charge la totalité des options mentionnées. Certaines options s'appliquent uniquement à un modèle d'imprimante, un système d'exploitation ou un type de pilote spécifique.

Pour utiliser le toner transparent :

  1. Dans la fenêtre Toner transparent, sélectionnez une option sous Options toner transparent.
    • Valeur imprimante par défaut : utilise les paramètres actuels de l'imprimante.
    • Couchage pleine page transparent : couvre toute la page de toner transparent.
    • Images uniquement : couvre uniquement les objets identifiés comme des images.
    • Texte uniquement : couvre uniquement le texte de toner transparent.
    • Graphiques uniquement : couvre uniquement les graphiques de toner transparent.
    • Texte et graphiques uniquement : couvre la page entière, le texte et les graphiques de toner transparent en omettant les images.
    • Images et graphiques uniquement : couvre la page entière, les images et les graphiques de toner transparent en omettant le texte.
    • Images et texte uniquement : couvre la page entière, les images et le texte de toner transparent en omettant les graphiques.
    • Supprimer l'encre transparente : empêche le couchage de la page par du toner transparent.
    • Utiliser les paramètres d'encre transparente incorporés : l'imprimante utilise les instructions trouvées dans le PDL pour appliquer du toner transparent.

      L'imprimante utilise le langage de description de page (PDL, Page Description Language) pour créer une page imprimée. Les langages PDL incluent PS (PostScript) et PCL (Printer Command Language).

  2. Pour spécifier la quantité de toner transparent à utiliser, en regard de Quantité de toner transparent, cliquez sur les flèches et sélectionnez une option.
  3. Cliquez sur OK.

Anti-aliassage

L'anti-aliassage lisse les bords crénelés des images et du texte.

Pour sélectionner une option anti-aliassage :

Dans la fenêtre Options image, sous Anti-aliassage, sélectionnez une option.

  • Valeur imprimante par défaut : utilise les paramètres actuels de l'imprimante.
  • Oui : active l'anti-aliassage.
  • Éteint : désactive l'anti-aliassage.
  • N 100% haute résolution : utilise une couleur noire unie nette. Vous pouvez utilisez cette fonction, par exemple, pour du texte ou des graphiques fins qui contiennent des aplats noirs.

Courbes tonales utilisateur

Les courbes tonales garantissent l'homogénéité des couleurs de l'image imprimée. Les courbes tonales utilisateur permettent d'affiner les couleurs imprimées en réglant les courbes tonales.

Pour imprimer à l'aide des courbes tonales utilisateur :

Dans la fenêtre Options image, sous Courbe tonale utilisateur, sélectionnez une option.

  • Valeur imprimante par défaut : utilise les paramètres par défaut de l'imprimante.
  • Sélectionnez un nom dans la liste contenant les noms des dernières courbes tonales (jusquà 10) qui ont été saisis.
  • Entrez un nouveau nom.

Résolution

L'option Résolution détermine la qualité des images graphiques. La résolution image s'exprime en points par pouce (ppp). Lorsque la résolution est fixée à 600 ppp, les images sont rendues selon une densité de 600 points (pixels) par pouce de l'image. Plus la résolution est élevée, meilleure est la qualité d'impression.

Pour spécifier la résolution, cliquez sur Résolution, puis choisissez une option ou sélectionnez Défaut imprimante.

Trame demi-teinte

L'impression en demi-teinte est une méthode dans le cadre de laquelle la teinte foncée, la densité et la taille relatives de points minuscules varient pour refléter des changements de couleur et de luminosité. Le niveau de détail d'une demi-teinte est proportionnel au nombre de points par pouce ; plus les points sont nombreux, plus le niveau de détail est élevé sur les impressions. Le mot « trame » est parfois utilisé pour désigner une demi-teinte.

L'option de trame demi-teinte stochastique utilise un procédé reposant sur une répartition aléatoire des points dans laquelle vous spécifiez le densité des points en fonction du niveau de gris voulu. Cette option est conçue pour éviter l'effet de moirage. La nature aléatoire du tramage stochastique est idéale pour le rendu des photographies.

Pour sélectionner les options de trames demi-teintes :

Dans la fenêtre Options image, cliquez sur Trame demi-teinte, puis sélectionnez une option.

  • Valeur imprimante par défaut : utilise les paramètres par défaut de l'imprimante.
  • Sélectionnez une valeur. Plus vous sélectionnez un nombre de points, de lignes ou de lignes par pouce (lpi) élevé, plus l'impression comportera de détails.
  • Sélectionnez Stochastique.

Économie de toner

Économie de toner réduit la quantité de toner utilisée pour les travaux d'impression et est utile pour une qualité d'impression brouillon. La qualité d'impression est légèrement inférieure à Standard.

Pour activer l'économie de toner, dans la fenêtre Options image, en regard de Économie de toner, cliquez sur les flèches et sélectionnez Activé(e).

Décalage image

Le décalage image vous permet de modifier l'emplacement de l'image sur la page. Vous pouvez décaler l'image imprimée selon les axes X et Y de chaque feuille de papier.

Pour définir les options de décalage image :

  1. En regard des Options de décalage image, utilisez les flèches et sélectionnez une option.
  2. Si vous avez sélectionné Impression recto dans la fenêtre Support/Réception, sous Options de décalage image, sélectionnez une option.
    • Valeur imprimante par défaut : utilise les paramètres par défaut de l'imprimante.
    • Différent : permet de régler les paramètres manuellement.
  3. Si vous avez sélectionné Impression recto verso dans la fenêtre Support/Réception, sous Options de décalage image, sélectionnez une option.
    • Valeur imprimante par défaut : utilise les paramètres par défaut de l'imprimante.
    • Différent : permet de régler les paramètres manuellement.
    • Miroir X : permet de déplacer les images sur les deux faces sur une distance égale mais dans des directions opposées, selon l'axe X, simultanément.
    • Miroir Y : permet de déplacer les images sur les deux faces sur une distance égale mais dans des directions opposées, selon l'axe Y, simultanément.
    • Miroir X et Y : permet de déplacer les images sur les deux faces sur une distance égale mais dans des directions opposées, selon les axes X et Y, simultanément.
    • Miroir X, Identique Y : permet de déplacer les images sur les deux faces sur une distance égale, dans des directions opposées selon l'axe X et dans la même direction selon l'axe Y, simultanément.
    • Miroir Y, Identique Y : permet de déplacer les images sur les deux faces sur une distance égale, dans des directions opposées selon l'axe Y et dans la même direction selon l'axe X, simultanément.
    • Identique X : permet de déplacer les images sur les deux faces sur une distance égale et dans la même direction, le long de l'axe X, simultanément.
    • Identique Y : permet de déplacer les images sur les deux faces sur une distance égale et dans la même direction, le long de l'axe Y, simultanément.
    • Identique X et Y : permet de déplacer les images sur les deux faces sur une distance égale et dans la même direction, le long des axes X et Y, simultanément.
  4. Sous Différent, effectuez les opérations suivantes :
    1. Pour le recto, entrez la valeur du décalage image selon les axes X et Y.
    2. Si l'option Verso est disponible, entrez la valeur du décalage image selon les axes X et Y.

      Remarque : L'option verso n'est disponible que si vous avez sélectionné Impression recto verso.

  5. Cliquez sur OK.

Couleur d'accompagnement

La couleur d'accompagnement permet de faire ressortir du texte ou des parties imprimées importants.

Pour sélectionner les méthodes pour la couleur d'accompagnement :

  1. Dans la fenêtre Options image, cliquez sur Couleur d'accompagnement.
  2. Sélectionnez une option.
    • Valeur imprimante par défaut : utilise les paramètres par défaut de l'imprimante.
    • Algorithme de mappage : détermine comment les couleurs sont imprimées à l'exception de la couleur sélectionnée dans Couleur d'accompagnement - Mappage. Ces sélections appliquent différents algorithmes pour des résultats différents.
    • Couleur d'accompagnement - Mappage : sélectionne une couleur spécifique comme couleur d'accompagnement.
    • Couleur d'accompagnement - Impression : imprime le document dans la couleur que vous sélectionnez.
    • Procédure en cas d'incompatibilité : définit comment gérer les incompatibilités entre la couleur sélectionnée et les fonctions couleur de l'imprimante.
