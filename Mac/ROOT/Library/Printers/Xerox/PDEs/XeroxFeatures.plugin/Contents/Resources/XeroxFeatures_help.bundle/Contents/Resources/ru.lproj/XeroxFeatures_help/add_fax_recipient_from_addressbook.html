Отправка факса получателю из телефонной книги

Получателей факса можно добавлять из телефонной книги.

Добавление получателя факса из телефонной книги:

  1. В окне «Бумага/Вывод» нажатием стрелок рядом с полем «Тип работы» выберите значение Факс.
  2. Нажмите кнопку Получатели.
  3. Нажмите пиктограмму адресной книги и выполните одно из следующих действий:
    • Выберите группу, затем получателя.
    • Чтобы найти получателя, введите его имя в поле для поиска и нажмите кнопку OK.
  4. Нажмите кнопку OK.
