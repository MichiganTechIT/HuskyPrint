Skapa block

Med Block kan du skapa block av enkelarksformulär, så att varje formulär kan rivas av. Det vanligaste är att ett block har ett bakre omslag i kartong och är limbundet i överkanten.

Så här skriver du ut block:

  1. Klicka på pilarna bredvid Efterbehandling i fönstret Papper/Utmatning och välj Block.
  2. Välj ett alternativ bredvid Block:
    • Har inaktiverats: Inaktiverar funktionen Skapa block.
    • Har aktiverats: Aktiverar blockutskrift.
  3. Välj ett av följande alternativ i Antal block:
    • Ange antalet block som ska skrivas ut.
    • Om du vill använda skrivarens grundinställningar, välj Skrivarens grundinställda typ.
  4. Välj ett av följande alternativ i Ark per block:
    • Ange antalet sidor för varje block.
    • Om du vill använda skrivarens grundinställningar, välj Skrivarens grundinställda typ.
  5. Välj som Bakre omslag typen av bakre omslag för blocken.
  6. Välj alternativ för Papper.
    • Annatformat: Här kan du ställa in pappersformat eller skapa ett eget format.
    • Annan färg: Här kan du välja färger från en lista eller egna färger som administratören anger.
    • Annan typ: Här kan du välja papperstyp eller en egen papperstyp som administratören anger.
  7. Klicka på OK.
