帐目

帐目将您的用户名和帐户标识信息与您打印的每份文档相关联。 将作业提交到打印机时，将收集和存储帐目数据。

注意：

  • 并非所有列出的选项在所有打印机上都受支持。 某些选项仅适用于特定打印机型号、配置、操作系统或驱动程序类型。
  • 根据您选择的帐目系统和帐目提示，并非所有选项都会出现。

设置帐目：

  1. 在“帐目”窗口中“帐目系统”的旁边单击箭头，然后选择一个选项。
    • Xerox® 标准帐目：跟踪每个用户的复印、打印、扫描和传真作业的数目。 它使用打印机中预定义的用户 ID 和帐户 ID。
    • Xerox® 网络帐目：从网络验证服务器中获取用户 ID 和帐户 ID。
    • 本地帐目：需要您的子网的帐目代码。 此帐目代码可以为部门开单代码，例如，0001。其他代码为可选。
    • 计费打印：需要打印作业的帐单 ID，例如您的部门的帐单 ID。 其他 ID 为可选。
    • 发送用户 ID 和帐目 ID：要求您键入用户 ID、帐目 ID 和密码来提交打印作业。
    • 只发送用户 ID：要求您键入用户 ID 和密码来提交打印作业。
    • 仅发送帐目 ID：要求您键入帐目 ID 和密码来提交打印作业。
    • 只发送帐单 ID：要求您键入帐单 ID 和密码来提交打印作业。
    • 发送全部：要求您键入用户 ID、帐目 ID、帐单 ID 和密码来提交打印作业。
  2. 在“帐目提示”旁边，选择一个选项。
    • 总是提示会在您每次提交打印作业时要求您输入用户 ID 和帐户 ID。
    • 不提示为每个打印作业使用您在“Account Codes”（帐户代码）下输入的信息。
    • 仅提示传真：在您每次发送传真时要求您输入用户 ID 和帐户 ID。
    • 仅提示彩色仅要求您为彩色打印作业输入用户 ID 和帐户 ID。
    • 仅提示彩色和传真仅要求您为彩色打印作业和传真键入用户 ID 和帐户 ID。
  3. 要在视图中隐藏用户 ID，请选择“屏蔽用户 ID”。 星号 * 将替换字段中键入的任何字符。
  4. 要在视图中隐藏帐户 ID，请选择“屏蔽帐户 ID”。 星号 * 将替换字段中键入的任何字符。
  5. 在“默认帐目代码”下，根据需要键入值。 如果选择“不提示”，则这些字段是必需的。 帐户 ID 和帐单 ID 可以为部门开单代码，例如，0001。
  6. 单击确定。
