Опции бумаги с именем

Бумага с именем — это набор параметров бумаги с присвоенным именем, который сохраняется для последующего использования. Бумагу с именем можно изменять и удалять только в списке бумаги пользователя.

Добавление бумаги с именем

  1. В окне «Бумага с именем» в верхнем меню выберите Список бумаги пользователя и нажмите кнопку Создать.
  2. В поле «Имя бумаги» выберите или введите имя.
  3. Чтобы использовать только бумагу с именем, выберите Исп. только имя (эксперт) и введите имя или выберите его в списке.

    Примечание: Если выбирается вариант Исп. только имя (эксперт), поля атрибутов становятся недоступны.

  4. Выполните одно из следующих действий.
    • Выберите параметры бумаги и настройки.
    • Для использования настроек по умолчанию выберите Настройка по умолч.
  5. Для добавления дополнительной информации об использовании бумаги с имением введите ее в поле «Комментарий (дополнительно)».
  6. Нажмите кнопку OK.

Копирование бумаги с именем

  1. В окне «Бумага с именем» в верхнем меню выберите Список бумаги пользователя.
  2. Выберите в таблице бумагу с именем для копирования и нажмите кнопку Копировать .

    Примечание: Кнопка становится доступна после выбора бумаги с именем в списке.

  3. В поле «Имя бумаги» выберите или введите имя.
  4. Чтобы использовать только бумагу с именем, выберите Исп. только имя (эксперт) и введите имя или выберите его в списке.
  5. Если выбирается вариант Исп. только имя (эксперт), поля атрибутов становятся недоступны.
  6. Выполните одно из следующих действий.
    • Сохраните параметры бумаги и настройки.
    • Выберите параметры бумаги и настройки.
    • Для использования настроек по умолчанию выберите Настройка по умолч.
  7. Для добавления дополнительной информации об использовании бумаги с имением введите ее в поле «Комментарий (дополнительно)».
  8. Нажмите кнопку OK.

Изменение бумаги с именем

  1. В окне «Бумага с именем» в верхнем меню выберите Список бумаги пользователя.
  2. Выберите в таблице бумагу с именем и нажмите кнопку Изменить .

    Примечание: Кнопка становится доступна после выбора бумаги с именем в списке.

  3. Чтобы использовать только бумагу с именем, выберите Исп. только имя (эксперт) и введите имя или выберите его в списке.

    Примечание: Если выбирается вариант Исп. только имя (эксперт), поля атрибутов становятся недоступны.

  4. Выполните одно из следующих действий.
    • Выберите параметры бумаги и настройки.
    • Для использования настроек по умолчанию выберите Настройка по умолч.
  5. Для добавления дополнительной информации об использовании бумаги с имением введите ее в поле «Комментарий (дополнительно)».
  6. Нажмите кнопку OK.

Удаление бумаги с именем

  1. В окне «Бумага с именем» в верхнем меню выберите Список бумаги пользователя.
  2. Выберите бумагу с именем для удаления и нажмите кнопку Удалить .
  3. Нажмите кнопку Закрыть.
