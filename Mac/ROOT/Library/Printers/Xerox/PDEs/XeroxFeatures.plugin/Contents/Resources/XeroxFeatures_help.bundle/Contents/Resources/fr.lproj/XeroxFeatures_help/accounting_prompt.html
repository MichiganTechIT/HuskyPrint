Invite de comptabilisation

Si la comptabilisation est activée et que l'invite de comptabilisation est définie sur travaux d'impression, de télécopie et d'impression couleur, l'invite de comptabilisation apparaît chaque fois que vous envoyez un de ces travaux.

Remarque : Toutes les options peuvent ne pas apparaître, selon votre système de comptabilisation et vos sélections d'invite de comptabilisation.

  1. Si une valeur par défaut est spécifiée pour l'ID utilisateur, vous pouvez l'utiliser ou taper une autre valeur.
  2. Si une valeur par défaut est spécifiée pour le code d'accès, vous pouvez l'utiliser ou taper un autre code d'accès.
  3. Sélectionnez un type de compte, le cas échéant.
  4. Si une valeur par défaut est spécifiée pour l'ID de compte, vous pouvez l'utiliser ou taper un autre ID de compte.
  5. Si une valeur par défaut est spécifiée pour le code d'accès ID de compte, vous pouvez l'utiliser ou taper un autre code d'accès.
