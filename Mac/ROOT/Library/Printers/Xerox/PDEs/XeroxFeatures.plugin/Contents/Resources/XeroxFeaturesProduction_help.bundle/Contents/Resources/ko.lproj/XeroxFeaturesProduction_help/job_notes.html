작업 메모

[작업 메모]에서 인쇄 작업에 추가 정보를 연결할 수 있습니다. 인쇄 작업 설정을 PDF 또는 PostScript로 저장한 후 필요에 따라 PDF를 미리 보거나 팩스 또는 메일로 보낼 수 있습니다.

참고: 일부 프린터에서는 여기 나열된 모든 옵션이 지원되지 않습니다. 일부 옵션은 특정 프린터 모델, 구성, 운영 체제 또는 드라이버 유형에만 적용됩니다.

작업 메모를 사용하여 인쇄:

  1. 작업 메모를 클릭합니다.
  2. 다음 작업 중 하나 이상을 수행하십시오.
    • 작업 이름을 선택합니다.

      작업 이름에는 인쇄 작업의 표지에 나타나는 문서의 이름이 포함됩니다. 문서 이름은 프린터의 인쇄 대기열에도 표시됩니다.

      작업 이름을 추가하려면 다음 중 하나를 수행합니다.

      • [작업 이름] 옆에 있는 화살표를 클릭한 후 문서 이름 사용을 선택합니다.
      • 오른쪽 화살표를 클릭한 후 이전에 입력한 작업 이름을 선택합니다.
      • 새로운 이름을 입력합니다.
    • 발신자 이름을 선택합니다.

      송신자 이름은 문서를 보낸 사람의 이름이고, 인쇄 작업 표지에 인쇄됩니다. 송신자 이름은 프린터의 인쇄 대기열에도 표시됩니다.

      • 오른쪽 화살표를 클릭한 후 로그인 이름 사용을 선택합니다.
      • 오른쪽 화살표를 클릭한 후 이전에 입력한 작업 이름을 선택합니다.
      • 새로운 이름을 입력합니다.
    • 수신자 이름을 선택합니다.

      송신자 이름은 문서를 받는 사람의 이름이고, 인쇄 작업 표지에 인쇄됩니다.

      • 오른쪽 화살표를 클릭한 후 로그인 이름 사용을 선택합니다.
      • 오른쪽 화살표를 클릭한 후 이전에 입력한 작업 이름을 선택합니다.

        참고: 이전에 입력한 이름의 목록을 지우려면 목록 지우기를 클릭합니다.

      • 새로운 이름을 입력합니다.
    • 배너 메시지를 입력합니다.

      배너 메시지를 사용하여 인쇄 작업의 배너 페이지에 포함할 모든 정보를 입력할 수 있습니다.

    • 작업 시작 메시지를 입력합니다.

      작업 시작 메시지를 사용하여 프린터 사용자에게 지침을 전달할 수 있습니다. 예를 들어, Print this file by 2:00 p.m.(오후 2시까지 이 파일 인쇄)을 사용할 수 있습니다.

    • 추가 계정 정보를 입력합니다.

      추가 계정 정보를 사용하여 인쇄 작업에 포함할 과금 또는 작업 관련 정보를 입력할 수 있습니다.

  3. 인쇄를 클릭합니다.
