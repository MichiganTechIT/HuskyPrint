Adicionar inserções

Adicione uma inserção em branco ou pré-impressa antes da primeira página do documento ou após as páginas especificadas de um documento. Por exemplo, para inserir um pedaço em branco de papel colorido após cada seção em um documento, observe os números das páginas nas quais cada seção termina. Use estes números de páginas para designar uma página de inserção para ser colocada após cada página especificada.

Nota: Nem todas as opções listadas são suportadas em todas as impressoras. Algumas opções são aplicáveis apenas a um modelo de impressora, configuração, sistema operacional ou tipo de driver específico.

Para adicionar uma inserção:

  1. Na janela Páginas especiais, clique em Adicionar inserções.
  2. Selecione o tipo de Opção de Inserção:
    • Antes da Página 1 coloca uma inserção antes da página 1.
    • Após página(s): coloca uma inserção após as páginas ou intervalos de páginas especificados.
  3. Caso selecione Após página(s), no campo Página(s), insira o número de página que vem antes de cada inserção.
  4. Para quantidade, selecione o número de folhas de inserção necessário.
  5. Para aceitar os padrões de trabalho para todas as configurações, clique em OK.
  6. Para selecionar o tamanho, cor ou tipo, clique nas setas ao lado de Papel e selecione opções.
  7. Clique em OK.
