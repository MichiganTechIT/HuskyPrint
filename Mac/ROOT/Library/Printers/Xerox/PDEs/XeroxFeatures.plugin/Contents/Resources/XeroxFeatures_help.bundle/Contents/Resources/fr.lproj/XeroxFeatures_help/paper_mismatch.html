Procédure de format papier inadapté

Si votre imprimante ne contient pas le type de papier voulu, utilisez l'option Procédure de format papier inadapté pour régler le format de l'image ou charger le format papier correct.

Pour sélectionner une procédure de format de papier inadapté :

  1. Dans la fenêtre Support/Réception, en regard de Support, cliquez sur les flèches et sélectionnez Procédure de format de papier inadapté.
  2. Sélectionnez une option.
    • Afficher sur l'imprimante une invite à charger le format support approprié : assure la correspondance des supports avant l'impression.
    • Mise à l'échelle automatique, puis impression sur les supports chargés : adapte le document au format du support chargé dans le magasin.
    • Pas de mise à l'échelle, mais impression sur les supports chargés : imprime sur des supports de formats différents.
    • Afficher sur l'imprimante une invite à charger le format support approprié : assure la correspondance des supports avant l'impression.
    • Ignorer la non-correspondance et imprimer sur les supports chargés : imprime sur des types de support qui sont différents.
