Neues Materialformat erzwingen

Diese Funktion druckt ein Dokument auf ein Materialformat, das anders als das Originalformat ist, ohne die Bildgröße, Proportionen oder das Layout zu beeinträchtigen.

  1. Im Fenster "Erweitert" neben "Neues Materialformat erzwingen" auf die Pfeile klicken und dann eine Option wählen.
  2. Zum Festlegen eines benutzerdefinierten Materialformats auf Benutzerdefiniert klicken.

    Hinweis: Wenn das Originalformat größer als das Ausgabeformat ist, wird das Bild für die Seite passend zugeschnitten.
