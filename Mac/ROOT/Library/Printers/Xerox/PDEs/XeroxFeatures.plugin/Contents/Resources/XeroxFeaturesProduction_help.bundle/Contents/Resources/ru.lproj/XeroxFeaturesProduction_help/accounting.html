Учет

Функция учета обеспечивает привязку каждого печатаемого документа к имени пользователя и идентификационным учетным данным. Данные учета работ собираются и сохранятся по мере поступления работ на принтер.

  1. Для доступа к настройкам учета в окне «Дополнительно» рядом с полем «Конфигурация» выберите Учет.
  2. Выберите один из вариантов в поле «Система учета».
    • Для настройки учета нажмите кнопку Включено.
    • Для отключения учета нажмите кнопку Нет.
  3. Выберите один из вариантов в поле «Сообщение системы учета».
    • Не выдавать сообщение — при отправке работы на печать указание о вводе идентификаторов пользователя и счета не выдается.
    • Всегда выдавать сообщение — при отправке каждой работы печати необходимо вводить идентификаторы пользователя и счета.
  4. При выборе варианта «Всегда выдавать сообщение»:
    1. Чтобы при печати работы идентификаторы пользователя и счета скрывались, установите флажки рядом с данными опциями.
    2. При необходимости введите значение в поле «ID пользователя по умолчанию».
    3. При необходимости введите значение в поле «ID счета по умолчанию» или код оплаты.
  5. Нажмите кнопку OK.
