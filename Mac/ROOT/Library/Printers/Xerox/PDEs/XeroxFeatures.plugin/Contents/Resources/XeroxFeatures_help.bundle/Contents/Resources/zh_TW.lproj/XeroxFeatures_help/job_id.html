工作 ID

工作 ID 會將您工作的詳細資料列印在標題頁上或在文件的邊界中。 在這裡所做的設定將會套用至所有列印工作，除非被「環保設定」的設定覆寫。

標題頁是列印有列印工作資訊的分隔頁。 標題頁中包含工作提交者的網路使用者名稱和文件名稱。 當使用標題頁時，此頁可分隔輸出槽中每個已完成的列印工作。 在某些印表機上，控制面板設定會覆寫列印驅動程式中選取的標題頁選項。

注意：

  • 環保設定提供停用標題頁和啟用在頁面邊界列印工作 ID 資訊的功能。
  • 並非所有印表機都支援所有列出的選項。 某些選項只適用於特定印表機機型、設定、作業系統或驅動程式類型。

如要選擇工作 ID 選項：

在「進階」視窗中，在「工作 ID」旁，按一下箭頭，然後選擇一個選項。

  • 停用工作 ID： 將不會列印任何工作 ID 資訊。
  • 在標題頁上列印 ID： 在標題頁上列印工作 ID。
  • 在邊界列印 ID - 僅第一頁： 會在文件第一頁的左上角邊界列印工作 ID。
  • 在邊界列印 ID - 所有頁面： 會在文件每一頁的左上角邊界列印工作 ID。
