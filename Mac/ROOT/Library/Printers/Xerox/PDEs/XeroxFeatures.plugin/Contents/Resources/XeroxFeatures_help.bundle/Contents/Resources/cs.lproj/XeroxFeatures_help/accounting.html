Účtování

Funkce účtování přidruží ke každému dokumentu, který vytisknete, vaše uživatelské jméno a identifikační informace o účtu. Účtovací data jsou shromažďována a ukládána současně s tím, jak jsou na tiskárnu odesílány úlohy.

Poznámky:

  • Ne všechny tiskárny podporují všechny uvedené volby. Některé volby se týkají pouze konkrétního modelu tiskárny, konfigurace, operačního systému nebo typu ovladače.
  • V závislosti na nastavení v částech Systém účtování a Výzva účtování se nemusí zobrazit všechny volby.

Nastavení účtování:

  1. V okně Účtování klikněte na šipky vedle položky Systém účtování a vyberte požadovanou možnost.
    • Standardní účtování Xerox®: Sleduje počet úloh kopírování, tisku, snímání a faxování u každého uživatele. Používá ID uživatelů a ID účtů, která jsou předdefinována v tiskárně.
    • Xerox® Účtování v síti: Získá ID uživatelů a ID účtů ze síťového ověřovacího serveru.
    • Místní účtování: Vyžaduje kód účtování pro vaši podsíť. Tento kód účtování může být účtovací kód oddělení, například 0001. Ostatní kódy jsou volitelné.
    • Zpoplatněný tisk: Vyžaduje k tisku úloh ID pro účtování, jako například ID pro účtování vašeho oddělení. Ostatní identifikátory jsou volitelné.
    • Posílat ID uživatele a ID účtování: Při odesílání tiskové úlohy vyžaduje zadání ID uživatele, ID účtování a hesla.
    • Posílat pouze ID uživatele: Při odesílání tiskové úlohy vyžaduje zadání ID uživatele a hesla.
    • Posílat pouze ID účtování: Při odesílání tiskové úlohy vyžaduje zadání ID účtování a hesla.
    • Posílat pouze ID pro účtování: Při odesílání tiskové úlohy vyžaduje zadání ID pro účtování a hesla.
    • Posílat vše: Při odesílání tiskové úlohy vyžaduje zadání ID uživatele, ID účtování, ID pro účtování a hesla.
  2. V části Výzva účtování vyberte požadovanou možnost.
    • Volba Vždy zobrazit výzvu požaduje zadání ID uživatele a ID účtu při každém odeslání tiskové úlohy.
    • Volba Nezobrazovat výzvu použije pro každou tiskovou úlohu informace zadané v části Kódy účtování.
    • Zobrazit výzvu jen pro fax: Požaduje zadání ID uživatele a ID účtu při každém odesílání faxu.
    • Volba Zobrazit výzvu jen pro barevný tisk požaduje zadání ID uživatele a ID účtu pouze pro barevné tiskové úlohy.
    • Volba Zobrazit výzvu jen pro barevný tisk a fax požaduje zadání ID uživatele a ID účtu pouze pro barevné tiskové úlohy a faxy.
  3. Chcete-li skrýt ID uživatele ze zobrazení, vyberte možnost Maskovat ID uživatele. Znaky zadané do tohoto pole budou nahrazeny hvězdičkami *.
  4. Chcete-li skrýt ID účtu ze zobrazení, vyberte možnost Maskovat ID účtu. Znaky zadané do tohoto pole budou nahrazeny hvězdičkami *.
  5. V části Výchozí kódy účtování zadejte potřebné hodnoty. Tato pole jsou povinná, pokud vyberete možnost Nezobrazovat výzvu. ID účtu a ID pro účtování může být účtovací kód oddělení, například 0001.
  6. Klikněte na tlačítko OK.
