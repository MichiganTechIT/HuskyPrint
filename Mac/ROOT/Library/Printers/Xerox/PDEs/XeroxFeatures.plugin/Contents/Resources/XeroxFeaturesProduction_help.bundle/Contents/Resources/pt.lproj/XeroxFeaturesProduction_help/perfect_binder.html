Encadernação perfeita

Utilize a Encadernação perfeita para imprimir e encadernar um livro em brochura. Ele embala e cola uma capa ao redor do corpo do livro. É possível também aparar o livro para o tamanho desejado.

Notas:

  • Esta opção aparece somente se você tiver o módulo de acabamento adequado instalado na sua impressora. Não aparece quando é selecionado capas, grampeamento, perfuração, encardenação, dobras ou layout de folhetos.
  • Nem todas as opções listadas são suportadas em todas as impressoras. Algumas opções são aplicáveis apenas a um modelo de impressora, configuração, sistema operacional ou tipo de driver específico.

Para utilizar a Encadernação perfeita:

  1. Na janela Papel/Saída, ao lado de Acabamento, clique nas setas e selecione Encadernação sem costura.
  2. Ao lado de Borda de encadernação, selecione uma opção:
    • Retrato - Esquerda: Configura a orientação para retrato e encaderna o livro na esquerda.
    • Retrato - Direita: Configura a orientação para retrato e encaderna o livro na direita.
    • Paisagem - Topo: Configura a orientação para paisagem e encaderna o livro no topo.
    • Paisagem - Base: configura a orientação para paisagem e encaderna o livro na base.
  3. Clique em Aparar, e selecione uma opção.
    • Não Aparar: Não apara as páginas.
    • Frente ou Face: Apara a borda oposta à borda de encadernação.
    • Sangramento Completo: Ativa o Tamanho para Aparar, permitindo escolher um tamanho para aparar. O Sangramento Completo permite que a imagem se estenda para a borda do papel.
  4. Para selecionar um tamanho para aparar, clique em Tamanho para Aparar, e selecione a partir de uma lista de tamanhos ou selecione Personalizar.
  5. Para selecionar o papel para a capa, selecione as seguintes opções na lista Capa.
    • Papel nomeado: Permite selecionar um papel nomeado de uma lista ou criar um novo Papel nomeado.
    • Outro tamanho: Essa opção permite selecionar o tamanho do papel ou criar um tamanho personalizado.
    • Outra cor: Permite selecionar de uma lista de cores disponíveis ou criar uma cor de papel personalizada.
    • Outro tipo: Permite selecionar o tipo de papel ou criar tipos de papel personalizados.
    • Outra gramatura: Permite selecionar a gramatura do papel ou criar uma gramatura de papel personalizada.
    • Opções de papel avançadas: permite selecionar atributos de papel avançados para a capa.
  6. Clique em OK.
