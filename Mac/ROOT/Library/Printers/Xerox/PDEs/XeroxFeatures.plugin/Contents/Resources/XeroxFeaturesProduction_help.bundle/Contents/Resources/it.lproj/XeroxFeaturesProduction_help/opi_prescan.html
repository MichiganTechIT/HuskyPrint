Prescansione OPI

Prescansione OPI (Open Prepress Interface) offre un'anteprima del lavoro di stampa.

Per selezionare le opzioni di Prescansione OPI:

Nella finestra Avanzate, accanto a Prescansione OPI, fare clic sulle frecce, quindi selezionare un'opzione.

  • Default stampante: utilizza le impostazioni stampante correnti.
  • No: disattiva la funzione.
  • Scansione: consente di individuare i file ad alta risoluzione e di utilizzare i percorsi dei file per collegarli al documento.
  • Scansiona e raccogli: consente di individuare i file ad alta risoluzione e di inserirli nel documento.
