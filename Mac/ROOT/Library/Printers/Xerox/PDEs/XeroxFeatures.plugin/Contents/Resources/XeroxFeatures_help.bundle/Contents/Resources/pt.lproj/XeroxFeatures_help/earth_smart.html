Configurações do Recurso Ecológico

As configurações do recurso ecológico permitem a você reduzir a poluição ambiental durante a impressão, reduzindo o uso de materiais e energia. Você pode selecionar as Configurações do Recurso Ecológico como o padrão para todos os trabalhos de impressão. Para acessar as Configurações do recurso ecológico, clique no botão Configurações do recurso ecológico.

O botão verde das Configurações do Recurso Ecológico encontra-se na parte inferior do driver de impressão. Use este botão para selecionar Configurações do Recurso Ecológico para todos os trabalhos de impressão.

Notas:

  • Nem todas as opções listadas são suportadas em todas as impressoras. Algumas opções são aplicáveis apenas a um modelo de impressora, configuração, sistema operacional ou tipo de driver específico.
  • Alterar as Configurações do Recurso Ecológico em um trabalho de impressão a partir de um aplicativo de software alterará as configurações para o trabalho atual somente. Alterar as Configurações do Recurso Ecológico no driver de impressão alterará as configurações padrão para todos os aplicativos.

Para selecionar as Configurações do Recurso Ecológico:

  1. Na janela Configurações do recurso ecológico, selecione opções conforme o necessário.
    • Para imprimir todos os trabalhos usando as duas faces do papel, selecione Definir impressão em 2 faces.
      • Impressão em 2 Faces imprime o trabalho em ambos os lados do papel para permitir a encadernação na borda longa do papel. As ilustrações a seguir mostram o resultado para documentos com orientação retrato e paisagem, respectivamente:

    • Para usar menos papel e toner ou tinta, selecione Jogo de Amostra.

      O tipo de trabalho Jogo de amostra imprime uma cópia de um trabalho de impressão de diversas cópias, antes da impressão das cópias restantes. Após revisar o jogo de amostra, você pode imprimir as cópias restantes ou excluí-las no painel de controle da impressora.

    • Para imprimir uma cópia de um trabalho de impressão de várias cópias e reter as cópias restantes na impressora, selecione Definir impressão de prova recurso ecológico como meu tipo de trabalho.
    • Para evitar que as páginas de identificação sejam impressas a cada trabalho de impressão, selecione Sem identificação ou Desligar páginas de identificação. Selecione uma opção:

      Uma página de identificação é uma folha separadora impressa com um trabalho de impressão. Uma página de identificação inclui o nome do usuário de rede do remetente do trabalho e o nome do documento. Ao usar uma página de identificação, a página separa cada jogo de impressão concluído no escaninho de saída. Em algumas impressoras, a configuração do painel de controle substitui a opção de página de identificação no driver de impressão.

      • Não imprimir ID do trabalho: impede que qualquer informação de identificação do trabalho seja impressa.
      • ID de impressão em margens - Primeira página apenas: imprime a identificação do trabalho na primeira página do documento na margem esquerda superior.
      • ID de impressão em margens - Todas as páginas: imprime a identificação do trabalho na margem superior esquerda de cada página do documento.
    • Para economizar toner ou tinta por meio de produzir imagens de menor qualidade, selecione Modo rascunho ou Definir o modo rascunho como minha qualidade de impressão.
  2. Clique em OK.
