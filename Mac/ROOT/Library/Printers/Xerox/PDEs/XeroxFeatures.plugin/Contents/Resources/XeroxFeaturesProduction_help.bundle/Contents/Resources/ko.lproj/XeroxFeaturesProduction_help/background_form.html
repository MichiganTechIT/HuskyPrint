배경 양식

배경 양식을 사용하면 프린터 하드 드라이브에 저장된 양식과 인쇄 작업 데이터를 병합할 수 있습니다. 배경 양식으로 작업을 인쇄하면 문서와 양식이 병합됩니다. 그 결과 얻게 되는 문서는 사전 인쇄 용지에 인쇄된 것과 비슷한 문서입니다.

배경 양식을 저장하려면 먼저 Microsoft Word와 같은 다른 응용 프로그램에서 이 양식을 만드십시오. 그런 다음 프린터 하드 드라이브에 배경 양식을 저장합니다.
  1. 배경 양식
  2. 전경 인쇄 작업
  3. 결과 인쇄 문서
  1. 배경 양식 만들기:
  1. 배경 양식을 클릭합니다.
  2. 양식 이름의 경우 다음 중 하나를 수행합니다.
    • 기존 배경 양식 목록을 표시하려면 [양식 이름] 옆에 있는 화살표를 클릭한 후 옵션을 하나 선택합니다.
    • 양식을 만들려면 이름을 입력합니다.
  3. 포그라운드 인쇄 작업에 배경 양식을 적용하려면 양식 페이지 적용에서, 다음 옵션 중 하나를 선택합니다.
    • 작업 전체는 포그라운드 인쇄 작업에 배경 양식의 페이지를 적용합니다. 이 프로세스는 전경 인쇄 작업의 모든 페이지가 인쇄될 때까지 반복됩니다.

    • 한 번: 포그라운드 인쇄 작업에 배경 양식의 페이지를 한 번만 적용합니다. 전경 인쇄 작업의 나머지 페이지는 배경 양식 없이 인쇄됩니다.

    • 한 번 그리고 반복: 포그라운드 인쇄 작업에 배경 양식의 페이지를 한 번 적용합니다. 그런 다음 배경 양식의 지정된 페이지에서 시작하여 전경 인쇄 작업의 모든 페이지가 인쇄될 때까지 반복합니다.

  4. X축과 Y축을 따라 페이지 이미지를 이동하려면 [이미지 이동]에 값을 입력합니다.
  5. 인치와 밀리미터 중에서 선택하려면 창 하단의 화살표를 클릭한 후 측정 단위를 선택합니다.
  6. 확인을 클릭합니다.
