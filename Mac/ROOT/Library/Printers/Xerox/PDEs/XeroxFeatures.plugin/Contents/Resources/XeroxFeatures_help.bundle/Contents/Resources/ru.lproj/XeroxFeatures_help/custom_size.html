Создание настраиваемого размера бумаги

  1. В окне «Настраиваемый формат» в поле «Ширина» введите значение ширины бумаги или выберите его с помощью стрелок.
  2. В поле «Высота» введите значение высоты бумаги или выберите его с помощью стрелок.
  3. Для указания единиц измерения нажатием стрелок выберите мм или дюйм в поле «Единицы».
  4. Нажмите кнопку OK.
