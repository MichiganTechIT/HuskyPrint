Sparat bakgrundsformulär

Med Bakgrundsformulär kan du sammanföra utskriftsjobbets data med formulär som är lagrade på skrivarens hårddisk. När du skriver ut ett jobb med ett bakgrundsformulär, sammanförs dokumentet och formuläret. Slutresultatet ser ut som om det var utskrivet på ett förtryckt formulär.

Innan du kan spara ett bakgrundsformulär måste du först skapa det i ett annat program, t.ex. Microsoft Word. Spara det sedan på skrivarens hårddisk.

Obs! Alla alternativ som visas stöds inte på alla skrivare. Vissa alternativ finns bara i en viss skrivarmodell, en viss konfiguration, ett visst operativsystem eller en viss drivrutinstyp.

Så här skriver du ut med ett bakgrundsformulär:

  1. Klicka på pilarna bredvid Jobbtyp i fönstret Papper/Utmatning och välj Sparat bakgrundsformulär.
  2. Välj ett namn för jobbet på något av följande sätt:
    • Ge jobbet ett namn.
    • Klicka på pilen till höger om fältet och välj ett namn i listan.
    • Klicka på pilen till höger om fältet och välj Använd dokumentets namn om du vill använda dokumentets namn.
  3. Klicka på OK.
