Toner transparente

Toner transparente pode fazer com que a sua imagem impressa pareça uma fotografia. Ou então pode ser utilizada para realçar uma imagem, um texto ou um gráfico.

Nota: Nem todas as opções listadas são suportadas em todas as impressoras. Algumas opções são aplicáveis apenas a um modelo de impressora, configuração, sistema operacional ou tipo de driver específico.

Para usar Toner transparente:

  1. Na janela Toner transparente, para Opções de transparência, selecione uma opção.
    • Padrão da impressora: usa as configurações atuais da impressora.
    • Preencher transparente: reveste a página inteira com toner transparente.
    • Somente imagens: Reveste somente objetos da página identificados como imagens.
    • Somente textos: reveste somente textos com toner transparente.
    • Somente gráficos: reveste somente gráficos com toner transparente.
    • Sem imagens (somente texto, gráficos): Reveste a página inteira, textos e gráficos com o toner transparente, omitindo as imagens.
    • Sem texto (somente imagens, gráficos): Reveste a página inteira, imagens e gráficos com o toner transparente, omitindo os textos.
    • Sem gráficos (somente imagens, texto): Reveste a página inteira, imagens e textos com toner transparente, omitindo os gráficos.
    • Suprimir transparente: impede o revestimento de qualquer parte da página com toner transparente.
    • Usar transparente embutido: A impressora usa as instruções encontradas na PDL para aplicar o toner transparente.

      A impressora usa a Linguagem de descrição de páginas (PDL) para criar uma página impressa. PDLs incluem PostScript (PS) e Printer Command Language (PCL).

  2. Dependendo do dispositivo, selecione uma das opções para especificar a quantidade de toner transparente.
    • Clique em Toner transparente novamente e selecione a quantidade de toner transparente a aplicar na sua seleção.
    • Na lista Quantidade de transparência, selecione a quantidade de toner transparente.
  3. Clique em OK.
