Autres paramètres

Pour définir d'autres paramètres :

  1. Dans la fenêtre Correction de couleur personnalisée, pour toute sélection, cliquez sur Autres paramètres.
  2. Pour utiliser les paramètres couleur de la fenêtre précédente, sélectionnez Utiliser les paramètres généraux.
  3. Pour spécifier la luminosité, le contraste, la saturation et la nuance, sélectionnez les options à l'aide des curseurs.
  4. Pour rétablir les paramètres système d'origine, cliquez sur Défaut.
  5. Cliquez sur OK.
