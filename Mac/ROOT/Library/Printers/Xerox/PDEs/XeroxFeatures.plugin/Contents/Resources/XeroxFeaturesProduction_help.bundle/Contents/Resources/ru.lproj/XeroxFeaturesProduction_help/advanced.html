Дополнительно

В окне «Дополнительно» можно выбрать дополнительные параметры печати. Выбранные в этом окне параметры применяются для всех работ, печатаемых с помощью драйвера принтера.

Для проверки версии установленного драйвера принтера перейдите вниз окна.

Порядок изображений в брошюре

В окне «Дополнительно» выберите один из вариантов в поле «Порядок изображений в брошюре».

  • Слева направо — макет брошюры, когда при ее складывании сгиб находится с левой стороны брошюры. «Слева направо» относится к направлению чтения текста.
  • Слева направо – установка такого макета брошюры, что при ее складывании сгиб находится с правой стороны. Данная опция применяется для языков, тексты на которых читаются справа налево, таких как арабский и фарси.

Повернуть изображение на 180 градусов

В окне «Дополнительно» выберите значение нажатием стрелок рядом с полем «Повернуть изображение на 180 градусов».

  • Настройка по умолч. — используются действующие настройки принтера.
  • Вкл. или Включено — автоматический поворот изображений на 180 градусов. При повороте изображений учитываются форматы бумаги, ориентация, направление подачи бумаги и выбранные опции финишной обработки.
  • Выкл. или Выключено — отключение данной функции.

Идентификация работы

Опции вывода

В окне «Дополнительно» выберите значение нажатием стрелок рядом с полем «Опции вывода».

  • Настройка по умолч. — печать выполняется с использованием настройки по умолчанию, заданной системным администратором с панели управления принтера.
  • Изображением вверх и в обратном порядке — страницы выводятся изображением вверх в убывающем порядке (3, 2, 1).
  • Изображением вниз (рекомендуется) – страницы выводятся изображением вниз в убывающем порядке (3, 2, 1).
  • Изображением вверх — страницы выводятся изображением вверх в возрастающем порядке (1, 2, 3).
  • Изображением вниз и в обратном порядке — страницы выводятся изображением вниз в возрастающем порядке (1, 2, 3).

Зеркальное изображение/Зеркальный вывод

В окне «Дополнительно» выберите один из вариантов в поле «Зеркальный вывод» или «Зеркальное изображение».

  • Включено или Да — переворот изображения страницы слева направо.
  • Выключено или Нет — отключение зеркального вывода.

Паспорт работы Xerox

Негативный отпечаток

В окне «Дополнительно» выберите значение нажатием стрелок рядом с полем «Негативный отпечаток».

  • Включено — инверсия светлых и темных областей.
  • Выключено — отключение данной функции.

Настроить плотность линий

Функция «Настроить плотность линий» служит для настройки яркости линий для работ печати.

Настройка плотности линий:

В окне «Дополнительно» выберите значение нажатием стрелок рядом с полем «Настроить плотность линий».

  • -2 пикселя — для работы печати используется наименьшая плотность линий.
  • -1 пиксель — для работы печати используется уменьшенная плотность линий.
  • Нет — для работы печати используется плотность линий, установленная на принтере по умолчанию.
  • +1 пиксель — для работы печати используется увеличенная плотность линий.
  • +2 пикселя — для работы печати используется наибольшая плотность линий.

Подстановка OPI

В окне «Дополнительно» выберите значение нажатием стрелок рядом с полем «Подстановка OPI».

  • Настройка по умолч. — используются действующие настройки принтера.
  • Включено — сервер печати обрабатывает комментарии OPI и заменяет в документе изображения с низким разрешением на изображения высокого разрешения.
  • Выключено — отключение функции подстановки OPI.

Предварительное сканирование OPI

В окне «Дополнительно» выберите значение нажатием стрелок рядом с полем «Предварительное сканирование OPI».

  • Настройка по умолч. — используются действующие настройки принтера.
  • Нет — отключение данной функции.
  • Сканирование — нахождение файлов высокого разрешения и использование путей к файлам для связывания их с документом.
  • Сканировать и собрать — нахождение файлов высокого разрешения и вставка их в документ.

В новый формат бумаги

  1. В окне «Дополнительно» выберите значение нажатием стрелок рядом с полем «В новый формат бумаги».
  2. Для указания настраиваемого формата бумаги выберите опцию Настройка пользователя.

    Примечание: Если размер оригинала больше формата отпечатка, часть изображения страницы обрезается.
