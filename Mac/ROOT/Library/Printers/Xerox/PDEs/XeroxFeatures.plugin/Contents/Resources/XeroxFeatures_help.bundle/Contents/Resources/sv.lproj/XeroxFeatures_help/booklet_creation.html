Häften

Med funktionen Häften skrivs två dokumentsidor ut på varje sida av papperet. Sidorna placeras i ordning och vänds så att de hamnar i nummerordning när de falsas. Därefter kan sidorna häftas så att de bildar ett häfte. Kom ihåg att häfteslayouten skiljer sig från layouten på ett dokument där varje sida skrivs ut på ena sidan av papperet. Också rad- och sidbrytningar skiljer sig åt.

Obs!

  • Alla alternativ som visas stöds inte på alla skrivare. Vissa alternativ finns bara i en viss skrivarmodell, en viss konfiguration, ett visst operativsystem eller en viss drivrutinstyp.
  • Med funktionen Häften kan du förminska bilden på sidan så att den kan skrivas ut på önskat pappersformat.

Så här skapar du ett häfte:

  1. Klicka på pilarna bredvid Efterbehandling för häften i fönstret Häften och välj ett alternativ.
    • Ingen häftesefterbehandling eller falsning: Utskriften varken häftas eller falsas.
    • Häftesfals eller Vikning: Utskriften viks eller falsas längs mittlinjen.
    • Häftesfals och klammer eller Vikning och klammer: Utskriften viks eller falsas längs mittlinjen och två klamrar placeras i falsen så att sidorna hålls ihop.
    • Inaktivera Häften: Alternativ för häften tillämpas inte på utskriften. Använd detta alternativ om utskriftsjobbet redan är formaterat som ett häfte.
  2. Om du vill att skrivaren ska skala en bild till ett nytt pappersformat klickar du på pilarna bredvid Skala till nytt pappersformat. Välj önskat pappersformat. Du anger ett anpassat pappersformat genom att klicka på Anpassat.
  3. Ställ in marginaler genom att klicka på pilarna bredvid Häftesmarginaler och välja ett alternativ.
    • Standard förminskar sidbilden så att dokumentet skrivs ut som ett häfte på ett standardpappersformat utan bildförlust. Om skrivaren inte kan skriva ut till kanten på papperet ska du välja Standard.
    • Ingen skriver ut dokumentet som ett häfte utan att sidbilderna förminskas. Detta kan medföra att bilden blir avklippt om det inte går att skriva ut till kanten på papperet.

      Obs! Exemplet visar två sidor med Letter-format som placerats på ett ark med Tabloid-format. I bilden till vänster används standardmarginaler. I bilden till höger används inga marginaler.

  4. Ange fästmarginal och förskjutning efter behov.
    • Fästmarginal: Anger bredden (i punkter) på mittmarginalen mellan sidbilderna. Skriv punktvärdet eller ange det med hjälp av pilarna. En punkt är 0,35 mm eller 1/72 tum.
    • Förskjutning: Anger hur mycket sidbilderna ska förskjutas utåt (i tiondelar av en punkt). Skriv förskjutningsvärdet eller ange det med hjälp av pilarna.
  5. Klicka på OK.
