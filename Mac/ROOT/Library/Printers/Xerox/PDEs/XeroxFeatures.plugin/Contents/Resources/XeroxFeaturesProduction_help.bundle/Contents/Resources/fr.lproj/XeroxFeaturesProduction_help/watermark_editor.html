Boîte de dialogue Créer/modifier filigrane

Un filigrane est une image ou un texte imprimé sous ou sur le texte et les images des pages d'un travail d'impression.

Remarque : Certains systèmes d'exploitation ne prennent pas en charge l'impression de filigranes.

Créer un filigrane texte

Un filigrane texte utilise les caractères saisis dans le champ Texte comme filigrane. Vous pouvez créer un filigrane texte à imprimer dans un document et sélectionner des options telles que la taille, le type et la couleur de la police, ainsi que l'angle du filigrane.

Pour créer un filigrane texte :

  1. Dans la fenêtre Cahier/Filigrane, sous Filigrane, cliquez sur les flèches et sélectionnez Nouveau.
  2. Dans la fenêtre Créer/Modifier filigrane, cliquez sur Texte.
  3. Dans le champ Texte, gardez le nom du filigrane ou tapez un nouveau texte.
  4. Pour mettre en forme le filigrane, cliquez sur Afficher les polices et sélectionnez vos options.
  5. Pour spécifier l'angle du filigrane sur la page, tapez ou sélectionnez un pourcentage ou modifiez l'angle à l'aide du curseur.
  6. Pour spécifier l'emplacement du filigrane sur la page, sélectionnez l'unité de mesure puis entrez les valeurs pour la position horizontale et la position verticale.
  7. Pour centrer le filigrane sur la page, cliquez sur Centrer.
  8. Cliquez sur OK.

Créer un filigrane image

Comme son nom l'indique, un filigrane image utilise une image comme filigrane. L'image peut être au format .bmp, .gif ou .jpg. Vous pouvez créer un filigrane image à imprimer dans un document et sélectionner des options, notamment la taille et la position du filigrane sur la page.

Pour créer un filigrane image :

  1. Dans la fenêtre Cahier/Filigrane, sous Filigrane, cliquez sur les flèches et sélectionnez Nouveau.
  2. Dans la fenêtre Créer/Modifier filigrane, cliquez sur Image.
  3. Tapez le chemin et le nom du fichier d'image ou cliquez sur Choisir l'image, puis recherchez et sélectionnez un fichier d'image. Cette option prend en charge les fichiers .bmp, .jpg et .gif.
  4. Pour mettre l'image à l'échelle, indiquez le taux d'agrandissement ou de réduction de l'image.
  5. Pour déplacer le filigrane qui se trouve au centre de la page, sélectionnez l'unité de mesure puis entrez les valeurs pour la position horizontale et la position verticale.
  6. Pour centrer le filigrane sur la page, cliquez sur Centrer.
  7. Cliquez sur OK.

Modifier un filigrane

Vous pouvez modifier un filigrane à l'aide de la boîte de dialogue Créer/Modifier filigrane.

Pour modifier un filigrane :

  1. Dans la fenêtre Cahier/Filigrane, sous Filigrane, cliquez sur les flèches et sélectionnez Modifier.
  2. Dans la fenêtre Créer/Modifier filigrane, modifiez les options sélectionnées selon vos besoins.
  3. Cliquez sur OK.

Supprimer un filigrane

  1. Dans la fenêtre Cahier/Filigrane, sous Filigrane, sélectionnez le filigrane à supprimer puis cliquez sur Supprimer.
  2. Cliquez sur Oui pour confirmer la suppression.
