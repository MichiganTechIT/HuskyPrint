Тип работы

В разделе «Тип работы» устанавливаются настройки печати работы. Можно задать пароль для работ печати, сохранять документы на принтере или указывать время для печати работы. Также можно отправлять работы факсом.

Примечание: Не все перечисленные опции поддерживаются всеми принтерами. Некоторые опции предназначены только для определенных моделей и конфигураций принтера, операционных систем и драйверов.

Обычная печать

При нормальной печати работа в принтере не сохраняется для последующего использования и для нее не требуется пароль. Обычная печать является типом работы по умолчанию. При обычной печати работы можно печатать сразу.

Использование обычной печати:

  1. В окне «Бумага/Вывод» нажатием стрелок рядом с полем «Тип работы» выберите значение Обычная печать.
  2. Выберите остальные настройки для работы печати.
  3. Нажмите кнопку Печать.

Пробный комплект

Функция "Пробный комплект" позволяет напечатать одну копию многокомплектной работы, прежде чем печатать оставшиеся копии. После проверки пробного комплекта можно распечатать остальные копии или удалить их с панели управления принтера.

Печать с использованием функции «Пробный комплект»:

  1. В окне «Бумага/Вывод» нажатием стрелок рядом с полем «Тип работы» выберите значение Пробный комплект.
  2. Нажмите кнопку Печать. Распечатается одна пробная копия.
  3. Если качество пробной копии будет приемлемым, вы можете распечатать остальные копии с панели управления принтера.
