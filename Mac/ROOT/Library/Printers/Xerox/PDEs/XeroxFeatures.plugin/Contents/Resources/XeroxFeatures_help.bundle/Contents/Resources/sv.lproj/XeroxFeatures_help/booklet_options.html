Alternativ för häften

  1. Välj ett alternativ för Häften.
  2. Du kan skala ett häfte till ett nytt format genom att klicka på pilarna bredvid Skala till nytt pappersformat och välja önskat format. Du skapar ett anpassat format genom att välja Anpassat.
  3. Ställ in marginaler genom att klicka på pilarna bredvid Häftesmarginaler och välja ett alternativ.
    • Standard förminskar sidbilden så att dokumentet skrivs ut som ett häfte på ett standardpappersformat utan bildförlust.
    • Ingen skriver ut dokumentet som ett häfte utan att sidbilderna förminskas. Detta kan medföra att bilden blir avklippt om det inte går att skriva ut till kanten på papperet.

  4. Ange fästmarginal och förskjutning efter behov.
    • Fästmarginal: Anger bredden (i punkter) på mittmarginalen mellan sidbilderna. Skriv punktvärdet eller ange det med hjälp av pilarna. En punkt är 0,35 mm eller 1/72 tum.
    • Förskjutning: Anger hur mycket sidbilderna ska förskjutas utåt (i tiondelar av en punkt). Skriv förskjutningsvärdet eller ange det med hjälp av pilarna.
  5. Klicka på OK.
