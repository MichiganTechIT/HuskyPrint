Contabilità

L'opzione Contabilità consente di associare nome utente e informazioni di identificazione del conto con ogni documento stampato. I dati di contabilità vengono raccolti e memorizzati all'invio dei lavori alla stampante.

  1. Per accedere alle impostazioni Contabilità, nella finestra Avanzate, accanto a Configurazione, fare clic su Contabilità.
  2. In Sistema di contabilità, selezionare un'opzione:
    • Per impostare la contabilità, fare clic su Abilitato.
    • Per disabilitare la contabilità, fare clic su No.
  3. Accanto a Richiesta contabilità, selezionare un'opzione.
    • Nessuna richiesta: non richiede di digitare un ID utente o un ID conto quando si invia un lavoro di stampa.
    • Richiedi sempre: richiede di inserire l'ID utente e l'ID conto ogni volta che si invia un lavoro di stampa.
  4. Se si seleziona Richiedi sempre:
    1. Per nascondere un ID utente o un ID conto quando si stampa il lavoro, accanto a queste opzioni, selezionare la casella di controllo.
    2. Se necessario, digitare un valore ID utente predefinito.
    3. Se necessario, digitare un ID conto predefinito o un valore codice di fatturazione.
  5. Fare clic su OK.
