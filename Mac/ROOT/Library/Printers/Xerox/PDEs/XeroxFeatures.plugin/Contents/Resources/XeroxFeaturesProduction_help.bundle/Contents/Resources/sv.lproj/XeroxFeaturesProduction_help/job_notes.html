Jobbanteckningar

Med Jobbanteckningar kan du bifoga ytterligare information till en utskrift. Du kan spara inställningarna för utskriftsjobbet som PDF eller PostScript och sedan granska, faxa eller mejla PDF-filen efter behov.

Obs! Alla alternativ som visas stöds inte på alla skrivare. Vissa alternativ finns bara i en viss skrivarmodell, en viss konfiguration, ett visst operativsystem eller en viss drivrutinstyp.

Skriva ut med Jobbanteckningar:

  1. Klicka på Jobbanteckningar.
  2. Gör minst ett av följande:
    • Välj ett Jobbnamn.

      Jobbnamnet inkluderar namnet på dokumentet som visas på försättssidan för utskriftsjobbet. Dokumentnamnet visas också i skrivarens utskriftskö.

      Gör något av följande om du vill lägga till ett Jobbnamn:

      • Klicka på pilarna bredvid Jobbnamn och välj Använd dokumentets namn
      • Klicka på pilen till höger och välj ett jobbnamn som angetts tidigare.
      • Skriv ett nytt namn.
    • Välj ett Avsändarnamn.

      Avsändarens namn är namnet på den person som sänder dokumentet och det skrivs ut på utskriftsjobbets försättsblad. Avsändarens namn visas också i skrivarens utskriftskö.

      • Klicka på pilen till höger och välj Använd inloggningsnamn.
      • Klicka på pilen till höger och välj ett jobbnamn som angetts tidigare.
      • Skriv ett nytt namn.
    • Välj ett Mottagarnamn.

      Mottagarens namn är namnet på den person som tar emot utskriftsjobbet och det visas på försättsbladet.

      • Klicka på pilen till höger och välj Använd inloggningsnamn.
      • Klicka på pilen till höger och välj ett jobbnamn som angetts tidigare.

        Obs! Klicka på Rensa lista när du vill rensa listan över namn som angetts tidigare.

      • Skriv ett nytt namn.
    • Skriv ett Meddelande på försättssidan.

      Använd Meddelande på försättssida när du vill skriva information som ska visas på utskriftsjobbets försättsblad.

    • Skriv ett Meddelande vid jobbstart.

      Använd Meddelande vid jobbstart när du vill skriva instruktioner till skrivaroperatören. Till exempel: Skriv ut den här filen före 14.00.

    • Skriv ytterligare kontoinformation.

      I Extra kontoinformation kan du skriva faktureringsinformation eller annan information som rör jobbet som du vill inkludera med utskriftsjobbet.

  3. Klicka på Skriv ut.
