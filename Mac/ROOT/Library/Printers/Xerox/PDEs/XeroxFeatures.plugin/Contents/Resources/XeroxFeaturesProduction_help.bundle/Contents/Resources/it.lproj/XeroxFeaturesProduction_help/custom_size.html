Creazione formato carta personalizzato

  1. Nella finestra Formato personalizzato, immettere un valore per la larghezza personalizzata o utilizzare le frecce per selezionare un valore.
  2. Accanto ad Altezza, immettere un valore per l'altezza personalizzata o utilizzare le frecce per selezionare un valore.
  3. Per specificare le unità di misura, nel campo Unità, fare clic sulle frecce, quindi selezionare poll. o mm.
  4. Fare clic su OK.
