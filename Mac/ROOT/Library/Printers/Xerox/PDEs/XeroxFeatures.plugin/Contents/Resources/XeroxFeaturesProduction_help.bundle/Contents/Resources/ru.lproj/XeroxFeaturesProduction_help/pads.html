Создание блока

Функция «Блоки» служит для создания однолистовых форм, которые при необходимости могут отрываться от блока. Обычно блок имеет картонную заднюю обложку и клееный переплет по верхней кромке.

Печать блоков:

  1. В окне «Бумага/Вывод» нажатием стрелок рядом с полем «Послепечатная обработка» выберите Блоки.
  2. Выберите один из вариантов в поле «Блоки».
    • Выключено — отключение функции «Создание блока».
    • Включено — включение печати блоков.
  3. Для параметра «Число блоков» выполните одно из следующих действий:
    • Введите количество блоков для печати.
    • Для использования настройки принтера по умолчанию выберите значение Настройка по умолч.
  4. Для параметра «Листов в блоке» выполните одно из следующих действий:
    • Введите число страниц в блоке.
    • Для использования настройки принтера по умолчанию выберите значение Настройка по умолч.
  5. В поле «Задняя обложка» выберите тип задней обложки для блоков.
  6. Выберите настройки в разделе «Бумага».
    • Бумага с именем — выбирается бумага в списке или создается новая бумага с именем.
    • Другой формат — формат бумаги выбирается в списке или создается пользовательский формат.
    • Другой цвет — выбирается цвет бумаги в списке или создается настраиваемый цвет бумаги.
    • Другой тип — выбирается тип бумаги или создается настраиваемый тип бумаги.
    • Другая плотность — выбирается плотность бумаги или создается пользовательская плотность бумаги.
    • «Выбор расширенных настроек бумаги» — выбираются дополнительные параметры бумаги для обложек.
  7. Нажмите кнопку OK.
