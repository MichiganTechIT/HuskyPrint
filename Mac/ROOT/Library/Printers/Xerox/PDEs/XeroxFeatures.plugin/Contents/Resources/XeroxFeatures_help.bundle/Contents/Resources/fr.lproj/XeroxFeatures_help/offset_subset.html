Décalage/Sous-jeu

L'option Résultat du décalage permet de contrôler le décalage des jeux dans le cadre de travaux d'impression requis en plusieurs exemplaires. Lorsqu'elle est activée, les jeux sont légèrement décalés les uns par rapport aux autres dans le bac récepteur. Cette option est disponible si le bac récepteur prend en charge le décalage.

Pour utiliser la fonction Décalage/Sous-jeu :

  1. Dans la fenêtre Support/Réception, en regard de Finition, cliquez sur les flèches et sélectionnez Décalage/Sous-jeu.
  2. Sélectionnez une option.
    • Aucun : désactive les décalages. Vous pouvez néanmoins toujours sélectionner des sous-jeux.
    • Chaque jeu : décale chaque jeu d'un travail imprimé en plusieurs exemplaires.
    • Chaque sous-jeu : décale un nombre de pages spécifié dans un même jeu.
    • Chaque jeu et chaque sous-jeu : décale un nombre de pages spécifié au sein d'un jeu, puis décale chaque jeu d'un travail imprimé en plusieurs exemplaires.
  3. Pour imprimer les sous-jeux d'un travail d'impression, sélectionnez Activer les sous-jeux.
  4. Dans Pages par sous-jeu, tapez le nombre de pages par sous-jeu. Tous les sous-jeux doivent contenir le même nombre de pages.
  5. Cliquez sur OK.
