Термопереплетное устройство

Термопереплетное устройство служит для печати и переплета книг в мягкой обложке. В нем книга заворачивается в обложку, которая затем приклеивается. Также возможна обрезка книги до нужного размера.

Примечания:

  • Данная опция отображается, если на принтере установлен соответствующий финишер. Она не отображается, если выбраны обложки, сшивание, перфорация, переплет, фальцовка или макет брошюры.
  • Не все перечисленные опции поддерживаются всеми принтерами. Некоторые опции предназначены только для определенных моделей и конфигураций принтера, операционных систем и драйверов.

Использование термопереплетного устройства:

  1. В окне «Бумага/Вывод» нажатием стрелок рядом с полем «Послепечатная обработка» выберите Термопереплет.
  2. Выберите один из вариантов в списке «Термопереплет».
    • Вертикальная - слева — вертикальная ориентация, переплет слева.
    • Вертикальная - справа — вертикальная ориентация, переплет справа.
    • Горизонтальная - вверху — горизонтальная ориентация, переплет сверху.
  3. Выберите бумагу для обложки в списке «Обложка» в следующих разделах.
    • Другой формат — выбирается формат бумаги в списке или создается настраиваемый формат.
    • Другой цвет — в списке выбирается один из стандартных цветов бумаги или настраиваемый цвет, созданный администратором.
    • Другой тип — выбирается один из стандартных типов бумаги или настраиваемый тип, созданный администратором.
    • Выбор по лоткам — выбор лотка для бумаги.
  4. Выберите Положение обложки.
  5. Для выбора способа печати обложек нажмите меню в разделе «Обложка» и выберите один из вариантов.
    • Пустые или с печатью — для обложек используются чистые листы или листы с печатью.
    • C печатью — печать обложки на выбранной бумаге.
  6. Нажмите кнопку Обрезка и выберите один из вариантов.
    • Без обрезки — обрезка листов не производится.
    • Fore или Лицевая сторона — обрезка выполняется по краю, противоположному кромке переплета.
    • Полные поля для обрезки — размер выбирается в меню «Размер обрезки». Это позволяет расширить изображение до краев листа.
  7. Для выбора размера обрезки нажмите кнопку Размер обрезки и выберите его в списке. Для выбора нестандартного размера выберите опцию Настройка пользователя.
  8. Для настройки обрезки отпечатков выберите Настройки обрезки.
  9. Нажмите кнопку OK.
