Tipo de trabajo

Tipo de trabajo controla cómo se imprime un trabajo. Se puede solicitar una clave para los trabajos de impresión, guardar un documento en la impresora o definir una hora específica para imprimir el trabajo de impresión. También puede enviar un fax.

Nota: No todas las opciones descritas están disponibles en todas las impresoras. Algunas opciones solo se aplican a un modelo de impresora, configuración, sistema operativo o tipo de controlador específico.

Impresión normal

Impresión normal no guarda los trabajos en la impresora para utilizarlos más adelante y no requiere una clave. Esta opción es el tipo de trabajo prefijado. Utilice la opción Impresión normal para imprimir un trabajo inmediatamente.

Para imprimir con la opción Impresión normal:

  1. En la ventana Papel/Salida, en Tipo de trabajo, haga clic en las flechas y, a continuación, seleccione Impresión normal.
  2. Realice otras selecciones para el trabajo de impresión según sea preciso.
  3. Haga clic en Impresión.
