Correção de cor personalizada

Cor automática personalizada abre a janela Correção de cor personalizada, que fornece aos usuários avançados um controle mais fino sobre os ajustes de cor.

Nota: Nem todas as opções listadas são suportadas em todas as impressoras. Algumas opções são aplicáveis apenas a um modelo de impressora, configuração, sistema operacional ou tipo de driver específico.

Para efetuar ajustes de cores:

  1. Execute um ou mais destes procedimentos:
    • Para Cor escritório, Press Match e Cor sólida, siga um destes procedimentos:
      • Selecione as opções do menu de perfil Texto e Gráficos e do menu Perfil de imagens.
      • Para ajustar os níveis de claridade, contraste, saturação e cor, clique em Mais ajustes.
    • Para Cor CIE, prossiga de uma das seguintes maneiras:
      • Selecione Sempre aplicar perfis incorporados.

        Sempre aplicar perfis incorporados instrui a impressora para usar os perfis incorporados do documento, com o risco de aplicar esquemas de cores não padrão. Usar perfis incorporados pode diminuir a qualidade de impressão. Em casos em que os dados forem incomuns, como ocorre com os dados de laboratórios, usar esquemas de cores proprietárias no perfil incorporado fornecerá a melhor qualidade de impressão.

        Um perfil incorporado é um Color Rendering Dictionary (CRD) incluído com os dados enviados do aplicativo, que descreve as características dos dados de cor.

      • Selecione Nunca aplicar perfis incorporados.

        Nunca aplicar perfis incorporados impede que a impressora use os perfis incorporados de um documento, o que reduz o risco de uso de esquemas de cores não padrão.

      • Para identificar e aplicar perfis de cores conhecidos no documento, selecione Aplicar automaticamente perfis incorporados quando apropriado. Essa opção ativa os menus de Texto, Gráficos e Imagens. Selecione as opções desejadas. Para efetuar alterações adicionais, clique em Mais ajustes.
        1. Selecione uma opção para Texto e gráficos:
          • Cor automática aplica a melhor correção de cores para texto, gráficos e imagens. Xerox recomenda esta configuração.
          • sRGB Vivo produz cores mais brilhantes e mais saturadas.
          • Preto e branco copia somente em preto e branco. As cores no original são convertidas em tons de cinza.
          • Colorimétrico tenta uma correspondência exata da cor. Este modo imita as condições de iluminação controlada.
          • Perceptivo fornece o melhor mapeamento para imagens fotográficas. É um mapeamento equilibrado que inclui ajustes subjetivos para cores da memória.
          • Saturação fornece o melhor mapeamento para gráficos. As cores são mais vivas e fortes, mas há menos fidelidade da cor de quando se usa a configuração Perceptivo.
        2. Selecione uma opção para Imagens. Consulte a etapa anterior para obter uma lista de opções.
        3. Para fazer ajustes adicionais à configuração de Imagens, clique em Mais ajustes.
      • Selecione as opções do menu de perfil Texto e Gráficos e do menu Perfil de imagens.
      • Para ajustar os níveis de claridade, contraste, saturação e cor, clique em Mais ajustes.
    • Para Espaço de cor cinza, prossiga de uma das seguintes maneiras:
      • Selecione Utilizar configurações de cor RGB para preto.

        Usar Configurações de cor RGB para Preto usa linhas cinzas para fazer a correspondência das configurações do preto do vermelho, verde e azul (RGB). Selecionar esta opção configura o campo Texto e gráficos e o campo Imagens às seleções atuais na guia Cor Office.

      • Selecione Utilizar configurações de cor CMYK para preto.

        Usar Configurações de cor CMYK para Preto usa canais de cinza para fazer a correspondência das configurações do ciano, magenta, amarelo e preto. Esta opção é a mais próximo do monocromático. Selecionar esta opção configura o campo Texto e gráficos e o campo Imagens às seleções atuais na guia Ajuste de impressão.

      • Para manipular configurações de cor preta, selecione Configurar a cor preta manualmente. Essa opção ativa os menus de Texto, Gráficos e Imagens. Selecione as opções desejadas. Para efetuar alterações adicionais, clique em Mais ajustes.
        1. Para fazer ajustes adicionais à configuração de Texto e gráficos, clique em Mais ajustes. Selecione uma opção para Texto e gráficos:
          • Gráficos monocromáticos é otimizada para fornecer um gradiente linear uniforme do preto para o branco.
          • Imagens monocromáticas possui uma curva de croma (gama) para resultados perceptivos mais nítidos em imagens, como um contraste melhor.
          • Comercial aplica a cor para impressão em uma impressora gráfica.
          • Impressão SWOP (especificações para a Impressora gráfica Web Offset) é um padrão de impressão profissional baseado em CMYK, geralmente usado na América do Norte.

            As cores ciano, magenta, amarelo e preto (CMYK) que a impressora usa para fazer todas as cores.

          • Impressão Euroscale corresponde à especificação de papel brilhante da FOGRA na Europa.

            FOGRA é um padrão de cor, projetado para ajudar a imprimir cores em tecnologias de impressão offset e digital comum.

          • Japan Color é a configuração de correção da cor para o Japão. Esta configuração corresponde à especificação Japan Color 2001.
          • ISO Revestido é uma emulação do padrão 27 da FOGRA para impressão em papel revestido. A impressão em papel revestido não é necessária, ao usar esta configuração.

            FOGRA é um padrão de cor, projetado para ajudar a imprimir cores em tecnologias de impressão offset e digital comum.

          • ISO Não Revestido é uma emulação do padrão 29 da FOGRA para impressão em papel não revestido.

            FOGRA é um padrão de cor, projetado para ajudar a imprimir cores em tecnologias de impressão offset e digital comum.

          • Impressão SNAP (especificações para a impressora Non-Heatset Advertising Printing - Impressão Não-a-quente de Anúncios) é um padrão de impressão de artes gráficas.
          • Tela sRGB faz com que a impressora emule as cores de uma exibição RGB calibrada, como uma tela de tubo de raios catódicos (CRT) ou de visor LCD de alta fidelidade.

            Configurações das cores vermelho, verde e azul (RGB).

          • Visor LCD aproxima as cores do trabalho de impressão às cores da tela do visor do computador.
          • sRGB Vivo produz cores mais brilhantes e saturadas usando o espaço de cor padrão vermelho, verde e azul.
          • Cor sólida mistura o ciano, magenta, amarelo e preto para se aproximar das cores de tinta sólida em uma impressora gráfica. Para alcançar a gama mais ampla e a melhor correspondência de espaço da cor da Pantone, selecione Cor sólida.
          • Nenhuma remove todas as correções de cores.
        2. Selecione uma opção para Imagens. Consulte a etapa anterior para obter uma lista de opções. Para fazer ajustes adicionais à configuração de Imagens, clique em Mais ajustes.
      • Selecione as opções do menu de perfil Texto e Gráficos e do menu Perfil de imagens.
      • Para ajustar os níveis de claridade, contraste, saturação e cor, clique em Mais ajustes.
  2. Para retornar às configurações originais do sistema, clique em Padrões.
  3. Clique em OK.
