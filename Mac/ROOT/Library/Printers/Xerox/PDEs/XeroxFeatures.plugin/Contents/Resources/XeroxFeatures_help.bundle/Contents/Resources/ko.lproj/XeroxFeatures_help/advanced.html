목차

고급

고급 창을 사용하여 고급 인쇄 옵션을 설정합니다. 이 창의 선택 항목은 인쇄 드라이버를 사용하여 인쇄하는 모든 작업에 적용됩니다.

설치된 인쇄 드라이버의 버전을 보려면 창 하단으로 스크롤합니다.

소책자 이미지 순서

고급 창의 [소책자 이미지 순서] 옆에 있는 옵션 중에서 하나를 선택합니다.

  • 왼쪽에서 오른쪽은 소책자를 접을 때 소책자의 왼쪽에 접기가 오도록 소책자를 설정합니다. 왼쪽에서 오른쪽은 텍스트의 읽는 방향을 나타냅니다.
  • 오른쪽에서 왼쪽으로를 선택하면 소책자를 접을 때 소책자 레이아웃이 오른쪽으로 제본되도록 설정됩니다. 아랍어나 페르시아어처럼 오른쪽에서 왼쪽으로 읽는 언어를 사용하는 경우 이 옵션을 선택합니다.

이미지 180도 회전

고급 창에서 [이미지 180도 회전] 옆에 있는 화살표를 클릭한 후 다음 옵션 중 하나를 선택합니다.

  • 프린터 기본값: 현재 프린터 설정을 사용합니다.
  • 설정 또는 사용: 프린터에서 이미지를 자동으로 180도 회전합니다. 이미지를 회전할 때 프린터에서 용지 크기, 방향, 용지 공급 방향 및 선택된 마감 옵션을 고려합니다.
  • 해제 또는 사용 안 함: 기능을 해제합니다.

작업 ID

고급 창에서 [작업 ID] 옆에 있는 화살표를 클릭한 후 다음 옵션 중 하나를 선택합니다.

  • 작업 ID 사용 안 함: 작업 ID 정보가 인쇄되지 않습니다.
  • 배너 시트에 ID 인쇄: 별도의 배너 페이지에 작업 ID를 인쇄합니다.
  • 여백에 ID 인쇄 - 첫 페이지만: 문서의 첫 번째 페이지 상단 왼쪽 여백에 작업 ID를 인쇄합니다.
  • 여백에 ID 인쇄 - 모든 페이지: 문서의 모든 페이지 상단 왼쪽 여백에 작업 ID를 인쇄합니다.

출력 옵션

고급 창에서 [출력 옵션] 옆에 있는 화살표를 클릭한 후 다음 옵션 중 하나를 선택합니다.

  • 프린터 기본값을 선택하면 시스템 관리자가 프린터 제어판에서 설정한 기본값을 사용하여 인쇄합니다.
  • 앞면을 위로 역순 배출은 내림차순(3, 2, 1)으로 페이지 앞면이 위를 향하도록 제공합니다.
  • 앞면을 아래로(권장함)는 내림차순(3, 2, 1)으로 페이지 앞면이 아래를 향하도록 제공합니다.
  • 앞면을 위로는 오름차순(1, 2, 3)으로 페이지 앞면이 위를 향하도록 제공합니다.
  • 앞면을 아래로 역순 배출은 내림차순(3, 2, 1)으로 페이지 앞면이 아래를 향하도록 제공합니다.

거울 이미지/좌우대칭 출력

고급 창에서 [좌우대칭 출력] 또는 [거울 이미지] 옆에 있는 옵션 중에서 하나를 선택합니다.

  • 설정 또는 예: 왼쪽에서 오른쪽으로 페이지 이미지를 반전시킵니다.
  • 사용 안 함 또는 아니요: 좌우대칭 출력을 해제합니다.

반전 출력

고급 창에서 [반전 출력] 옆에 있는 화살표를 클릭한 후 다음 옵션 중 하나를 선택합니다.

  • 사용: 밝은 영역과 어두운 영역을 반전시킵니다.
  • 사용 안 함: 반전 출력을 해제합니다.

간단하게 인쇄 모드

고급 창에서 [초안 이미지] 옆에 있는 화살표를 클릭한 후 다음 옵션 중 하나를 선택합니다.

  • 설정 또는 사용: 간단하게 인쇄합니다.
  • 해제 또는 사용 안 함: 간단하게 인쇄 모드를 해제합니다.

작업 오프셋/오프셋 요청

고급 창에서 [작업 오프셋] 또는 [오프셋 요청] 옆에 있는 화살표를 클릭한 후 다음 옵션 중 하나를 선택합니다.

  • 오프셋 각각 설정: 여러 부 인쇄 작업의 각 세트가 서로 약간 오프셋 상태가 됩니다.
  • 오프셋 없음: 인쇄 작업에 대해 여러 부 인쇄 오프셋을 비활성화합니다.

RGB 중립 회색

고급 창에서 [RGB 중립 회색] 옆에 있는 화살표를 클릭한 후 다음 옵션 중 하나를 선택합니다.

  • 프린터 기본값: 기본 프린터 설정을 사용합니다.
  • 자동 검정색을 선택하면 검정색 및 회색 텍스트와 라인 아트에 검정색 토너를 사용합니다. 그리고 사진과 그래픽의 검정색 및 회색 부분에는 혼합 검정색을 사용합니다.

    혼합 검정색은 사이안, 마젠타, 노란색 색조와 검정색 색조가 혼합된 색조입니다.

  • 순검정색 사용을 선택하면 텍스트를 포함한 페이지의 모든 검정색 및 회색 요소에 검정색 토너를 사용합니다.
  • 혼합 검정색 사용을 선택하면 텍스트, 라인 아트, 사진 및 그래픽의 검정색 및 회색 부분에 혼합 검정색을 사용합니다.

이미지 보강

고급 창에서 [이미지 보강] 옆에 있는 화살표를 클릭한 후 다음 옵션 중 하나를 선택합니다.

  • 자동을 선택하면 각 인쇄 모드에 맞게 속도와 품질의 균형이 가장 적절한 설정 모음을 적용합니다.
  • 켬을 선택하면 인쇄 품질을 향상시키기 위해 사용 가능한 모든 향상 옵션을 사용합니다. 이 옵션을 사용하는 작업은 인쇄 시간이 더 오래 걸립니다.
  • 끔을 선택하면 이미지 향상 옵션이 적용되지 않습니다.

검정색 트래핑/트래핑

고급 창에서 [트래핑] 옆에 있는 화살표를 클릭한 후 다음 옵션 중 하나를 선택합니다.

  • 설정 또는 사용: 컬러 영역의 가장자리가 검정색 영역의 가장자리와 겹쳐집니다. 이 설정은 흰색 영역에는 영향을 주지 않습니다.
  • 해제 또는 사용 안 함: 기능을 해제합니다.
  • 검정색만: 컬러 영역이 검정색 영역의 가장자리와 겹쳐집니다. 이 설정은 흰색 영역에는 영향을 주지 않습니다.

하프톤/프린터 하프톤 사용

고급 창에서 [프린터 하프톤 사용] 또는 [하프톤] 옆에 있는 화살표를 클릭한 후 다음 옵션 중 하나를 선택합니다.

  • 소프트웨어의 하프톤 관리 허용을 선택하면 응용 프로그램에서 하프톤 도트 패턴을 조정할 수 있습니다. 이 설정을 사용하여 낮은 해상도 인쇄를 사용할 때 나타나는 조건과 같은 여러 조건을 시뮬레이션할 수 있습니다. 이 설정은 인쇄 작업이 있는 프린터로 하프톤을 다운로드할 때 포스트스크립트 드라이버 하프톤, 프린터 응용 프로그램 하프톤 또는 둘 다를 사용합니다.
  • 프린터의 하프톤 관리 허용을 선택하면 프린터에 저장되어 있는 하프톤 도트 패턴이 사용됩니다.
  • 메뉴에서 값을 하나 선택합니다.

트루 그레이 그래픽

빈 페이지 건너뛰기

고급 창에서 [빈 페이지 건너뛰기] 옆에 있는 화살표를 클릭한 후 다음 옵션 중 하나를 선택합니다.

  • 설정: 인쇄 작업의 모든 빈 페이지를 무시합니다.
  • 해제: 인쇄 작업의 모든 빈 페이지를 포함합니다.

별색

고급 창에서 [별색] 옆에 있는 화살표를 클릭한 후 옵션을 선택합니다.

  • 소프트웨어의 별색 관리 허용을 선택하면 포스트스크립트 드라이버, 프린터 응용 프로그램 또는 둘 모두에서 인쇄를 위해 별색을 CMYK로 변환하는 작업을 제어할 수 있습니다. 특수 별색으로 예상 음영을 인쇄하지 못하는 경우 이 설정을 사용합니다.

    프린터에서 모든 컬러를 만드는 데 사용하는 사이안, 마젠타, 노란색, 검정색(CMYK) 컬러입니다.

  • 프린터의 별색 관리 허용을 선택하면 프린터에서 인쇄를 위해 별색을 CMYK로 변환하는 작업을 최적화할 수 있습니다.

    프린터에서 모든 컬러를 만드는 데 사용하는 사이안, 마젠타, 노란색, 검정색(CMYK) 컬러입니다.

제본변

고급 창에서 [테두리 지우기] 옆에 있는 화살표를 클릭한 후 다음 옵션 중 하나를 선택합니다.

  • 자동(권장)은 장변 또는 단변에서 인쇄 작업을 제본합니다(그러나 두 변 모두에서는 불가능).
  • 장변으로 제본은 페이지의 장변에서 인쇄 작업을 제본합니다.
  • 단변으로 제본은 페이지의 단변에서 인쇄 작업을 제본합니다.

광택 레벨

고급 창에서 [광택 레벨] 옆에 있는 화살표를 클릭한 후 다음 옵션 중 하나를 선택합니다.

  • 보통은 원본 광택 이미지를 무광택 버전으로 인쇄합니다.
  • 고급은 원본 광택 이미지를 복제합니다. 이 옵션을 사용하면 프린터의 인쇄 속도가 더욱 느려집니다.

용지함 5(수동 용지함) 급지변

고급 창에서 [용지함 5(수동 용지함) 급지변] 옆에 있는 화살표를 클릭한 후 다음 옵션 중 하나를 선택합니다.

  • 자동 선택을 선택하면 인쇄 작업에 필요한 크기와 종류의 용지가 공급된 용지함이 자동으로 선택되어 인쇄됩니다.
  • 장변 급지(일반)를 선택하면 용지가 프린터에 장변으로 보급됩니다.
  • 단변 급지를 선택하면 용지가 프린터에 단변으로 보급됩니다. 프린터 라벨에 이 설정을 사용하거나 페이지의 다른 부분에 스테이플을 사용하거나 천공 용지에 인쇄합니다.

인쇄 최적화

인쇄 최적화를 클릭한 후 다음 옵션 중 하나를 선택합니다.

  • 절약 최적화: 속도가 느리지만 소모품을 적게 사용하여 인쇄합니다.
  • 속도 최적화: 속도는 빠르지만 일반적인 양의 소모품을 사용하여 인쇄합니다.

해상도

고급 창에서 [해상도] 옆에 있는 화살표를 클릭한 후 해상도 옵션을 선택합니다.

얇은 선 향상

고급 창에서 [얇은 선 향상] 옆에 있는 화살표를 클릭한 후 다음 옵션 중 하나를 선택합니다.

  • 사용: 얇은 선에 두께를 추가합니다.
  • 해제: 얇은 선 향상을 해제합니다.

용지 보정

고급 창에서 [용지 보정] 옆에 있는 화살표를 클릭한 후 다음 옵션 중 하나를 선택합니다.

  • 사용: 인쇄 프로세스 속도를 낮춰서 인쇄 작업이 구김을 일으키지 않도록 합니다.
  • 사용 안 함(권장): 용지 보정 기능을 해제합니다.
