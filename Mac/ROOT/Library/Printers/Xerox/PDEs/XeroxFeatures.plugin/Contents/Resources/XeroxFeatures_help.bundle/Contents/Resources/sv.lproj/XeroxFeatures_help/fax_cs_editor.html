Bildredigeringsprogram för försättsblad

Du kan lägga till text eller bilder till faxförsättsblad eller använda text eller bilder som är lagrade i skrivaren.

  1. Så här lägger du till eller redigerar Text till ett försättslad.
    1. Välj Text i fönstret Bildredigeringsprogram för försättsblad.
    2. Om du vill formatera texten klickar du på Visa fonter och väljer önskat alternativ.
    3. Om du vill ange textens vinkel anger du ett värde under Vinkel. Använd pilarna om du vill justera värdet. Välj ett värde med reglaget.
    4. Du kan förskjuta vattenstämpeln från sidans mitt genom att välja måttenheten och sedan ange värdena för horisontell och vertikal position.
    5. Klicka på Centrera om du vill centrera bilden.
    6. Klicka på OK.
  2. Så här lägger du till eller redigerar en Bild till ett försättslad.
    1. Välj Bild i fönstret Bildredigeringsprogram för försättsblad.
    2. Skriv bildfilens sökväg bredvid Välj fil eller hämta en fil genom att klicka på Välj fil.

      Obs! Filer med formatet .bmp, .jpg eller .gif kan väljas.

    3. Du kan ändra bildens storlek genom att skriva eller välja en procentsats under Skala eller justera skalan med reglaget.
    4. Du kan förskjuta vattenstämpeln från sidans mitt genom att välja måttenheten och sedan ange värdena för horisontell och vertikal position.
    5. Klicka på Centrera om du vill centrera bilden.
    6. Klicka på OK.
