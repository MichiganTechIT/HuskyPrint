Lavori salvati

I lavori salvati sono documenti inviati alla stampante e memorizzati per l'utilizzo futuro.

Nota: Non tutte le opzioni elencate sono supportate su tutte le stampanti. Alcune opzioni si applicano solo a modelli di stampante, configurazioni, sistemi operativi o tipi di driver specifici.

Per stampare un lavoro salvato:

  1. Nella finestra Carta/Copie, accanto a Tipo di lavoro, fare clic sulle frecce, quindi selezionare Lavori salvati.
  2. Per selezionare un nome per il lavoro, eseguire una delle operazioni seguenti:
    • Digitare un nome per il lavoro.
    • Fare clic sulla freccia a destra del campo e poi selezionare un nome dall'elenco.
    • Fare clic sulla freccia a destra del campo e poi selezionare Usa nome documento per utilizzare questo valore.
  3. Selezionare un'opzione. È possibile salvare un lavoro oppure stampare e salvare un lavoro. Dopo averlo salvato, il lavoro può essere stampato quando è richiesto.
  4. Accanto a Formato, selezionare un'opzione di formato per il lavoro salvato.
    • Default stampante utilizzare il valore di default impostato per i formati file.
    • TIFF a pagina singola consente di salvare il lavoro di stampa in formato .tif. Ogni pagina del lavoro di stampa viene salvata come un file .tif separato.
    • TIFF multipagina consente di salvare il lavoro di stampa in formato .tif. Tutte le pagine del lavoro di stampa vengono salvate in un solo file .tif.
    • PDF consente di salvare il lavoro di stampa come un file .pdf.
    • Ristampa rapida consente di salvare il lavoro di stampa nel formato FRF (Fast Reprint Format).
    • Ristampa rapida con DRI consente di salvare il lavoro di stampa nel formato FRF (Fast Reprint Format) per abilitare la ristampa rapida. L'impostazione DRI (Display Resolution Image) consente di visualizzare o modificare il lavoro dal server di stampa.
  5. Fare clic su OK.
  6. Fare clic su Stampa.
