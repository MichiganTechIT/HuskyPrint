Ειδικές σελίδες

Οι Ειδικές σελίδες είναι εξώφυλλα, ένθετα και διαφορετικές σελίδες που προστίθενται σε μια εργασία εκτύπωσης. Οι ρυθμίσεις για τις διαφορετικές σελίδες διαφέρουν από τις ρυθμίσεις για τις υπόλοιπες σελίδες της εργασίας εκτύπωσης.

Διαχείριση ειδικών σελίδων

Οι ειδικές σελίδες που ρυθμίστηκαν προηγουμένως για τις εργασίες εκτύπωσης εμφανίζονται στον πίνακα περίληψης Ειδικές σελίδες. Από αυτόν τον πίνακα μπορείτε να επιλέξετε μια σελίδα για επεξεργασία ή διαγραφή. Επίσης, μπορείτε να αλλάξετε τη σειρά των ενθέτων σε μια ομάδα ενθέτων.

Για επεξεργασία, διαγραφή ή μετακίνηση καταχωρήσεων:

  1. Ορίστε μια επιλογή στη λίστα αποθηκευμένων εργασιών. Τα κουμπιά ενεργοποιούνται.
  2. Για να επιλέξετε την επιθυμητή ενέργεια, κάντε κλικ στα πλήκτρα.
