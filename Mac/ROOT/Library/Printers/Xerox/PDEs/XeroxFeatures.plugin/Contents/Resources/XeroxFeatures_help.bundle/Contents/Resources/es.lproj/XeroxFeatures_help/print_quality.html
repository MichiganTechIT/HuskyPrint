Calidad de impresión

Utilice esta opción para controlar el aspecto de una impresión en cuanto a color, registro, claridad y grado de similitud con el original.

Nota: No todas las opciones descritas están disponibles en todas las impresoras. Algunas opciones solo se aplican a un modelo de impresora, configuración, sistema operativo o tipo de controlador específico.

Para seleccionar una calidad de impresión:

En la ventana Papel/Salida, en Calidad de impresión, haga clic en las flechas y, a continuación, seleccione una opción.

  • Mejorado proporciona la mejor calidad de imagen, 525 x 450 ppp, para aplicaciones CAD y detalles finos. Se mejora la suavidad de líneas y bordes. Se mejora la riqueza del negro y de los colores oscuros.
  • Color rápido o Alta velocidad: es el modo a todo color más rápido e imprime a 188 x 400 ppp. Utilice este modo cuando tenga muchas imágenes y para una vista previa de su trabajo. Color rápido produce rápidamente documentos de revisión para trabajos urgentes. Se usa con papel de tamaño estándar en la bandeja 1. No se recomienda para documentos con texto pequeño, detalles finos o grandes zonas con relleno sólido brillante.
  • Alta resolución utiliza medios tonos e imprime a 525 x 400 ppp. Proporciona la mejor calidad de imagen para dibujos lineales con detalles, como por ejemplo archivos CAD. Se recomienda para impresiones en color vibrantes y saturadas o para fotos. Es posible que el trabajo de impresión tarde más en procesarse cuando se utiliza este modo de calidad de impresión.
  • Alta calidad: proporciona líneas y detalles finos. Se recomienda para impresiones en color vibrantes y saturadas.
  • Modo de fotografía proporciona el mejor detalle y suavidad para las fotografías.
  • Estándar produce impresiones nítidas y brillantes a gran velocidad y es perfecta para texto en negro. Esta opción proporciona la mejor relación entre rendimiento y calidad de imagen en la mayoría de los trabajos. Imprime los medios tonos a 300 x 450 ppp.

    Nota: Utilice siempre el modo Estándar para imprimir transparencias.

  • Ahorro de tóner: reduce la cantidad de tóner usado para imprimir los trabajos. Es una función útil para la salida con calidad de borrador. La calidad de impresión es ligeramente inferior a la de Estándar. Ahorro de tóner utiliza medios tonos e imprime a 600 x 600 ppp.
