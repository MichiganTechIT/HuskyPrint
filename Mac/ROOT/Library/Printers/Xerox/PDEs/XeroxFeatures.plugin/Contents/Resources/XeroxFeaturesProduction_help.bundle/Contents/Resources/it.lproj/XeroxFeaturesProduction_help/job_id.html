Identificazione lavoro

Identificazione lavoro consente di stampare i dettagli del lavoro entro i margini del documento.

Nota: Non tutte le opzioni elencate sono supportate su tutte le stampanti. Alcune opzioni si applicano solo a modelli di stampante, configurazioni, sistemi operativi o tipi di driver specifici.

Per selezionare le opzioni Identificazione lavoro:

Nella finestra Avanzate, accanto a Identificazione lavoro, fare clic sulle frecce, quindi selezionare un'opzione.

  • Disabilita ID lavoro: impedisce la stampa di qualsiasi informazione di identificazione del lavoro.
  • Stampa ID sui margini solo della prima pagina: consente di stampare l'identificazione del lavoro sulla prima pagina del documento nel margine in alto a sinistra.
  • Stampa ID sui margini di tutte le pagine: consente di stampare l'identificazione lavoro sul margine superiore sinistro di tutte le pagine del documento.
  • Utilizza pagine iniziali: consente di immettere i numeri di pagina che includono un ID lavoro. Il conteggio pagine inizia in corrispondenza della prima pagina stampata.
