Suspendre le travail

Suspendre le travail conserve le travail sur l'imprimante jusqu'à ce que vous le libériez à partir du panneau de commande. Le travail n'est pas enregistré sur le serveur d'impression après avoir été libéré pour impression.

Pour suspendre un travail d'impression :

  1. Dans la fenêtre Support/Réception, en regard de Type de travail, cliquez sur les flèches et sélectionnez Suspendre l'impression.
  2. Faites les autres sélections nécessaires pour votre travail d'impression.
  3. Cliquez sur Impression. Le travail reste sur le serveur d'impression jusqu'à sa libération pour impression.
