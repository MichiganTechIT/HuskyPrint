Configuration de l'impression protégée

Utilisez la fonction d'impression protégée pour imprimer des informations sensibles ou confidentielles. Après envoi, le travail est conservé sur l'imprimante jusqu'à ce que vous saisissiez votre code d'accès sur le panneau de commande de l'imprimante.

Remarque : Toutes les imprimantes ne prennent pas en charge la totalité des options mentionnées. Certaines options s'appliquent uniquement à un modèle d'imprimante, un système d'exploitation ou un type de pilote spécifique.

Pour imprimer à l'aide de l'option Impression protégée :

  1. Dans la fenêtre Support/Réception, en regard de Type de travail, cliquez sur les flèches et sélectionnez l'option Impression protégée.
  2. Entrez puis confirmez le code d'accès à 4-10 chiffres dans les champs correspondants. Lorsque vous envoyez un travail d'impression protégé, celui-ci est suspendu jusqu'à ce que vous le libériez à l'aide d'un code d'accès sur le panneau de commande.
  3. Tapez de nouveau le code d'accès dans le champ Confirmer le code.
  4. Cliquez sur OK.
  5. Cliquez sur Impression.
  6. Pour imprimer le travail, entrez le code d'accès sur le panneau de commande de l'imprimante.
