Creazione blocco note

L'opzione Blocchi note consente di creare blocchi note di moduli a foglio singolo, in modo da poter strappare ciascun modulo singolarmente. In genere un blocco note ha una copertina posteriore a cartoncino e una rilegatura incollata sul bordo superiore.

Per stampare blocchi note:

  1. Nella finestra Carta/Copie, accanto a Finitura, fare clic sulle frecce, quindi selezionare Blocchi note.
  2. Accanto a Blocchi note, selezionare un'opzione.
    • Disabilitato: consente di disattivare la funzione Creazione blocco note.
    • Abilitato: consente di attivare la stampa blocco note.
  3. Per N. blocchi note, eseguire una delle operazioni seguenti:
    • Inserire il numero di blocchi note da stampare.
    • Per utilizzare il default stampante, selezionare Default stampante.
  4. Per Fogli per blocco note, eseguire una delle operazioni seguenti:
    • Inserire il numero di pagine per ogni blocco note.
    • Per utilizzare il default stampante, selezionare Default stampante.
  5. Per Copertina posteriore, selezionare il tipo di copertina posteriore per i blocchi note.
  6. Per Carta, selezionare le opzioni.
    • Carta con nome: consente di selezionare una carta con nome dall'elenco o creare una nuova Carta con nome.
    • Altro formato: questa opzione consente di selezionare il formato carta o di creare un formato personalizzato.
    • Altro colore: consente di scegliere da un elenco di colori disponibili o di creare un colore personalizzato.
    • Altro tipo: consente di selezionare il tipo di carta o di creare tipi di carta personalizzati.
    • Altra grammatura: consente di selezionare la grammatura della carta o creare una grammatura personalizzata.
    • Opzioni carta avanzate: consente di selezionare attributi carta avanzati per la copertina.
  7. Fare clic su OK.
