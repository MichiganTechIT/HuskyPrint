Contabilidade

Contabilidade associa seu nome de usuário e as informações de identificação da conta a cada documento que imprimir. Os dados de contabilidade são coletados e armazenados à medida que os trabalhos são enviados à impressora.

Notas:

  • Nem todas as opções listadas são suportadas em todas as impressoras. Algumas opções são aplicáveis apenas a um modelo de impressora, configuração, sistema operacional ou tipo de driver específico.
  • Conforme seu Sistema de contabilidade e as seleções de Solicitação de contabilidade, nem todas as opções aparecem.

Para configurar a Contabilidade:

  1. Na janela Contabilidade, ao lado de Sistema de contabilidade, clique nas setas e selecione uma opção.
    • Xerox® Contabilidade padrão: controla os números dos trabalhos de cópia, impressão, digitalização e fax de cada usuário. Utiliza IDs de usuário e IDs de contabilidade pré-definidas na impressora.
    • Xerox® Contabilidade da rede: obtém IDs de usuário e IDs de contabilidade a partir de um servidor de autenticação em rede.
    • Contabilidade local: requer um código de contabilidade para a sua sub-rede. Esse código de contabilidade pode ser um código de faturamento do departamento, por exemplo, 0001. Outros códigos são opcionais.
    • Impressão de cobrança: requer uma ID de faturamento para os trabalhos de impressão, como uma ID de faturamento para o seu departamento. Outras IDs são opcionais.
    • Enviar ID de usuário e ID de contabilidade: requer que se digite a ID de usuário, a ID de contabilidade e a senha para enviar um trabalho de impressão.
    • Enviar apenas ID de usuário: requer que se digite a ID de usuário e senha para enviar um trabalho de impressão.
    • Enviar apenas ID de contabilidade: requer que se digite a ID de contabilidade e senha para enviar um trabalho de impressão.
    • Enviar apenas ID de faturamento: requer que se digite a ID de faturamento e senha para enviar um trabalho de impressão.
    • Enviar tudo: requer que se digite a ID de usuário, a ID de contabilidade, a ID de faturamento e a senha para enviar um trabalho de impressão.
  2. Selecione uma opção ao lado de Solicitação de contabilidade.
    • Sempre solicitar requer que você digite a sua ID de usuário e ID de conta toda vez que enviar um trabalho de impressão.
    • Não solicitar usa as informações que você digita em Códigos da conta para cada trabalho de impressão.
    • Solicitar apenas para Fax: requer que você digite a sua ID de usuário e ID de contabilidade toda vez que enviar um fax.
    • Solicitar apenas para cores requer que você digite a sua ID de usuário e ID de conta apenas para trabalhos de impressão em cores.
    • Solicitar apenas para cores e fax requer que você digite a sua ID de usuário e ID da conta apenas para trabalhos em cores de impressão e fax.
  3. Para ocultar a ID do Usuário, selecione Ocultar ID do Usuário. Os asteriscos * substituem qualquer caractere digitado no campo.
  4. Para ocultar a ID da Conta, selecione Ocultar ID da Conta. Os asteriscos * substituem qualquer caractere digitado no campo.
  5. Sob Códigos de contabilidade padrão, digite os valores conforme o necessário. Estes campos são necessários quando você seleciona Não solicitar. A ID de conta e ID de Cobrança podem ser um código de faturamento do departamento, por exemplo, 0001.
  6. Clique em OK.
