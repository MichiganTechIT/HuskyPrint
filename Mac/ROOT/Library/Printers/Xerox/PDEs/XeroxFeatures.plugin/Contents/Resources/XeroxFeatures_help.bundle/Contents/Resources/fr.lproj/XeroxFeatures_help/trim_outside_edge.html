Découpage bord extérieur

La fonction Découpage bord extérieur définit le découpage à partir du bord extérieur d'un cahier.

Pour rogner le bord extérieur :

  1. Pour activer le Découpage bord extérieur, sélectionnez la case à cocher correspondante.
  2. En regard de Découpage bord extérieur, tapez une valeur ou aidez-vous des flèches pour sélectionner une valeur de rognage du bord.
  3. Pour spécifier les unités de mesure, dans le champ Unités, cliquez sur les flèches et sélectionnez pouces ou mm.
  4. Cliquez sur OK.
