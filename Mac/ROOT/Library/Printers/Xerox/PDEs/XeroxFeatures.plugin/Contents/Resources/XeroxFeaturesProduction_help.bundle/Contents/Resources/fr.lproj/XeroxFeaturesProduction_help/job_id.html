Identification du travail

L'option Identification du travail permet d'imprimer les détails du travail dans les marges du document.

Remarque : Toutes les imprimantes ne prennent pas en charge la totalité des options mentionnées. Certaines options s'appliquent uniquement à un modèle d'imprimante, un système d'exploitation ou un type de pilote spécifique.

Pour sélectionner les options d'identification du travail  :

Dans la fenêtre Avancé, en regard de Identification du travail, cliquez sur les flèches et sélectionnez une option.

  • Désactiver ID travail : aucune information d'identification du travail n'est imprimée.
  • Imprimer ID dans la marge - 1ère page : imprime l'identification du travail sur la première page du document dans la marge, en haut à gauche.
  • Imprimer ID dans les marges – Toutes les pages : imprime l'identification du travail dans la marge, en haut à gauche de chaque page du document.
  • Utilisez des pages au début : permet d'entrer le nombre de pages comprenant un ID de travail. La pagination commence à la première page imprimée.
