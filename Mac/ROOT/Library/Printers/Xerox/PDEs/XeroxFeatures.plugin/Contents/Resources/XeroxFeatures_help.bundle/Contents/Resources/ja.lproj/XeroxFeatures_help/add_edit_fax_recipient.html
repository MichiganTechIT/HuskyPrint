ファクス受信者を追加または編集する

ファクス受信者のレコードを作成すると、レコードをパーソナル電話帳に追加できます。

受信者レコードを追加するには:

  1. [用紙/出力]ウィンドウの[ジョブタイプ]の隣にある矢印をクリックし、[ファクス]を選択します。
  2. [受信者]をクリックします。
  3. [受信者に追加]ボタンをクリックします。
  4. 連絡先情報を入力します。
  5. [OK]をクリックします。

電話帳のレコードを編集せずに、ファクス受信者を編集するには:

  1. [用紙/出力]ウィンドウの[ジョブタイプ]の隣にある矢印をクリックし、[ファクス]を選択します。
  2. [受信者]をクリックします。
  3. 一覧から受信者を選択して、[編集]をクリックします。
  4. 必要に応じて情報を変更します。
  5. [OK]をクリックします。
