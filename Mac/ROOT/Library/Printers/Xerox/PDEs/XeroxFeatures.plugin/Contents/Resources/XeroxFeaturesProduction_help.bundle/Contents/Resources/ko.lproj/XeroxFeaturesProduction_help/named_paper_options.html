명명된 용지 옵션

명명된 용지는 재사용하기 위해 명명되고 저장된 용지 특성의 집합입니다. 내 용지 목록에서만 명명된 용지를 수정하거나 삭제할 수 있습니다.

명명된 용지 추가

  1. 명명된 용지 창에서 상단 메뉴를 클릭하고, 내 용지 목록과 새로 만들기 버튼 을 차례로 클릭합니다.
  2. [용지 이름]에서 이름을 선택하거나 입력합니다.
  3. 용지 이름만 사용하려면 이름만 사용(전문가)을 선택한 후 이름을 입력하거나 목록에서 이름을 선택합니다.

    참고: 이름만 사용(전문가)을 선택한 경우 특성 필드를 사용할 수 없습니다.

  4. 다음 작업 중 하나를 수행하십시오.
    • 필요한 용지 특성 및 옵션을 선택합니다.
    • 프린터 기본값을 사용하려면 메뉴를 클릭하고 프린터 기본값을 선택합니다.
  5. 명명된 이 용지 사용에 대한 자세한 정보를 추가하려면 설명(옵션)에서 정보를 입력합니다.
  6. 확인을 클릭합니다.

명명된 용지 복사

  1. 명명된 용지 창에서 상단 메뉴를 클릭하고, 내 용지 목록을 선택합니다.
  2. 표에서 명명된 용지를 선택한 후 복사 버튼 을 클릭합니다.

    참고: 목록에서 명명된 용지를 선택하면 단추가 활성화됩니다.

  3. 용지 이름 필드에서 이름을 선택하거나 이름을 입력합니다.
  4. 용지 이름만 사용하려면 이름만 사용(전문가)을 선택한 후 이름을 입력하거나 목록에서 이름을 선택합니다.
  5. 이름만 사용(전문가)을 선택한 경우 특성 필드를 사용할 수 없습니다.
  6. 다음 작업 중 하나를 수행하십시오.
    • 용지 속성과 옵션을 유지합니다.
    • 필요한 용지 특성 및 옵션을 선택합니다.
    • 프린터 기본값을 사용하려면 메뉴를 클릭하고 프린터 기본값을 선택합니다.
  7. 명명된 이 용지 사용에 대한 자세한 정보를 추가하려면 설명(옵션)에서 정보를 입력합니다.
  8. 확인을 클릭합니다.

명명된 용지 편집

  1. 명명된 용지 창에서 상단 메뉴를 클릭하고, 내 용지 목록을 선택합니다.
  2. 표에서 명명된 용지를 선택한 후 편집 버튼 을 클릭합니다.

    참고: 목록에서 명명된 용지를 선택하면 단추가 활성화됩니다.

  3. 용지 이름만 사용하려면 이름만 사용(전문가)을 선택한 후 이름을 입력하거나 목록에서 이름을 선택합니다.

    참고: 이름만 사용(전문가)을 선택한 경우 특성 필드를 사용할 수 없습니다.

  4. 다음 작업 중 하나를 수행하십시오.
    • 필요한 용지 특성 및 옵션을 선택합니다.
    • 프린터 기본값을 사용하려면 메뉴를 클릭하고 프린터 기본값을 선택합니다.
  5. 명명된 이 용지 사용에 대한 자세한 정보를 추가하려면 설명(옵션)에서 정보를 입력합니다.
  6. 확인을 클릭합니다.

명명된 용지 삭제

  1. 명명된 용지 창에서 상단 메뉴를 클릭하고, 내 용지 목록을 선택합니다.
  2. 제거할 명명된 용지를 선택한 후 삭제 버튼 을 클릭합니다.
  3. 닫기를 클릭합니다.
