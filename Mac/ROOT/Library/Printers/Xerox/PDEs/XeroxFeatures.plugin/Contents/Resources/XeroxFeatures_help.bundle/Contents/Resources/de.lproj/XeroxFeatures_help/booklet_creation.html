Broschürenerstellung

Mit der Broschürenfunktion werden zwei Seiten auf jede Blattseite aufgedruckt. Die Seiten werden vom Druckertreiber so geordnet und ausgerichtet, dass sie nach dem Falten in der richtigen Reihenfolge erscheinen. Die Broschüre kann dann geheftet werden. Das Layout der Broschüre unterscheidet sich vom Layout von Dokumenten, bei denen auf jede Blattseite eine Seite aufgedruckt wird. Entsprechend ergeben sich Unterschiede beim Zeilen- und Seitenumbruch.

Hinweise:

  • Nicht alle aufgelisteten Optionen werden von allen Druckern unterstützt. Einige Optionen stehen nur für bestimmte Druckermodelle, Konfigurationen, Betriebssysteme oder Treibertypen zur Verfügung.
  • Mit der Broschürenfunktion können Dokumentseiten so verkleinert werden, dass sie auf das angeforderte Materialformat passen.

Broschüre erstellen:

  1. Im Fenster "Broschürenerstellung" neben "Broschürenendverarbeitung" auf die Pfeile klicken und dann eine Option wählen.
    • Keine Broschürenendverarbeitung oder Falzung: Mit dieser Funktion wird der Druckauftrag weder geheftet noch gefalzt.
    • Broschürenfalz oder Einbruchfalz: Falzt den Druckauftrag in der Mitte.
    • Einbruchfalz und Heftung oder Broschürenfalz und Heftung Mit dieser Funktion wird der Druckauftrag in der Mitte gefalzt und mit zwei Heftklammern auf dem Falz geheftet.
    • Broschürenlayout deaktivieren: Wendet keine Broschürenformatoptionen auf den Druckauftrag an. Diese Option verwenden, wenn der Druckauftrag bereits als Broschüre formatiert ist.
  2. Zum Einstellen des Druckers, um ein Bild auf ein neues Materialformat zu skalieren, neben "In neues Format einpassen" auf die Pfeile klicken. Das gewünschte Materialformat wählen. Zum Festlegen eines benutzerdefinierten Materialformats auf Benutzerdefiniert klicken.
  3. Zum Festlegen der Randgrößen neben "Broschürenränder" auf die Pfeile klicken und dann eine Option wählen.
    • Standard bewirkt, dass die Seite so verkleinert wird, dass das Dokument ohne Bildverlust als Broschüre gedruckt wird. Bietet der Drucker keine randlose Ausgabe, die Option Standard auswählen.
    • Aus: Dokument wird ohne Verkleinerung der Seiten als Broschüre gedruckt. Auf Druckern, auf denen randloser Druck nicht möglich ist, kann es am Rand zu Bildverlusten kommen.

      Hinweis: Das Beispiel zeigt zwei Seiten des Formats "Letter" auf einem Blatt des Formats "Tabloid". Das Bild links zeigt Standardränder. Beim Bild rechts wurde die Option "Aus" gewählt.

  4. Bundsteg und Falzausgleich wie gewünscht festlegen.
    • Bundsteg: Legt die Breite des Rands zwischen den Druckseiten in Punkt fest. Einen Wert in Punkt eingeben oder mithilfe der Pfeile auswählen. Ein Punkt entspricht 0,35 mm bzw. 1/72 Zoll
    • Falzausgleich: Legt in 1/10 Punkt fest, wie weit die Druckbilder je Seite nach außen verschoben werden sollen. Zum Festlegen, um wie viel die Seitenbilder verschoben werden sollen, einen Wert eingeben oder mithilfe der Pfeile auswählen.
  5. Auf OK klicken.
