Опции брошюры

  1. Выберите вариант брошюровки.
  2. Для масштабирования брошюры по новому размеру нажатием стрелок выберите значение размера рядом с полем «По новому размеру бумаги». Для создания нестандартного формата выберите опцию Настройка пользователя.
  3. Для установки размеров полей нажатием стрелок выберите значение рядом с полем «Поля брошюры».
    • Стандартный — уменьшение размера изображения страницы для печати документа в виде брошюры на бумаге стандартного формата без потери изображения.
    • Нет — печать документа в виде брошюры без уменьшения размера изображений страниц. Если принтер не печатает по всей ширине страницы, возможна обрезка изображения.

  4. Укажите сдвиг изображения и поле переплета.
    • Поле переплета — указывается центральное поле между изображениями страниц (в пунктах). Введите значение в пунктах или выберите его с помощью стрелок. Пункт равен 0,35 мм или 1/72 дюйма.
    • Сдвиг изображения — указывается сдвиг изображения наружу относительно переплета (в десятых долях пункта). Для указания сдвига изображений страниц введите значение или выберите его с помощью стрелок.
  5. Нажмите OK.
