Сдвиг изображения

Функция «Сдвиг изображения» служит для изменения положения изображения на странице. Изображение может сдвигаться по осям X и Y на каждом листе бумаги.

Настройка сдвига изображения:

  1. В поле «Опции сдвига изображения» выберите один из вариантов нажатием стрелок.
  2. Если в окне «Вывод бумаги» выбирается опция «1-сторонняя печать», следует выбрать один из вариантов в разделе «Опции сдвига изображения».
    • Настройка по умолч. — используются настройки принтера по умолчанию.
    • Независимо — установка настроек вручную.
  3. Если в окне «Вывод бумаги» выбирается опция «2-сторонняя печать», следует выбрать один из вариантов в разделе «Опции сдвига изображения».
    • Настройка по умолч. — используются настройки принтера по умолчанию.
    • Независимо — установка настроек вручную.
    • Зеркально по X — одновременный сдвиг изображений на одинаковое расстояние в обе стороны по оси X.
    • Зеркально по Y — одновременный сдвиг изображений на одинаковое расстояние в обе стороны по оси Y.
    • Зеркально по обеим осям — одновременный сдвиг изображений на одинаковое расстояние в обе стороны по осям X и Y.
    • Зеркально по X, вместе по Y — одновременный сдвиг изображений на одинаковое расстояние в обе стороны по оси X и в одном направлении по оси Y.
    • Зеркально по Y, вместе по X — одновременный сдвиг изображений на одинаковое расстояние в обе стороны по оси Y и в одном направлении по оси X.
    • Вместе по X — одновременный сдвиг изображений на одинаковое расстояние по обеим сторонам в одном направлении по оси X.
    • Вместе по Y — одновременный сдвиг изображений на одинаковое расстояние по обеим сторонам в одном направлении по оси Y.
    • Вместе по обеим осям — одновременный сдвиг изображений на одинаковое расстояние по обеим сторонам в одном направлении по осям X и Y.
  4. Для настройки «Независимо» выполните следующее:
    1. В поле «Сторона 1» введите величину сдвига изображения страницы по осям X и Y.
    2. В используется сторона 2, введите величину сдвига изображения страницы по осям X и Y.

      Примечание: Опция «Сторона 2» доступна только в режиме 2-сторонняя печать.

  5. Нажмите кнопку OK.
