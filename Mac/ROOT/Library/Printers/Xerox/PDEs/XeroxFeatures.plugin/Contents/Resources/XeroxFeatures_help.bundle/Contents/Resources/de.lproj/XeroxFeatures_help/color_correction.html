Benutzerdefinierte Farbkorrektur

Mit „Benutzerdefinierte automatische Farbe“ wird das Fenster „Benutzerdefinierte Farbkorrektur“ geöffnet, das erfahrenen Benutzern eine genauere Abstimmung der Farbeinstellungen ermöglicht.

Hinweis: Nicht alle aufgelisteten Optionen werden von allen Druckern unterstützt. Einige Optionen stehen nur für bestimmte Druckermodelle, Konfigurationen, Betriebssysteme oder Treibertypen zur Verfügung.

Farbeinstellung vornehmen:

  1. Einen oder mehrere der folgenden Schritte ausführen:
    • Für "Bürofarbe", "Wie Druckerpresse" und "Schmuckfarbe" einen der folgenden Schritte ausführen:
      • Optionen aus dem Menü "Profil für Text und Grafik" und dem Menü "Bildprofil" wählen.
      • Zum Einstellen der Helligkeit, des Kontrasts, der Sättigung und der Farbstufen auf Weitere Einstellungen klicken.
    • Für "CIE-Farbe" einen der folgenden Schritte ausführen:
      • Eingebettete Profile immer anwenden wählen.

        Eingebettete Profile immer anwenden bewirkt die Verwendung der im Dokument eingebetteten Profile, auch wenn dadurch vom Standard abweichende Farbschemata zur Anwendung kommen. Durch die Verwendung eingebetteter Profile kann die Druckqualität beeinträchtigt werden. Bei ungewöhnlichen Daten, zum Beispiel bei Labordaten, wird durch Verwendung der proprietären Farbschemata im eingebetteten Profil die beste Druckqualität erzielt.

        Ein eingebettetes Profil ist ein Color Rendering Dictionary (CRD), das zusammen mit den Daten aus der Anwendung übermittelt wird und die Eigenschaften der Farbdaten beschreibt.

      • Eingebettete Profile nie anwenden wählen.

        Eingebettete Profile nie anwenden verhindert, dass der Drucker die eingebetteten Profile eines Dokuments verwendet, wodurch das Risiko, dass vom Standard abweichende Farbschemata verwendet werden, reduziert wird.

      • Zum Identifizieren und Anwenden bekannter Farbprofile im Dokument Eingebettete Profile ggf. automatisch anwenden wählen. Diese Option aktiviert die Menüs "Text und Grafik" und "Bilder". Die gewünschten Optionen auswählen. Zum Vornehmen weiterer Änderungen auf Weitere Einstellungen klicken.
        1. Eine Option für „Text und Grafiken“ auswählen:
          • Automatische Farbe: In diesem Modus wird automatisch die optimale Farbkorrektur auf Text, Grafiken und Bilder angewendet. Xerox empfiehlt diese Einstellung.
          • Vivid: produziert strahlendere und kräftigere Farben.
          • Schwarzweiß, wenn die Kopien schwarzweiß ausgegeben werden sollen. Die Farben von Farbvorlagen werden in Grautöne umgewandelt.
          • Kolorimetrisch: Eine genaue Wiedergabe der Farbe wird versucht. Dieser Modus ahmt kontrollierte Beleuchtungsbedingungen nach.
          • Fotometrisch: bietet die beste Farbzuordnung für Fotos. Hierbei handelt es sich um eine ausgeglichene Zuordnung, die subjektive Einstellungen für Gedächtnisfarben umfasst.
          • Sättigung: bietet die beste Farbzuordnung für Grafiken. Die Farben wirken lebendiger und kräftiger, aber die Farbtreue ist geringer als bei er Einstellung „Fotometrisch“.
        2. Eine Option für „Bilder“ auswählen: Eine Liste von Optionen ist im vorigen Schritt zu finden.
        3. Um weitere Einstellungen für Bilder vorzunehmen, auf Weitere Anpassungen klicken.
      • Optionen aus dem Menü "Profil für Text und Grafik" und dem Menü "Bildprofil" wählen.
      • Zum Einstellen der Helligkeit, des Kontrasts, der Sättigung und der Farbstufen auf Weitere Einstellungen klicken.
    • Für "Graustufen-Farbraum" einen der folgenden Schritte ausführen:
      • Für Schwarz RGB-Farbeinstellungen verwenden wählen.

        Für Schwarz RGB-Farbeinstellungen verwenden bewirkt die Verwendung grauer Linien entsprechend den RGB-Schwarzeinstellungen (Rot, Grün, Blau). Wird diese Option gewählt, werden die Felder „Text und Grafiken“ und „Bilder“ auf die aktuellen Werte auf der Registerkarte „Bürofarbe“ eingestellt.

      • Für Schwarz CMYK-Farbeinstellungen verwenden wählen.

        Für Schwarz CMYK-Farbeinstellungen verwenden bewirkt die Verwendung von Graukanälen entsprechend den Cyan-, Magenta-, Gelb- und Schwarzeinstellungen. Diese Option kommt der Monochrom-Option am nächsten. Wird diese Option gewählt, werden die Felder „Text und Grafiken“ und „Bilder“ auf die aktuellen Werte auf der Registerkarte „Entsprechender Druck“ eingestellt.

      • Zum Ändern der Einstellungen für Schwarz Schwarz manuell konfigurieren wählen. Diese Option aktiviert die Menüs "Text und Grafik" und "Bilder". Die gewünschten Optionen auswählen. Zum Vornehmen weiterer Änderungen auf Weitere Einstellungen klicken.
        1. Um weitere Einstellungen für Text und Grafiken vorzunehmen, auf Weitere Anpassungen klicken. Eine Option für „Text und Grafiken“ auswählen:
          • Schwarzweißgrafiken: optimiert zur Erzeugung eines glatten linearen Übergangs von Weiß zu Schwarz.
          • Schwarzweißbilder: hat eine Gammakurve, die optimiert ist für schärfer wirkende Ergebnisse bei Bildern, zum Beispiel besseren Kontrast.
          • Commercial: Farben zum Drucken auf einem kommerziellen Drucksystem werden angewandt.
          • SWOP-Druck: „Specifications for Web Offset Printing Press“ ist ein professioneller, auf CMYK basierender Druckstandard, der in Nordamerika gängig ist.

            Die Farben Cyan, Magenta, Gelb und Schwarz (CMYK), die der Drucker zur Erzeugung aller Farben verwendet.

          • Druck nach Euroskala: entspricht der europäischen FOGRA-Spezifikation für Hochglanzpapier.

            FOGRA ist ein Farbstandard für das Drucken von Farben im Offset- und in gängigen Digitaldruckverfahren.

          • Japan Color ist die Farbkorrektureinstellung für Japan. Diese Einstellung entspricht der Spezifikation „Japan Color 2001“.
          • ISO gestrichen: eine Emulation des Standards „FOGRA 27“ zum Drucken auf gestrichenem Papier. Bei Wahl dieser Einstellung ist das Drucken auf gestrichenem Papier nicht zwingend.

            FOGRA ist ein Farbstandard für das Drucken von Farben im Offset- und in gängigen Digitaldruckverfahren.

          • ISO gestrichen: eine Emulation des Standards „FOGRA 29 zum Drucken auf gestrichenem Papier.

            FOGRA ist ein Farbstandard für das Drucken von Farben im Offset- und in gängigen Digitaldruckverfahren.

          • SNAP-Druck: „Specifications for Non-Heatset Advertising Printing Press“ ist ein Druckstandard für die Grafikbranche.
          • sRGB-Anzeige bewirkt die Emulation der Farben einer kalibrierten RGB-Anzeige, zum Beispiel eines CRT- oder eines hochentwickelten, abgestimmten LCD-Monitors, durch den Drucker.

            Farbeinstellungen für Rot, Grün, Blau (RGB).

          • LCD-Anzeige bewirkt die Annährung der Farben im Druckauftrag an die Farben des Computermonitors.
          • Vivid sRGB: produziert hellere und sattere Farben unter Verwendung des Rot-Grün-Blau-Standardfarbraums.
          • Schmuckfarbe bewirkt die Annährung an Schmuckfarben auf einem Drucksystem durch Mischung von Cyan, Magenta, Gelb und Schwarz. Zur Erzielung der breitesten Farbpalette und der besten Übereinstimmung mit dem Pantone-Farbraum die Option „Schmuckfarbe“ wählen.
          • Aus: Farbkorrektur deaktiviert.
        2. Eine Option für „Bilder“ auswählen: Eine Liste von Optionen ist im vorigen Schritt zu finden. Um weitere Einstellungen für Bilder vorzunehmen, auf Weitere Anpassungen klicken.
      • Optionen aus dem Menü "Profil für Text und Grafik" und dem Menü "Bildprofil" wählen.
      • Zum Einstellen der Helligkeit, des Kontrasts, der Sättigung und der Farbstufen auf Weitere Einstellungen klicken.
  2. Zum Zurückkehren zu den ursprünglichen Systemeinstellungen auf Standardwerte klicken.
  3. Auf OK klicken.
