Нейтральные серые RGB

Функция «Нейтральные серые RGB» служит для указания способа печати серых участков без заметного цветного оттенка.

Использование функции «Нейтральные серые RGB»:

В окне «Дополнительно» выберите значение нажатием стрелок рядом с полем «Нейтральные серые RGB».

  • Настройка по умолч. — используются настройки принтера по умолчанию.
  • Автоматически черный – для черного и серого текста и штриховой графики. Для черных и серых участков фотографий и изображений используется композитный черный цвет.

    Составной черный – смесь черного тонера с голубым, пурпурным и желтым.

  • Использовать чистый черный – для всех черных и серых элементов на странице, включая текст, используется только черный тонер.
  • Использовать композитный черный – для черных и серых участков фотографий, изображений, штриховой графики и текста используется композитный черный цвет.
