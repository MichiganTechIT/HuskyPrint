Optimisation image

La fonction d'amélioration de l'image permet à l'imprimante de déterminer quand appliquer les options Amélioration des bords, Amélioration des traits fins ou Recouvrement à une image.

Pour sélectionner les options d'amélioration de l'image :

Dans la fenêtre Avancé, en regard de Amélioration de l'image, cliquez sur les flèches et sélectionnez une option.

  • Automatique : applique un jeu de paramètres offrant le meilleur compromis entre performances et qualité pour chaque mode d'impression.
  • Activé(e) : applique toutes les améliorations disponibles pour une qualité d'impression optimisée. L'impression des travaux est plus lente lorsque cette option est sélectionnée.
  • Désactivé(e) : aucune amélioration image n'est appliquée.
