Contabilidad

Contabilidad asocia la información de identificación de nombre de usuario y cuenta con cada documento que se imprime. Los datos de contabilidad se recopilan y guardan cuando se envían trabajos a la impresora.

  1. Para acceder a las opciones de contabilidad, en la ventana Avanzadas, junto a Configuración, haga clic en Contabilidad.
  2. En Sistema de contabilidad, seleccione una opción.
    • Para configurar la contabilidad, haga clic en Activada.
    • Para desactivar la contabilidad, haga clic en No.
  3. Junto a Indicador de contabilidad, seleccione una opción.
    • No indicar: no solicita que escriba una ID de usuario o una ID de cuenta al enviar el trabajo de impresión.
    • Indicar siempre: requiere la introducción de las ID de usuario y de cuenta cada vez que se envía un trabajo de impresión.
  4. Si selecciona Indicar siempre:
    1. Para ocultar la ID de usuario o la ID de cuenta cuando imprime el trabajo, junto a estas opciones, seleccione la casilla de verificación.
    2. Si es preciso, escriba un valor en ID de usuario prefijada.
    3. Si es preciso, escriba una ID de cuenta prefijada o el código de facturación.
  5. Haga clic en Aceptar.
