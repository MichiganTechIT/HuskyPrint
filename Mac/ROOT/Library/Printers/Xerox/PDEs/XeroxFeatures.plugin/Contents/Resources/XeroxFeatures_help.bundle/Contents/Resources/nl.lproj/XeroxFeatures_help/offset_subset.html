Staffelen/subset

Met Gestaffelde aflevering bepaalt u hoe exemplaren worden verschoven bij afdrukopdrachten die uit meerdere exemplaren bestaan. Indien de optie is ingeschakeld, worden sets afdrukken iets verschoven van elkaar in de opvangbak afgeleverd. Deze optie is beschikbaar als de opvangbak over staffelmogelijkheden beschikt.

Staffelen/subset gebruiken:

  1. Klik in het venster Papier/aflevering op de pijlen naast Afwerking en selecteer vervolgens Staffelen/subsets.
  2. Selecteer een optie.
    • Geen: hiermee schakelt u staffelen uit. U kunt nog wel subsets selecteren.
    • Elke set: elke set van een afdrukopdracht die uit meerdere sets bestaat, wordt gestaffeld.
    • Elke subset: een specifiek aantal pagina's in een enkele set wordt gestaffeld.
    • Elke set en elke subset: een specifiek aantal pagina's binnen een set wordt gestaffeld en elke set van een afdrukopdracht die uit meerdere sets bestaat, wordt gestaffeld.
  3. Als u subsets wilt afdrukken binnen één afdrukopdracht, selecteert u Subsets inschakelen.
  4. Typ bij In Pagina's per subset een aantal pagina's dat in elke subset van het document moet worden opgenomen. Elke subset moet uit hetzelfde aantal pagina's bestaan.
  5. Klik op OK.
