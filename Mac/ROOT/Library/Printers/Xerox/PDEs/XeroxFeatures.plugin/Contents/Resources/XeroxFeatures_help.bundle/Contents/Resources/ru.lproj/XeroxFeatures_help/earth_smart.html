Настройки рационального режима

Настройки рационального режима позволяют сократить загрязнение окружающей среды за счет меньшего расхода материала и энергии при печати. Настройки рационального режима можно установить по умолчанию для всех работ печати. Для доступа к настройкам рационального режима нажмите кнопку Настройки рационального режима.

Кнопка «Настройки рационального режима» — это зеленая кнопка, расположенная в нижней части окна драйвера принтера. Эта кнопка служит для выбора настроек рационального режима для всех работ печати.

Примечания:

  • Не все перечисленные опции поддерживаются всеми принтерами. Некоторые опции предназначены только для определенных моделей и конфигураций принтера, операционных систем и драйверов.
  • При изменении настроек рационального режима для работы печати, отправляемой из программного приложения, изменяются настройки только для данной работы печати. При изменении настроек рационального режима в драйвере принтера изменяются настройки по умолчанию для всех приложений.

Выбор настроек рационального режима:

  1. Выберите настройки в окне «Настройки рационального режима».
    • Для печати всех работ на обеих сторонах листа выберите опцию Установить 2-стороннюю печать.
      • 2-сторонняя печать – печать работы на обеих сторонах листа с возможностью брошюровки по длине. На рисунках ниже показаны результаты печати для вертикальной и горизонтальной ориентаций соответственно.

    • Для экономии бумаги и тонера или чернил выберите опцию Пробный комплект.

      Функция "Пробный комплект" позволяет напечатать одну копию многокомплектной работы, прежде чем печатать оставшиеся копии. После проверки пробного комплекта можно распечатать остальные копии или удалить их с панели управления принтера.

    • Чтобы печаталась одна копия многокомплектной работы, а печать оставшихся копий задерживалась, выберите Установить пробную печать в рациональном режиме для типа работы.
    • Для того чтобы титульные листы не печатались с каждой работой, выберите опцию Без титульных листов или Отключить титульные листы. Предусмотрены следующие варианты:

      Титульный лист представляет собой страницу-разделитель, которая печатается вместе с работой. На титульном листе указывается сетевое имя пользователя, отправившего работу, и имя документа. Если используются титульные листы, они разделяют полные комплекты напечатанных работ в выходном лотке. На некоторых принтерах с панели управления можно изменить опцию титульного листа, выбранную в настройках драйвера принтера.

      • Не печатать ID работы — идентификационная информация о работе не печатается.
      • Печать ID на полях - только на первой стр. — идентификационная информация для работы печатается на первой странице документа, в верхней части левого поля.
      • Печать ID на полях - на всех стр. – идентификационная информация для работы печатается на всех страницах документа, в верхней части левого поля.
    • Для экономии тонера или чернил за счет снижения качества изображений выберите опцию Режим черновика или Установить черновой режим для качества печати по умолчанию.
  2. Нажмите кнопку OK.
