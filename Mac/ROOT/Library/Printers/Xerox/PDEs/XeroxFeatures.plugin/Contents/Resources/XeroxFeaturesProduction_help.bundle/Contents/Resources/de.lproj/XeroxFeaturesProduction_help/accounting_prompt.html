Kostenzählungsaufforderung

Wenn die Kostenzählung aktiviert ist und die Kostenzählungsaufforderung auf "Immer" gesetzt ist und ein Druckauftrag übermittelt wird, erscheint die Kostenzählungsaufforderung.

  1. Wenn als Benutzerkennung ein Standardwert festgelegt ist, kann dieser verwendet werden, oder es kann ein neuer Wert eingegeben werden.
  2. Wenn als Kostenstellennummer ein Standardwert festgelegt ist, kann dieser verwendet werden, oder es kann eine neue Kostenstellennummer eingegeben werden.
  3. Auf OK klicken.
