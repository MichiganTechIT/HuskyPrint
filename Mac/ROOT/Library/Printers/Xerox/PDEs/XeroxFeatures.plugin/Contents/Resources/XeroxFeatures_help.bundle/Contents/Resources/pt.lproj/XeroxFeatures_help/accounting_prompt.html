Solicitação de contabilidade

Se Contabilidade estiver ativada e Solicitação de contabilidade estiver configurada para solicitar trabalhos de impressão, fax e impressão em cores, a Solicitação de contabilidade é exibida quando se envia um desses trabalhos.

Nota: Conforme seu Sistema de contabilidade e as seleções de Solicitação de contabilidade, nem todas as opções aparecem.

  1. Para ID de usuário, se um valor padrão estiver especificado, é possível usá-lo ou digitar um novo valor.
  2. Para Senha, se um valor padrão estiver especificado, é possível usá-lo ou digitar uma nova senha.
  3. Se necessário, selecione um tipo de conta.
  4. Para ID de contabilidade, se um valor padrão estiver especificado, é possível usá-lo ou digitar uma nova ID de contabilidade.
  5. Para Senha de ID de conta, se um valor padrão estiver especificado, é possível usá-lo ou digitar uma nova senha.
