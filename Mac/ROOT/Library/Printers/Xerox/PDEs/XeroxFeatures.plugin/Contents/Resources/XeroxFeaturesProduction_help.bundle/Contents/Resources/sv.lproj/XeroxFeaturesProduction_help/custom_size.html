Skapa ett eget pappersformat

  1. Skriv bredden på det anpassade formatet bredvid Bredd eller ange den med pilarna.
  2. Skriv höjden på det anpassade formatet bredvid Höjd eller ange den med pilarna.
  3. Ange måttenhet genom att klicka på pilarna i fältet måttenhet och välja mm eller tum.
  4. Klicka på OK.
